Across

Bug Base Product Area:

  • TM Across

 

Description:

One of Lingos method of using its Translation Memory, is to use Across. Across is an existing Translation Memory system that Lingo ties into. Lingo can upload to the Across server as well as use suggestions given by Across. Across also has its own dictionary, that Lingo uses and displays in its dictionary pane.

The purpose of Translation Memory is to assist the translator and make their job easier. The translator can upload translated segments to the TM and in turn the TM will offer suggestions for future translations.
