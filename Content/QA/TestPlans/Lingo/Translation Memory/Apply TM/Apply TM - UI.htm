Test Cases:

  • If invalid file it notifies you of the file but still lets you continue. You can save the log.

  • Suggestions were applied and file status updated

  • Cancel Apply TM to Project before it completes and make sure it cancels and allows you to close the dialogue and continue.
  • Apply TM to a variety of projects(Diff, Flare, Dita, .htm files, word, large projects, etc.)
  • Make sure Apply TM is applied to all types of files. Use FlareForAllProject, translate it manually, upload everything, export the TMX file. Then using a clean TM open up the original FlareForAllProject, import the TMX file you saved and Apply TM to project. All Files should be translated.

 

Note: tools>options: you can set the match percentage that is displayed or automatically translated.
