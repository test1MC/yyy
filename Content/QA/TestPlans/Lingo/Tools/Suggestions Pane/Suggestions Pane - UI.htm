UI

Suggestions Window Pane. Default View (Docked at the bottom of the screen)

 

 

 

 

Controls:

  • Insert Translation: Click this button to replace the selected content with the suggested translation.
  • Tags>Copy:It lets you copy the specified markup tag from the source segment into the editing window.
  • Insert Translation:Opens the 'Add Term to Termbase' dialog window

  • Toggle Show Space: Turns paragraph and space markers on and off. When you turn this feature on, you can see markers that show where paragraphs end and where space exists before and after content.

     

    •  

Columns:

  • Source: The area in the upper-left part of the window pane displays the text in the source language.

  • Translation: [Text area] The area in the lower-left part of the window pane displays the text in the target language (if you have translated it). You can click in this area and type translated text directly. When you click anywhere out of the Suggestions window pane, a message displays, asking if you want to accept your changes as the translation.

    Suggestions Section:

    • %Match: Displays a numerical ranking for the match. For example, if "100" is displayed, the content matches 100%; if "85" is displayed; the content matches 85%; and so on.
    • Source: Displays the word(s) in the source language.
    • Translation: Displays the translated word(s) in the target language.

Test: Make sure you can re-arrange the tags around inside the ‘Translation’ area by dragging/dropping them

 

How to use translation suggestions

The following steps can be used in grid mode only. If you are translating a content file such as a topic or snippet, make sure you first switch from visual mode to grid mode if necessary (CTRL+M).

  1. Open the file you want to translate.
  2. Do one of the following:
    1. • Press ALT+S on your keyboard or...
    2. • Select Translate>Suggest.
    3. The file is scanned against the TM database for matching suggestions.
      1. If exact or partial matches are found, a message indicates this. In addition, the Suggestion Info cells are populated with indications of matches. If an exact match is found, the cell has a green background with the phrase "Exact Match." If a partial match is found, the cell has a red background with percent match shown (e.g., "75% Match").
      2. If no matches are found, a message indicates this as well.
  3. If exact matches are found and you want to use all of those translations, click Yes in the message. The translations replace the source text on the segments that have exact matches. However, for partial matches, you still need to insert the translation manually or provide a new one.
  4. Press CTRL+S or click to save your work.

 

Insert Top Matches dialog window:

 

Yes- Will insert the 100% matches automatically

No- Closes dialog and ignores suggestions
