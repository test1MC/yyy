Concept Editor

Bug Base Product Area:

  • Index / Concept Editor

 

Description:

This editor is used to translate Concept keywords in a project. Authors use Concept keywords as a way to "tie" topics together in a Lingo or Blaze project when those topics discuss the same concepts. You might see Concept keyword markers when you are translating a topic; they are the dark green rectangles containing short terms. Rather than translating these keywords in the Topic Editor, you translate them in the Concept Keyword Editor.
