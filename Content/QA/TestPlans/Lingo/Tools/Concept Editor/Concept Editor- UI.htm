Concept Keyword Editor - UI

Concept Map files are located in the File List.

 

Opening an Concept Map file:

Double-click the file in the File List

Right-click the file in the File List and select Open

Select the file in the File List and press [Enter]

 

Editing Concept Map files

Local Toolbar:

Button (Tooltips in Parentheses):

  • (Hide/show the Info Bar): Hides or shows special information pertaining to the editor or window pane.
  • (Translate):Opens a menu that lets you select an option related to translating content.
  • (Update Keywords and Concepts in Topics): Updates the topics throughout the project, replacing the index and Concept keywords in those topics with the new, translated index and Concept keywords..

UI Controls

  • Concept Keyword Editor grid: Displays and lets you translate and edit Concept Map files.

    Columns

    • Source Keyword: Displays the text in the source language.
    • Translated Keyword: Displays a copy of the text. After you translate the text in the target language, it is displayed in this cell.
    • Difference: If you are working in a "difference" project, the "Difference" column displays any changes that have been made to an item since the last version of the project.

Test Cases

For Translated Keyword, enter single words, multiple words, numbers, words with international characters, etc.

Right-click menu

Right-clicking a row in the Concept Keyword Editor should produce the following menu:

Right-click menu:

 

Right-click menu items:

  • Info Bar: Hides or shows special information pertaining to the editor or window pane.
  • Properties: Opens the Properties dialog window for the selected file.
  • Translate: Opens a menu that lets you select an option related to translating content.

 

Drag and Drop

There is currently no drag/drop editing functionality within the Concept Keyword Editor (as of Lingo V3)

Cut/Copy & Paste – You should be able to do so in the 'Translated Keyword' column.

Undo/Redo works and commands display correctly in the Standard Toolbar.

Deleting

Item:

You should not be able to delete any segments from the Concept Keyword Editor.

Tooltips

If any Source/Translated Keyword is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.
