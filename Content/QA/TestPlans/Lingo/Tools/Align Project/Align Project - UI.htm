Note: In order to use this feature, you must first install a translation memory database.

Controls:

 

  • In the Source language section, select the original language that is used in the project you are translating. In the Target language section, select the language that you want to use for the translation. Doing this does not determine or limit which language you can use for translating content. You can translate in any left-to-right or right-to-left languages, regardless of the language you select. Instead, selecting a language in this wizard accomplishes the following:
  • Indication of language: Selecting a language provides an easy way to identify the project with a particular language. When you are finished creating the project, the target language will be shown in the Lingo Project Editor. You can also indicate the target language by including it when you name the project (e.g., "My Flare Project Italian").
  • Dictionary: Use If you plan to use the dictionary feature in Lingo, the language that you select allows you to work with a dictionary in the same language.
  • Spell check: This affects the language that is used for spell checking.

Note: The languages displayed in bold are supported for spell check. This does not mean that you cannot translate in a particular language if it is not displayed in bold font in this page of the wizard. You can translate in any left-to-right or right-to-left languages in the Lingo interface. The bold font simply indicates which languages are supported with spell checking.

Aligning Files:

 

When you create an alignment project, Lingo matches the files and aligns them by name. Although it might not be a common occurrence, there are a variety of reasons that a file from the original project might not be aligned with the same file in the translated project. The important thing is that you identify such misalignments and then correct them. You can do this by using the File Map window pane.

In an alignment project, the File Map window pane is used instead of the standard File List window pane in a regular Lingo project. It groups files found in both projects by file type. For each type, you can match the files together when necessary. There are a few operations for aligning files; you can shift the selected cells up and down, or swap two selected cells. After the files are aligned, you can manually upload them to your translation memory database.

 

After creating an alignment project, let's say you open the File Map window pane and notice that all of the topic file rows are aligned with identical names, except for two—"Welcome.htm" and "Introduction.htm."

To fix this you can select one of the misaligned cells (say, the "Introduction.htm" file in the first column) and click the appropriate "swap" button to align the files.

 

See Lingo's Help for more information.

Note: You can only “Swap” segments in the File Map. You can Swap, Split and Join segments in the Alignment Tool Editor.

 

 

 

How to align files

 

  1. Open the aligned project. Unless you specify otherwise, these types of projects are stored in the My Documents\My Aligned Projects folder on your computer. (Lingo creates the My Aligned Projects folder when you install the program).

    Note: In Vista, the folder is called "Documents" instead of "My Documents."

  2. The File Map window pane should already be open. If it is not, do one of the following: Select View>File Map. OR Press CTRL+SHIFT+R on your keyboard. OR In the Standard toolbar, click .
  3. In the File Map window pane, select the appropriate cell(s) on either the source language side or the target language side. You can hold down your SHIFT or CTRL key to click and select multiple cells.
  4. In the local toolbar, click one of the buttons to perform an alignment action (so that the source files on the left are aligned with the proper target files on the right).
    1. If you have selected two cells (one above and one below) and click this button, the positions of those two cells will be swapped.
    2. Click this button if you want to swap the selected cell(s) with those below it.
    3. Click this button if you want to swap the selected cell(s) with those above it.

 

 

Opening the Alignment Tool Editor

To open the Alignment Tool, first open the File Map window pane (View>File Map). Then double-click on the associated pair of files. Like the File Map window pane, the Alignment Tool lets you shift selected cells up and down, or swap two selected cells. In addition, you can split and join segments.

UI

Alignment Tool Editor

 

 

Controls:

 

  1. If you have selected two cells (one above and one below) and click this button, the positions of those two cells will be swapped.
  2. Click this button if you want to swap the selected cell(s) with those below it.
  3. Click this button if you want to swap the selected cell(s) with those above it.
  4. Click this button if you have selected one cell and want to split its content into two cells. This button is disabled until you double-click in the cell that you want to split. After double-clicking the cell, place your cursor where you want the split to occur in the segment. Then click this button.
  5. Click this button if you have selected multiple cells (above and below one another) and you want to join the content within them into a single cell.
  6. After you align segments, you should upload them to your translation memory database. If you select specific segments and click this button in the local toolbar, only those translations will be uploaded.
  7. If you click this button in the local toolbar, all translations in the file will be uploaded.

 

See Lingo's Help for more information and examples.
