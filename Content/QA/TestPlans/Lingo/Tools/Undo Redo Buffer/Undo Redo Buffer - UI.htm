Undo & Redo Buffer - UI

Opening the UI

! View | Toolbar | Standard and notice the <Undo> and <Redo> buttons

! Edit | Undo or !Edit | Redo

CTRL<Z> to undo, CTRL<Y> to redo

UI

 

Controls:

  • Undo: Undo the previous action

  • Redo: Redo the previous undone action

  • : pulls the drop-down list of actions to undo/redo

 

                

 

  • Dialog Title is correct 
  • Spelling and Grammar are correct
  • Keyboard can navigate and change settings
  • Dialog is easy to use and understand
  • Dialog is resizable and redraws correctly
  • Dialog remembers setting and sizes from previous session
  • Dialog opens correctly on a dual monitor system
  • Default options are displayed in the screenshot
  • Drop-Down box should be resizable
  • Drop-Down box’s bottom area should list “Undo x actions”/”Redo x actions”

 

Test Cases:

  • Test multiple buffer functionality; undo/redo buffer for couple of documents
  • Test resizing the buffer drop-down box
  • Test to make sure that undo/redo is listing the names of the action
  • Test undo/redo on multiple files (i.e. Browse Seq, Variable Sets, Filter Sets, Glossaries, Topics, Stylesheets, etc)
  • Test to undo/redo multiple actions at once
