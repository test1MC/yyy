UI

Startup File Dialog

 

How to open this dialog:

 

  1. Open the document that you want to use as the startup file.
  2. Select File>Startup File.

 

Explain the buttons and sections:

 

New startup file: Displays the path to the new startup file.

Current startup file: Displays the path to the startup file that is being replaced (if any).

Reset: Clears the "New startup file" field.

 

Test:

  • Able to change the start up file location.
  • Can not enter unsupported characters: ? : * " < >| % #
