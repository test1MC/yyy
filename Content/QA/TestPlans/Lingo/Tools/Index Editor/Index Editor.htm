Index Editor

Bug Base Product Area:

  • Index / Concept Editor

 

Description:

This editor is used to translate index keywords in a project. Index keywords are used to create a generated index for the project. However, they are not contained in any one file. Instead, index keywords are markers inserted into topics. When translating a topic, you might see some index keywords, which are contained in light green rectangles. Rather than translating these keywords with the rest of the topic content, you use the Index Keyword Editor, which displays the keywords in a grid.
