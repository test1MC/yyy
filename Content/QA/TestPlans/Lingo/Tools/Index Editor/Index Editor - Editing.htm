Search Filter Sets - Editing

Search Filter Set files are saved to the Project/Advanced folder.

 

Opening a Search Filter Set file:

Double-click the file in the Project Organizer

Right-click the file in the Project Organizer and select Open

Select the file in the Project Organizer and press <Enter>

Target Editor: In the Advanced tab of the Target Editor, in the Search Filters section, select a Filter File and click the "Edit" button.

 

Editing Search Filter Sets

Local Toolbar:

Button (Tooltips in Parentheses):

  • (New item): Adds a new line to the Search Filter Set Editor using the default filter name "NewFilter."
  • Cut: Cuts the selected filter(s) to the clipboard.
  • Copy: Copies the selected filter(s) to the clipboard.
  • (Paste): Pastes the filter(s) from the clipboard.
  • (Delete): Deletes the selected filter(s)

UI Controls

  • Search Filter Set Editor grid: Displays and lets you create and edit Auto-index Phrases.

    Columns

    • Filter Name: Lists the search filters as they will appear in the filter list in the output.
    • Concept: Displays the concept(s) associated with this filter which will be used to limit the search results.
    • Comment: Allows you to add a comment about the filter. Comments should not affect the output.

Test Cases

For Filter Name, enter single words, multiple words, numbers, words with international characters, etc.

Right-click menu

Right-clicking a row in the Search Filter Set Editor should produce the following menu:

Right-click menu:

 

Right-click menu items:

  • New: Adds a new line to the Search Filter Set Editor using the default filter name "NewFilter."
  • Cut: Cuts the selected filter(s) to the clipboard..
  • Copy: Copies the selected filter(s) to the clipboard.
  • Paste: Pastes the filter(s) from the clipboard.
  • Delete: Deletes the selected filter(s)

 

Select Concepts For Search Filter dialog:

UI Controls

  • Selected Concepts: Displays all concepts associated with the selected search filter.
  • All Concepts: Displays all concepts contained in all topics throughout the project.
  • [ << ] : Adds the concept(s) selected in the All Concepts list to the list of Selected Concepts on the left.
  • [ >> ] : Removes the concept(s) selected in the Selected list from that list.

 

Test Cases

  • Select Concepts and All Concepts lists are sortable.
  • Double-clicking an item in the All Concepts list adds it to the Selected Concepts list.
  • <Shift>-Click and <CTRL>-Click allow for multi-selection in both lists and multi-selections can be added to and removed from the Selected Concepts list.
  • Create concepts with various characters, symbols and international letters and make sure all concepts used in your project appear in the All Concepts list.
  • Create concepts in all files that allow them (topics, snippets, etc) and make sure all concepts used in your project appear in the All Concepts list.

Drag and Drop

There is currently no drag/drop editing functionality within the Search Filter Set Editor (as of Flare V5)

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

Select one or more filter rows (CTRL-Click) from any combination of rows and then:

Right-click on the filter row | Cut/Copy Paste menu options

Cut/Copy Paste Standard Toolbar options

Cut/Copy Paste Keyboard keys (CTRL-X, CTRL-C, CTRL-V)

Cut/Copy Paste !Edit Menu options

 

Undo/Redo works and commands display correctly in the Standard Toolbar.

Deleting

Item:

Select the filter row and press <Delete> on the keyboard

Select the filter row and click the Delete button on the Standard Toolbar

Right-click the filter row and select Delete

Select the filter row | !Edit | Delete

Select the filter row and “Cut”

Tooltips

If any Filter Name, Concepts list or Comment is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.

Conditional Tags

Conditional Tags cannot be applied to individual Search Filter Set file entries. Conditional Tags can be applied to the actual Search Filter Set file (*.flsfs), however, as of Flare V5, this tag can only be used for the purposes of Advanced Global Project importing.

Code

Default Search Filter Set:

<?xml version="1.0" encoding="utf-8"?>
<CatapultSearchFilterSet>
<SearchFilter Name="My Subset"></SearchFilter>
</CatapultSearchFilterSet>
 

Search Filter Set with two filters, one with two concept associations and a comment:

<?xml version="1.0" encoding="utf-8"?>
<CatapultSearchFilterSet>
<SearchFilter
Name="My Subset"
Concepts="concept1"></SearchFilter>
<SearchFilter
Name="NewFilter"
Comment="Two concept associations"
Concepts="concept2;concept3" />
</CatapultSearchFilterSet>
