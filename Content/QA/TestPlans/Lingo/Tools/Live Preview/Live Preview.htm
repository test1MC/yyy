Live Preview

Bug Base Product Area:

  • Live Preview

 

Description:

You can display a live preview of the translated file next to the source content in the Translation Editor. View changes to translated content as it would appear in the output in real-time. Toggle live preview pane to the right or bottom of Translation Editor

You can live preview MS Word files, PowerPoint files, Topics, Snippets Master Pages, Capture Images with text and Contributor Files.
