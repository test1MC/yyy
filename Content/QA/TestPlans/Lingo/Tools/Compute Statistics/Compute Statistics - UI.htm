UI

 

 

How to compute statistics:

 

Do one of the following:

  1. Select File>Compute Statistics. Or..
  2. In the Project Editor, click Compute Statistics.
  3. The statistics report opens as an HTML file, showing the following information (both for the entire project, as well as for individual documents in the project):
  • Number of segments
  • Number of words
  • Number of duplicates (These are segments that occur multiple times within the project. For example, the first time a segment occurs in the project and there is no match in the translation memory for it, it is counted as "No Match." If it shows up again 20 times in other places, it is counted as a "Duplicate.")
  • Number of translated segments and words
  • Matching percentages of segments and words in translation memory (e.g., the number of segments and words that have 100% matches, those that have 85%-94% matches, those that have no matches)
  • Number of no matches (These are segments that are not translated and have no useful suggestions in the translation memory.)

 

 

UI-Local Toolbar

 

This toolbar is taken from Flare and has un-used UI items which should be removed at some point.

 

Controls:

 

  • Browser is there to let you know that this is an html file that is stored in the Lingo project and can be opened and viewed in a browser.
  • Back and Forward buttons are grayed out and not used
  • The stop button is active but there is nothing to stop
  • The refresh button is active but it does not refresh anything nor does it compute statistics again.
  • Print: Allows you to print the statistics report and opens the print dialog.
  • Save To: Allows you to choose where you want to save this statistics report and open the save as dialog.

 

Test:

  • Print and Save To buttons work correctly.
  • Compute statistics on a variety of projects to make sure it is functioning correctly.
  • Compare the statistics to the translation bundle and they should match.
  • Make some changes by uploaded some more translations and compute statistics again to make sure statistics are updated.
