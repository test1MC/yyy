Diff Icons:

 

If you are working in a "difference" project, special icons display next to the areas in the left window pane (in visual mode) or next to the rows (in grid mode) where changes have been made since the last version of the project. The type of icon shown depends on the type of change that has been made. In grids, a "Difference" column also provides an indication of what was changed in the content.

Creating Difference Projects:

 

Let's say you have already created a Lingo project for an earlier version of a particular Flare project, Blaze project, Mimic project, Mimic movie, MadCap document, DITA file, Microsoft Word document, or Capture image. You have already translated it, exported it to a new project or file, and given it to the author.Sometime after this, the author makes many changes to the project or file (in the source language). Now the author needs you to translate those changes into the target language.When this occurs, you can "Update" the project to translate the new version of the project or file. This allows you to easily locate the differences between the old and new versions, rather than looking through each piece of text. If a file or string has been modified since the previous version, an icon displays next to that content (the type of icon depends on the type of change made). In addition, the "Difference" column in an editor's grid provides an indication of the change. You can then compare the two versions in order to determine what was modified and translate the changes as necessary. If the author has deleted content from the new project or file, that content is automatically deleted from the new Lingo difference project when you create it.

 

How to create a difference project:

  1. Open the Lingo project that you already translated and exported. See Opening Projects.
  2. Select File>Update Project.
    • It opens up the File folders on the computer
  3. In the dialog that opens, navigate to the updated project or file that you want to translate, and double-click the appropriate file. The main extensions for the various types of files that you can translate are: .flprj (Flare projects), .blprj (Blaze projects), .miprj (Mimic projects), .mimov (Mimic movies), .mdcoct or .mcdoc (MadCap documents), .txt (text files), .dita (DITA files), .doc (Microsoft Word files), various image file types (Capture files). Files with other extensions are also supported (e.g., .flprjzip, .ditacon, .docx, .htm, .xml).
    1. Note: You will need to select the appropriate file type from the file type list to view files of that type in the explorer.
  4. Click Open.
  5. The project will stat to update.
