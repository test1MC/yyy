Diff Project

Bug Base Product Area:

  • Diff

 

Description:

Create New Diff Project Wizard was removed as of V4. We now Update projects instead. It allows you to easily locate the differences between the old and new versions of the Flare or Blaze project, rather than looking through each piece of text. If a file or string has been modified since the previous version, an icon displays next to that content (the type of icon depends on the type of change made). In addition, the "Difference" column in an editor's grid provides an indication of the change.
