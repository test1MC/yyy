Find & Find and Replace

Bug Base Product Area:

  • Find|Find & Replace

 

Description:

Find and Replace allows user to find text in topics or stylesheets and replace it with different text. Users are given the options to replace specific instances of text in documents or replace all instances at once. Users are also given the option to look in current document only or in other documents, to match the case of the text, to use regular or wildcard expressions, and to include source code in the search.

Find in Files allows user to find text in topics and stylesheets. The limitation of these features is the same as those of Find and Replace, but Find in Files allows user to see a list of all document containing the text that you want to find. These features also include Load Find Selection that copies selected text to the Find column, Find Next that finds next instance of the text, Find Result Windows that displays the search results from Find in Files window.
