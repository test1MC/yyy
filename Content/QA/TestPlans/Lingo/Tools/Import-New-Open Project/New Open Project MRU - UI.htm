UI

Recent File Manager

 

  • All Scroll bars work correctly.

 

Button/Section:

 

  • Recent Files:Lists the names of the most recently opened Lingo files.
  • Path:Provides the location of the file.
  • Open File: Opens the first selected file in the list.
  • Remove All:Removes all files from the list.
  • Remove:Removes the selected file(s) from the list. You can hold the SHIFT key to select a range, or you can hold the CTRL key to select individual items.
  • OK: Accepts changes made in the recent file manager and closes the dialog.
  • Cancel: Closes the recent file manager without saving any of the changes made.

 

Recent File Manager

 

  • All Scroll bars work correctly.

 

Button/Section:

 

  • Recent Projects:Lists the names of the most recently opened Lingo project files.
  • Path:Provides the location of the project.
  • Open File: Opens the first selected project in the list.
  • Remove All:Removes all projects from the list.
  • Remove:Removes the selected project(s) from the list. You can select multiple projects in the list by holding down the SHIFT or CTRL keys and clicking a range of projects or individual ones.
  • OK: Accepts changes made in the recent project manager and closes the dialog.
  • Cancel: Closes the recent project manager without saving any of the changes made.
