Termbase Editor Dialog

This termbase will show you suggestions for the selected segment if the selected segment has matches in the currently selected termbase. If there are no matches nothing will be displayed in this pane. You can also quickly add terms to the termbase using the Quick add Terms button in this pane.

 

 

Columns:Display only if there is a suggested term, otherwise the columns will not display.

  • Term: Shows the term in the selected termbase
  • Source:Shows the source of the term as written in the Termbase
  • Context: Displays the context for that term as written in the Termbase. It may be an example sentence of the term.
  • EXAMPLE: If the term is "motorcycle," you might enter "My motorcycle is very fast."

Batch Add Terms to Termbase Dialog

"Quick Add Terms" option (add multiple terms at once) In the Termbase window pane you can click the "Quick Add Terms" button, which lets you add many different terms at the same time to a termbase. For example, you might need to add a list of corporate terms from an email. The obvious advantage of this method is that you can quickly add lots of different terms. In addition, you do not need to have any files open for translation at the same time.

 

 

Controls:

  • Language pickers: Select the source and target languages for the term(s)
  • Data grid view: Enter the Original and Translated term (s) on each row
  • Termbase picker: Select the termbase to insert quick term(s) to

 

To add terms using the Batch Add Terms to Termbase Dialog or 'Quick Add Terms',

Do one of the following:

  1. Select View>Termbase or...
  2. In the Standard toolbar click .
    1. The Termbase window pane opens.
      1. In the local toolbar, click Quick Add Terms. The Batch Add Terms to Termbases dialog opens.
      2. In the Language 1 field, make sure your source language is selected.
      3. In the Language 2 field, make sure your target language is selected.
      4. In the first row click in the Original cell and type the term in the source language and press Enter on your keyboard.
      5. In the first row click in the Translation cell and type the term in the target language and press Enter on your keyboard.
      6. In the next row repeat steps 4 and 5. Do this until you've added all the terms you need.
      7. In the field at the bottom of the dialog, select the termbase that you want to add the terms to.
      8. Click OK. The next time you reopen the Termbase Editor, you will see the terms you added.

 

Test Cases:

  • Dialog opens when “Quick Add Terms...” Button clicked.
  • Button is inactive when there are no termbases
  • Languages are set to the languages of the project if there is a project open, or to the most recently chosen languages otherwise.
  • Data is saved to the chosen termbase when closing the dialog.
  • Duplicates are only entered once.
  • The termbase is unchanged on Cancel.
