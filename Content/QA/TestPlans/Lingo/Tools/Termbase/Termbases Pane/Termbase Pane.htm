Termbase Pane

Bug Base Product Area:

  • TermBase Pane

 

Description:

This termbase pane will show you suggestions for the selected segment if the selected segment has matches in the termbase. If there are no matches nothing will be displayed in this pane. You can also quickly add terms to the termbase using the Quick add Terms button in this pane.
