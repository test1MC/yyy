Adding Termbases

You can add as many termbases as you need. For example, you might use one termbase for Client A, a different termbase for Client B, and yet another for Client C. When you add a new termbase, it can be used in any of your Lingo projects.

 

How to add a termbase.

 

Do one of the following:

1. Select View>Termbase. OR

2. In the Standard toolbar click .

  1. The Termbase window pane opens.
  2. In the local toolbar, click Manage Termbases.
    1. The Manage Termbases dialog opens.
    2. Click Add.
      1. The Enter New Termbase Name dialog opens.
      2. Type a name for the termbase

 

Importing TBX-Basic Files

If you have created a termbase in Lingo, you can import a TermBase eXchange (TBX) file into it. TBX is an open, XML-based standard used for exchanging structured terminological data. It has been approved as an international standard by the Localization Industry Standards Association (LISA) and the International Organization for Standardization (ISO).

One difference between a TBX file and a Translation Memory eXchange (TMX) file is that a TBX file handles multiple languages, whereas a TMX file handles only two languages.

 

 

How to import TBX files

 

Do one of the following:

  1. Select View>Termbase. OR
  2. In the Standard toolbar click .
    1. The Termbase window pane opens.
    2. In the local toolbar, click Manage Termbases.
      1. The Manage Termbases dialog opens.
      2. Select the termbase that you want to import the TBX file into.
        1. Click Import TBX-Basic…
        2. The Import TBX dialog opens.
          1. Click the Browse button.
          2. Navigate to the TBX file that you want to import and double-click it.
          3. Click Start Import. – The TBX-Basic Progress dialog window appears.

        3. Click OK

 

Note: Lingo supports TBX-Basic files (not the "Full" version).

 

Exporting TBX-Basic Files

If you have created a termbase in Lingo, you can export it to a TermBase eXchange (TBX) file. TBX is an open, XML-based standard used for exchanging structured terminological data. It has been approved as an international standard by the Localization Industry Standards Association (LISA) and the International Organization for Standardization (ISO). One difference between a TBX file and a Translation Memory eXchange (TMX) file is that a TBX file handles multiple languages, whereas a TMX file handles only two languages.

 

 

How to export TBX files

 

Do one of the following:

  1. Select View>Termbase. OR
  2. In the Standard toolbar click .
    1. The Termbase window pane opens.
    2. In the local toolbar, click Manage Termbases.
      1. The Manage Termbases dialog opens.
      2. Select the termbase that you want to export.
      3. Click Export TBX.
        1. The Export to TBX dialog opens.
        2. Click the Browse button.
        3. Navigate to the folder where you want to export the file.
          1. In the File name field, enter a name for the new TBX file.
          2. Click Start Export. – The TBX-Basic Progress dialog window appears.

Adding Terms to Termbases

After you add a termbase, you can add translated terms (or phrases) to it. You can do this through the Termbase window pane (without needing to have any files open for translation at the same time) or through the Suggestions window pane.

 

 

To add terms via the Termbase window pane, do one of the following:

  1. Select View>Termbase. OR
  2. In the Standard toolbar click .
    1. The Termbase window pane opens.
      1. In the local toolbar, click Manage Termbases.
      2. The Manage Termbases dialog opens.
        1. Select the termbase that you want to work with.
        2. Click Edit.
        3. The Termbase Editor opens.
          1. Select the Search tab.
          2. In the local toolbar, click .
          3. The Add Term to Termbase dialog opens.
            1. In the top drop-down list, make sure your source language is selected.
            2. In the field below, type the source term or phrase.
            3. In the next drop-down list, make sure your target language is selected.
            4. In the field below, type the translated term or phrase.
            5. (Optional) If you want to add more detailed information about the term (e.g., source, definition, related term, context sentence, usage), click Advanced.

:Adding terms via the Suggestions pane

  1. Open a file.
  2. In the editor that opens, select a segment of text.
  3. Do one of the following:
    1. Press CTRL+G on your keyboard. OR
    2. Select View>Suggestions.
    3. The Suggestions window pane opens.

 

  1. In the Suggestions window pane, highlight the term you want to add to the termbase. You can highlight the term in the Source section, as well as in the Translation section.
  2. In the local toolbar, click Add New Term.
  3. The Add Term to Termbase dialog opens. The terms that you highlighted are also highlighted in the corresponding fields in this dialog.(Optional) If you want to add more detailed information about the term (e.g., source, definition, related term, context sentence, usage), click Advanced.

 

Multi-Term selection:

Holding Control while highlighting words or phrases in a BoxView (particularly the Suggestions Pane in Lingo) create multiword selections, like in Word. Unlike Word, the subsequent selected areas have different colors, in order to indicate pairing between source and target language phrases.

Test Cases:

  1. Select a word or phrase in the Suggestions pane Source box. Hold Control and select another word in the same box. The second area should also become highlighted, but in a different color. Repeat as often as desired.
  2. Translate the segment and highlight the corresponding terms in the target language box, in the same order.
  3. Click the “Add New Term…” button. Notice the highlighting is preserved in the dialog that pops up.
  4. Click OK. Open the Termbase in the Termbase editor and verify that all the terms that were highlighted are in the Termbase, with the appropriate context phrase.
  5. It is also possible to do all the highlighting work in the Add Multiple Terms dialog itself, instead of in the Suggestions pane.
    1. The colors should become lighter when the Suggestions pane or Lingo loses focus.
    2. Unbalanced numbers of source or target terms should be ignored silently (e.g., 3 highlights on the source side vs. 6 on the target, only the first three valid pairs will be added).

Editing Termbases

You can edit the words in a termbase, which is not possible in a dictionary. Not only that, but you can also add much more detailed information about each term (e.g., source, definition, related term, context sentence, usage) click Advanced. That way, whenever you are using the termbase to perform translation work, you can refer to this extra information if necessary.

 

How to edit a termbase.

Do one of the following:

  1. Select View>Termbase. OR
  2. In the Standard toolbar click .
    1. The Termbase window pane opens.
    2. In the local toolbar, click Manage Termbases.
      1. The Manage Termbases dialog opens.
      2. Select the termbase that you want to work with.
        1. Click Edit.
        2. The Termbase Editor opens.
          1. Select the Search tab.
          2. (Optional) You can use the filter field in the local toolbar at the top to quickly find a particular term. Simply type the term you want to find and press Enter. You can also use wildcards (asterisks) in your search.
            1. EXAMPLE: If you want to find the term "Delete," you can enter the following:
            2. All terms that start with "Dele" will be found. Each time you type something in the filter field, it will be saved in the drop-down for future searches.
            3. Double-click on the term that you want to edit.
              1. The Edit Termbase Term dialog opens.
              2. Using the fields in this dialog, you can change the text for either the source or target language.

 

Insert Terms from Termbase:

 

After you create a termbase and add terms to it, you can insert those terms into your translation content. You can do this by highlighting the word you want to replace and then double-clicking the translated term in the Termbase window pane.

 

How to insert terms from a termbase:

  1. Make sure you have created a termbase and added terms to it.
  2. Do one of the following:
    1. Select View>Termbase. OR
    2. In the Standard toolbar click .
    3. The Termbase window pane opens.
      1. Open an editor to translate content. You can be working in visual or grid mode.
      2. Highlight a word or a segment that you want to translate.
        1. If a translation of the word is found in a termbase, it is shown in the Termbase window pane.
          1. Note: In addition, if you have the Suggestions window pane open and highlight a word in it, the Termbase window pane displays its translation (if it exists in the termbase).
        2. In the Termbase window pane, double-click the translated term.
          1. The translated term replaces the non-translated word.
