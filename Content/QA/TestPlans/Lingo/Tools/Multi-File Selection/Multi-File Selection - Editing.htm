Multi-File Selection - Editing

This window pane lets you view all of the content files in your project in alphabetical order. If you want to see a comprehensive list of the files in the project that should be translated, you can select the "Translation Candidates" option from the drop-down list.

 

When you want to translate a particular file, you can double-click it in the File List window pane to open it.
