Multi-File Selection - UI

Opening the UI

!File> New Project or Press [Ctrl] + [Shift] + [N] on your keyboard

From the Start Page, click "Create a New Project"

Click in the Standard toolbar

UI

 

  • Test that you can add all the supported file types.
  • Selecting multiple files and clicking "Remove" should only remove the selected files
  • [Ctrl] + click should select individual files
  • [Shift] + click should select multiple files (drag)
  • When selecting an individual Flare, Blaze or Mimic project, the "Add" button should be disabled

UI Controls

  • Add: Will open the ‘Select file(s) or project to translate’ dialog window.
  • Remove: Will remove the selected file(s) or project.
  • Back: This button is disabled in this window.should be sortable and resizable (#10134).
  • Next: Opens the next page of the wizard.
  • Finish: This button is disabled in this window.
