Relevant Menu Items:

  • (Alternate Spelling Suggestions): A list of suggested words to replace the current misspelled words. Select one to replace the existing text. Suggestions should also draw from Dictionary files within the project.
  • Ignore All: Adds the word to a list of ignored words list for this topic.
  • Add To Dictionary: Add the current misspelled word to the appropriate language Dictionary

 

 

 

Language Markers in Lingo Editor

Lingo will spell check based on the language of your content. Language markers appear in the editor to show what language the project was set to use the source and target areas.

In this case, the Lingo project is set to be an English to Spanish project. So an English language marker appears in the source topic and a Spanish language marker appears in the translated topic despite the fact both topics are written in German.

 

 

Test Cases

Verify that words added to the Dictionary are added to the appropriate dictionary. That is, if the word is contained in a topic with a Swedish language tag, the word should be added to the Swedish language dictionary.
