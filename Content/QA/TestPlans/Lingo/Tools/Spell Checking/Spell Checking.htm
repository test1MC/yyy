Spell Checking

Bug Base Product Area:

  • Tools | Options

 

Description:

Spell Checking allows user to spell check contents in Lingo. This features includes spell checking while typing, spell checking manually by opening the spell check window and adding words into the “Ignore” spell check list.

 

Misspelled words can be ignored or added to a project Dictionary file (one per language). Ignored words are ignored per topic. Words added to the Dictionary file will not be determined to be misspelled in any of the project content and should also be used when Flare makes spelling correction suggestions.

 

Spell Checking is only fully supported for a specific set of languages. To see which languages are supported, set the language of a project, a topic or any element in Flare and select the "Hide languages with no spell check support" box. The languages which remain are the only languages fully supported. You can create dictionary files for unsupported languages, however, you would essentially need to build the entire dictionary from scratch to make it useful.
