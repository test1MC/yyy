Dictionary Pane

Bug Base Product Area:

  • Dictionary

 

Description:

You can use dictionaries in various languages to assist you when translating content. MadCap provides many dictionaries with English terms translated to another language (e.g., an English-to-German dictionaries, an English-to-French dictionary, an English-to-Spanish dictionary, and so on). In addition, if using "across" for your translation memory (TM) system (see their website at http://www.across.net), you can use any dictionary that you have set up in the across TM database. When you click on a particular word, the dictionary provides one or more translations of the term in the other language. You can then double-click the translated word to insert it into the project.
