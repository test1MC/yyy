Opening the UI

  1. Make sure you have installed a dictionary.
  2. Do one of the following:
    1. Select View>Dictionary. Or...
    2. In the Standard toolbar, click .

 

UI File List Pane

 

Controls:

 

  • Term: Displays a translation of the term in the target language.
  • Dictionary: Displays abbreviations for the dictionary that you are using. For example, if you are using an English-to-German dictionary, this cell displays "en_de."
  • Class: Displays the type of dictionary that is providing the translation. If "Basic Dictionary" is displayed, it means the words are those that you might find in everyday language. If "Software - Microsoft Terminology" is displayed, it means the words are those from a special Microsoft dictionary used for software terms.
  • Path: Displays the location where the dictionary is installed on your computer.
  • CRTL + #:Add a term.

 

 

Following are the basic steps for using a dictionary in Lingo:

 

  1. Install SQL Server In order to use features such as translation memory and dictionary, you need to install Microsoft SQL Server.
  2. Install TM In order to use the dictionary feature, you must have a TM database installed.

  3. Download/set up dictionaries Download and install a dictionary provided by MadCap Software.

    You can download language dictionary from the following link:

    http://www.madcapsoftware.com/downloads/lingo010-p9nn.aspx

    You can also set up a dictionary in your across database (if you are using across instead of the built-in Lingo TM system).

  4. Select appropriate target language When you create a new Lingo project, you select a target language. This target language must match the language of the dictionary that you intend to use. To ensure the correct languages are associated with the project, open the Project Editor (select File>Project Properties). The Source Language field should display "English," and the Target Language field should display the same language as the dictionary that is to be used.
