Flare file translation

- If the Index Links is a “See Also” type, we should NOT let them translate the ‘Term’ and ‘Linked Term’ as they should get automatically updated when translating the Index

- If the Index Link is a “See” type, we should only let them translate the ‘Term’. The ‘Linked Term’ should be automatically updated when translating the Index

- The only thing that should be translated in the ‘Search Filter Set’ is the ‘Filter Name’ column. The ‘Concepts’ column should be automatically updated when translating the Concepts

- We should let the user translate both the ‘Phrase’ and the ‘Index Term’ in the ‘Auto Index Set’ file (No change required in Lingo)

- For “See Also” (Concept Link Control), the ‘Selected Concepts’ should automatically update when translating the Concepts

- For “Search Index” (Keyword Link Control), the ‘Selected Keywords’ should automatically update when translating the Index
