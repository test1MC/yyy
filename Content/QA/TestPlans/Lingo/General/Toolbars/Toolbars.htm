Toolbars

Bug Base Product Area:

  • Toolbars / Menus

 

Description:

Lingo toolbar is made up of a single Global Standard Toolbar as well as local toolbars for different editors. Global Toolbar are always available, regardless of where the user are working in the interface, this includes the Standard Toolbar. Local Toolbars are intended for a particular editor and are covered test plans for each specific module that has a Local Toolbar.
