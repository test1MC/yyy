Getting Started Wizard

Bug Base Product Area:

  • Toolbars / Menus

 

Description:

The Getting Started Wizard helps users to become familiar with Lingo quickly and to get started on a project. It provides links to the most common high-level tasks and is designed to help new users begin working with the product right away i.e. Creating a new project, opening an existing project, Launch align tool, tutorials, links to recent projects (min4 and max7 projects).
