UI

Recent File Manager

 

  • All Scroll bars work correctly.

 

Controls:

 

  • Open File: Opens up the selected file in Lingo's Image viewer showing you the properties of that file. You can then click Open Folder to open the files folder location.
  • Remove All: Removes all recent files from the recent file manager.
  • Remove: Removes only the selected file from the recent file manager.
  • OK: Accepts changes made in the recent file manager and closes the dialog.
  • Cancell: Closes the recent file manager without saving any of the changes made.

 

Recent File Manager

 

  • All Scroll bars work correctly.

 

Controls:

 

  • Open File: Opens up the selected file in Lingo's project editor showing you the properties of that project. You can then click Open Folder to open the files folder location.
  • Remove All: Removes all recent projects from the recent project manager.
  • Remove: Removes only the selected project from the recent project manager.
  • OK: Accepts changes made in the recent project manager and closes the dialog.
  • Cancel: Closes the recent project manager without saving any of the changes made.
