UI

New Updates Available Dialog-This dialog is used to find and download any Lingo program updates.

 

 

How to open this dialog:

Select Help>Check for Updates.

 

Controls:

 

  • Download: Displays the update available (if any).
  • Description: Displays a description of the update.
  • Download Update(s): Downloads the updates from the list above. Follow the instructions on the screen to install the updates.
  • Close: Closes the dialog.

 

If no updates are available this dialog will appear:

 

 

This prompt will appear every time user opens up Lingo if the updates hasn’t been downloaded and installed.

Clicking on <Yes> button will bring us to the New Updates Available Dialog listing the entire available dialog.
