TMX - Import

Description:This dialog is used to import TMX files from tools such as Trados, CATALYST, and across. This allows you to leverage all prior translations, whether or not they occurred on the same project or document.

Opening the UI

Select Translate>Import TMX.

How to import TMX files:

 

  1. Select Translate>Import TMX. The Import TMX File dialog opens.
  2. Click Browse.
  3. In the dialog that opens, navigate to the TMX file that you want to import, and double-click it.
  4. (Optional) By default, the current database is shown in the Translation Memory field. If you are working with other databases and want to use a different one instead, select it from this field. You might use this field, for example, if you forgot to change to the database you want before starting the import process.

  5. Click Start Import. A message lets you know the results of the import.

  6. Click OK.
