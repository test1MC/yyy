TMX Import and Export

Bug Base Product Area:

  • Import TMX and Export TMX

 

Description:

TMX stands for Translation Memory eXchange. It is an open XML standard used as a way of exchanging translation memory data between tools and translation vendors without losing any data. A TMX file contains previous translations made by a user and uploaded to the TM. It can then later be used to for updating documents as the user will not have to repeat a translation that has already been done before and is stored in the TMX.
