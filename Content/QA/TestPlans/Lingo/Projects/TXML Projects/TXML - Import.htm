  • Add Files:Will open the ‘Select file(s) or project to translate’ dialog window.

When the source /target language of the selected TXML file does not match the project source/target language, you should get the following messages:

 

Target Language:

 

  • Dialog Title is correct
  • Spelling and Grammar are correct
  • No duplicate accelerators
  • All accelerators function
  • Keyboard can navigate and change settings
  • 'Yes' should update the target language to match the file
  • 'No' should keep the selected target language

 

 

Source Language:

 

  • Dialog Title is correct
  • Spelling and Grammar are correct
  • No duplicate accelerators
  • All accelerators function
  • Keyboard can navigate and change settings

  • 'Yes' should update the source language to match the file

  • 'No' should keep the selected source language

 

Note:

Custom segmentation rules do not work on SDLXLIFF, TTX, and TXML files, because these files should be segmented already by other tools before they are imported.
