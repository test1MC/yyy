Dita Project Import and Export

Bug Base Product Area:

  • For Import testing use: Import/New/Open Project
  • For Export testing use: Exporting

 

Description:

Import: You can create a new Lingo translation project using the Start New Project Wizard. This wizard lets you select from many different kinds of files (e.g., Flare projects, Word documents, DITA files), as well as specify a target language.

 

Export: When you finish translating all of the content in a project or file, you can export your work to the appropriate application, file, or location. For example,if you have translated a DITA file, you can export your work out to a new DITA file. The author can then open this translated project or file and create output for it.
