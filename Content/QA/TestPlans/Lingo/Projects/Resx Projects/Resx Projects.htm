Resx Project Import and Export

Bug Base Product Area:

  • For Import testing use: Import/New/Open Project
  • For Export testing use: Exporting

 

Description:

 

Import:You can create a new Lingo translation project using the Start New Project Wizard. This wizard lets you select from many different kinds of files (e.g., InDesign projects, Word documents, DITA files), as well as specify a target language.

 

Export: When you finish translating all of the content in a project or file, you can export your work to the appropriate application, file, or location. For example, if you translated a Mimic project, you can export your work out to a new Mimic project. The author can then open this translated project or file and create output for it.

 

Get in the habit of computing statistics and translating bundles and exporting to Resx with all test projects you work with. Don't just test the bug but run through other test as well.

 

Export to Resx and test to see it exports correctly. Open the exported project using Visual Studio (or Notepad ++)and make sure the contents are translated.
