How to use this matrix

  1. Copy the project to your test PC (Do not open from the network)
  2. Print out this Matrix and mark progress accordingly:
    1. Import (File> New Project...)
    2. Apply TM (These projects include a TMX file which you should import: Translate> Import TMX...) **
      1. After importing the TMX do: Translate> Apply TM to Project...) **
    3. Bundle (File> Prepare Translation Bundle...)
    4. Statistics ( File> Compute Statistics)
    5. Export (File> Export to Flare...)

 

** Complete this step only for projects in bold. When importing the project (FLPRJ), make sure to choose the correct source and target languages as specified in the Source/Target column. When opening the project (LIPRJ) you don't need to worry about selecting the correct language.
