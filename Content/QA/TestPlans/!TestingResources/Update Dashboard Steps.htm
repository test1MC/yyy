Update Dashboard Steps

When working on a new version (or DOT release) follow these steps to update your products dashboard

  1. Remote into \\build2\
  2. Update <product name>.bat file to the new version
  3. Update dashboard.bat file to the new version
  4. Create a new <product name>.xls file on \\winesap\nonsourcs VSS
  5. Open the bug base (Hold shift + Click) and update the Current Version of your product
  6. You may double-click the <product name>.bat file to verify everything works
