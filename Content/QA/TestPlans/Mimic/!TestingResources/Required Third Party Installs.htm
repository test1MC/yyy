Required Third Party Installs

Mimic 4 contains a completely new Flash compiler based on the FLEX engine. It presented many benefits to Mimic's Flash output, but also introduced some required Third Party utilities.

 

Java Runtime Environment (JRE) 6.0.15 or later

This is a mandatory download for all Operating Systems. Flash will not be able to compile without this installed

  • 6.0.15 of the JRE is located at \\dc2\software\Java JRE\
  • The latest version is available at the Java web site. http://java.sun.com/javase/downloads/index.jsp

Install by double-clicking the .exe.

 

LAME mp3 Codec

This is an Optional install, only needed if you are running Mimic on Windows Vista or Windows 7. Both of these Operating Systems do not come with an MP3 codec built in, therefore a third party MP3 codec is needed when working with sound and Flash output. Mimic 4 was tested using the free LAME codec.

  • \\dc2\Software\Lame 3.98.2 mp3 codec\

 

To install Lame Codec on 32 bit Windows

  1. Close Mimic
  2. Unzip the Lame zip file
  3. Right-Click theLameACM.inf file
  4. Choose Install
  5. Open Mimic and compile Flash. WAV files should convert to MP3.

To Install Lame Codec on 64 bit Windows

  1. Close Mimic
  2. Unzip the Lame zip file to c drive (ex c:\tempfolder)
  3. Open up the RUN command window (Windows key + R)
  4. Type CMD in the window
  5. Hit Ctrl+Shift+Enter (to ensure the command window opens in Administrator mode)
  6. In the command window, type cd C:\Windows\SysWOW64
  7. At this path, type rundll32.exe setupapi.dll,InstallHinfSection defaultinstall 0 c:\tempfolder\LameACM.inf
  8. Codec should install. You may see a quick install dialog pop-up while the files are installed and copied to the directory
  9. Open Mimic and compile Flash. WAV files should convert to MP3.
