Toolbars - UI

Standard Toolbar

Opening/Closing the Standard Toolbar

From a menu:

!View | Toolbars | Standard

Click the button in the upper-left corner (next to the File menu) and select Standard.

Additionally, you can close the toolbar by clicking the button which appears to the left of the toolbar.

Standard Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (There is MadCap News available... ): Launches the default web browser and opens http://www.madcapsoftware.com/mcnews/

    (Once this option is clicked, it will not appear until there is new MadCap News. To restore this button delete the registry data at HKEY_CURRENT_USER/Software/MadCap Software/Shared/News)

  • (Open the Start Page): Opens the Start Page, which provides links to high-level tasks and information.
  • (Open the Project Properties editor): Opens the Project Properties in the main editor area.
  • (Starts the New Project Wizard): Opens the Start new Project Wizard, which user can use to create a new Mimic project.
  • (Open File): Opens the Open dialog, which lets user search for and select a Mimic project to open.
  • (Save File): Saves changes to the active file.
  • (Save All): Saves changes to all of the open files.
  • (Send To): Opens a submenu, which lets user open the active file in another application (e.g., Dreamweaver, Word, Notepad), attach it to an email, or open the Windows folder where it is stored.
  • (Create a new movie): Opens a submenu, which lets the user choose how to record/capture/Import a new movie.
  • Cut: Cuts the selected item(s) to the clipboard.
  • Copy: Copies the selected item(s) to the clipboard.
  • (Paste): Pastes the item(s) from the clipboard.
  • (Delete): Deletes the selected item(s)
  • (Undo): Lets user reverse the most recent action(s) in a file. The button label will vary based on the most recent action.
  • (Redo): Lets user repeat the most recent action(s) that user reversed in a file. The button label will vary based on the most recent Undo action.
  • (Display properties for the selected item): Opens the Properties dialog for the object that is selected in the main editor area.
  • (Open the Project Movies window): Opens the project movies pane. Will list all the movies in the currently opened project.
  • (Open the Frames explorer window): Opens the Frames pane. Will list all the frames for the active movie.
  • (Open the Master Frames window): Opens the Master Frames pane. Will list all master frames for the active movie.
  • (Open the Link Browser window): Opens the link browser in the main editor area. Active movie will have its frames links displayed.
  • (Open the Timeline window): Opens the Timeline pane.
  • (Open the Palettes window): Opens the Palette pane.
  • (Open the Frame Libraries window): Opens the frames library pane.
  • (Build Targets...): Opens the Build Targets dialog.
  • (Build the active Movie/Project): Will generate the opened Movie or Project.
  • (View the Generated Movie/Movie Collection): Will View the generated movie or movie collection. If output is not up-to-date, a prompt will be given.

 

Format Toolbar

Opening/Closing the Format Toolbar

From a menu:

!View | Toolbars | Format

Click the button in the upper-left corner (next to the File menu) and select Format.

Additionally, you can close the toolbar by clicking the button which appears to the left of the toolbar.

Format Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (Font Family): Lets user select a font type (e.g., Arial, Times New Roman), applying it to the content selected in the topic.
  • : Lets user select a font size, applying it to the content selected in the topic.
  • (Bold): Applies bold typeface to the content selected in the topic.
  • (Italic): Applies italic typeface to the content selected in the topic.
  • (Underline): Underlines the content selected in the topic.
  • (Unformat): Resets the formatting applied to content in a topic.
  • (Color): Lets user choose and apply a color to the selected content.
  • (Background Color): Lets user choose and apply a color to the background of the selected content.
  • (Enable/Disable Shadow): Enables or disables the shadow effect on the selected object.
  • (Transparency Value): Increases and Decreases the intensity of the transparency on the selected object.
  • (Background Color): Sets the background color for the selected object.
  • (Color): Sets the color of the text for the selected object.
  • (Line Width): Sets the width of the border for the selected object.
  • (Line Color): Sets the border color.
  • (Align Left): Aligns the selected content to the left. 
  • (Align Center): Aligns the selected content in the center.
  • (Align Right): Aligns the selected content in the right.
  • (Align Top): Aligns the selected content to the top of the object.
  • (Align Middle): Aligns the selected content to the center of the object.
  • (Align Bottom): Aligns the selected content to the bottom of the object.
  • (Rotate): Opens the rotate submenu, which lets the user rotate the object.

 

 

Frames Toolbar

Opening/Closing the Review Toolbar

From a menu:

!View | Toolbars | Frames

Click the button in the upper-left corner (next to the File menu) and select Frames.

Additionally, you can close the toolbar by clicking the button which appears to the left of the toolbar.

Frames Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (Edit Frame Properties): Opens the active frames properties dialog.
  • (Go to the first frame): Brings up the first frame in the main editor.
  • (Go to previous frame): Brings up the previous frame in the main editor.
  • (Go to next frame): Brings up the next frame in the main editor.
  • (Go to last frame): Brings up the last frame in the main editor.
  • (Remove selected frame): Removes the selected frame(s).
  • (Copy Selected frame): Copies the selected frame(s).
  • (Paste frame): Pastes the frame(s) that are on the clipboard.
  • (Duplicate selected frame): Duplicates the selected frame(s).
  • (Copy background of selected frame): Copies the background image of the selected frame. Grayed out if the frame has no image.
  • (Paste frame background): Pastes the current frame background on the clipboard, to the currently selected frame.
  • (Insert a frame using the same bitmap as the selected one): Will create a new frame, insert it after the selected frame and will use the same background image as the selected frame.
  • (Insert a blank frame after the current one): Will insert a blank frame (no background image) after the currently selected frame.
  • (Insert PowerPoint slides after the selected frame): Imports and inserts PowerPoint slides after the currently selected frame.

 

Layout Toolbar

Opening/Closing the Layout Toolbar

From a menu:

!View | Toolbars | Layout

Click the button in the upper-left corner (next to the File menu) and select Layout.

Additionally, you can close the toolbar by clicking the button which appears to the left of the toolbar.

Layout Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (Sink): Sinks the selected object one layer lower.
  • (Float): Floats the selected object one layer higher.
  • (Same Left): Sets the selected objects to use the same left coordinates.
  • (Same Right): Sets the selected objects to use the same right coordinates.
  • (Same Bottom): Sets the selected objects to use the same bottom coordinates.
  • (Same Top): Sets the selected objects to use the same top coordinates.
  • (Same width): Sets the selected objects to use the same width.
  • (Same height): Sets the selected objects to use the same height.
  • (Same size): Sets the selected objects to use the same size.
  • (Vertically align objects according to the center): Will align selected objects vertically according to the center of the frame.
  • (Horizontally align objects according to the middle): Will align selected objects horizontally according to the middle of the frame.
  • (Center the selected object(s) on the canvas): Centers the selected objects on the canvas.
  • (Horizontally center the selected object(s) on the canvas): Horizontally centers the selected objects on the canvas.
  • (Vertically center the selected object(s) on the canvas): Vertically horizontally centers the selected objects on the canvas.
