MovieViewer - UI

Opening the UI

Output a movie to MMF format. The resulting playing movie is in the MovieViewer.

UI Test Cases

  • :Opens the previous movie (if you are viewing a project).
  • :Opens the next movie (if you are viewing a project).
  • :Lets you open a specific section (movie) if you are viewing a project. The current movie name (in this example, "Lesson 2") is displayed in the button.
  • :Restarts the movie from the first frame.
  • :Opens the previous frame in the movie.
  • :Opens the next frame in the movie.
  • :Pauses the movie.
  • :Displays the current frame number as the movie plays.
  • :Displays the time for each frame as the movie plays.
