Getting Started Wizard - UI

Accessing the Getting Started Wizard UI

By default, when Mimic is first launched, the Getting Started Wizard should open. There is an option in the Getting Started Wizard to show it on launch. If that option is turned off, the Getting Started Wizard can be opened by:

!File | Getting Started Wizard

Click the "Launch the Getting Started Wizard" option in the Getting Started section of the Start Page

Getting Started Wizard UI

UI Controls

  • Action: Offers a series of options used to get started using Mimic. As each option is selected, an explanation of the option should appear in the window to the right. Explanations are included below in italics.

    • Watch tutorial "How to Create Movies": This option pens the 45-Minute interactive video tutorial "How to Create Movies", which shows you step-by-step how to create, enhance, and build Mimic projects and movies.

      Clicking this option will activate the <Finish> and <Cancel> button. Click <Finish> to launch the video tutorial in Mimics internal Movie Viewer.

    • Open a sample Movie: This option guides you through steps for opening a sample standalone Mimic movie (not contained in a project). To become familiar with Mimic, you can then work inside the sample movie just as you would any other movie.

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click Finish to open an Untitled sample Mimic movie.

    • Open a sample Project: This option guides you through steps for opening a sample Mimic project, which contains multiple movies in a collection. To become familiar with Mimic, you can then work inside the sample project (and movies) just as you would any other project.

      Clicking this option will activate the <Next> and <Cancel> button. After you select this option, click <Next> to move to the Getting Started Wizard; Open Sample Project dialog.

    • Start a new Project: This option opens the Start New Project Wizard, which guides you through the basic steps for creating a new Mimic project, which is designed to hold multiple movies in a collection. When you finish providing information in the Start New Project Wizard, the new project is loaded into Mimic. (Note: You can also create standalone movies in Mimic, without creating a project.).

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click <Finish> to launch the Start New Project Wizard.

  • Show this dialog on launch: If there is a check mark in the box, this wizard will open each time you open Flare in the future. If you do not want to see the Getting Started Wizard each time you open Flare, remove the check mark from the box.
  • Back: This button is disabled in this window.
  • Next: Opens the next page of the Getting Started Wizard ("Open a sample project" page 1) if you have selected the "Open a sample project" option. Otherwise, this button is disabled.
  • Finish: Completes the option that you selected (unless you selected "Open a sample project," in which case this button is disabled).
  • Close/Cancel: Closes the Wizard.

Open a Sample Project - Page 1

UI Controls

  • Sample Projects: Displays a list of Sample Projects provided by Mimic. Select one to use as a template for the new project.
  • Project Name: Type any name for the project, or keep the one provided for you.
  • Project Folder: Displays the folder to which the new project will be saved. By default, Mimic will use My Documents\My Mimic Projects\Samples.
  • Pick Folder: Opens the Browse For Folder dialog, which lets you select a different folder to store the sample project files.
  • Show this dialog on launch: Same as above.
  • Back: Returns to the previous page of the wizard ("Welcome to MadCap Flare").
  • Next: Disabled at this point
  • Finish: Creates the sample project and loads the files into Mimic.
  • Cancel: Closes the Wizard and does not create a new project.
