Menus - UI

Menus are located at the top of the application, above all the toolbars

 

Menus - Test Cases

Open a Toolbar

  • Clicking the Open a Toolbar menu button will present the following options:
    • Standard: Will open/close the Standard toolbar.
    • Format: Will open/close the Format toolbar.
    • Frames: Will open/close the Frames toolbar.
    • Layout: Will open/close the Layout toolbar.

 

File Menu

To access:

  • Click File
  • ALT + F

 

Verify:

  • Getting Started Wizard...: Launches the Getting Started Wizard
  • New Project...: Opens the New Project dialog
  • New Movie: Expand the New Movie menu
  • Open: Opens the Open menu
    • Can filter for .miprj, and mimov file formats
  • Reload: Will reload the current movie, discarding any unsaved changes.
  • Test Files: Expands the Test Files menu
    • This is removed for release
    • When active, only appears on MadCap domain machines
  • StartUp File...: Opens the StartUp File dialog
    • Used to select what movie or project the user wishes to open by default when Mimic opens
  • Close: Will close the currently open movie
  • Close Project: Will close the currently open project
  • Save: Will save the movie or project that is currently in focus
  • Save As: Will open the Save As dialog
    • This option is disabled when a Project Editor has focus
    • Options available are: .mimov
  • Save To: Will open the Save To dialog
    • This option is disabled when a Project Editor has focus
    • Options available are: .pdf, .xps, .mimov
  • Save All:  Will save all currently dirty movies/projects
  • Send To:  Will provide a list (provided by windows) of applications to send the currently active project / movie to.
  • Print: Will bring up the standard windows Print dialog.
  • Properties:  Will open up the properties Dialog for the active movie or project
  • Recent Files: Will open the Recent files sub menu, listing all the recently used movies.
  • Recent Projects: Will open the Recent projects sub menu, listing all the recently used projects
  • Exit: Will close Mimic

 

Project Menu

To access:

  • Click Edit
  • ALT + E

 

Verify:

  • Build the movie collection:  Will build all the project movies.
  • View Generated Movie Collection: Will view the generated project
  • Publish the Generated Movie Collection: Will open the publish dialog
  • Clean Project Output...: Will delete any output files in the output folder
  • Open Output Folder: Will open the output folder in a Windows explorer window
  • Project Properties: Will open the project properties editor

 

Movie Menu

To access

  • Click Movie
  • Alt + M

 

Verify:

  • New Movie: Expands the New Movie Menu
  • Build the Output Movie: Builds the movie
  • View Generated Movie: Will view the generated Movie
  • Publish the Generated Movie: Will bring up the publish dialog
  • Quick-Preview Movie: Expands the Preview menu
  • Clean Movie Output...:  Will delete all output files in the output folder
  • Open Output Folder: Will open the output folder in a Windows explorer window
  • Edit Skin...: Will open the skin editor
  • Movie Properties...: Will open the movie properties dialog

New Movie Sub Menu(s)

Sub menu(s) of the Movie menu:

 

Verify:

  • Record Movie: Expands the Record Movie Submenu
    • Record Application: Records actions in an open application
    • Record Window: Records actions in an application window
    • Record Region: Records actions in a rectangular region that is defined by the user
    • Record Full Screen: Records actions within the full computer screen
    • Hide On record:  When checked, Mimic will minimize during a recording operation. This is checked by default.
    • Options: Opens the recording options dialog
  • Create Movie From Image: Expands the Record Movie submenu
    • Capture Application: Captures an open application as a single image
    • Capture Window: Captures an application window as a single image
    • Capture Region: Captures a designated region as a single image.
    • Capture Full Screen: Captures the full computer screen as an image
    • Create from File...: Invokes the Create from File dialog
  • Create Blank Movie...: Opens the Create Blank movie dialog
  • Import Mimic Movie...: Opens the Import Mimic Movie Dialog
  • Import PowerPoint Presentation...: Opens the Import PowerPoint Presentation dialog

Quick-Preview Movie Sub Menu

Verify:

  • Movie: Previews the entire movie
  • This Frame: Previews the current frame only
  • Start at this Frame: Previews from the current frame to the last frame of the movie
  • Next 5 Frames: Previews the current frame and the next 5 frames
  • Next 10 Frames: Previews the current frame and the next 10 frames

Edit Menu

To access:

  • Click Edit
  • Alt + E

 

Verify:

  • Undo: Undo is available when an editing action has been done to a movie frame. Test depth of undo list.
  • Redo: Is abvailable after undo action has been executed. Test depth of redo list.
  • Cut: Will cut the active object/frame
  • Copy: Will copy the active object/frame
  • Paste: Will paste whatever is currently on the clipboard
  • Delete: Deletes the active selection
  • Edit Text:  Will open the edit text window of an object
  • Insert Variable...: Will bring up the Insert Variable dialog
  • Select All:  Will select all objects on the frame
  • Properties: Will open the properties for the active selection

View Menu

To access:

  • Click View
  • Alt + V

 

Verify

  • Toolbars:  Will expand the toolbar submenu
    • Standard: Will open/close the Standard toolbar.
    • Format: Will open/close the Format toolbar.
    • Frames: Will open/close the Frames toolbar.
    • Layout: Will open/close the Layout toolbar.
  • Show Anchor arrows: Will toggle on/off the display of anchor arrows on objects in the main editor
  • Draw Trajectories: Will toggle on/off the display of object trajectories in the main editor
  • Show Conditional Indicators:  Will toggle on/off the display of conditional indicators on objects in the main editor
  • Project Movies: Will open/focus on the project movies pane
  • Frames: Will open/focus on the frames pane
  • Master Frames: Will open/focus on the master frames pane
  • Link Browser: Will open/focus on the link browser
  • Timeline:  Will open/focus on the timeline pane
  • Object Layers: Will open/focus on the Object layers pane
  • Palettes: Will open/focus on the Palettes pane
  • Frame Libraries: Will open/focus on the Frame Libraries pane
  • Start Page: Will open/focus on the start page
  • Messages: Will open/focus on the Messages pane
  • Status Bar: Will toggle the stats bar on/off

Insert

To access:

  • Click Insert
  • Alt + I

 

Verify:

  • Add Image: Opens a standard windows Open dialog
    • Filters for all image files
    • Navigating to an image file and pressing OK will insert that Image as an image object on the frame
  • Add Cursor from File:  Will open a standard windows Open dialog
    • Filter for .cur file types
    • Navigating to a cursor file type and pressing OK will insert the Cursor as an image object on the frame
  • Insert Capture Region: Capture a rectangular region on your screen and insert it into the current frame as an image object.
  • Insert Captured UI Element: Capture a fixed area of a window and insert it into the current frame as an image object.

Format

To access:

  • Click Format
  • Alt + O

 

Verify:

  • Object Format Toolbar: Opens the object format toolbar
  • Layout Toolbar: Opens the layout toolbar
  • Align Text: Will open the align text sub-menu (not shown)
    • Left:  Will align the text of the selected object(s) to the left
    • Center: Will align the text of the selected object(s) to the Center
    • Right: Will align the text of the selected object(s) to the right
    • Top: Will align the text of the selected object(s) to the top
    • Middle: Will align the text of the selected object(s) to the middle
    • Bottom: Will align the text of the selected object(s) to the bottom
  • Order: Will open the order sub-menu (not shown)
  • Make Same: Will open the make same sub-menu (not shown)
  • Align on Canvas: Will expand the align on canvas sub-menu (not shown)
    • Center on canvas: Will set the objects to the center of the canvas
    • Vertically center on canvas: Will set the objects vertically center on the canvas
    • Horizontal center on canvas: Will set the objects horizontally center on the canvas
  • Rotate: Will expand the Rotate sub-menu (not shown)
    • Each rotate option will rotate the selected object the amount indicated
    • Reset Rotation: will put the object back to its default location (zero degrees rotated)
  • Shadow: Applies/removes a shadow to the selected object
  • Bold:  Bolds selected objects text
  • Italic:  Italicizes a selected objects text
  • Underline: Underlines a selected objects text
  • Color : Lets you select a color for the selected content
  • Background Color: Lets you select a background color to the selected content
  • Use Default Look...:  Will set the object to use the set default attributes
  • Set Default Look...: Will set the currently selected objects attributes as the default for that shape
  • Make All Look The Same: Selected objects attributes are applied to all similar objects on the screen
  • Pick Style...: Lets you pick a style of the selected shape. Any style you have added to your palette, or a factory style can be chosen
  • Trajectory Mode: Will let you set a trajectory for the object
  • Reset Trajectory: Will remove the trajectory for the object
    • Will be grayed out if no trajectory is present

Frames Menu

To access:

  • Clicking the Frames menu
  • Alt + R

 

Verify:

    • Copy Frame: Will copy the selected frame to the clipboard.
    • Paste Frame: Will paste the frame currently located on the clipboard.
    • Duplicate Frame:  Will make a duplicate of the selected Frame
    • Remove Frame: Will remove the selected frame.
    • Add Frame to Library: Will add the current frame to the active Frame Library.
    • Add Frame to Master Frames: Will add the currently selected frame to the master frames list.
    • Copy Frame Background: Will copy the current frames background to the clipboard.
    • Paste Frame Background: Will paste the current background on the clipboard to the selected frame.
    • Frame Properties...: Will open up the frame properties dialog.
    • Previous Frame: Go to the previous frame
    • Next Frame: Go to the next frame
    • First Frame: Go to the first frame
    • Last Frame: Go to the last frame
    • Insert Frame: Will Insert a frame below the selected frame using the selected frames background.
    • Insert Blank Frame: Will insert a blank frame.
    • Insert PowerPoint Slide: Will bring up the insert PowerPoint dialog.
    • Capture New Frame...:  Will capture a new frame and insert it into the movie
    • Record New Frames...: Will begin the recording process to record new frames and insert them into the movie
