UI Controls

Misspelled Word

  • (misspelled word box): Displays the word that Flare considers misspelled. The word is also highlighted in the topic.
  • Find Next: Click this button to initiate the spell check search. This button is replaced by the Skip button after user clicks it.
  • Skip: Temporarily ignores the spelling of the word (this occurrence only) and moves on to the next misspelled word. If other occurrences of the word are found in the topic, they will be flagged as questionable spellings.
  • Add To Dictionary: Adds the word to the project dictionary so that it is not flagged as a questionable spelling in the future. Flare then moves on to the next misspelled word.
  • Ignore: Ignores the spelling of the word and moves on to the next misspelled word. Flare also adds the spelling of this word to the Ignored Words dialog (for this topic only). This means that if other occurrences of the word are found in the topic, they will not be flagged as questionable spellings.
  • Ignore in All Files: Ignores the spelling of the word and moves on to the next misspelled word. Flare also adds the spelling of this word to the Ignored Words dialog (for all topics included in the search). This means that if other occurrences of the word are found in any other topics included in the search, they will not be flagged as questionable spellings.
  • Start Spell Check: Click the down arrow and select one of the following:
    • (current document): Flare checks the spelling in the active topic only.
    • (whole project): Flare checks the spelling in every topic in the project.
    • (all open documents): Flare checks the spelling in all topics currently open in the interface.
    • (documents in same folder): Flare checks the spelling in all topics located in the same folder as the current one in the Content Explorer.
  • [ >> ] : Skips searching in the current topic and moves on to the next file.

Suggestions:

  • (suggestion box): Displays a suggested word to replace the misspelled word. Additional suggestions are listed below. You can use the suggested term, type user’s own corrected word in this field, or select a word from the list (which is then automatically added to the Suggestions field).
  • Change: Changes only that occurrence of the misspelled word to the word in the suggestion box..
  • Change in All Files: Changes that occurrence of the misspelled word, as well as any other occurrences of it in the rest of the project.
  • (further suggestions box): Lists additional suggestions. Clicking any suggestion in this lower box will put that suggestion in the upper suggestion box where it can be used to replace the misspelled word.

 

Setting the Language

Setting the language of your project

Set the project/movie language in the by viewing Project Properties/Movie Properties (Language tab). Mimic will use this language when spell checking your project. Test that Spell Check works correctly with different project/movie languages set.

Select Language - UI Test Cases

To open the dialog:

  • Click on the Edit File Properties buttonin the Frame Editor | Language Tab
  • !File | Properties | Language Tab
  • !Movie | Movie Properties | Language Tab
  • Click on the Project Properties buttonin the Standard Toolbar | Language Tab
  • Click on the Project Properties buttonon the local toolbar of the Movies Pane | Language Tab

UI Controls

  • (Language list): Displays a list of all languages that can be used to tag content.
  • Hide unsupported languages: Reduces the list of visible languages to those which have spell check support.
