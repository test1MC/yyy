Master Frames - UI

Opening the Master Frames pane:

  • !View | Master Frames
  • Click the Master Frames icon on the standard toolbar

UI Test Cases:

  • Default location of the Master Frames pane is on the left edge of the application.

Local Toolbar

  • Frame Properties: Will open the frame properties for the selected master frame (See Editing).
  • Insert Blank Frame: Will insert a new blank frame into the master frames list. Essentially adding a new master frame to choose from when applying master frames to a movie or individual frames.
  • Insert Frame using the same background as the selected frame: Inserts a new frame into the master frames list that uses the same background as the currently selected frame.
    • This frames background is linked to the previous frames background. If the previous frames background changes, this frames background will change as well.
  • Remove Frame: Will remove the selected frame from the master frames list.

Main Editor

  • The main editor contains a master frames filter on the bottom toolbar:
    • Show All Master Frames: When this is selected, all master frames that are applied to this frame will be displayed in the main editor.
    • Show Bottom Layer Master Frame: When this is selected, the Master Frame that is set as the frames bottom layer will be displayed only. The frame that is set as the top layer will be hidden.
    • Show Top Layer Master Frame: When this is selected, the Master Frame that is set as the frames top layer will be displayed only. The frame that is set as the bottom layer will be hidden.
    • Hide Master Frames: When this is selected, the main editor will not display any master frames in the main editor.
    • Note: These are for editor display purposes only. The output will always show the master frames that are applied to the frame, regardless of which setting above was set.
  • When focusing on a Master Frame (via clicking on one in the Master Frames pane), the main editor background will change color to a light blue. This indicates to the user that they are now looking at a Master Frame and not a frame in the movie.

Misc.

  • The Master Frames pane will display all the Master Frames.
    • By default, a new movie will have one master frame named 'Navigation'.

 

  •  Right-Clicking a frame in the Master Frames pane:
    • Copy Frame: Will copy the frame to the clipboard.
    • Paste Frame: Will paste the frame that is on clipboard into the master frames list.
    • Remove Frame: Will remove the selected frame from the master frames list.
    • Add Frame to Library: Will add the currently selected frame to the current Frames library.
    • Copy Frame Background: Will copy the current frame background to the clipboard.
    • Paste Frame Background: Will paste the background currently on the clipboard to the selected Master Frame.
    • Frame Properties...: Will open up the Frame properties for the selected Master Frame (See editing).
    • Insert Frame: Inserts a new frame into the master frames list that uses the same background as the currently selected
    • Insert Blank Frame: Will insert a new blank frame into the master frames list. Essentially adding a new master frame to choose from when applying master frames to a movie or individual frames.
    • Insert PowerPoint Slide: Currently will bring up the Import PowerPoint dialog.

 

  • Right-Clicking on a Frame in the main editor: 
    • Add Frame to Master Frames: Will add the current frame to the Master Frames list.
    • Pick Master Frames: Will bring up the Pick Master Frames dialog (See editing).
    • Remove Master Frames: Will remove the master frames from the current frame. Essentially un-apply the master frame selection.

     

  • Right-Clicking on a frame in the Frame Libraries pane:
    • Add Frame to Master Frames: Will add the selected frame to the Master Frames list
