Master Frames - Editing

Editing and assigning master frames are covered here

Editing - Test Cases

Pick Master Frames dialog is accessed by right-clicking on a frame in the main editor, a frame in the frames pane, or in the Frame's properties dialog.

Pick Master Frames dialog contains two drop-down areas:

  • Bottom Layer:
    • Contains all of the movies Master Frames
      • Selecting a Master Frame from this drop-down will set that Master Frame as this Frames Bottom Master Frame.
      • Selecting a Master from here, at the Frame level, will take precedence over a Movie level Master Frame that has been set.
    • Contains the option No Bottom Layer Master Frame
      • Will set this Frame to have no bottom layer Master Frame, not even if one is set at the Movie level
    • Contains the option Use Movie's Bottom Layer Master Frame
      • Will use the Bottom Layer Master Frame that has been set at the Movie Level
  • Top Layer:
    • Contains all of the movies Master Frames
      • Selecting a Master Frame from this drop-down will set that Master Frame as this Frames Top Master Frame
      • Selecting a Master from here, at the Frame level, will take precedence over a Movie level Master Frame that has been set.
    • Contains the option No Top Layer Master Frame
      • Will set this Frame to have no top layer Master Frame, not even if one is set at the Movie level
    • Contains the option Use Movie's Top Layer Master Frame
      • Will use the Top Layer Master Frame that has been set at the Movie Level

 

Master Frame frame property dialog is accessed by focusing on a Master Frame (via the Master Frame Pane) and selecting Frame Properties.

Master Frame frame properties contain three tabs:

  • Appearance: Controls the general look and border of the Master Frame. See Frame Properties test plan for more information.
  • Bitmap: Sets the background image for the Master Frame. See Frame Properties test plan for more information.
  • Conditions:  Sets what conditions to exclude / include for the Master Frame. See the conditions test plan for more information.

Clicking the Movie button will open up the movie properties dialog. See the Movie Properties test plan for more information.

 

Misc.

  • Adding objects to the Master Frame will appear on any frame that applies that Master Frame
  • Adding a variable to an object on a Master Frame will update as needed depending on what frame has applied that Master Frame
    • For example: Putting a FrameName variable on a Master Frame, that variable will show a different value for each frame that it is applied to
