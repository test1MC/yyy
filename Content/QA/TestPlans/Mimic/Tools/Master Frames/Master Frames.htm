Master Frames

Bug Base Product Area:

  • Master Frames

 

Description:

Master Frames are special frames that allow objects to be applied to many frames at once. Using Master Frames in a movie will allow the user to set up a bottom and a top layer that may be applied to any frame of the movie. For example, setting a common background for frames to share or a company logo that you wish to appear on all frames in the same spot. Using the master frame will allow you to make a change in one location, that will update across the movie.

 

Master Frames can be applied at 2 layers, the Top Layer and the Bottom Layer. The bottom layer will put everything on the master frame below any object that exists on the applied frame. The top layer will put everything on the master frame on top of any object that exists on the applied frame. Each frame in a movie can have 2 Master Frames applied to it, one in each layer.
