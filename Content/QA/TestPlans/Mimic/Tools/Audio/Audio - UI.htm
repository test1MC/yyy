Audio - UI

Opening the Audio Tab:

Opening the Audio Tab:

  1. !Project| Project Properties or !Movie | Movie Properties or Object Properties
  2. Click the Audio tab item

UI - Enabling Audio:

  • Enable Audio: This will toggle the sound on/off for the selected Movie/Frame/Object

    • All audio options will be disabled if this is unchecked
  • Audio: Will list the currently selected audio
    • The file will be preceded by system if the sound is a system sound
    • The file will be preceded by movie if the sound has been imported from an external source
  • Audio Controls:
    • Play: Will play the selected audio file
    • Stop: Will stop the sound from playing (Disabled if the sound is not playing)
    • Edit...: Will launch the audio editing tool (essentially Echo) See Audio - Editing
  • Pick Built-In Sound...: Opens the Pick System Sound dialog
  •  Pick From Movie Sound Library...: Opens the Manage Movie Sound Library dialog
  • Import Into Sound Library...: Allows the user to browse to external audio files
    • Selected files are then imported (copied) into the Mimic movie/project, into the Sounds folder
    • If the folder does not exist, we create one automatically
    • User can import .mp3 or .wav files only
  • Record a New Audio Clip...: Opens the Record dialog. See Audio - Recording
  • Loop Sound: If this is checked the sound file will repeat continuously in a loop for the duration of the movie/frame/object
  • Stop Sound of Previous Frame: This option is only visible in the Audio tab of Frame properties
    • Selecting this will stop all sounds that were initiated on previous frames
  • Selecting OK will apply the chosen sound
  • Selecting Cancel will discard changes

Pick System Sound

  • System Sound Area: Contains a list of audio files included with mimic

    • These sounds are included in Movie Viewer
    • These sounds are included in Help Viewer
    • Selecting a sound and pressing Ok will assign that sound to the movie/frame/object
  • Play on Select: If this is checked, when selecting a sound from the list, the sound will play. Uncheck this to disable this function.
  • Play: Press this button to play the currently selected sound
  • Stop: If a sound is playing, this will stop it.
    • Disabled if no sound is playing

Manage Movie Sound Library

  • Movie Library Sound: This column will list all the sounds you have imported into the movie

    • Sounds imported on movie/frame/object will appear here
    • Sounds no longer "used" will still appear here. They are not removed once you stop using that sound
  • Use Count: Will display how many times this sound has been used in the movie
  • Play on Select: If this is checked, when selecting a sound from the list, the sound will play. Uncheck this to disable this function.
  • Play: Press this button to play the currently selected sound
  • Stop: If a sound is playing, this will stop it.
    • Disabled if no sound is playing
  • Remove: Will remove the selected sound from the movie
