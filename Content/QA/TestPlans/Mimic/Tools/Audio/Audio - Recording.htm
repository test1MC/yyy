Audio - Recording

Opening the Recording Dialog:

Opening the Audio Tab See Audio - UI first:

  1. Click Record a New Audio Clip...

UI - Creating and Editing Conditions:

  • Sound Name: Field for giving the recording a name

    • All audio options will be disabled if this is unchecked
  • Input Source: Sets the input source for recording audio
    • Will list the available input sources provided by your system
  • Encoding Frequency:Sets the frequency that the recording will be encoded in
    • Verify on disk that the correct frequency was used
    • Recordings are recorded in .mp3 format
  • Pick Built-In Sound...: Opens the Pick System Sound dialog
  •  Pick From Movie Sound Library...: Opens the Manage Movie Sound Library dialog
  • Import Into Sound Library...: Allows the user to browse to external audio files
    • Selected files are then imported (copied) into the Mimic movie/project, into the Sounds folder
    • If the folder does not exist, we create one automatically
    • User can import .mp3 or .wav files only
  • Record Time: During the recording process, this will display the amount of time that has passed
  • Record: Initiates the recording process.
  • Pause: Pauses/Resumes the recording process.
  • Stop: Stops the recording process
  • Recording window: Will display the sound wave while recording
    • Louder sounds will be indicated by a higher spike in the wave form
