Palettes Pane

Bug Base Product Area:

  • Palettes

 

Description:

A palette is a collection of objects and object groups. This collection is stored in a file type on disk with the extension of .capal. This file is stored in the My Documents\My Pallets\ folder. All your palettes are accessible from within Mimic or Capture via the palette pane. Palettes are edited directly inside of Mimic (or capture).
