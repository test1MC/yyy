Palettes Pane - UI

Opening the Palette Pane:

  1. !View | Palettes or
  2. Click the Palette button located on the standard toolbar

UI Test Cases:

Default location of the palette pane is on the left edge of the application.

Local Toolbar

  • Palette: Displays the name of the currently shown Palette. Clicking the down arrow will display a list of all local Palette files (.capal files). Selecting a Palette from this list will display that Palette in the Palette Pane.
  • Create New Palette: Will bring up the New Palette dialog.

      • Entering a name into the Field and pressing OKwill create a new Palette
      • The new Palette will have focus in the main Palette Pane
      • Entering a duplicate Palette name or a blank name will result in an error
  • Delete Palette: Will delete the currently displayed palette. This will invoke a simple Yes/No confirmation dialog, which lists the name of the palette you wish to delete.

Misc.

Right-Clicking on an object from within the Palette pane:

  • Cut: Cuts the object
  • Copy: Copies the object
  • Paste: If an object exists on the clipboard, this will be active and will paste the object into the Palette. Otherwise the option is disabled.
  • Delete: Will delete the object from the Palette

 

 

Right-Clicking on an object in the main editor:

  • Selecting Add To Palette will add the currently selected object to the currently active Palette

 

Dragging an object from the Palette to the main editor will place a copy of the object on the frame

Double-Clicking an object on the Palette will place a copy of the object on the frame
