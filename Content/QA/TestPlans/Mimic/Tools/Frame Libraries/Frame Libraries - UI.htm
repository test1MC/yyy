Frame Libraries - UI

Opening the Frame Libraries pane:

  1. !View | Frame Libraries
  2. Click the Frame Libraries icon on the standard toolbar

UI Test Cases:

Default location of the Frame Libraries pane is on the left edge of the application.

Local Toolbar

  • Library: Displays the name of the currently shown Library. Clicking the down arrow will display a list of all local Library files (.miflib files). Selecting a library from thi slist will display that Library in the Frame Libraries Pane.
  • Create a new Library: Will bring up the New Library dialog
    • Entering a name into the Field and pressing OKwill create a new Library
    • The new Library will have focus in the main Frame Libraries Pane
    • Entering a duplicate Library name or a blank name will result in an error
  • Delete Library: Will delete the currently displayed library. This will invoke a simple Yes/No confirmation dialog, which lists the name of the library you wish to delete.
  • Add Frame to end of the current movie: Will add the currently selected frame to the bottom of the currently active movie
  • Add Frame as a Master Frame to the current movie: Will add the selected frame to the Master Frame list of the currently active movie.

Misc.

Right-Clicking on a frame in the frames pane: 

  • Add Frame to Library will add the selected frame to the currently active Frame Library

 

Right-Clicking on a Frame in the main editor: 

  • Add Frame to Library will add the current frame to the currently active Frame Library

 

Right-Clicking on a frame in the master frames pane:  

  • Add Frame to Library will add the current frame to the currently active Frame Library

 

Right-Clicking on a frame in the frames library pane:  

  • Add Frame to Movie will add the selected frame to the currently active movie
  • Add Frame to Master Frames will add the selected frame to the master frame list
  • Copy Frame will copy the selected frame to the clipboard
  • Paste Frame will paste any frame that is currently on the clipboard, into the currently active Frame Library
  • Remove Frame will remove the currently selected frame from the Frame library
