Bug Base Product Area:

Frame Libraries

 

Description:

A Frames Library is a collection of Frames. This collection is stored in a file type on disk with the extension of .miflib. This file is stored in the My Documents\My Mimic Frame Libraries\ folder. All your Frame Libraries are accessible from within Mimic via the Frame Libraries pane.
