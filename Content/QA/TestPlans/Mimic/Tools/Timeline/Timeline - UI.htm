Timeline - UI

Opening the Timeline pane:

  1. !View | Timeline
  2. Click the timeline button on the toolbar
  3. Ctrl + T

UI - Test Cases

Default location of the Timeline pane is on the bottom of the application

Local Toolbar

  • Properties: View properties dialog of selected object.
    • Selecting an object on the timeline and hitting F4 will also bring up the properties dialog of that object.
  • Toggle Time spanned property: Toggles on / off the Time Span property of the selected object.
    • Turning off will make the object appear for the entirety of the Frame.
    • Turning this on will allow the user to specify the duration the object remains on screen during the Frames time span.
      • Manually changing the objects duration will automatically turn this feature on.
      • Toggling this off and then back on, Mimic will remember the previous time span that was set.
      • Default state of an object is to have this turned off (i.e. an object by default will appear on the frame for the entire duration of the frame).
  • Adjust shape time span automatically when frame time span changes: Toggles on /off the automatic adjust of a shapes time span when the frames time span changes.
    • Toggling this On will automatically change the time span of the object whenever the frames time span changes. Essentially keeping the ratio of the objects duration to that of the frames total time span.
    • Toggling this Off will maintain the shapes time span, no matter what the length of the frame is.
    • Default state of an object is to have this property turned off.
  • Move item to the left: Will move the selected item to the left 0.1 second on the time line. Essentially subtracting 0.1 second from the beginning and end of the objects time span.
    • This will not effect (and should show as disabled) an object that has its time span property turned off
  • Move item to the right: Will move the selected item to the right 0.1 second on the time line. Essentially adding 0.1 second from the beginning and end of the objects time span.
    • This will not effect (and should show as disabled) an object that has its time span property turned off
  • Frame Time span: Sets the time span for the current frame.
    • Using the arrows will increase/decrease the time span by 1.0 second in the appropriate direction.
    • Updating the frames time span via the frame properties should be reflected here immediately.
    • Updating the frames time span here should be reflected in the frame properties immediately.
  • Movie Length: Shows the length of the entire movie. This area has no interaction capabilities.
    • Updating any time for a frame should update this field immediately.

Misc

  • Under the local toolbar will be a list of whole numbers, in seconds, depicting the length of the frame.

    • For Example: A 10 second movie will show seconds 0 through 9.
  • Every object on the frame should be represented on the Timeline with an icon and name appropriate to the object type listed.
  • Selecting an object on the current frame will highlight that object on the timeline.
  • Selecting an object on the timeline will highlight that item on the current frame.
  • Any object that has the time span property turned on, will have the amount of time (in seconds) it will be displayed on the frame next its name.
  • Any object that has the time span property turned off, will display no number next to its name

  • Right-Clicking on an object in the Timeline will bring up a 2 option menu.

    • Time Spanned: Turns on / off the time span property of the selected object. (See above)
    • Properties: Will bring up the properties dialog of the selected object. (See above)
  • An object with a condition on it, will shot that color of the condition in the toolbar. The entire object bar in the timeline will be the condition color.
