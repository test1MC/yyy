Conditions

Bug Base Product Area:

  • Conditions

 

Description:

Conditions in Mimic function as they do in Flare, however there are no Condition Sets in Mimic. Conditions are used in order to reuse a same set of content, but have different results on output depending on which conditions have been set and applied. All conditions are defined at either the Project or Movie level. In addition, a Project or Movie can link to a Flare project in order to access that Flare projects condition sets. A condition can be applied to an object, frame or movie.
