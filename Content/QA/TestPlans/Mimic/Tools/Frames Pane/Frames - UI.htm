Frames Pane - UI

Opening the Frames Pane:

  • !View | Frames
  • Ctrl + J
  • Clicking the Frames toolbar button

UI - Test Cases

  • By default, the frames pane opens on the left of the application.

Local Toolbar

Misc.

 

  • Dragging frames in this pane will reorder the sequence of the frames.
    • You can drag a frame (or multiple selected frames) to anywhere in the frame sequence.
  • Selecting a frame in this pane will open that frame up in the main editor.
  • Double-clicking a frame in the pane will bring up that frames properties.
  • Select multiple frames by shift-click or ctrl-click.
  • Using the arrow keys while in the Frames pane, will navigate up or down the frame sequence.
