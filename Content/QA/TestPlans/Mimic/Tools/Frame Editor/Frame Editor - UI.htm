Frame Editor Local Toolbar - UI Controls

Toolbar Items (Tooltips in parentheses)

  • Frame: Frame Name.

  • (Go back to the Previous Frame in view): Go back to the last viewed frame.
  • (Go to the First Frame): Go to the first frame.
  • (Go to the Previous Frame): Go back to the previous frame.
  • (Go to the Next Frame): Go to the next frame.
  • (Go to the Last Frame): Go to the last frame.
  • (Quick Preview Mode): Generate a quick preview of X amount of frames.
    • Movie
    • This Frame
    • Start at This Frame
    • Next 5 Frames
    • Next 10 Frames
  • (File Properties): Opens the properties of the movie/project that is open.
  • (Frame Properties): Opens the properties of the current frame.
  • (Shape Properties): Opens the properties for the currently selected shape.
  • (Open the Object Format Toolbar): Opens the Object Format Toolbar.
  • (Open the Object Layout Toolbar): Opens the Object Layout Toolbar.
  • (Opens the Frames Toolbar): Opens the Frames Toolbar.
  • (Draw Animation Trajectories):Show the animation trajectories for shapes that have the 'Move' animation effect.
  • (Delete Point): Deletes the selected point on an object
  • (Hand Mode): Lets you drag areas of a frame around.
  • (Select Mode): Converts the cursor to Select mode, which lets you select any objects or areas in the frame.
  • (Bubble Mode): Converts the cursor to Bubble mode, which lets you create a callout by clicking in the frame and dragging a rectangle area. When you release the cursor, a "bubble" shape is created, which consists of a rectangular or square area for holding text and a pointy area for pointing toward an area of the frame that you want to explain or emphasize.
  • (Text Box Mode): Converts the cursor to Text Box mode, which lets you create a square or rectangle shape with text by clicking in the image and dragging in any direction. When you release the cursor, default text is shown in a popup editor. You can replace the text with your own and click outside the area to see the finished shape. In addition, this mode adds a background to the box (as opposed to a transparent background), which makes it easier to see the text.
  • (Arrow Mode): Converts the cursor to Arrow mode, which lets you create a curvy arrow shape. After you select this button, you can click anywhere in the frame to identify the starting point for the arrow. When you move the cursor, a straight line shows where the arrow will be placed. When you release the cursor, an arrow is created.
  • (Loop Mode): Converts the cursor to Loop mode, which lets you create a "loop" shape by clicking in the image and dragging a rectangle area. When you release the cursor, a loop shape is created. This is a useful shape for circling parts of an image that you want to emphasize.
  • (Button Mode): Converts the cursor to New Button mode, which lets you create a button with text (e.g., "Click Me"). You can double-click the button to open the Button Properties dialog, which lets you assign an action to a button (e.g., go to next frame) that occurs when a user clicks the button in the final movie. You can also use the appropriate tabs in that dialog (Mouse Hover and Mouse Pressed) to change the look of the button when the mouse hovers over it or presses it.

    • (New Button Mode): Lets you create a button.

    • (Rollover Caption Mode): Lets you create a rollover caption. This is an effect where you designate an area on the frame for the user to hover the mouse over. When the user does this, a caption with text that you provide appears on the frame.

    • (Typing Box Mode): Lets you create an animated typing box. When you record a movie, Mimic captures any typing actions that you perform on screen. This typing is therefore animated in the movie output. In other words, each letter of the content is typed as the user watches it. Not only are these animated text boxes added automatically when you record movies, but you can also add them manually when you edit the movie.

    • (Input Box Mode): Lets you create an input box. If you want the user to enter text at a certain frame in the movie, you can accomplish this by adding an input box. You can then enter the exact content that you want the user to type. You can also format the text, as well as specify captions and actions based on the user's input.

    • (Input Success Caption Mode): Lets you create an input success caption. After you add an input box to a frame, you can add a success caption to accompany it. This is a message that displays to users after they complete the input box successfully.

    • (Input Failure Caption Mode): Lets you create an input failure caption. After you add an input box to a frame, you can add a failure caption to accompany it. This is a message that displays to users if they fail to complete the input box successfully within the specified number of attempts.

    • (Input Hint Caption Mode): Lets you create an input hint caption. After you add an input box to a frame, you can add a hint caption to accompany it. This is a message that displays to users if they hover over the input box for help.

  • (Shade Effect Mode):Click the down arrow to select one of the following: Shade Effect Mode, Gray Scale Effect Mode, Blur Effect Mode, Zoom Effect Mode. Each of these modes lets you create a square or rectangle shape by clicking in the frame and dragging in any direction. When you release the cursor, the shape is created. The effect that you see depends on which mode you selected.
    • (Shade Effect): The area inside the shape is clear, while the area outside the shape is shaded. The shade color is based on the color selected in the Appearance tab of the Frame Properties dialog. From that dialog, you can change the color, as well as the darkness of the shading. If you want to create this effect with other closed-off shapes (e.g., oval, polygon), first draw the shape in the frame, then enable the shade effect from the Image Effects tab in the properties dialog for that object.

    • (Gray Scale Effect): The area inside the shape is displayed in its original color, while the area outside the shape is displayed in gray. If you want to create this effect with other closed-off shapes (e.g., oval, polygon), first draw the shape in the image, then enable the gray scale effect from the Image Effects tab in the properties dialog for that object.

    • (Blur Effect:) The area inside the shape is clear, while the area outside the shape is blurred. You can change the amount of the blurriness from the Frame Properties dialog. If you want to create this effect with other closed-off shapes (e.g., oval, polygon), first draw the shape in the frame, then enable the blur effect from the Image Effects tab in the properties dialog for that object.

    • (Zoom Effect): The area inside the original rectangle is enlarged and displayed in another rectangle that is larger and seemingly closer to the user (a three-dimensional effect). You can adjust the zoom settings from the properties dialog for that object.

  • (Rectangle Mode): Opens a submenu that lets you select additional shape options. The image on this button actually changes, depending on the last shape from the list that you used.

    • (Rectangle Mode): Converts the cursor to Rectangle mode, which lets you create a square or rectangle shape by clicking in the frame and dragging in any direction. When you release the cursor, the shape is created.

    • (Oval Mode): Converts the cursor to Oval mode, which lets you create a circle or oval shape by clicking in the frame and dragging in any direction. When you release the cursor, the shape is created.

    • (Polygon Mode): Converts the cursor to Polygon mode, which lets you create a closed plane shape bounded by three or more line segments. After you select this button, you can click anywhere in the frame to identify the starting point for the shape. When you move the cursor, a straight line segment is drawn. If you click the left mouse button, it signifies the end of that line segment and the start of another line segment. When you move the cursor, another straight line segment is drawn. When you double-click, a final line segment joins your starting and ending points, thus closing off the shape.

    • (Polyline Mode): Converts the cursor to Polyline mode, which lets you create a straight or angled line with an arrow at the end (which you can change or remove by using the Polyline Properties dialog). After you select this button, you can click anywhere in the frame to identify the starting point for the line. When you move the cursor, a straight line segment is drawn. If you click the left mouse button, it signifies the end of that line segment and the start of another line segment. When you move the cursor, another straight line segment is drawn. You can identify the end of the line by double-clicking.

    • (Cursor Mode): Converts the cursor to Cursor mode, which lets you add a cursor bitmap to the frame by clicking in the frame and dragging in any direction. When you release the mouse button, a cursor bitmap is added within a shaded oval shape. You can change the cursor type and adjust the color and transparency of the oval background from the Cursor Properties dialog. You can also use the Cursor Properties dialog to add animation to the cursor (causing it to move), and you can use the "Set Animation Trajectory" button to determine where

  • Enter 'Select Animation Trajectory Mode' for selected shape): Lets you determine the trajectory (path) of an object. For example, if you have added a cursor to the frame, you can click this button and then drag a path from the cursor to its destination on the frame. When you view the final movie, the object will move along the path that you created.
  • (Group Selected Items): This button is enabled if you have selected more than one object. You can group multiple objects into an object group, which then acts as a single object. And at any point, you can ungroup the objects so that they are separate once again..
  • (Shows the current warning, if any): If you have created an object that requires an action (such as a button or input box), or if you have created a caption that requires a connection to an object (such as a rollover caption), this button displays the number of objects or captions on the frame that contain issues. If you click this button, a small dialog opens, displaying the number of missing actions, broken actions, unconnected captions, spelling errors, or miscellaneous actions. A similar icon will display on the object(s) in question. After you fix the issue, this button and the object icon will no longer display.
