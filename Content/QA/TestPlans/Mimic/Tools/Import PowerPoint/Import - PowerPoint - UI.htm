Import PowerPoint Wizard

UI Controls

  • PowerPoint File: Displays the path and name of the *.ppt or *.pptx file you are going to import.
  • : Produces the Open dialog which lets you browse for *.ppt or *.pptx
  • Movie Name: Set a name of the resulting movie from the imported PowerPoint slides.
  • Import Slides as Background Images: Slides will be imported as background images. With this selected, no objects will be created (such as text boxes).
  • Change Movie Size: Allows you to change the dimensions of the resulting movie from the original dimensions of the PowerPoint Presentation.

Import PowerPoint Wizard dialog 2: Select Slides to Import

UI Controls

  • Select All: Lets you enter a name for the project that will be created from the imported documents.
  • Deselect All: Lets you set the location of the project.
  • Show Images: Show images of the imported PowerPoint Slides.
  • Import:Imports the selected slides.
