Skin Editor - UI Test Cases

Basic

UI Controls

  • Options:
    • Toolbar at Top: Toolbar at the top of the movie in output
    • Toolbar at Bottom:Toolbar at the bottom of the movie in output
    • Center Screen Progress Bar: Moves the progress bar from the normal navigation toolbar to the middle of the screen (does not apply to embedded skins)
  • Import/Export:
    • Import Skin...:Import a skin (Opens the Pick Skin dialog)
    • Export Skin...:Export a skin (Opens the Save Skin dialog)

Toolbar Top

UI Controls

  • Top Toolbar Buttons:
    • Fit to Movie:Resizes the toolbar to fit to the movie.
    • Enable Custom Layout: If this option is not checked, the default toolbar buttons will be used. When checked, the user can set which toolbar buttons to show on the toolbar, including their own custom buttons.
    • Selected: With the above option selected, these are the buttons that will show up on the toolbar.
    • Available: The buttons that are available for the user to select from when the 'Enable Custom Layout' option is checked.
  • :Adds the selected button(s) in the "Available" list to the "Selected" list
  • :Removes the selected button(s) from the "Selected List"
  • : Moves the selected button in the "Selected" list up in the sequence
  • : Moves the selected button in the "Selected" list down in the sequence

Custom Script

UI Controls

  • Edit...:Opens the Toolbar JavaScript dialog that allows you to add a javascript script to your toolbar.

About Box

UI Controls

  • About Box:
    • Browse: Opens a generic open dialog that lets you select an image that displays when you click the About button on the toolbar (you have to manually add this button via the skin editor)
    • Reset: Removes the image

Styles

UI Controls

Add Class: Click to add a new class to the selected style in the Style Tab.

  • Styles: All available styles for items that appear in the skin
  • Show Property Groups/Alphabetical List: Choose how you want the Properties to be displayed.
  • Preview: The Preview now shows below and the Style item that is selected will be highlighted with a label in the preview.
