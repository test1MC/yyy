Project Movies - UI

Opening the Project Movies Pane:

  • !View | Project Movies
  • Clicking Movies icon on the standard toolbar .
  • Clicking the Movies icon on the Project Editor toolbar .

UI Test Cases:

  • By default the Project Movies Pane appears on the left side of the application

Local Toolbar

  • Project Properties Editor: Opens the Project Properties Editor. (See Project Properties test plan for more info)
  • Create Blank Movie....: Will open the New Movie dialog.
  • Import Mimic Movie...: Will open up the Import Movie dialog.(See Below)
  • Import PowerPoint Presentation...:  Will open up the Import PowerPoint dialog.

Misc.

  • All current movies in the project will be displayed in the pane
  • Double-clicking a movie in the pane will open that movie in the main editor

Import Movie dialog:

  • Source File: Enter in the path to the desired movie here
  • Pressing the browse button will bring up a standard windows Open dialog. Use that to browse to the desired movie. (*.mimov)
  • File Name: Enter desired file name for the imported movie here
    • By default, the file name will be the name of the movie you are importing. This will be populated in the field once you select a Source File.
  • Right-Clicking a movie in this pane will give you one option, Delete
    • Selecting Delete will invoke a standard confirmation (yes/no) dialog.
      • Selecting Yes will remove the movie from the project.
      • Selecting No will cancel the operation and return you to the editor.
