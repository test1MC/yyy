Context Menu

Right-clicking a frame or the white space in the Link Browser while a frame is selected should yield the following context menu:

UI Controls
  • Copy Frame:Copy the selected frame
  • Paste Frame:Paste the copied frame
  • Remove Frame:Remove the selected frame
  • Copy Frame Background:Copy just the background of the selected frame
  • Paste Frame Background:Paste the copied frame background
  • Frame Properties...:Opens the Frames Properties dialog
  • Insert Frame:Insert a duplicate frame of the selected frame after the selected frame
  • Insert Blank Frame:Insert a blank frame after the selected frame
  • Insert PowerPoint Slide(s)...:Insert PowerPoint slides after the selected frame
