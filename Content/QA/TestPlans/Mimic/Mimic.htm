MIMIC

MadCap Mimic is a straightforward, easy-to-use software simulation tool that lets you quickly create fully interactive movies, simulations, presentations, and tutorials of software or systems without any programming knowledge.

  • Create fully editable, interactive movies of software applications
  • Allow your users to view and listen to demonstrations/presentations and (optionally) interact with simulations
  • Ideal for use in Technical Support, eLearning, Sales & Marketing, and in Help systems

 

Quickly Generate Multiple File Outputs

  • Microsoft® Silverlight™ and XPS
  • Adobe® Flash®, AIR™, PDF
  • Mimic Movie Format
