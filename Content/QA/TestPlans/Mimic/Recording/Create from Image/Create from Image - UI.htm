Recording Options

Options Context menu:

 

  • Snap to window... will open the Select Window dialog
  • Move Window into Rectangle... will open the Select Window dialog
  • Options... will open the options dialog

 

 

Snap to Window... brings up the Select Window dialog:

 

  • Displays all the currently opened windows
  • Selecting one and hitting OK will move the capture region box to the selected window
    • This will most likely change the size of the capture region box
  • Pressing Cancel will close the dialog and return you to capture region box

Move Window Into Rectangle... brings up the Select Window dialog:

 

  • Displays all the currently opened windows
  • Selecting one and hitting OK will move the selected window into the region box
    • This will most likely change the size of the selected window
  • Pressing Cancel will close the dialog and return you to capture region box

Options... brings up the Recording Options Dialog:

 

  • Recording Mode area:
    • Automatic: Every mouse Click will record a new frame
    • Manual: New frames are only recorded if Print-Screen is pressed
  • Options area:
    • Show Magnifier: Will toggle on/off the magnifier region box, while doing a Record Region operation
    • Play Sound effects: Toggles on/off sound effects during recording (mouse clicks, typing, etc...)
    • Add callouts: Will toggle on/off Mimic's automatic insertion of callouts where mouse clicks are recorded
    • Record cursor movements: Will toggle on/off Mimic's automatic insertion of mouse cursor movements when clicks are recorded
  • Callout Style: Will show what style of callout will be used for your callouts
  • Cursor Style: Will show what style of cursor will be used for your cursors
  • Pick Style...: Launches the Pick Style dialog for the respective object. The dialog consists of a palette of the selected object type for the user to select from.
    • Verify the chosen callouts are used in the recording

 

Adding new captured frames to a Movie

The Framesmenu contains a Capture New Frames... option:

 

  • Selecting this option will bring up a capture region that is set to the current size of your movie.
    • Changing the size of this region will capture the new size. However once those new frames are inserted into the Mimic movie, they will be condensed/stretched to fit into the current movie size.
  • All frames captured via this option, will appear below the currently selected frame in the Frames Pane.
