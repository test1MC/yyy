Hide on Record

  • By default, it is checked
  • When unchecked, when Mimic initiates a record region or record full screen, the Mimic application will not hide itself.
  • When checked, Mimic will hide itself

Record Application

Record Application brings up the following dialog:

  • All currently opened applications will be listed

  • Selecting an application and hitting OK will minimize mimic (by default) and bring into focus the selected application
  • OK button remains deactivated if no application is selected
  • Cancel will return you to mimic

Record Application - Record Region

When the selected application is brought into focus, Mimic will put a record region around the application. Here is an example:

 

  • The record region box will have 8 resize points located around it
  • Each point can be used to resize the record region
    • Middle boxes can be used to resize either left/right or up/down
    • Corner boxes can be used to resize the entire record region
  • The region box will also have 2 pixel count boxes
    • Box in the upper left contains location of the top left corner of the record region box
    • Box in the center displays the size of the region box, in pixels
      • Clicking and dragging this box will move the record region accordingly
  • Verify resizing the record region box will still result in accurate recordings
  • Three buttons will be displayed in the lower right corner
    • OK: Will set the region box and allow recording to begin
    • Cancel: Will cancel the operation and return the user to Mimic
    • Options: Brings up the options context menu

Recording Options

Options Context menu:

 

  • Snap to window... will open the Select Window dialog
  • Move Window into Rectangle... will open the Select Window dialog
  • Options... will open the options dialog

 

 

Snap to Window... brings up the Select Window dialog:

 

  • Displays all the currently opened windows
  • Selecting one and hitting OK will move the record region box to the selected window
    • This will most likely change the size of the record region box
  • Pressing Cancel will close the dialog and return you to record region box

Move Window Into Rectangle... brings up the Select Window dialog:

 

  • Displays all the currently opened windows
  • Selecting one and hitting OK will move the selected window into the region box
    • This will most likely change the size of the selected window
  • Pressing Cancel will close the dialog and return you to record region box

Options... brings up the Recording Options Dialog:

 

  • Recording Mode area:
    • Automatic: Every mouse Click will record a new frame
    • Manual: New frames are only recorded if Print-Screen is pressed
  • Options area:
    • Show Magnifier: Will toggle on/off the magnifier region box, while doing a Record Region operation
    • Play Sound effects: Toggles on/off sound effects during recording (mouse clicks, typing, etc...)
    • Add callouts: Will toggle on/off Mimic's automatic insertion of callouts where mouse clicks are recorded
    • Record cursor movements: Will toggle on/off Mimic's automatic insertion of mouse cursor movements when clicks are recorded
  • Callout Style: Will show what style of callout will be used for your callouts
  • Cursor Style: Will show what style of cursor will be used for your cursors
  • Pick Style...: Launches the Pick Style dialog for the respective object. The dialog consists of a palette of the selected object type for the user to select from.
    • Verify the chosen callouts are used in the recording

 

Adding new recorded frames to a Movie

The Framesmenu contains a Record New Frames... option:

 

  • Selecting this option will bring up a record region that is set to the current size of your movie.
    • Changing the size of this region will record the new size. However once those new frames are inserted into the Mimic movie, they will be condensed/stretched to fit into the current movie size.
  • All frames recorded via this option, will appear below the currently selected frame in the Frames Pane.

 

 

Recording Windows Tray Menu

Record Application: Opens a select window dialog that allows you to select an application to record.

Record Window:Opens a select window dialog that allows you to select a window to record.

Record Region: Select a region to record using a resizable recording window.

Record Full Screen: Records the full screen of your desktop.

Hide on Record: When enabled, will minimize the recording menu when recording begins so that it does not appear in your recording.

Recording Options...: Invokes the Recording Options Dialog.

About: Invokes the generic MadCap About splash screen.

Exit: Exits recording mode.
