Movie Properties - UI

Opening the UI

  • Click on the Edit File Properties buttonin the Frame Editor
  • !File | Properties
  • !Movie | Movie Properties

Movie Output

UI controls
  • Default Output:
    • MadCap Movie Player
      • Insert Mark of the Web (in generated HTML Pages) (not enabled for this output)
      • Single File Output:
    • Microsoft Silverlight
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • Silverlight Options...:(Only available when this output is selected) Invokes the Silverlight Options Dialog
    • Adobe Flash
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • Flash Options...:(Only available when this output is selected) Invokes the Flash Options Dialog
    • Adobe AIR
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • AIR Options...:(Only available when this output is selected) Invokes the AIR Options Dialog
    • PDF
      • Page Setup...:(Only available when this output is selected) Invokes the Page Setup Dialog
    • XPS
      • Page Setup...:(Only available when this output is selected) Invokes the Page Setup Dialog
  • Skin:Silverlight, Flash, and Air only
    • Home Page:Sets the homepage for the home button in the skin
  • Edit Skin...:Invokes the Edit Skin Dialog
    • Enable Embedded Skin (Flash Only): Controls selected for the skin are embedded directly into the swf.
      • When enabled, the edit skin button switched to the Edit Embedded Skin button that invokes the Embedded Skin Options Dialog.

End Action Tab

UI controls
  • Action:
    • (as is)
    • (default)
    • GotoFrame: Go to specified frame
    • GotoFirst: Go to first frame
    • GotoNext: Go to next frame
    • GotoPrevious: Go to previous frame
    • StartCursor: Start the animation cursor
    • GotoFirstSection: Go to first section
    • GotoPreviousSection: Go to previous section
    • GotoNextSection: Go to next section
    • GotoSection: Go to specified section
    • Pause: Pause/Resume movie
    • Accept: Accept current input
    • Open: Open URL in local viewer/browser
    • OpenWithDefault: OpenURL in a new browser window

For GotoFrame and GotoSection, you choose which specific frame/section at the bottom of the dialog. For the Open actions, you enter the URL at the bottom

Transition Tab

UI controls
  • Transition Effects:
    • (default)
    • (as is)
    • (no effect)
    • Fade
    • Slide
    • Push
  • Effect Properties: Invokes the Effect Properties dialog when fade, slide, or push transition effect is selected.
  • Preview:Shows a preview of the transition
    • Play: Plays the sample transition
    • Stop: Stops the sample transition

Effect Properties Dialog

UI controls
  • Options
    • Duration: Pauses the movie at the end of the frame and will proceed on user action (such as an input box input or button)
    • Direction:
      • Left to Right
      • Right to Left
      • Top to Bottom
      • Bottom to Top
    • Preview:Shows a preview of the transition
      • Play: Plays the sample transition
      • Stop: Stops the sample transition

Image Effects tab

UI controls
  • Blur Factor: Adjusts the blur factor of any shapes that use the blur effect.
    • Preview:Shows a preview of the blur factor.
  • Shade Factor: Adjusts the shade factor of any shapes that use the shade effect.
    • Preview:Shows a preview of the shade factor.

Audio tab

UI controls
  • Enable Audio:Enables audio (Audio that is set in this dialog will be applied across the entire movie.
  • Settings:
    • Audio: Displays the name of the audio clip selected
      • : Plays the selected audio (only enabled after you have picked or recorded an audio clip)
      • :Stops the selected audio that is playing (only enabled after you have picked or recorded an audio clip)
      • Edit...:Edit the selected audio clip
  • Pick Built-In Sound...:Invokes the Pick System Sound Dialog (Covered in the Audio section of the test plan)
  • Pick From Movie Sound Library...:Invokes the Manage Movie Sound Library Dialog (Covered in the Audio section of the test plan)
  • Import Into Sound Library...:Opens a generic windows dialog to select audio files to import into the sound library
  • Record a New Audio Clip...Invokes the Record Dialog (Covered in the Audio section of the test plan)
  • Loop Sound:Loops the selected audio track in the output

Variables tab

UI controls

  • Variables: Displays a list of all Variables

    Columns:

    • Name: Displays the name of all Variables
    • Definition: By default, displays the definition of the variable. Double-clicking the Definition here lets you edit the variable definition specifically for this target.
    • Comment: Displays the comments

Condition Tags Tab

UI controls

  • Conditions: Displays a list of all Conditions

    Columns:

    • Name: Displays the name of all Conditions
    • Background: Displays the background color of the condition. You can also choose the background color here
    • Comment: Displays the comments, also editable in this grid

Conditions Tab

UI controls
  •  

    • Targets tab

    UI controls
    • New...:Invokes the New Target dialog.
    • Remove:Removes the selected target.
    • Edit...:Edits the selected target.

    Project tab

    UI controls
    • Project Link:Link to a Flare project. It allows the user to share the variables and conditions available in the Flare project.
