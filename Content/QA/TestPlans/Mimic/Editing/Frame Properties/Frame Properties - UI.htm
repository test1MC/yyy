Frame Properties - UI

Opening the Frame Properties

  • Double-click the frame in the Frame Editor
  • Double-click the frame in the Frames Pane
  • With the frame open in the Frame Editor, click on the time ruler in the Timeline pane
  • Right-Click the frame in the Frame Editor and select Frame Properties...
  • Right-Click the frame in the frames pane and select Frame Properties...
  • Right-click the frame in the Link Browser and select Frame Properties...
  • Click on the Edit Frame Properties buttonin the Frame Editor
  • Click on the Edit Frame Properties buttonin the Frame Pane
  • Click on the Edit Frame Properties buttonin the Frames Toolbar
  • Select Frame Properties... from the Frames menu

Note: Full motion video frames do not have the Image Tab.

Appearance Tab

UI controls
  • Frame Name: Give a name to the frame
  • Background Section: Controls the look of the background
    • (default): Will set the background back to the default state
    • (as is): Will leave the background as it currently is. (Useful for multiple selections)
    • Solid:  Will set the chosen color as the only color for the background
    • Top to Bottom: Will set a top to bottom gradient, based on the two colors chosen
    • Bottom to Top: Will set a bottom to top gradient, based on the two colors chosen
    • Left to Right: Will set a left to right gradient, based on the two colors chosen
    • Right to Left: Will set a right to left gradient, based on the two colors chosen
    • Top Left to Bottom Right: Will set a top left to bottom right gradient, based on the two colors chosen
    • Bottom Left to Top Right: Will set a bottom left to top right gradient, based on the two colors chosen

Image Tab

UI controls
  • Use same background image as previous frame
  • Insert Image File...:Opens a generic Windows dialog to select an image from your computer to be the background image for the frame.
  • Insert Capture...:Invokes capture functionality to take a screen capture or a region capture.
  • Remove Image:Removes the background image that has been inserted.

Transition Tab

UI controls
  • Specify the display time for this printer:Select the number of seconds that the frame will display in the movie.
  • Pause movie at end of frame: Pauses the movie at the end of the frame and will proceed on user action (such as an input box input or button)
  • Transition Effects:
    • (default)
    • (as is)
    • (no effect)
    • Fade
    • Slide
    • Push
  • Effect Properties: Invokes the Effect Properties dialog when fade, slide, or push transition effect is selected.
  • Preview:Shows a preview of the transition
    • Play: Plays the sample transition
    • Stop: Stops the sample transition

Effect Properties Dialog

UI controls
  • Options
    • Duration: Pauses the movie at the end of the frame and will proceed on user action (such as an input box input or button)
    • Direction:
      • Left to Right
      • Right to Left
      • Top to Bottom
      • Bottom to Top
    • Preview:Shows a preview of the transition
      • Play: Plays the sample transition
      • Stop: Stops the sample transition

Image Effects tab

UI controls
  • Blur Factor: Adjusts the blur factor of any shapes in the frame that uses the blur effect.
    • Same as movie:Sets the blur factor to be the same as the movie level setting.
    • Preview:Shows a preview of the blur factor.
  • Shade Factor: Adjusts the shade factor of any shapes in the frame that uses the shade effect.
    • Same as movie:Sets the shade factor to be the same as the movie level setting.
    • Preview:Shows a preview of the shade factor.
