Multi-Select Object Props - Properties

To access:

  • Select Multiple Objects by
    • Dragging a selection box around some objects
    • Ctrl + Clicking multiple objects
  • Clicking the shape properties toolbar button
  • !Edit | Properties
  • Right-Click the objects | Properties
  • Double-Click the objects

 

Note: Selecting multiple of the same object, will show the standard properties pane for that object type. The list below assumes you selected multiple objects that have at least 2 different types of objects.
