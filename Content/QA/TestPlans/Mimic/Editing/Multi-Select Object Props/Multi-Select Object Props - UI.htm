Selecting multiple objects allows the user to apply properties to many objects at once. There are two distinct set of property windows a user will see:

  • Selecting all of the same type of object, will result in the normal property options for that object displaying
    • Using theAs isoption (where applicable) will maintain each objects specific setting for that property
  • Selecting different types of objects, will result in a special set of property options
    • Using theAs isoption (where applicable) will maintain each objects specific setting for that property
