Polygon - UI

General
  • Clicking the Polygon toolbar item on the local frame toolbar will let you draw a polygon object on a frame
    • After clicking the object, click to add a new segment of the polygon. Each additional click will add a new segment, connected by a resize point
    • Object is realizable by the resize points around the object.
  • Clicking and dragging the polygon will drag the polygon object to a new location
  • Pressing F2 will bring up a rich text window
  • Typing with the object selected will bring up a rich text window
