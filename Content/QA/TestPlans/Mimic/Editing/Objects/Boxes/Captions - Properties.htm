Pointer tab

UI controls
  • Pointer Section
    • X Position: Controls where on the X axis the pointer tip will be located
    • Y Position: Controls where on the Y axis the pointer tip will be located
    • Curved Sides: Toggles on/off the curved sides effect (will round out the pointer arrow)
      • Curve Factor: Sets the degree of the curve
      • The higher the value, the more extreme the curve
  • Pointer Base Section: Controls the base of the pointer (the part that is actually touching the main part of the bubble
    • Edge: Locks what edge of the object the pointer base will be located on
      • Auto: Mimic will automatically choose which side, depending on where the pointer is located
      • Left:  Will lock the base of the point to the left edge of the object
      • Right: Will lock the base of the point to the right edge of the object
      • Top: Will lock the base of the point to the Top edge of the object
      • Bottom: Will lock the base of the point to the bottom edge of the object
    • Base Width: Sets the width of the base
    • Base Minimum Width: Sets a minimum width for the base. The base will not be shorter then this value.

  • Preview Section: Shows a real time preview of the object as you make changes in the settings above
