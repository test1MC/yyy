Boxes - UI

General

Typing Boxes
  •   Clicking the Button toolbar dropdown item on the frame toolbar and selecting insert Typing Box Mode will let you draw a box object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
  • Pressing F2 will allow you to type in the typing box
  • Typing with the object selected will bring up a cursor to allow you to start typing
Input Boxes
  •   Clicking the Button toolbar dropdown item on the frame toolbar and selecting insert Input Box Mode will let you draw an input box object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
  • You have to double click the object to bring up its options to select correct entries and necessary options (covered in the properties test plan Boxes - Properties)
Input Success Caption
  •   Clicking the Button toolbar item on the frame toolbar will let you draw a success caption object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
    • The single resize point at the pointer of the input success caption can be used to move the pointer to a new location and/or change its length
  • Clicking and dragging the input success caption will drag the whole object, including the pointer
  • Pressing F2 will bring up a rich text window
  • Typing with the object selected will bring up a rich text window
