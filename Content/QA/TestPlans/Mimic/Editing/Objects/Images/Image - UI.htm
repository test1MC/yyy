Image - UI

General
  • Images are inserted via the !Insert menu
  • Images that can be inserted : Existing images, Cursor images, Captured Region, Captured UI
    • Object is realizable by the 8 resize points around the object
  • Clicking and dragging the cursor will drag the whole object, including the pointer

 

Inserting
  • Add Image: Brings up a standard windows dialog
    • File types supported are .bmp, .jpg, .jpeg, .gif, .png, .tif, .tiff
    • Image will be inserted at full size
  • Add Cursor from File: Brings up a standard windows dialog
    • File types supported are .cur
    • Image will be inserted at full size
    • Note: Even though you are inserting a cursor, the cursor object you insert is treated as an image file by mimic
  • Insert Captured Region will invoke the targeting cursor shown to the right
  • By default, Mimic should minimize during this step
  • The Pixel box shows the exact pixel location that the center of the targeting cursor is hovering over
  • Hitting ESC will return you to Mimic, canceling the operation
  • A zoom box will appear in the upper left corner of your monitor,
  • Clicking and holding the mouse button, while dragging the mouse, will create a region box
  • The pixel box in the center of the region box shows the dimensions (in pixels) of the rectangle
  • The Pixel box in the upper left corner, shows the pixel location of where the region box started being drawn from
  • Releasing the mouse button, the region box is created
  • The pixel box in the center of the region box shows the dimensions (in pixels) of the rectangle
  • Clicking and dragging the center pixel box will relocate the region box
  • The pixel box in the upper left corner, shows the pixel location of where the Region box started being drawn from
  • The image can be resized by the 8 resize points around the image
  • Clicking OK will take the capture
  • Clicking Cancel will cancel the operation and return you to mimic
  • Insert Captured UI Element will minimize mimic, by default and invoke the UI element capture feature
  • Hovering over various UI elements, a red region capture box will appear, as shown to the right
  • Clicking will capture that UI element and insert it as an image object in Mimic
  • Hitting ESC will cancel the operation and return you to Mimic
