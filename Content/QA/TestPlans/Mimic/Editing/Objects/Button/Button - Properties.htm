Mouse Hover

UI controls
  • Background Section: Controls the look of the background
    • (default): Will set the background back to the default state
    • (as is): Will leave the background as it currently is. (Useful for multiple selections)
    • Solid:  Will set the chosen color as the only color for the background
    • Top to Bottom: Will set a top to bottom gradient, based on the two colors chosen
    • Bottom to Top: Will set a bottom to top gradient, based on the two colors chosen
    • Left to Right: Will set a left to right gradient, based on the two colors chosen
    • Right to Left: Will set a right to left gradient, based on the two colors chosen
    • Top Left to Bottom Right: Will set a top left to bottom right gradient, based on the two colors chosen
    • Bottom Left to Top Right: Will set a bottom left to top right gradient, based on the two colors chosen
  • Fill Start:Sets the start color of the button background
  • Fill End:Sets the end color of the button background
  • Line Section: Controls the look of the border
    • Color:  Sets a color for the border
    • Width:  Sets the thickness, in pixels, for the border
  • Preview Section: Will display a preview of the object
    • This area will update on the fly as properties are selected
  • Color: Sets a color for the text
  • Bold: Will bold the text
  • Italic: Will italicize the text
  • Underline: Will underline the text
