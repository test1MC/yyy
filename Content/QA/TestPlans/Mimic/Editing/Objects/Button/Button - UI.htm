Button - UI

Buttons
  •   Clicking the Button toolbar dropdown item on the frame toolbar and selecting Button will let you draw abutton on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
  • Pressing F2 will allow you to type in the button
  • Typing with the object selected will bring up a cursor to allow you to start typing
