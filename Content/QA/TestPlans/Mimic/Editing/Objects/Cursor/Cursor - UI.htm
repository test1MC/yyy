Cursor - UI

General
  •   Clicking the Cursor toolbar item on the frame toolbar will let you draw a Cursor object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
  • Clicking and dragging the cursor will drag the whole object, including the pointer
  • Hovering over a cursor object will show a trajectory icon
    • Clicking this icon will allow you to change the cursor trajectory
    • When you first click this icon, your mouse cursor will move to the current destination of the existing trajectory (if there is one)
    • A confirmation box will present itself, when setting a new trajectory. Yes will set it, Cancel will cancel the action
