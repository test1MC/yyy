Rectangle - UI

General
  • Clicking the Rectangle toolbar item on the local frame toolbar will let you draw a rectangle object on a frame
    • After clicking the object, click and drag on the frame to set the object size
    • Object is realizable by the 8 resize points around the object
  • Clicking and dragging the rectangle will drag the rectangle object to a new location
    • Pressing F2 will bring up a rich text window
    • Typing with the object selected will bring up a rich text window
  • Clicking the Text Box toolbar item on the local frame toolbar will draw a rectangle object on a frame. After the rectangle is drawn, the rich text window will appear automatically.
    • The text box is a rectangle object in all aspects save for this one difference.
  • Clicking the Shade Effect Mode toolbar item on the local frame toolbar will draw a rectangle object on a frame. The object will have the Shade Effect already selected in its properties.
    • The Shade Effect Mode box is a rectangle object in all aspects save for this one difference.
  • Clicking the Gray Scale Effect Mode toolbar item on the local frame toolbar will draw a rectangle object on a frame. The object will have the Gray Scale Effect already selected in its properties.
    • The Gray Scale Effect Mode box is a rectangle object in all aspects save for this one difference.
  • Clicking the Blur Effect Mode toolbar item on the local frame toolbar will draw a rectangle object on a frame. The object will have the Blur Effect already selected in its properties.
    • The Blur Effect Mode box is a rectangle object in all aspects save for this one difference.
  • Clicking the Blur- Inside Effect Mode toolbar item on the local frame toolbar will draw a rectangle object on a frame. The object will have the Blur-Inside Effect already selected in its properties.
    • The Blur-Inside Effect Mode box is a rectangle object in all aspects save for this one difference.
  • Clicking the Zoom Effect Mode toolbar item on the local frame toolbar will draw a rectangle object on a frame. The object will have the Zoom Effect already selected in its properties.
    • The Zoom Effect Mode box is a rectangle object in all aspects save for this one difference.
