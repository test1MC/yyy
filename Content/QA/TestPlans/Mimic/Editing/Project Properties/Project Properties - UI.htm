Project Properties - UI

Opening the UI

Project Properties only available when you have a project open

  • Click on the Project Properties buttonin the Standard Toolbar
  • Click on the Project Properties buttonon the local toolbar of the Movies Pane
  • !Project | Project Properties

UI Test Cases

Project Editor Toolbar

UI controls
  • : Invokes the Project Movies window
  • Create Blank Movie...:Creates a blank movie in that project
  • Import Mimic Movie...:Imports a mimic movie into that project
  • Import PowerPoint Presentation...:Imports a PowerPoint Presentation into that project

Sections Tab

UI controls
  • Rename...: Rename a section (movie) in the project
  • Change Title...:Changes the title of the section (movie) in the project
  • Open:Opens the section (movie)
  • Delete...:Deletes the section (movie)
  • Up:Moves the section (movie) up in the order of playback on output
  • Down:Moves the section (movie) down in the order of playback on output

Options

UI controls
  • Default Output:
    • MadCap Movie Player
      • Insert Mark of the Web (in generated HTML Pages) (not enabled for this output)
      • Single File Output:
    • Microsoft Silverlight
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • Silverlight Options...:(Only available when this output is selected) Invokes the Silverlight Options Dialog
    • Adobe Flash
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • Flash Options...:(Only available when this output is selected) Invokes the Flash Options Dialog
    • Adobe AIR
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • AIR Options...:(Only available when this output is selected) Invokes the AIR Options Dialog
    • PDF
      • Page Setup...:(Only available when this output is selected) Invokes the Page Setup Dialog
    • XPS
      • Page Setup...:(Only available when this output is selected) Invokes the Page Setup Dialog
  • Home Page:Sets the homepage for the home button in the skin
  • Comment:A place to make comments about your project

Skin

UI controls
  • Edit Skin...:Invokes the Edit Skin Dialog
  • Preview: Shows a preview of the skin

UI Test Cases

Variables tab

UI controls

  • Variables: Displays a list of all Variables

    Columns:

    • Name: Displays the name of all Variables
    • Definition: By default, displays the definition of the variable. Double-clicking the Definition here lets you edit the variable definition specifically for this target.
    • Comment: Displays the comments

Condition Tags Tab

UI controls

  • Conditions: Displays a list of all Conditions

    Columns:

    • Name: Displays the name of all Conditions
    • Background: Displays the background color of the condition. You can also choose the background color here
    • Comment: Displays the comments, also editable in this grid

Targets tab

UI controls
  • New...:Invokes the New Target dialog.
  • Remove:Removes the selected target.
  • Edit...:Edits the selected target.

Project Link tab

UI controls
  • Project Link:Link to a Flare project. It allows the user to share the variables and conditions available in the Flare project.
    • Linked Project:Link to a project by clicking the ... button. Displays the file path of the project file once you have linked to it.
