Output Format Tab

UI controls
  • Default Output:
    • MadCap Movie Player
      • Insert Mark of the Web (in generated HTML Pages) (not enabled for this output)
      • Single File Output:
    • Microsoft Silverlight
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • Silverlight Options...:(Only available when this output is selected) Invokes the Silverlight Options Dialog
    • Adobe Flash
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • Flash Options...:(Only available when this output is selected) Invokes the Flash Options Dialog
    • Adobe AIR
      • Insert Mark of the Web (in generated HTML Pages)
      • Single File Output: (not enabled for this output)
      • AIR Options...:(Only available when this output is selected) Invokes the AIR Options Dialog
    • PDF
      • Page Setup...:(Only available when this output is selected) Invokes the Page Setup Dialog
    • XPS
      • Page Setup...:(Only available when this output is selected) Invokes the Page Setup Dialog
  • Home Page:Sets the homepage for the home button in the skin
  • Comment:A place to make comments about your project

Skin

UI controls
  • Edit Skin...:Invokes the Edit Skin Dialog
  • Preview: Shows a preview of the skin

Variables tab

UI controls

  • Variables: Displays a list of all Variables

    Columns:

    • Name: Displays the name of all Variables
    • Definition: By default, displays the definition of the variable. Double-clicking the Definition here lets you edit the variable definition specifically for this target.
    • Comment: Displays the comments
