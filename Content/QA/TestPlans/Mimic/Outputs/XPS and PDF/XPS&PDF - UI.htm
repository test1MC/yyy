Page Setup

UI controls

  • Paper: 

    • Size: Select the size of your outputted document.
    • Source: Select the source of the paper.
  • Orientation: 
    • Portrait:Set the orientation of the output to be Portrait.
    • Landscape:Set the orientation of the output to be landscape.
  • Margins (inches):
    • Left:Set the left margin.
    • Right:Set the right margin.
    • Top:Set the top margin.
    • Bottom:Set the bottom margin.
