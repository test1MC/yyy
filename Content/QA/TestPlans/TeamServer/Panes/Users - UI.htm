Users - UI

Opening the Projects pane:

  • Click the button in the Standard Toolbar
  • !View | Users
  • <CTRL><Shift>+U

Users pane:

Default view:

 

MDI view:

 

UI Controls

Local Toolbar Buttons (Tooltips in parentheses)

  • (Hide/show the info bar): Lets you hide or show the Info Bar blurb
  • (Refresh): Refreshes the User list to display the latest data
  • (Delete): Deletes the selected User(s) from the Team Server
  • Add User... (Add a user to the Team Server): Open a new empty user profile.

Columns

  • Name: Displays the user's Team Server display name.
  • Login: Displays the user's Team Server login name.
  • Email: Displays the user's e-mail address.
  • First Name: Displays the user's first name.
  • Last Name: Displays the user's last name.
  • Workgroup: Should be removed from Team Server V1.
  • Member of: Displays a list of all of the Groups that of which the user is a member.

 

User viewer:

Basic tab:

UI Controls

Controls

  • Login Name: Lets enter or edit the user's Team Server login name.
  • First Name: Lets enter or edit the user's first name.
  • Last Name: Lets enter or edit the user's last name.
  • Display Name: Lets enter or edit the user's Team Server display name.
  • E-mail: Lets enter or edit the user's e-mail address.
  • Password: Lets enter or edit the user's Team Server login password.
  • Confirm Password: Requires you to enter the user's Team Server login password whenever the Password is modified.
  • Member of: Displays a list of all of the Groups that exist on the Team Server and lets you add the User to one or more of those Groups.
