Groups - UI

Opening the Projects pane:

  • Click the button in the Standard Toolbar
  • !View | Groups
  • <CTRL><Shift>+G

Users pane:

Default view:

 

UI Controls

Local Toolbar Buttons (Tooltips in parentheses)

  • (Hide/show the info bar): Lets you hide or show the Info Bar blurb
  • (Refresh): Refreshes the Group list to display the latest data
  • (Delete): Deletes the selected Group(s) from the Team Server
  • Add Group... (Add a user group to the Team Server): Open a new empty group profile.

Columns

  • Name: Displays the name of every Group created on the Team Server.

 

Group viewer:

Basic tab:

UI Controls

Columns

  • (Checkbox): Lets you add (checked) or remove (unchecked) a User from the Group
  • User: Displays a list of all Users attached to the Team Server.
  • Email: Displays the user's e-mail address.
  • First: Displays the user's first name.
  • Last: Displays the user's last name.
