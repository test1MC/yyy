Projects - UI

Opening the Projects pane:

  • Click the button in the Standard Toolbar
  • !View | Projects
  • <Alt><Shift>+P

Projects pane:

Default view:

 

MDI view:

 

UI Controls

Local Toolbar Buttons (Tooltips in parentheses)

  • (Hide/show the info bar): Lets you hide or show the Info Bar blurb
  • (Refresh): Refreshes the Project list to display the latest data
  • Add Project... (Add a project to the Team Server): Opens the Add Project From Source Control Wizard which lets you browse for a project to copy to the Team Server.
  • Start Download... (Start downloading selected projects): Lets you manually start a download of any selected Flare and/or Blaze project(s).
  • (Delete): Deletes the selected Project(s) from the Team Server

Columns

  • Project: Displays each Flare and Blaze project that has been copied to the Team Server.
  • Source: Displays the path of the Flare or Blaze project from which Team Server will download the Project.
  • Download Frequency: Displays the download frequency (Daily, Monday, etc.) for the Project.
  • Download Time: Displays the download time for the Project.
  • Last Download: Displays the time that Team Server last downloaded the Project.

 

Adding Projects to Team Server:

Adding a Project:

  • Click the "Add Project..." button in the Projects pane
  • !File | Add Project...
  • <CTRL><Shift>+A
Add Project From Source Control Wizard (Select Source Control Provider...):

UI Controls

Controls

  • (radio buttons): Let you select the source location of the project you want to copy to the Team Server.
  • Browse/Advanced: Lets you browse for the project on the selected Source Control provider.
  • < Back: Returns to the previous dialog in the wizard. (Should be grayed out in this dialog.)
  • Next > : Proceeds to the next dialog in the wizard.
  • Finish: Closes the Wizard. (Should be grayed out in this dialog.)
  • Cancel: Closes the Wizard and does not add the project.
Add Project From Source Control Wizard (Additional Project Settings...):

UI Controls

Controls

  • Project Name: Lets you set the name of the Project. By default, Team Server should use the project file name minus the file extension.
  • (Download Frequency) Frequency: Lets you set the frequency of the auto-download functionality.
  • (Download Frequency) Time of Day: Lets you set the time that the auto-download should occur.
  • < Back: Returns to the previous dialog in the wizard.
  • Next > : Proceeds to the next dialog in the wizard. (Should be grayed out in this dialog.)
  • Finish: Closes the Wizard, adds the project to the Team Server and prompts if you would like to download the project now.
  • Cancel: Closes the Wizard and does not add the project.

Project viewer:

Local Toolbar Buttons (Tooltips in parentheses)

  • Download Now... (Download the project): Lets you manually start a download of any selected Project(s).
Basic tab:

UI Controls

Controls

  • Source Project: Displays the name and path of the Project.
  • Name: Displays the name of the Project.
  • (Download Frequency) Frequency: Lets you set the frequency of the auto-download functionality.
  • (Download Frequency) Time of Day: Lets you set the time that the auto-download should occur.
Publishing tab:

This tab displays all of the Destination files that exist in the selected Flare or Blaze project. The purpose for this UI is to allow the Team Server user to set authentication username and password information for each destination so that Team Server will be able to publish output to these Destinations.

 

UI Controls

Columns

  • Type: Displays the type of Destination (file or FTP).
  • Name: Displays the name of the Destination file.
  • Path: Displays the Destination directory.
  • Username: Lets you set the authentication username for the Destination.
  • Password: Lets you set the authentication password for the Destination.

Controls

  • Save: Lets the user save any changes made to the Username and Password columns
