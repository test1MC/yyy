Reports - UI

Opening the Reports pane:

  • Click the button in the Standard Toolbar
  • !View | Reports
  • <CTRL><Shift>+R

Projects pane:

Default view:

 

MDI view:

 

UI Controls

Local Toolbar Buttons (Tooltips in parentheses)

  • (Hide/show the info bar): Lets you hide or show the Info Bar blurb
  • (Refresh): Refreshes the Reports list to display the latest data
  • (Delete): Deletes the selected Report(s) from the Team Server
  • Add Report... (Add a report to the Team Server): Opens the Add Report to Server dialog. (See below)
  • Generate Report Now... (Generate the report now): Generates the selected Report and opens it in Team Server as an *.htm file.

Columns

  • Title: Displays the title of each Report.
  • Project: Displays the name of the Flare or Blaze project to which the Report is attached.
  • Last Generated: Displays the time that Team Server last generated the Report.
  • Generate Frequency: Displays the generation frequency (Daily, Monday, etc.) for the Report.
  • Generation Time: Displays the generation time for the Report.
  • Destination: Displays the publishing destinations for the Report
  • Publish: Displays whether or not the Report is set to auto-publish. (You need to open the Report to toggle this option.)
  • Email: Displays whether or not the Report is set to auto-email. (You need to open the Report to toggle this option.)

 

Add Report to Server dialog

UI Controls

Controls

  • Select Project for new Report:  Lets you select with which project the new report should be associated.

    Columns

    • Project: Displays the names of all Flare or Blaze projects attached to the Team Server.
    • Source: Displays the path of the source Flare or Blaze project.
  • Report Title: Lets you enter a Title (Name) for the Report.

Report viewer:

Basic tab:

UI Controls

Controls

  • Title: Displays the title of the Report.
  • Tables: Lists all available types of data which can be included in a report. Check the box next to each table you would like to include in your report.
  • Select All: Selects and deselects all tables.
  • Open in Browser: Opens the generated report in your default browser.
  • Generate: Generates the Report locally (This button does not publish or email the Report)
  • Report: Displays the generated Report.
Frequency tab:

UI Controls

Controls

  • (Generation Frequency) Frequency: Displays the auto-generate frequency (Daily, Monday, etc.) for the Report.
  • (Generation Frequency) Time of Day: Displays the auto-generate time for the Report.
  • (Generation Frequency) Min Count: Lets you set a minimum number of results for any individual table which must be matched for the Report to be e-mailed and/or published. (If any selected table produces more results than the Min Count value or that value, the report will be e-mailed and/or published.)
  • (Generation Frequency) Max Count: Lets you set a maximum number of results for any individual table which must not be exceeded for the report to be e-mailed and/or published. (If any selected table produces fewer results than the Max Count value or that value, the report will be e-mailed and/or published.)

  • (Recipients) Recipients: Lets you select which Team Server Users and/or Groups will receive the report when it is emailed.

    Columns

    • Type: Displays the type of Recipient (User or Group).
    • User: Displays the User or Group name
    • Email: Displays the User's email address (empty if Group)
    • First: Displays the User's first name (empty if Group)
    • Last: Displays the User's last name (empty if Group)
  • (Recipients) Send Current Report... : Sends the Report which is currently generated in the Basic tab to the selected Recipients.
  • (Publish) Publish Report: Lets you enable/disable auto-publish functionality.
  • (Publish) Publish generated report to selected destination: Publishes the Report which is currently generated in the Basic tab.
  • (Publish) Select...: Opens the Select Destination dialog.
Select Destination dialog

UI Controls

Controls

  • Select a Destination for this Report:  Lets you select a publishing destination for the report. (Currently only uses Destinations which exits in the Flare/Blaze project.)

    Columns

    • Name: Displays the name of the Destination file.
    • Target Location: Displays the Destination directory.
