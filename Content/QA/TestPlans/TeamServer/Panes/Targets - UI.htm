Targets - UI

Opening the Targets pane:

  • Click the button in the Standard Toolbar
  • !View | Targets
  • <CTRL><Shift>+T

UI Controls

Local Toolbar Buttons (Tooltips in parentheses)

  • (Hide/show the info bar): Lets you hide or show the Info Bar blurb
  • (Refresh): Refreshes the Target list to display the latest data
  • Start Build... (Start building and publishing selected target(s)): Lets you manually start a build of any selected Target(s). Additionally, any Targets which have the Publish option enabled will get published.

Columns

  • Project: Displays the Flare or Blaze project to which each Target belongs.
  • Target: Displays each Target contained in each Flare or Blaze project copied to the Team Server.
  • Last Build: Displays the time that Team Server last built the Target.
  • Last Publish: Displays the time that Team Server last published the Target.
  • Build: Displays whether or not the Target is set to auto-build. (You need to open the Target to toggle this option.)
  • Publish: Displays whether or not the Target is set to auto-publish. (You need to open the Target to toggle this option.)
  • Build Frequency: Displays the auto-build frequency (Daily, Monday, etc.) for the Target.
  • Build Time: Displays the auto-build time for the Target.
  • Destination: Displays the publishing destinations for the Target (as set in the Flare/Blaze project).

 

Target viewer:

Default view in Main Dock:

 

UI Controls

Local Toolbar Buttons (Tooltips in parentheses)

  • Start Build Now... (Start building and publishing selected target(s)): Lets you manually start a build of any selected Target(s). Additionally, any Targets which have the Publish option enabled will get published.

Controls

  • Source Project: Displays the name of the project to which the Target belongs.
  • Name: Displays the name of the Target.
  • (Build Frequency) Build Target: Lets you enable/disable auto-build functionality.
  • (Build Frequency) Publish Output: Lets you enable/disable auto-publish functionality.
  • (Build Frequency) Frequency: Lets you set the frequency of the auto-build/auto-publish functionality.
  • (Build Frequency) Time of Day: Lets you set the time that the auto-build/auto-publish should occur.
