Toolbars - UI

Standard Toolbar

Opening/Closing the Standard Toolbar

From a menu:

!View | Toolbars | Standard

Click the button in the upper-left corner (next to the File menu) and select Standard.

Additionally, you can close the toolbar by clicking the button which appears to the left of the toolbar.

Standard Toolbar - UI Controls

State #1: No SQL Server set up. The Standard Toolbar will appear in this state if you click Cancel in the Feedback Server Setup dialog or after you uninstall the Feedback Server (SQL Server).

State #2: SQL Server set up. The Standard Toolbar will appear in this state when Feedback Server Admin is set up with a SQL Server database.

 

Toolbar Items (Tooltips in parentheses)

  • (Install Feedback Server): Invokes the Feedback Server Setup dialog which lets you select a SQL Server.
  • (Uninstall Feedback Server): Invokes a warning dialog indicating that all data will be lost if you proceed. Clicking "Yes" will proceed to uninstall the linked SQL database. In the process, this will also permanently delete the linked database.
  • (Import Data): Invokes the Select SQL Server and Database dialog which lets you select a SQL Server and database from which you will import data.
