Active Users - UI

Opening the Active Users view

  • !View | Active Users
  • Press <CTRL>-U on the keyboard
  • Click the Active Users button on the Standard Toolbar

Active Users - UI Test Cases

Local Toolbar and Columns

 

Local Toolbar Buttons (Tooltips in parentheses):

  • (Hide/Show the Info Bar): Displays or hides the Info Bar.
  • Export the data to a csv file: Exports the data to a csv file that can be opened in Microsoft Excel.

Active Users grid

Columns (NOTE: The exact set of columns that appear in this view are dependant on the data that exists in the Feedback Database. This is entirely dependant on which fields the Flare author has determined to be included in the Feedback User Profile dialog. Still, the columns may vary if a particular profile item is set to be "optional." Until at least one user opts to provide information in an optional field, no column will appear in this view.)

  • Username: Displays the user's username. (This column will always be present)
  • EmailAddress: Displays the user's e-mail address. (This column will always be present)
  • RegisteredSince: Displays the date and time the user registered an account on the Feedback Server. (This column will always be present)
  • Status: Shows the status of the user, "OK" or "Pending" (This column will always be present)
  • (Other): Any item that is available as a Feedback User Profile item in Flare's Skin Editor may appear in this grid as a column. Examples: FirstName, LastName, PostalCode, Country, Gender, Custom1, etc.

 

Right-Click Menu

Menu items

  • Info Bar: Displays or hides the Info Bar.

Keyboard Navigation

<Up Arrow> moves the selection to the row above

<Down Arrow> moves the selection to the row below

<Enter> moves the focus to the row below

<Page Up> will move the selection one page at a time upward

<Page Down> will move the selection one page at a time downward

<Left Arrow> and <Right Arrow> do nothing

<Home> and <End> do nothing

<Insert> does nothing

<Delete> does nothing
