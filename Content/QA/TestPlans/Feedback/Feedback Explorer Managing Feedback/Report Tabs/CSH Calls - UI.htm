Context-Sensitive Help Calls - UI

Opening the Context-Sensitive Help Calls view

  • !View | Context-Sensitive Help Calls
  • Press <CTRL>-L on the keyboard
  • Click the Context-Sensitive Help Calls button on the Standard Toolbar

Context-Sensitive Help Calls - UI Test Cases

Local Toolbar and Columns

 

Local Toolbar Buttons (Tooltips in parentheses):

  • (Hide/Show the Info Bar): Displays or hides the Info Bar.
  • (Refresh): Refreshes the data in the Report Tab.
  • (Date filter): Invokes the Date Filter dialog which lets you limit the visible data to data collected within a particular date range.
  • Export the data to a csv file: Exports the data to a csv file that can be opened in Microsoft Excel.
  • (Select columns to show/chart): Opens the Options dialog which lets you set which columns are visible and which columns should be included when charting data.
  • Chart Options: Open the Chart Options dialog which allows you customize the row colors of the grid.

    • Add a custom chart color button: tooltip: Add a custom chart color: Opens the color picker dialog.

    • Delete the custom chart color button: Tooltip: Delete the custom chart color: removes the selected custom chart color.

  • (Select the chart type): Lets you select the type of chart which displays for this Report Tab.
  • (Select the data for the chart): Lets you select which rows in the grid view should be considered in the generated chart.
  • (Page Setup): Invokes the Page Setup dialog which lets you select paper size, orientation and margins.
  • (Print Preview): Invokes the Print Preview dialog which displays a preview of the chart which can be printed from the Report Tab.
  • (Print): Prints the chart from the Report Tab.
  • (Reset CSH call counts): Resets the CSH call counts for the selected CSH IDs to zero.

Context-Sensitive Help Calls grid

Columns

  • CSH ID: Displays all CSH IDs called by the end user.
  • Path: Displays the path of the topic file within the project.
  • Call Count: Displays the total number of times the CSH ID to the left was called.

Right-Click Menu

Menu items

  • Info Bar: Displays or hides the Info Bar.
  • Properties: Invokes the (Comment) Properties dialog.

 

Options dialog - UI Test Cases

UI Controls

  • Move Up: Moves the selected row(s) up in the sequence which in turn moves the corresponding column to the left in the Context-Sensitive Help Calls view.
  • Move Down: Moves the selected row(s) down in the sequence which in turn moves the corresponding column to the right in the Context-Sensitive Help Calls view.
  • Reset to Factory Settings... : Removes all custom column settings and restores factory defaults.

Options dialog grid

Columns

  • Show: Lets user show/hide the associated column in the Context-Sensitive Help Calls view.
  • Chart: Lets user determine if the column should be included in charting. Only one column can be selected to chart at any one time.
  • Column: Displays the name of each column that can be displayed in the Context-Sensitive Help Calls view.
  • Width: Lets user set the width in pixels of each column in the Context-Sensitive Help Calls view.
  • Color: Lets user open the Color Picker dialog to set the font color for each column in the Context-Sensitive Help Calls view.
  • Background: Lets user open the Color Picker dialog to set a background color for each column in the Context-Sensitive Help Calls view.

Grid Test Cases

  • Columns dialog columns should not be sortable
  • Columns dialog columns should not be resizable
  • Columns dialog columns cannot be moved by clicking on a column heading and dragging
  • Horizontal and vertical scrollbars should be working correctly
  • Columns dialog rows should not be able to be multi-selected with Shift- or CTRL-click

Date Filter dialog - UI Test Cases

UI Controls

  • Enable Filter: Enables/disables the date filter for the data set.
  • Filter Settings: Lets you select between a user-defined date range or a preset date range.
    • Range:  Select this option to set a specific date and time for the filter. Click on the down arrow in the "From" and "To" fields to set the specific month and date. You can also click inside the "From" and "To" fields and type to specify a particular time.
    • Preset: Lets you set the Date filter to one of a set of factory presets. These include: Today, Yesterday, Last 7 days, and Last 30 days

Keyboard Navigation

<Up Arrow> moves the selection to the row above

<Down Arrow> moves the selection to the row below

<Enter> moves the focus to the row below

<Page Up> will move the selection one page at a time upward

<Page Down> will move the selection one page at a time downward

<Left Arrow> and <Right Arrow> do nothing

<Home> and <End> do nothing

<Insert> does nothing

<Delete> does nothing
