Feedback Configuration

Bug Base Product Area:

  • Explorer - Configuration

 

Description:

The Feedback Configuration file (*.fbconf) lets the user define how the features of Feedback Server should be implemented. The primary elements in a *.fbconf file are the Feedback Server URL and the Feedback Service key. The user can then customize the other features of the Feedback Service to suit there needs. These features include setting administrator and reviewer emails, allowing anonymous comments, requiring comments to be reviewed, and setting excluded IPs.
