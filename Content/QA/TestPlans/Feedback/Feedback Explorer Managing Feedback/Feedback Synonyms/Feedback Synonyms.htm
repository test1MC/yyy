Feedback Synonyms

Bug Base Product Area:

  • Explorer - Synonyms

 

Description:

"Feedback" Synonyms function exactly the same as Synonyms contained within a Flare project. The difference being the synonyms created in the Flare project are built into the output. Synonyms created on the Feedback Server will only function when the published output is connected to the Feedback Server. The primary purpose of "Feedback" Synonyms is to improve the end user's search experience by adding synonyms "on-the-fly" without having to rebuild the output of the original Flare project. Because a Feedback Synonym exists on the Feedback Server, it will only function when the output has a connection to the server.

Synonym sets created in Feedback Explorer can be saved out to a *.flsyns file which can then, if desired, be incorporated into the Flare project the next time it is built.
