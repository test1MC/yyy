Feedback Synonyms - UI

Opening:

Opening a Feedback key's Synonym set:

  • !View | Synonyms
  • Press CTRL-W on the keyboard
  • Click the Synonyms button in the Standard Toolbar

Editing Synonyms

Directional Tab

UI Controls

  • Directional Synonyms grid: Displays and lets you create and edit directional Synonyms.

    Columns

    • Word: Enter a term that users might use instead of the term that actually occurs throughout your content.
    • Synonym: Enter the term that exists in your content that should be found when the corresponding "Word" is used as a search term
    • Stem: Enable this option to if you want Flare to accept other variations of the search term that have the same stem.

Groups Tab

UI Controls

  • Groups Synonyms grid: Displays and lets you create and edit Synonym Groups.

    Columns

    • Group: Enter a collection of terms (synonyms) that will produce the same search results for all of the words in the group.
    • Stem: Enable this option to if you want Flare to accept other variations of the search term that have the same stem.

Drag and Drop

There is currently no drag/drop editing functionality within the Synonyms Editor (as of Flare V6 and Feedback Explorer V3)

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

Cut, Copy and Paste functionality is currently unavailable in the Synonyms Editor (as of Flare V6 and Feedback Explorer V3) with regard to the actual synonym rows. You can cut, copy or past text within an individual text box in the grid.

Deleting

Item:

Select the synonym row and press <Delete> on the keyboard

 

(Bugs are entered to add more methods of deletion)

Tooltips

If any Word, Synonym or Group is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.

 

Exporting Synonyms

Synonyms created on the Feedback Server using Feedback Explorer can be exported to a *.flsyns file which can then be added to a Flare project for future use.  To do this:

  1. Open the Synonyms Editor
  2. !File | Save As
  3. Browse to the location where you want to save the file and save
