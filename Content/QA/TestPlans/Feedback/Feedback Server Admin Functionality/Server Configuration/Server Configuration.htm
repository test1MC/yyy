Server Configuration

Bug Base Product Area:

  • Server - Configuration

 

Description:

This test plan will cover the primary UI in Feedback Server Admin, the Feedback Server Configuration tab. The Feedback Server Configuration tab provides UI to test the SQL Server and Website, to set up and test the SMTP Server, to set User Profile activation "success" and "failure" pages and (for local SQL Express only) to set up scheduled backups of the database.

 

Test Plans:

  • Server Configuration - UI
  • Server Configuration - Test Cases

  

Dependencies:

  • Access to SQL Server (either local or another machine on the domain)
