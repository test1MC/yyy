Server Configuration - UI

This test plan will cover the UI associated with configuring your Feedback Server.

Feedback Server Configuration

After successful installation of the Feedback Server SQL Database, Feedback Server Admin will open the Feedback Server Configuration tab in the MDI area.

Feedback Server Configuration - Basic tab

UI Controls

  • Security:
    • Require password from Feedback Explorer: Select the toggle box enable or disable this option. Toggling the option on will invoke the Set Server Password dialog.
    • Set new password… : This option will only be active if a password is already set. Clicking this option will invoke the Set Server Password dialog.
  • SQL Server: Displays information about the SQL database
    • Server Name: Displays the name of the machine hosting the SQL database
    • Database Name: Displays the name of the SQL database
    • Test: Click this button to test the connection to the SQL database. Be sure to test a broken connection as well as a good one. Verify the messages are appropriate.
  • Website: Displays information about the Feedback Server Website
    • Location: Displays the location of the Feedback Server website
    • Test: Click this button to test the connection to the Feedback Server website. Be sure to test a broken connection as well as a good one. Verify the messages are appropriate.
  • SMTP Server: Displays information about the SMTP Server used to support Feedback comment emails. Additional information for setting up a third party SMTP Server is available here: "Setting up a 3rd party SMTP service on Windows Vista"
    • Server: Enter the name of the server
    • User Name: Lets you enter a username if the SMTP server you've identified requires specific credentials. In most cases, this can be left blank. Test both cases.
    • Password: Lets you enter a password if the SMTP server you've identified requires specific credentials. In most cases, this can be left blank. Test both cases. If you do enter a password here, it should display an asterisk for each typed character.
    • "From" Address: Enter the email address that you want email notifications to contain in the "From" field
    • Test… : Invokes the SMTP Test dialog which allows you to send a test email from the Feedback Server

Feedback Server Configuration - Advanced tab

UI Controls

  • Activation success page: Lets you enter a URL to be used as the Web page that displays when the user registration process is successful. The page should display in the area immediately below.
  • Activation failure page: Lets you enter a URL to be used as the Web page that displays when the user registration process is not successful. The page should display in the area immediately below.

Feedback Server Configuration - Maintenance tab

NOTE: The tasks on the Maintenance tab are applicable to SQL Server Express when it is located on the same computer as IIS only. These options should be tested with all supported version of SQL Express. See the Help for more information.

UI Controls

  • Location: Enter the path where you want backups of the database and transaction log to be stored.
    • Browse: Opens a browse dialog which lets you browse for a backup location.
  • Day & Time: Three pulldowns which let you specify the day and time you want the backup process to run.
  • SQL Server Express Maintenance Windows Service (Start/Stop): By default, the Start button should be engaged which enables backups to be created. Click Stop to disable the backup process.
