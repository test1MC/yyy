Server Email Notification

Bug Base Product Area:

  • Server - SMTP/Email Notifications

 

Description:

This test plan will cover the various email notification settings that can be set for Administrators, Reviewers and registered users.

Administrators and Reviewers may receive an email from the server for the following:

  • User submitted comment or reply to a comment
  • User submitted rating
  • Change of a comment status
  • Various milestones. For example, a term is searched with no results 10 times. (not implemented yet, should be added in V3)

Registered users may receive an email from the server for the following:

  • Another registered user replies to one of their comments
  • Another registered user adds a comment to a topic they have commented on
  • Any comment is added anywhere in the Help system
