Test Cases

  • Test that no email notifications are sent when no options are selected.

  • Test each email notification option independently. Verify that only the appropriate email is sent.

    Option Email Verified
    Reply Reply  
    Topic Topic  
    Help System Help System  

  • Test that each combination of options work appropriately. For each combination of options, verify that only one email is sent and that it is the appropriate email.

    Options Action Email Verified
    Reply & Topic Reply made to a comment Reply  
    Reply & Topic Comment added to same topic Topic  
    Reply & Topic Comment added to another topic in the Help System None  
    Reply & Help System Reply made to a comment Reply  
    Reply & Help System Comment added to same topic Help System  
    Reply & Help System Comment added to another topic in the Help System Help System  
    Topic & Help System Reply made to a comment Topic  
    Topic & Help System Comment added to same topic Topic  
    Topic & Help System Comment added to another topic in the Help System Help System  
    Reply, Topic & Help System Reply made to a comment Reply  
    Reply, Topic & Help System Comment added to same topic Topic  
    Reply, Topic & Help System Comment added to another topic in the Help System Help System
