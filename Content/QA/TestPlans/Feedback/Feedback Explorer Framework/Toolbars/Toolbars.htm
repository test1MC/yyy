Toolbars

 

Bug Base Product Area:

  • Explorer - Toolbars / Menus

 

Description:

Feedback Explorer's UI contains Global Toolbars and Local Toolbars. Global Toolbars are always available, regardless of where the user are working in the interface. In Feedback Explorer, there is only on Global Toolbar and that is the Standard Toolbar. Local Toolbars are intended for a particular editor and are covered test plans for each specific module that has a Local Toolbar.
