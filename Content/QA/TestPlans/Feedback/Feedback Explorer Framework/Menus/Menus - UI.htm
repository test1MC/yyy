Menus - UI

File Menu

To access:

  • Click the File menu
  • Press [ALT] + F on the keyboard

File Menu Items (Tooltips in parentheses)

  • New: Opens the New Local Target dialog, which you can use to create a new Feedback file configured with a Flare target.
  • Open: Invokes an Open dialog which lets you browse for appropriate file types to open in the application
  • StartUp File…: Invokes the Startup File dialog which lets you set the file that should open each time the application opens
  • Close (Close File): Closes the current file
  • Save (Save File): Saves the current file
  • Save As: Invokes a Save As dialog which lets you save the current file with a new filename, and opens the new file in place of the current file.
  • Save To: Invokes a Save As dialog which lets you save the current file with a new name but does not open the new file.
  • Save All: Saves all open documents
  • Send To: Invokes a submenu, which lets you open the active file in another application, attach it to an email, or open the Windows folder where it is stored
  • Page Setup… (Page setup): Blah
  • Print (Print): Prints the current file
  • Print Preview… (Print preview): blah
  • Properties (File Properties): Opens the Properties dialog for the selected item.
  • Recent Files: Invokes the Recent Files sub-menu which lets you select a recently opened file to open again or manage the list of files that are considered "recent"
  • Exit (Exit the application): Exits the application

 

Edit Menu

To access:

  • Click the Edit menu
  • Press [ALT] + E on the keyboard

Edit Menu Items (Tooltips in parentheses)

  • Undo (Undo): Undoes the most recent action
  • Redo (Redo): Redoes the action that was most recently undone
  • Cut (Cut): Cuts the current selection to the clipboard
  • Copy (Copy): Copies the current selection to the clipboard
  • Paste (Paste): Paste the contents from the clipboard
  • Delete (Delete): Delete the current selection
  • Select All (Select All): Selects all
  • Properties (Run properties dialog for this item): Invokes the Properties dialog for the selected item

 

View Menu

To access:

  • Click the View menu
  • Press [ALT] + V on the keyboard

 

Test cases:

Make sure all data is incremented correctly. For example, if topic A is viewed 9 times that the Topics pane increments by 1 every time topic A is viewed. Test all data in all panes. CHS calls, Comments, Active Users, Search Phrases, Search Phrases with No Results, Browser Statistics(test with IE 7,8,9 Firefox 3.5, 3.6, 4.0, Chrome, Safari etc.) Operating System Statistics (XP, Vista, Win7 etc.), and Synonyms.

View Menu Items (Tooltips in parentheses)

  • Toolbars: Invokes the Toolbars sub-menu
  • Topics (Open the Topics List window): Opens the Topics List Window
  • Context-Sensitive Help Calls (Open the Context-Sensitive Help Calls window): Opens the Context-Sensitive Help Calls Window
  • Comments (Open the Comments window): Opens the Comments Window
  • Active Users (Open the Active Users window): Opens the Active Users Window
  • Search Phrases (Open the Search Phrases window): Opens the Search Phrases Window
  • Search Phrases with No Results (Open the Search Phrases with No Results window): Opens the Search Phrases with No Results Window
  • Synonyms (Open the Synonyms window): Opens the Synonyms Editor
  • Start Page (Open the Start Page): Opens the Start Page
  • Status Bar (Hide/show the Status Bar): Hides or shows the Status Bar

 

 

Topics

 

 

 

Open Topic Pane:

  • !View | Topics or Ctrl+Shift+T

 

 

Topic Pane Local Toolbar:

  • Info Bar: displays the info bar
  • Refresh: Refreshes the pane displaying the latest information
  • Date Filter: Opens the Date Filter dialog
  • Date Filter Dialog:

    • Enable Filter: Enables/disables the date filter

    • Filter Settings:
      • Range: set the date range you want to filter by.
      • Preset: Select a preset range you want to filter by.
  • Options: Opens the options dialog where you can select what data to display
  • Chart Type Dropdown: Dropdown list of different types of charts you can display.
  • Data for the chart: Select the number or rows of data for the chart to display. All, Selected Rows, Top 10, Top 25, Top 50, Custom.
  • Custom: Opens the custom data filter dialog where you can select a custom number of rows to display.
  • Page Setup: Opens the page setup dialog
  • Preview: Opens the Print Preview dialog.
  • Print:
  • Reset Topic Views: Resets the topic view data to 0. New topic view data will begin with 1.
  • Reset Topic Ratings: Resets the topic rating data to 0. New topic rating data entries will begin with 1.

Statistics:

Let you see the which browsers or operating systems are being used to view your output. Each window pane automatically displays a chart with the statistics. These window panes let you see statistics on all browsers or operating systems used, including the number of views for each. You can also see that browser or operating system's overall percentage of views compared with other browsers or operating systems.

Synonyms Editor

Directional View:

 

Group View:

 

Open Synonyms Editor:

  • View | Synonyms or Ctrl+W

 

Synonyms Editor Local Toolbar:

  • Info Bar: displays the info bar.

 

Directional synonyms: When a user searches for a term in the "Word" column, the help system will also include results for the term in the "Synonym" column.

Group synonyms: Enter Synonym groups in the following format: car=automobile bicycle=tricycle=bike=trik.

If the stem box is checked, all forms of each word will be included.
