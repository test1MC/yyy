Server Import Data - UI

This test plan will cover the UI associated with importing data from an existing Feedback Server SQL database into an established database. As opposed to importing a previous version database at the point of creating a new database (on the same machine), this process creates the new database first and then allows you to import the SQL database from any machine on the network.

Importing the SQL Database

If you are installing Feedback Server Admin over an existing install, you can import data from the previous installation at the point the new database is created. That process is covered in the test plan "Install SQL Database - UI". Alternatively, once you've set up a new database, you can import data from any database on the network by clicking the button in the Standard Toolbar. This will invoke the Select SQL Server and Database dialog.

Select SQL Server and Database dialog

UI Controls

  • SQL Server: Displays a list of all instances of supported SQL and SQL Express available on the network. Select the instance that contains the SQL database you want to import.

  • Database: Upon clicking this drop-down the first time, Feedback Server Admin will retrieve all appropriate database files and the following dialog will appear:

    Cancel: Cancels the process and closes the dialog.

    Once this process complete, all "MadCapFeedbackServer" SQL databases attached to the selected SQL Server should appear in the Database drop-down.

  • OK: Begins the import process. This button should be grayed out until both the SQL Server an Database fields above are populated.
