Server Import Data

Bug Base Product Area:

  • Server - Install/Setup

 

Description:

This test plan will cover importing data from an existing Feedback Server SQL database into an established database. The Feedback Server database can be imported at the point at which the Feedback Server is installed (if installing over a previous installation) or you can import a SQL database from any machine on the network.
