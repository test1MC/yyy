Compatibility with Flare and Feedback Explorer

The latest version of Flare and Feedback Explorer as well as the output created from the latest version of Flare should all be tested against previous versions of Feedback Server Admin. Use FEEDBACKV1, FEEDBACKV2 and FEEDBACKV3 as your Feedback Serve machine to complete this testing.

 

Use this matrix to test all configurations of a previous version of Feedback Server with the latest version of Flare and Feedback Explorer. Use the Feedback Testing Matrix to verify all applicable features work with each configuration.

Compatibility with Feedback Server

The latest version of Feedback Server Admin should be tested with all previous versions of Feedback Explorer as well as at least one version of Flare released in support of each previous version of Feedback. In other words, Flare V4.2 was released to support Feedback V2. All releases of Flare from V4.2 through V6.0.1 were also shipped with support for Feedback V2. Only one version in that range of releases needs to be tested against the latest server. Use FEEDBACKV4 as your Feedback Server to complete this testing.

 

Use this matrix to test all previous versions of Feedback Explorer, Flare and Flare output against the latest version of Feedback Server. Use the Feedback Testing Matrix to verify all applicable features work with each configuration
