Installing IIS - Windows 7

This document will provide step-by-step instructions for installing Internet Information Services (IIS) on Windows 7.

Installing IIS via the Control Panel

  1. From the Start menu, select Control Panel
  2. Select Programs. Then, under Programs and Features click "Turn Windows features on or off"

  3. The Windows Features dialog will open. Select the box for Internet Information Services

  4. If installing IIS for MadCap Feedback, you will also need to enable IIS 6 Metabase Compatibility
    1. Expand Internet Information Services
    2. Expand Web Management Tools

    3. Expand IIS 6 Management Compatibility and enable IIS Metabase and IIS 6 configuration compatibility

  5. Click OK and Windows will configure IIS

  6. When finished, the dialog above will simply disappear and IIS will be installed.
