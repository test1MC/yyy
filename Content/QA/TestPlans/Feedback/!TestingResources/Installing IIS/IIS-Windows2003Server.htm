Installing IIS - Windows 2003 Server

This document will provide step-by-step instructions for installing Internet Information Services (IIS) on Windows 2003 Server.

Installing IIS via the Manage Your Server dialog

  1. Open the Manage Your Server dialog from the Start menu.

  2. Click "Add or remove a role"

  3. This will open the Configure Your Server Wizard. Click Next.

  4. In the next dialog, select "Application server (IIS, ASP.NET)". Click Next.

  5. In the next dialog, select the box next to "Enable ASP.NET". Click Next.

  6. The next dialog displays a summary of your selections. Click Next.

  7. The Windows Component Wizard will open and display the status of the install.

  8. When the install completes, the final dialog of the Configure Your Server Wizard will display. Click Finish.

 

Installing IIS via the Control Panel

  1. From the Start menu, select Control Panel > Add or Remove Programs

  2. In the Add or Remove programs dialog, click Add/Remove Windows Components

  3. In the Windows Components Wizard, select Application Server and click Details…

  4. This will open the Application Server dialog. Enable Internet Information Services (IIS) and click OK.

  5. This will retunr you to the Windows Component Wizard. You will now see that Application Server appears enabled. Click Next.

  6. IIS will now be installed. The Windows Components Wizard displays the install progress.

  7. When the install completes, the final dialog of the Windows Components Wizard will display. Click Finish.
