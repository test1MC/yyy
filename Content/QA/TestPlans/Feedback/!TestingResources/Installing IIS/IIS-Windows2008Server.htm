Installing IIS - Windows 2008 Server

This document will provide step-by-step instructions for installing Internet Information Services (IIS) on Windows 2003 Server.

Installing IIS via the Server Manager dialog

  1. Open the Server Manager dialog from the Start menu.

  2. Click "Add Roles"

  3. This will open the Add Roles Wizard. Click Next.

  4. In the next dialog, click the toggle box next to "Web Server (IIS)".

  5. A separate dialog will open to confirm you want to add IIS. Click "Add Required Features".

  6. This will return you to the Add Roles Wizard, but now the toggle box for Web Server (IIS) will appear checked. Click Next.

  7. At the "Introduction to Web Server (IIS)" dialog, click Next again.

  8. If installing IIS for MadCap Feedback, you will also need to enable IIS 6 Metabase Compatibility. In the "Select Role Services" dialog, expand Management Tools > IIS 6 Management Compatibility and enable IIS 6 Metabase Compatibility.

  9. At the "Confirm Installation Selections" dialog, click Install.

  10. The Installation Progress dialog will open during installation  

  11. When installation completes, the "Installation Results" dialog will open. Click Close.
