Installing IIS - Windows 2012 Server

This document will provide step-by-step instructions for installing Internet Information Services (IIS) on Windows 2012 Server.

Installing IIS via the Server Manager dialog

  1. Open the Server Manager > Dashboard

  2. Click "Add roles and features"

  3. This will open the Add Roles and Features Wizard. Click Next.

  4. Click Next two more times until you get to "Server Roles"

  5. At "Server Roles," click the toggle box next to "Web Server (IIS)".

  6. A separate dialog will open to confirm you want to add IIS. Click "Add Features".

  7. This will return you to the Add Roles Wizard, but now the toggle box for Web Server (IIS) will appear checked. Click Next.

  8. At the Features tab, expand ".NET Framework 4.5 Features" and enable ASP.NET 4.5. Click Next.

  9. At the "Web Server Role (IIS)" tab, click Next again.

  10. If installing IIS for MadCap Feedback, you will also need to enable IIS 6 Metabase Compatibility. In the "Role Services" tab, expand Management Tools > IIS 6 Management Compatibility and enable IIS 6 Metabase Compatibility.

  11. At the "Confirm Installation Selections" dialog, click Install.

  12. The Installation Progress dialog will open during installation  

  13. When installation completes, the "Installation Results" dialog will open. Click Close.
