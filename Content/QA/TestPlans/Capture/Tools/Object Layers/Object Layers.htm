Object Layers

Bug Base Product Area:

  • Object Layers

 

Description:

The Object Layers pane lists all the objects on the current frame. All objects are listed in the layer order from top to bottom. Using the Object Layers pane, a user can have many layers of objects on top of each other and easily keep track of where each object is. Also the user can access the object properties without having to find the object on the main editor, which could be buried beneath many other objects. The user can also change the layer the object appears on via this pane as well.
