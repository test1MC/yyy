Find Groups

Select this menu command to search through the list of groups in Pulse.

  • New Groups: Shows the most recently created groups.
  • Most Followed Groups: Shows the groups with the largest combined number of followers and tags.
  • Most Active Groups: Shows the groups with the greatest amount of activity.
  • Followed by Me
  • Owned by Me
  • All Groups: Shows a list of all groups.
  • Create Group
  • Search Groups: Type a query in the Search field. Then press Enter. Matches appear in the Find Groups page.
