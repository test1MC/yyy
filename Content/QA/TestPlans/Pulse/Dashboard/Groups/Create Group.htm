Create Group

The Groups menu in the dashboard deals with groups that are different from those accessed from the Administration menu.

Basic:

  • Name: Give the group a name
  • Description: Enter a short description for the group
  • Audience: Select Employees or Employees and Customers to determine which members can be invited to join.
  • Moderation: Select if you want all group posts to be approved by a moderator before being displayed
  • Visibility: Select if you want the group to be visible to users who perform a search for the group.

Privacy:

  • Use the drop-downs to set the access level for each person or group.

  • Add a user or group...: Add a user or group to this group.
    • Type a name to select a user or group
    • Add: Adds the selected users or groups
    • Cancel: Exits the dialog without adding the user or group.
  • Create this Group: Saves settings and creates the group.

Test Cases:

  • Messaging to certain groups
  • Changing permissions for groups
  • Moderating groups
  • Subscribe to a group feed
  • Join multiple groups
