Home

Home is where you can manage your apps, your news feed, and your lists.

Note: Pulse Dashboard will appear slightly different in different browsers.  For example, the Open dialog in Google Chrome is called a File Upload dialog in Mozilla Firefox.  Screenshots for this test plan were taken using Google Chrome. During testing, ensure functionality in all browsers.

News Feed

News Feed displays a stream of news about activity on your Pulse server.

 

  • Status: Type content here.  
    • Share Movie: Opens the Open dialog where you can browse for a movie. Video must be of type mp4.
    • Share Image: Opens the Open dialog where you can browse for an image. Images must be of type PNG, JPG, or GIF.
    • Share File:
      • Upload a local file...: Opens the Open dialog where you can browse for a file.  Certain file types are not allowed (ex: .exe)
      • Link to a file...: Opens the Link Details dialog where you provide a Name and Url for your link.
  • Question: Type question here.  This text input field can be re sized by dragging the lower right corner.
  • Article:
    • Title: a title is required when posting an article.
    • Formatting: Bold, Italic, Underline, Bullet List and Numbered List. 

      • Note: These options will show in the preview, but are lost once you post (Bug 54742:Lists are losing their formatting when inserted via the "Article" link in the Pulse feed).

  • Task: Opens the task pane:
    • Enter a name...: Type a name for the task.
    • Description: Type a description of the task.
    • Assigned To: Type the user's name to assign them the task.
    • Tags: Type a tag name to associate the task with a tag.
    • Due: Click in the text field to see a calendar, then select a date.
    • Importance: Assign an importance level of low, medium, or high.
    • Status: Assign a status of Pending or Completed.
    • Save: Save the task
    • Cancel: Exits the task pane
  • Share: Once content has been typed in the Status, Question, or Article field, the Share button appears. Clicking it will publish your status to the News Feed.
  • Search this Feed: As you type in the search field, the contents of the feed will be filtered and your search term(s) will be highlighted in yellow.

    • Note: Wildcard, regular expression and partial word searching are not supported.

Files

Lists all files that you have uploaded to the server.  There are four columns: Name, Author, Size, and Created.

  • Download Folder: Downloads a .zip of all files listed
  • Upload Files: Opens the Upload Files dialog
    • Choose Files: Opens the Open dialog to browser for a file.

    • Upload Files: Uploads the selected files.
    • Cancel: Exits the dialog without uploading a file.
  • New Folder: Opens the New Folder dialog
    • Folder Name: Type the folder name here
    • Create Folder: Creates a folder with the name specified above.
    • Cancel: Exits the dialog without creating a folder.
  • Name: Clicking the file name will open a preview of the file
  • Author: Clicking the author currently takes you to the default IIS page (Bug 90859:Clicking the author in Files directs the site to default IIS page.)
  • Box: Clicking the square box at the end of the row produces this drop down menu:
    • Open: ?
    • Download: Downloads the file to your computer's default download location.
    • Rename: Opens the Rename File dialog:
    • Cut: ?
    • Delete: Opens the Confirm Delete Activity dialog.

Notifications: 

Pulse provides users with various notification types.  You can filter your notifications by clicking the filters in the upper right corner.

Moderation

Moderation allows you to: specify that new posts need to be approved, be notified when a post is pending approval, and approve or reject post. New Pulse communities will initially be set for moderation. Moderators will receive an email notification when there is a new post pending approval. You can then click the link to that topic and accept or reject the post.

  • Time stamp: Clicking the time stamp will hide all other posts.
  • Approve: Approves the post, it will now have the standard options below the post: Comment, Link, Follow, Tag, Share.  The user will get an email that their post has been approved.  If you refresh this page, the post will no longer be listed.
  • Reject: Rejects the post. The Reject Activity dialog will appear.

    You must select one of the predetermined reasons for rejecting the comment. The reason "Reject by the administrator." is available by default. The user will receive an email that the post was rejected.
  • All Pending: This view will show all active posts that need to be accepted/rejected.
  • Previously Rejected: This view will show all previously rejected posts.  You can approve any previously rejected posts here by clicking Approve.

Test Cases

  • By default, moderation is only available to Administrators and Employees, because they have status as moderators. However, when configuring the privacy settings for the community (i.e., system), you can assign moderation rights to other groups as well.  Test that those other groups have appropriate access to moderation, based on this setting.

Tasks:

All registered users can create, assign, and manage tasks with Pulse. This gives members of the Customers group the ability to assign a task to any other registered user. It also gives members of the Administrators and Employees groups the ability to assign tasks to each other in any topic, group, tag, and user feed that you have access permission to view.

Tasks will show "Tasks Assigned to me by Status" by default.  The progress bar will show green as an indicator of what percentage of the tasks have been complete.

  • Click the task name to open the task pane, here you can make changes and save.
  • Accept?: This task is pending you and will show as "Accept?". Click to show a drop down with the choices: Accept Task, Decline Task, Mark Complete.
  • Declined: This task has been declined. Click to show a drop down with the choices: Accept Task, Mark Complete.
  • Complete?: This task has already been accepted. Click to show a drop down with the choice to Mark Complete.
  • Completed: This task has already been completed. Click to show a drop down with the choice to Mark Incomplete.

The filter icon allows to change which tasks are displayed by filtering by people, tags or view.

  • Assigned to Anyone: Shows tasks assigned to you and tasks assigned to others by you.
  • Assigned to Me: Shows tasks assigned to you
  • Assigned by Me: Shows tasks that you assigned to other users.
  • Assigned to My Direct Reports: Shows tasks assigned to your Direct Reports folder. Direct Reports is an option you can select in the Notifications tab of the Settings page that deliver daily reports to your email.
  • Assigned to...: Shows tasks assigned to a specific person or people.
  • Tags: Shows tasks belonging to a certain tag.
  • Status: Shows tasks assigned by current status.
  • Upcoming: Shows tasks with upcoming due dates.

  • Ensure notification emails get sent correctly when assigning/modifying tasks

Mentions

You can use the Mentions feature to mention other registered users by name when posting status updates or creating articles. Like many popular social media sites, this feature lets you type the @ sign and then a user name to mention a user in your update.

Top News:

Top News are the user-specific followed Topics or comments that are receiving the most likes and comments. So if something “exciting or hot” gets posted on your News Feed, you can filter that by looking at your Top News first.

Read it Later

The Read it Later app is available in your user feed and gives you the ability to view the post activity that you bookmarked earlier. If you want to remove the bookmark, click the blue icon until it turns gray. A gray check mark indicates the next time you open or refresh the page, the post will no longer appear on your Read it Later page.

Questions:

The Questions app gives you the ability to read the questions that have been posted to a specific group, tag, topic, or user feed. If you are a member of the Administrators group, you also have the ability to delete the questions and answers that have been posted. Filter questions by: Recent, Unanswered, Uncategorized, Asked by Me, Followed by Me.

Articles

The Articles app gives you the ability to read the articles that have been posted to a specific group, tag, topic, or user feed. In addition, it gives you the ability to edit the articles you've posted to a feed. If you post an article, you have the ability to delete it. If you are a member of the Administrators group, you can also delete the articles that others post.
