Find Systems

Select this menu command to search through the list of people in Pulse.

  • New Systems: Displays the most recently created systems.

  • Most Active Systems: Displays the systems with the most user activity.

  • All Systems: Displays all systems that exist in the Pulse database.

  • [System Name]: Click the community's name to open the community page.
  • Search Systems...: Type a query in the Search field. Then press Enter. Matches appear in the Find Systems page.
