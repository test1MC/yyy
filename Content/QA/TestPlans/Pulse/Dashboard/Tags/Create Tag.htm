Create Tag

Basic:

  • Name: Give the tag a name
  • Description: Enter a short description for this tag.

Privacy:

  • Use the drop-downs to set the access level for each person or group.

  • Add a user or group...: Add a user or group to this group.
    • Type a name to select a user or group
    • Add: Adds the selected users or groups
    • Cancel: Exits the dialog without adding the user or group.
  • Create this Tag: Saves settings and creates the tag.
