Menus - UI

For the purposes of this test plan, verify all menu items function as they should and perform the Menu Test Cases for each menu.

File Menu

To access:

  • Click the File menu
  • Press [ALT] + F on the keyboard

File Menu Items (Tooltips in parentheses)

  • Import Feedback Data (Import Feedback Data): Opens the Select SQL Server and Database dialog
  • Set Admin Password (Set the password for the PulseAdmin account): Opens the Set PulseAdmin Password dialog
  • Save (Save File (Ctrl + S)): Saves the file
  • Exit (Exit the Application (Alt + F4)): Exits the application

Edit Menu

To access:

  • Click the Edit menu
  • Press [ALT] + E on the keyboard

Edit Menu Items (Tooltips in parentheses)

  • Undo (Undo): Undoes the most recent action
  • Redo (Redo): Redoes the action that was most recently undone
  • Cut (Cut): Cuts the current selection to the clipboard
  • Copy (Copy): Copies the current selection to the clipboard
  • Paste (Paste): Paste the contents from the clipboard
  • Delete (Del): Delete the current selection
  • Select All (Select All): Selects all
  • Properties (Display properties for the selected item (F4)): Opens Properties dialog

Help Menu

To access:

  • Click the Help menu
  • Press [ALT] + H on the keyboard
