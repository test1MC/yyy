Database Errors

Bug Base Product Area:

  • Tools

 

Description:

This window pane is used to view miscellaneous errors that may occur regarding the Analyzer database. These happen when your project contains data not compatible with the Analyzer database. If a file is listed in this pane, it will not be included in any analysis or suggestions. If your project contains database errors, please contact MadCap Software's technical support team for help.
