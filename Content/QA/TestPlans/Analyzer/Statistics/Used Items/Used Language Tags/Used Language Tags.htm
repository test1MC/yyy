Used Language Tags

Bug Base Product Area:

  •  

 

Description:

This window displays a list of all files where a specific language tag has been manually applied. For example, if you want the spell check feature to work in a particular language for a few topics, you can open the properties for those topic files and select the language. Those topics will then be included in this window pane. The list shows each file where a language tag is found, the content tag to which it has been applied (e.g., html, span), and the language used. You can double-click a row to open the topic or other file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
