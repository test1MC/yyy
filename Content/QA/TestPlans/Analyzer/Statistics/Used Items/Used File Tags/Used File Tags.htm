Used File Tags

Bug Base Product Area:

  •  

 

Description:

This window displays a list of all file tags that have been applied to files throughout your project. You can quickly rename or remove multiple file tag occurrences in a single batch. Make sure a check mark is next to the rows you want to affect, and click the "Rename" or "Delete" button. You can double-click a row to open the file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
