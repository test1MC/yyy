Undefined File Tags

Bug Base Product Area:

  •  

 

Description:

This window displays a list of all files with file tags applied to them, but those file tags have not been defined. This might occur, for example, if you have applied a file tag to a topic and then later change the name of the tag or file tag set. You can quickly rename or remove multiple file tag occurrences in a single batch. Make sure a check mark is next to the rows you want to affect, and click the "Rename" or "Delete" button. You can double-click a row to open the file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
