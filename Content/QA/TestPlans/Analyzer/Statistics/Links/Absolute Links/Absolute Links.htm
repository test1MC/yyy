Absolute Links

Bug Base Product Area:

  • Absolute Links

 

Description:

This window displays files in your project containing absolute links to other files. An absolute link contains the full path to the target file. For example, let’s say you have a text hyperlink that points to a file in your project like this: file://C:\MyProject\Content\MyFile.htm. Because users do not have access to your local drive, they will not be able to use the link successfully in the output. Instead, the link should be relative to the file in your project. This window allows you to find absolute links so that you turn them into relative links. You can double-click a row to open the file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
