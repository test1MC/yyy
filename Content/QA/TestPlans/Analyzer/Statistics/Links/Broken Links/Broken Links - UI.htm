  • Columns

    • File: Displays the files name of the file
    • Folder: Displays the full path of the folder containing the file
    • Link: Displays the (broken) link contained in the file
    • Type: Displays the type of broken link (Missing File, Missing Bookmark, etc.)
    • Link Text: Displays the text to which the broken link is associated
    • Link Tag: Displays the tag to which the broken link is associated
    • Extension: Displays the extension of the (broken) link
