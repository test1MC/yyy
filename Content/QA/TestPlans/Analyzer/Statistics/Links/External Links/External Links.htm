External Links

Bug Base Product Area:

  •  

 

Description:

This window displays files in your project containing links to external files. For example, if you have a text hyperlink to a website, the file where the link exists is included in this window. By clicking the "Check" button, you can verify that all external links point to valid destinations. You can double-click a row to open the file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
