Description:

This window pane displays a list of files associated with a frequent phrase, frequent segment, or similar segment (as selected in the Frequent Phrases window pane, Frequent Segments window pane, or Similar Segments window pane, respectively). You can double-click a row to open the file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
