UI Controls

Local Toolbar options

  • Hide/Show the Info Bar
  • Refresh
  • Export the data to a CSV file
  • Export the selected data to a CSV file
  • Print
  • Prints a report of the selected rows
  • Print Preview
  • Create (dropdown):
    • Create a Variable
    • Create a Snippet
  • Open the "Topics for Selected" Window
  • Search Properties:
    • This opens the "Frequent Segment Search Properties" dialog
      • Min Word Count; Max Word Count; Min Segment Count
  • Show All/Show Pages button:
    • Toggle between paged view and viewing all
  • Go to the first page (button)
  • Go to the previous page (button)
  • Go to the next page (button)
  • Go to the last page (button)
  • Page Count:
    • ex: Page 1 of 1
      • Regarding the "Page 1" part, this can be edited to a different value
      • There are also up/down arrows (buttons) to scroll through values

       

Columns

  • Segment: Displays the segment that occurs multiple times throughout your project.
  • Count: Displays the number of times a particular segment occurs throughout your project.
  • Word Count: Displays the number of words contained in a particular segment.
  • Character Count: Displays the number of characters contained in a particular segment.
