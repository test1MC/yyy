Replace Local Style Suggestions

Bug Base Product Area:

  •  

 

Description:

This window displays places in your project where you have used local formatting. You can replace these instances of formatting with existing styles in your style sheet that match the formatting that you have applied manually. Make sure a check mark is next to the rows you want to convert, and click the "Apply" button.
