Index Keyword Suggestions

Description:

This window pane displays a list of files containing text that matches existing index keywords in your project. You can quickly add those index keywords to the relevant files. Make sure a check mark is next to the rows where you want to add keywords, and click the "Apply" button. Alternatively, you can open each topic and add the index keywords manually.
