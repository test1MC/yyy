Writing Suggestions

Bug Base Product Area:

  •  

 

Description:

This window displays places in your project where you can make writing improvements based on industry-wide standards. For example, it can locate files where an h1 or h2 heading has only one subheading (i.e., an h1 heading with only one h2 heading under it, or an h2 heading with only one h3 heading under it). It is recommended that you have more than one subheading under a parent heading. It can also locate files where a heading level has been skipped (e.g., an h1 heading is followed by an h3 heading, rather than h2). You can use the "Search Properties" button to customize which types of suggestions to display in the window pane. You can double-click a row to open the topic or other file in the Analyzer interface. You can also select the row and click "Send to Flare" or "Send to Blaze" to open the file in the appropriate application.
