Duplicate TOC Items

Bug Base Product Area:

  • Duplicate TOC Items

 

Description:

This window pane displays files in your project that have been added more than once to a table of contents (TOC). You can double-click a row to open the TOC file in the grid view of the TOC Editor. The duplicate TOC entries related to that file are highlighted, making it easy to identify them. You can also select the row and click "Send to Flare" to open the file in the Flare project. This feature is used for Flare but not for Blaze.
