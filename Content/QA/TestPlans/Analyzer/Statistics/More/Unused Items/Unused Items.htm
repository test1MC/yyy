Description:

This window displays a list of all context-sensitive Help identifiers that have been created in your project but have not yet been assigned to any topics. You can double-click a row to open the alias file in the Analyzer interface. You can also select the row and click "Send to Flare" to open the header file in the Flare project. This feature is used for Flare but not for Blaze.
