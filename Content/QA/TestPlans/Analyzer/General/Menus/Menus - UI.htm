Menus - UI

For the purposes of this test plan, verify all menu items function as they should and perform the Menu Test Cases for each menu. All menu items will be tested in more detail in the test plans of the modules which they support.

File Menu

To access:

  • Click the File menu
  • Press [ALT] + F on the keyboard

File Menu Items (Tooltips in parentheses)

  • Open (Open File): Invokes an Open dialog which lets you browse for appropriate file types to open in the application
  • Reload (Reload the file): Returns an open file to its previous state, discarding any unsaved changes if you instruct it to.
  • StartUp File… (Set/reset the StartUp file) : Invokes the Startup File dialog which lets you set the file that should open each time the application opens
  • Close (Close File): Closes the current file
  • Close Project (Close project): Closes the current project, but the application itself does not close.
  • Analyzer Summary (Open the Analyzer Summary Window):
  • Project Properties… (Runs the Project Properties dialog):
  • Save (Save File): Saves the current file
  • Save As (Save As): Invokes a Save As dialog which lets you save the current file with a new filename, and opens the new file in place of the current file.
  • Save To (Save To): Invokes a Save As dialog which lets you save the current file with a new name but does not open the new file.
  • Save All (Save All): Saves all open documents

  • Backups: Invokes the Backups sub-menu

    • Send Copy to the "Backups Folder" (Copy the file to the Backups Folder): Creates a temporary copy of the file that is open and active in the interface in the Backups folder.
    • Clear the "Backups Folder"… (Delete all files in the Backups Folder): Removes all temporary backup files from the Backups folder.
    • Backups (Open the Backups Folder window): Opens the Backups pane
  • Source Control: Invokes the Source Control sub-menu
  • Send To: Invokes a submenu, which lets you open the active file in another application, attach it to an email, or open the Windows folder where it is stored
  • Print (Print): Prints the current file
  • Recent Projects: Invokes the Recent Projects sub-menu which lets you select a recently opened project to open again or manage the list of projects that are considered "recent"
  • Exit (Exit the application): Exits the application

 

Edit Menu

To access:

  • Click the Edit menu
  • Press [ALT] + E on the keyboard

Edit Menu Items (Tooltips in parentheses)

  • Undo (Undo): Undoes the most recent action
  • Redo (Redo): Redoes the action that was most recently undone
  • Cut (Cut): Cuts the current selection to the clipboard
  • Copy (Copy): Copies the current selection to the clipboard
  • Paste (Paste): Paste the contents from the clipboard
  • Delete (Delete): Delete the current selection
  • Select All (Select All): Selects all

  • Find and Replace: Invokes the Find and Replace sub-menu

    • Find and Replace: Opens the Find and Replace window pane
    • Find in Files: Opens the Find in Files window pane
    • Load Find Selection: Copies the selected text in the editor to the "Find what" field in the Find and Replace window pane or the Find in Files window pane.
    • Find Next: Initiates the search for the next instance of the search term.
    • Find Results 1: Opens the Find Results 1 window pane
    • Find Results 2: Opens the Find Results 2 window pane

  • IntelliSense: Invokes the IntelliSense sub-menu

    • List Matches (List the current matches): Select this option to display existing words that match the text in relation to where the cursor is placed in the content.
    • List Frequent Phrases (List the current frequent phrases): Select this option to display phrases that you have already typed elsewhere in your project and that match the text in relation to where the cursor is placed in the content.
    • List Variables (List variables): Select this option to display variables that you have already added in your project.
    • IntelliSense Properties… (Display the IntelliSense Properties dialog): Invokes the IntelliSense dialog, which is used to modify the settings for IntelliSense
    • Enable IntelliSense (Enable/disable IntelliSense): Lets you turn IntelliSense on or off.
  • Properties (Run properties dialog for this item): Invokes the Properties dialog for the selected item

Tools Menu

To access:

  • Click the Tools menu
  • Press [ALT] + T on the keyboard

Tools Menu Items (Tooltips in parentheses)

  • Update Cross-References… (Update Cross-References in the current document): Updates Cross-References in the current document if any require updating
  • Edit Master Stylesheet (Edit the associated master stylesheet document): Opens the Master Stylesheet in the Stylesheet Editor
  • Stylesheet Links (Manage links to stylesheets): Invokes the Stylesheet Links dialog, unless a Master Stylesheet is used by the project or primary target, in which case a message indicating as much will appear

  • Index: Invokes the Index submenu

    • Index Window (Open the Index Window): Opens the Index window
    • Insert "[selection]" as Index Keyword:
    • Index Entry Mode (Toggle Index Entry Mode): Enables or disables Index Entry Mode
    • Auto-complete Index Entries (Auto-complete Index and Concept entries in the Index and Concept windows): Enables or disables auto-complete of index and concept entries in the Index and Concepts panes

  • Concepts: Invokes the Concepts submenu

    • Concept Window (Open the Concept Window): Opens the Concepts window
    • Concept Entry Mode (Toggle Concept Entry Mode): Enables or disables Concept Entry Mode
  • Glossary Terms (Open the Glossary Terms Window): Opens the Glossary Terms pane
  • Page Frame Contents Pane (Open the Page Frame Contents Entry Window): Opens the Page Frame Contents pane
  • Edit Footnote Pane (Open the Edit Footnote pane): Opens the Edit Footnote pane
  • Footnotes (Open the Footnotes window): Opens the Footnotes pane
  • Edit Annotation Pane (Open the Edit Annotation pane): Opens the Edit Annotation pane
  • Annotations Pane (Open the Annotations Pane): Opens the Annotations pane
  • Spell Check While Typing (Spell check while typing): Enables or disables spell check while typing
  • Spell Check Window (Open the Spell Check window): Opens the Spell Check window
  • Ignored Words… (Edit the 'ignored words' associated with the current document): Invokes the Ignored Words dialog for the current document
  • Options (Change Global Application Options): Invokes the Options dialog
