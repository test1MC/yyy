Mobile Emulators

Installation Location: \\DC2\QA\Testing Tools\Mobile Emulators

iPhone

Network access is normal. You can use UNC paths to access network locations.

How to install:

  • Install iPhone SDK

How to use:

  • Open “iPhone Simulator” from the dock.
  • Open Safari browser.
  • Navigate to the Help system

 

Windows Mobile

Emulator images are based on Microsoft Virtual PC.

Network access is normal. You can use UNC paths to access network locations.

State can be saved and cleared.

How to install:

  • Install Virtual PC.
  • Install Windows Mobile SDK.
  • Install additional emulator images for V6.1, V6.5, and eventually V7.

How to use:

  • Launch Device Emulator Manager
    • C:\Program Files\Microsoft Device Emulator\1.0\dvcemumanager.exe
  • Right-click the emulator image you want to use and select “Connect”.
  • Set up network access (this can be done once and then saved to the state of the image):
    • File | Configure... | Network
    • Check “Enable NE2000 PCMCIA network adapter and bind to:”
    • Click “OK”
    • Start | Settings | Connections | Advanced | Select Networks
    • In the first drop-down, select “My Work Network”.
    • Click “Edit…”
    • Click “Proxy Settings”
    • Check “This network connects to the Internet”
    • Click “OK”, “OK”, “OK”
    • At this point, you will probably want to save the state of the emulator image.
      • Close the emulator window.
      • When you’re prompted to save the state, select “Yes”.
  • Open the browser.
  • Navigate to the Help system.
  • To clear the save state, launch Device Emulator Manager, right-click the emulator image and select “Clear Save State”.

Palm webOS

Emulator images are based on Sun Virtual Box.

Network access maps http://10.0.2.2 to http://localhost.

Multiple states can be saved, reverted to, and cleared.

How to install:

  • Install Virtual Box. Must be version 3.0.12.
  • Install Palm SDK.

How to use:

  • Launch Palm Emulator
    • Start | Programs | Palm | SDK | Palm Emulator
  • Select the emulator you need to test (different resolutions) and click “Start Emulator”.
  • Open the browser.
  • Navigate to the Help system.

 

BlackBerry

Network access is normal. You can use UNC paths to access network locations.

How to install:

  • Install JDK.
  • Install Email and MDS Services Simulator.
  • Install each phone simulator that you need to test.

How to use:

  • Launch Email and MDS Services Simulator
    • Start | Programs | Research In Motion | BlackBerry Email and MDS Services Simulators 4.1.4 | MDS
  • Launch the particular phone simulator that you need to test.

 

Android

Network access maps http://10.0.2.2 to http://localhost

State can be saved and cleared.

How to install:

  • Unzip Android SDK.
  • Launch “SDK Setup.exe”.
  • There may be an error regarding “Failed to fetch URL” due to “HTTPS SSL error”. If so follow these steps:
    • Click “Close” on the error message.
    • When prompted to “Choose Packages to Install”, click “Cancel”.
    • Select “Settings”.
    • Enable “Force https://... Sources to be fetched using http://...”.
    • Click “Save & Apply”.
    • Select “Installed Packages”.
    • Click “Update All…”.
  • When prompted to “Choose Packages to Install”, select all SDK Platforms (1.1, 1.5, 1.6, 2.0).
  • Select “Virtual Devices”.
  • Click “New…”
  • Enter a name for the virtual device.
  • Select the target in the drop-down (Android 2.0).
  • Click “Create AVD”.
  • You should get a confirmation that it was successfully created. Click “OK”.

How to use:

  • Launch “SDK Setup.exe” from the Android SDK folder.
  • When prompted to “Choose Packages to Install”, click “Cancel”.
  • Select “Virtual Devices”.
  • Select the AVD you created previously and click “Start…”
  • In the Launch Options dialog, there is an option to wipe user data.
  • Click “Launch”
