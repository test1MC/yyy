Start Page

Bug Base Product Area:

General or Documentation

 

Description:

The Start Page opens by default in the MDI area of Flare. It’s designed to give users shortcuts to last used projects, Quick links to getting started and help, and a news area. The news area is fed by an XML file on our server so it can be changed when needed..
