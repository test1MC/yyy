Getting Started Wizard - UI

Opening the Getting Started Wizard

By default, when Flare is first launched, the Getting Started Wizard should open. There is an option in the Getting Started Wizard to show it on launch. If that option is turned off, the Getting Started Wizard can be opened by:

  • !File | Getting Started Wizard
  • In the Start Page, click the "Launch the Getting Started Wizard" option in the "Getting Started" section

Getting Started Wizard - UI Test Cases

UI Controls

  • Action: Offers a series of options used to get started using Flare. As each option is selected, an explanation of the option should appear in the window to the right. Explanations are included below in italics.

    • Watch tutorial 6 Steps to Online Help: This option opens the 30-minute interactive video tutorial "6 Steps to Online Help," which shows you step-by-step how to create, develop, build, and distribute a Flare project.

      Clicking this option will activate the <Finish> and <Cancel> button. Click <Finish> to launch the video tutorial in Flare's internal Movie Viewer.

    • Open a sample project: This option guides you through steps for creating a sample Flare project. To become familiar with Flare, you can then work inside the sample project just as you would any other project.

      Clicking this option will activate the <Next> and <Cancel> button. After you select this option, click Next. The "Open a sample project" page of the wizard opens.

    • Start a new project: This option opens the Start New Project Wizard, which guides you through the basic steps for creating a new Flare project. When you finish providing information in the Start New Project Wizard, the new project is loaded into Flare.

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click <Finish> to launch the Start New Project Wizard.

    • Import a RoboHelp project: This option opens the Import Project Wizard, which guides you through the basic steps for importing a RoboHelp project and therefore creating a new project as a result. When you finish providing information in the Import Project Wizard, the new project is loaded into Flare with the imported RoboHelp files.

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click <Finish>.

    • Create a project from MS Word files: This option opens the Import Microsoft Word Wizard, which guides you through the basic steps for importing Microsoft Word documents and therefore creating a new project as a result. When you finish providing information in the Import Microsoft Word Wizard, the new project is loaded into Flare with the imported Word files.

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click <Finish> to launch the Import Microsoft Word Wizard.

    • Create a project from FrameMaker files: This option opens the Import FrameMaker Wizard, which guides you through the basic steps for importing FrameMaker documents and therefore creating a new project as a result. When you finish providing information in the Import FrameMaker Wizard, the new project is loaded into Flare with the imported FrameMaker files.

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click <Finish> to launch the Import FrameMaker Wizard.

    • Create a project from DITA files: This option opens the Import DITA Document Set Wizard, which guides you through the basic steps for importing DITA documents and therefore creating a new project as a result. When you finish providing information in the Import DITA Document Set Wizard, the new project is loaded into Flare with the imported DITA files.

      Clicking this option will activate the <Finish> and <Cancel> button. After you select this option, click <Finish> to launch the Import DITA Wizard.

  • Show this dialog on launch: If there is a check mark in the box, this wizard will open each time you open Flare in the future. If you do not want to see the Getting Started Wizard each time you open Flare, remove the check mark from the box.
  • Back: This button is disabled in this window.
  • Next: Opens the next page of the Getting Started Wizard ("Open a sample project" page 1) if you have selected the "Open a sample project" option. Otherwise, this button is disabled.
  • Finish: Completes the option that you selected (unless you selected "Open a sample project," in which case this button is disabled).
  • Close/Cancel: Closes the Wizard.

Open a Sample Project - Page 1

UI Controls

  • Sample Projects: Displays a list of Sample Project provided by Flare. Select one to use as a template for the new project.
  • Project Name: Type any name for the project, or keep the one provided for you.
  • Project Folder: Displays the folder to which the new project will be saved. By default, Flare will use My Documents\My Projects\Samples.
  • Pick Folder: Opens the Browse For Folder dialog, which lets you select a different folder to store the sample project files.
  • Show this dialog on launch: Same as above.
  • Back: Returns to the previous page of the wizard ("Welcome to MadCap Flare").
  • Next: Opens the next page of the Getting Started Wizard ("Open a sample project" page 2).
  • Finish: The Finish button should be disabled at this point.
  • Cancel: Closes the Wizard and does not create a new project.

 

Open a Sample Project - Page 2

UI Controls

  • Available Targets: Select one of the targets shown to serve as the primary target for the project.
  • Type Description: Displays a description of the target that you selected.
  • Show this dialog on launch: Same as above.
  • Back: Returns to the previous page of the wizard ("Welcome to MadCap Flare").
  • Next: Opens the next page of the Getting Started Wizard ("Open a sample project" page 3).
  • Finish: Creates the sample project and loads the files into Flare.
  • Cancel: Closes the Wizard and does not create a new project.

 

Open a Sample Project - Page 3

UI Controls

  • Create the project: Creates the project (after you click Finish).
  • Create the project and generate the selected output: Creates the project AND generates output for the project right away (after you click Finish).
  • Show this dialog on launch: Same as above.
  • Back: Returns to the previous page of the wizard ("Open a sample project" page 2).
  • Next: The Next button should be disabled at this point.
  • Finish: Creates the sample project and loads the files into Flare. If you selected the second option above, it also generates output for the project immediately.
  • Cancel: Closes the Wizard and does not create a new project.
