Toolbars

Bug Base Product Area:

  • Toolbars / Menus

 

Description:

Flare toolbar is made up of Global Toolbars and Local Toolbars. Global Toolbar are always available, regardless of where the user are working in the interface, this includes the Standard Toolbar, Text Formatting Toolbar, Project Toolbar and Review Toolbar. Local Toolbars are intended for a particular editor and are covered test plans for each specific module that has a Local Toolbar.

 

With the new version with a ribbon interface, these toolbars will only be selectable when in Standard Tool Strip mode.
