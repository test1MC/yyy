Toolbars - UI

Standard Toolbar

Opening/Closing the Standard Toolbar

From a menu:

  • !View | Toolbars | Standard
  • Click the button in the upper-left corner (next to the File menu) and select Standard.

Standard Toolbar - UI Controls

Toolbar Items (Tooltips in parentheses)

  • (Open the Start Page): Opens the Start Page, which provides links to high-level tasks and information.
  • (Starts the New Project Wizard): Opens the Start new Project Wizard, which user can use to create a new Flare project.
  • (Open File): Opens the Open dialog, which lets user search for and select a Flare project to open.
  • (Save File): Saves changes to the active file.
  • (Save All): Saves changes to all of the open files.
  • (Send To): Opens a submenu, which lets user open the active file in another application (e.g., Dreamweaver, Word, Notepad), attach it to an email, or open the Windows folder where it is stored.
  • (Source Control): Opens the Source Control Menu.
  • (Locate file in TOC(s)): Opens the TOC Editor and highlights the location of the active topic in the TOC. If the selected file is not in any TOC, a prompt will indicate "Can't find the file in the TOC." If the selected files is contained in more than one TOC file, a dialog will open allowing you to select which TOC to open.
  • (Locate file in Content Explorer): Opens the Content Explorer and highlights the location of the selected content file within it.
  • Cut: Cuts the selected item(s) to the clipboard.
  • Copy: Copies the selected item(s) to the clipboard.
  • (Paste): Pastes the item(s) from the clipboard.
  • (Delete): Deletes the selected item(s)
  • (Undo): Lets user reverse the most recent action(s) in a file. The button label will vary based on the most recent action.
  • (Redo): Lets user repeat the most recent action(s) that user reversed in a file. The button label will vary based on the most recent Undo action.
  • (Display properties for the selected item): Opens the Properties dialog for selected file (in the Content Explorer or Project Organizer).

 

Project Toolbar

Opening/Closing the Project Toolbar

From a menu:

  • !View | Toolbars | Project
  • Click the button in the upper-left corner (next to the File menu) and select Project.

Project Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (Runs the Project Properties dialog): Opens the Project Properties dialog.
  • (Build the Primary Target): Builds (generates) the output for the target that is designated as the Primary Target.

    • Drop-down menu displays all Target files in the project and lets you Build any one of them.  Tooltip for each of these menu items should be "Build this Target":

  • (View the Primary Target): Lets user view the output for the target that is designated as the Primary Target.

    • Drop-down menu displays all Target files in the project and lets you View any one of them.  Tooltip for each of these menu items should be "View this Target":

  • (Publish the Primary Target): Opens the Publish Target dialog, which lets user publish the output for the Primary Target to its destination(s). If there is no generated output, you should receive a prompt to generate the output.

    • Drop-down menu displays all Target files in the project and lets you Publish any one of them.  Tooltip for each of these menu items should be "Publish this Target":

  • (Open the Primary Target File): Opens the primary target in the Target Editor.
  • (Open the Master TOC): Opens the Master TOC file in the TOC Editor.
  • (Open the Master Stylesheet): Opens the Master Stylesheet in the Stylesheet Editor.
  • (Open the Master Page Layout): Opens the Master Page Layout file in the Page Layout Editor.
  • (Create a new Topic file): Opens the Add New Topic dialog which lets you create a new topic in the project.

 

Review Toolbar

The items in this toolbar will be tested in depth in the Annotations/Reviewing test plan.

Opening/Closing the Review Toolbar

From a menu:

  • !View | Toolbars | Review
  • Click the button in the upper-left corner (next to the File menu) and select Review.

Review Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (Send document to reviewer(s)): Opens the Send Topic for Review dialog which lets you email the current file to reviewers.
  • (Open Topic Reviews Window): Opens the Topic Reviews pane.
  • (Import a reviewed package to the inbox): Opens the Open dialog, which lets user search for and select a Flare review package file (*.fltrev) to open.
  • (Toggle tracking changes): Turn on/off track changes; open the review options dialog.
  • (Toggle showing changes): Toggle show changes on/off.
  • (Navigate to the previous change): Navigates to the previous change.
  • (Navigate to the next change): Navigates to the next change.
  • (Accept the current change): Accepts the currently selected change.
  • (Reject the current change): Rejects the currently selected change.
  • (Insert an annotation): Inserts an annotation around the selection.
  • (Delete the annotation): Deletes an annotation around the selection.

 

Format Toolbar

Opening/Closing the Format Toolbar

From a menu:

  • !View | Toolbars | Text Format
  • Click the button in the upper-left corner (next to the File menu) and select Text Format.

Format Toolbar - UI Controls

 

Toolbar Items (Tooltips in parentheses)

  • (Style): Lets user select a style class from any style sheet associated with the current topic, applying it to the content selected in the topic.
  • (Font Family): Lets user select a font type (e.g., Arial, Times New Roman), applying it to the content selected in the topic.
  • : Lets user select a font size, applying it to the content selected in the topic.
  • (Bold): Applies bold typeface to the content selected in the topic.
  • (Italic): Applies italic typeface to the content selected in the topic.
  • (Underline): Underlines the content selected in the topic.
  • (Superscript): Raises the selected content in the topic above the normal line of text (e.g., X2).
  • (Subscript): Lowers the selected content in the topic below the normal line of text (e.g., X2).
  • (Font Properties): Opens the Font Properties dialog, which lets user apply multiple kinds of formatting to the selected content in the topic all at once.
  • (Unformat): Resets the formatting applied to content in a topic.
  • (Color): Lets user choose and apply a color to the selected content.
  • (Background Color): Lets user choose and apply a color to the background of the selected content.
  • (Mark text for redaction): Marks the selected text for redaction in the output.
  • (Align Left): Aligns the selected content to the left.
  • (Align Center): Aligns the selected content in the center.
  • (Align Right): Aligns the selected content in the right.
  • (Align Justify): Spaces the selected content so that both the left and right sides are aligned.
  • (Align Default): Aligns the selected content to default.
  • (the tooltip will be the last selected menu item; Bullet List, Square Bullet List, Numbered List, etc): Lets user select one of several kinds of bullet or numbered lists, applying it to the selected content.
    • Bullet List (Bullet List): Applies a bulleted list to the selected content.
    • Circle Bullet List (Circle Bullet List): Applies a bulleted list to the selected content. The bullets are circles.
    • Square Bullet List (Square Bullet List): Applies a bulleted list to the selected content. The bullets are squares.
    • Numbered List (Numbered List): Applies a numbered list to the selected content.
    • Lower-alpha Numbered List (Lower-alpha Numbered List): Applies a numbered list to the selected content where lowercase alpha characters are used (a., b., c., d., etc.)
    • Upper-alpha Numbered List (Upper-alpha Numbered List): Applies a numbered list to the selected content where uppercase alpha characters are used (A., B., C., D., etc.)
    • Lower-roman Numbered List (Lower-roman Numbered List): Applies a numbered list to the selected content where lowercase Roman numerals are used (i., ii., iii., iv., etc.)
    • Upper-roman Numbered List (Upper-roman Numbered List): Applies a numbered list to the selected content where uppercase Roman numerals are used (I., II., III., IV., etc.) 
  • (Outdent Items): Lets user shift indented content to the left, thus "outdenting" it.
  • (Indent Items): Lets user shift selected content to the right, thus "indenting" it.
  • (Drop down menu of assorted list actions): Lets user apply miscellaneous actions to selected content in a list. These actions are as follows:
    • Merge With Previous List: Combines a list with another list immediately before it.
    • Merge With Next List: Combines a list with another list immediately following it.
    • Sort List: Reorders the items in the list alphabetically.
    • Reverse List: Reorders the items in the list so that they appear in reverse order (i.e., first item is last, last item is first).
    • Unbind List: Removes the list designation from the content so that it displays as regular text.
    • Continue Sequence: Ensures that the next list user create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content).
    • List Start Number: Lets user specify a number to start a numbered list. (After user enter a number in this selection, user need to click the blue "enter" arrow at the end to apply the effect.)
    • Item Number: Lets user specify a number to start a selected item in a numbered list.
    • Move Up: Moves the selected item in the list above the item before it.
    • Move Down: Moves the selected item in the list below the item after it.
    • Make Simple Item(s): Lets user return a line in a list from a paragraph item to a simple item (an "li" tag without a "p" tag following it). Paragraph items can be used in numbered or bulleted lists, for example, to add comments without disrupting the flow of the list.
    • Make Paragraph Item(s): Lets user add a "p" tag after the "li" tag in a numbered or bulleted list. User can then add more lines after a list item without adding numbers or bullets to those lines. Paragraph items can be used in numbered or bulleted lists, for example, to add comments without disrupting the flow of the list.
  • Insert Table:When clicking on the drop down, allows user to drag and select the size of a table that you want to insert. When clicking on the table icon, it will bring up the Insert Table dialog box.
