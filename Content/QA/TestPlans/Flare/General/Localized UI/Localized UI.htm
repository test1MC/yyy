Bug Base Product Area:

Localized UI

 

Description:

The entire Flare UI can be viewed in English, German, French or Japanese. When Flare is initially launched, this language option can be set. When testing localized UI, there are a few main things you need to look for: all strings have been translated, all translated strings fit inside the UI, special characters appear correctly, and a few other items which will show up as Test Cases.

 

Test Plans:

  • Localized UI - UI
  • Localized UI - Baseline

 

Dependencies:

None. However, it is best that this test plan is done after all new strings have been translated to all supported languages and the latest strings files are included in the build.
