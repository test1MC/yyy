Menus - Tool Strip UI

For the purposes of this test plan, verify all menu items function as they should and perform the Menu Test Cases for each menu. All menu items will be tested in more detail in the test plans of the modules which they support.

Accessing the Tool Strip (If Tool Strip is not enabled)

  • Go to !File | Options
  • In the "Interface" tab, select "Tool Strip"
  • Click OK

File Menu

To access:

  • Click the File menu
  • Press [ALT] + F on the keyboard

File Menu Items (Tooltips in parentheses)

  • New Project… (Starts the New Project Wizard): Starts the New Project Wizard

  • Import Project: Invokes the Import Project sub-menu

    • Import MS Word Documents (Starts the Import MS Word Documents Wizard): Starts the Import Microsoft Word Wizard
    • Import FrameMaker Documents (Starts the Import Adobe FrameMaker Documents Wizard): Starts the Import FrameMaker Wizard
    • Import Robohelp Project (Starts the Import Robohelp Wizard): Starts the Import Robohelp Wizard
    • Import HTML Help Project (HHP) (Starts the Import HHP Wizard): Starts the Import HHP Wizard
    • Import HTML Help File (CHM) (Starts the Import CHM Wizard): Starts the Import CHM Wizard
    • Import DITA Document Set (Starts the Import DITA Document Set Wizard): Starts the Import DITA Wizard
    • Import HTML File Set (Starts the import HTML File Set Wizard): Starts the Import HTML File Set Wizard
    • Import From Source Control (Starts the Import Project from Source Control Wizard): Starts the Import Project from Source Control Wizard

  • Zip: Invokes the Zip sub-menu

    • Zip Project… (Zip the project into a single file archive): Invokes the Create Package dialog
    • Zip and Email Project… (Zip and email the project): Automatically zips the project, opens your email program, and attaches the zip file to a new message.
  • Open (Open File): Invokes an Open dialog which lets you browse for appropriate file types to open in the application
  • Close (Close File): Closes the current file
  • Close Project (Close project): Closes the current project, but the application itself does not close.
  • Save (Save File): Saves the current file
  • Save As (Save As): Invokes a Save As dialog which lets you save the current file with a new filename, and opens the new file in place of the current file.
  • Save To (Save To): Invokes a Save As dialog which lets you save the current file with a new name but does not open the new file.
  • Save All (Save All): Saves all open documents
  • Save As Template… (Save As Template): Invokes the Save as Template dialog which lets you save the current project as a template.

  • Backups: Invokes the Backups sub-menu

    • Send Copy to the "Backups Folder" (Copy the file to the Backups Folder): Creates a temporary copy of the file that is open and active in the interface in the Backups folder.
    • Clear the "Backups Folder"… (Delete all files in the Backups Folder): Removes all temporary backup files from the Backups folder.
    • Backups (Open the Backups Folder window): Opens the Backups pane

  • Source Control: Invokes the Source Control sub-menu

    • Project: Invokes a submenu, which lets you apply changes at project level (check in all, check out all, get latest version all, undo checkout all)
    • Add (Add file(s) to Source Control): Add a new file to source control
    • Check In (Check file(s) into Source Control): Check file into source control after it's been checked out
    • Check Out (Check file(s) out of Source Control): Check a file out in order to make edits and changes
    • Get Latest Version (Get the latest version from Source Control): Copies the most current files stored in source control to your local Flare project
    • Undo Check Out (Undo Check Out): Revert pending changes to checked out files
    • Show Differences (Show Differences): Open a pane displaying the first version selected on the right side and the second version selected on the left side. The differences between the versions are highlighted.
    • View History (View History): Opens a dialog that displays the history of changes made to the file
    • Request Check In (Send request to check in file to user(s)): Opens the Send Request Check-In Message dialog, and provides option to send instant message or email to user
    • Request View File (Send request to view file to user(s)): Opens the View File Request dialog where user can notify another author to review a file and provide comments
    • Undelete (Undelete): Undoes what was deleted
    • Refresh (Refresh source control status): Refresh source control status
    • Pending Check-Ins (Open Pending Check-Ins window): Opens Pending Check-Ins window to display all the files marked to be checked into source control

  • SharePoint: Invokes the SharePoint sub-menu

    • Check In: Check file(s) into SharePoint
    • Check Out: Check file(s) out of SharePoint
    • Undo Check Out: Undo check out
    • SharePoint Explorer: Opens SharePoint Explorer window

     

     

  • Send To: Invokes a submenu, which lets you open the active file in another application, attach it to an email, or open the Windows folder where it is stored
  • Print (Print): Prints the current file
  • Print Preview (Print preview): Opens the Print Preview dialog, which lets you see how a document looks before you send it to a printer.
  • Properties (File Properties): Opens the Properties dialog for the selected item.
  • Recent Files: Invokes the Recent Files sub-menu which lets you select a recently opened file to open again or manage the list of files that are considered "recent"
  • Recent Projects: Invokes the Recent Projects sub-menu which lets you select a recently opened project to open again or manage the list of projects that are considered "recent"
  • Exit (Exit the application): Exits the application

View | Show Menu

View | Show Menu Items (Tooltips in parentheses)

  • Show Caret Tag Neighborhood (Toggle show Caret Tag Neighborhood): Turns the Caret Tag Neighborhood feature on or off.
  • Show Space (Toggle show space): Turns paragraph and space markers on and off in the active topic.
  • Show tags (Toggle show tags): Changes the XML Editor to "Show Tags" mode
  • Show Markers (Toggle show markers): Turns markers on or off in the active topic or other content file.
  • Show Inline Markers (Show markers for all inline elements): Lets you show or hide markers for all inline elements in the active topic.
  • Show invisible tags (Toggle show invisible tags): Turns invisible tags (e.g., <title>) on or off in the active topic. This option is enabled only if you have selected "Show tags."
  • Show Comments (Toggle show comments): Turns comment tags on or off in the active topic.
  • Show All Images as Thumbnails (Toggle show image thumbnails): Lets you toggle between displaying images as thumbnails or full size in the XML Editor.
  • Show tags in mono-color (): Lets you choose whether to display tags in a color or in black. This option is enabled only if you have selected "Show tags."
  • Show tags in mono-font (Toggle show tags in mono-font): Lets you choose whether to display all tags in the same font or in the font actually applied to each area. This option is enabled only if you have selected "Show tags."
  • Enable Object Positioning (Enable Object Positioning): Turns object positioning on and off.
  • Enable Orphan/Widow Control (Enable Orphan/Widow Control): Turns widow and orphan control on or off.
  • Show Hidden Elements (Toggle show hidden elements): Shows content that has been designated as "hidden." (the "display" property of the element is set to "none")
  • Show Conditional Indicators (Hide/show conditional indicators): Lets you show or hide conditional indicators in the XML Editor.
  • Show Variable Names (Toggle show variable names): Lets you show or hide variable names in your variable markers.
  • Highlight Variables (Highlight variables): Lets you choose whether to see a gray background behind variable definitions in a topic.
  • Show Bookmark Names (Toggle show bookmark names): Lets you show or hide the names of bookmarks in your markers (if you have inserted bookmarks) in the active topic.
  • Show Named Elements (Toggle show named elements): Lets you show or hide markers for all inline elements in the active topic.
  • Locks (Disabled/Enabled) (Switch to locked/unlocked mode): Locks or unlocks the content designated "locked"
  • Show Locked Elements (Toggle show locked elements): Allows you to show or hide the gray background associated with elements that have been tagged as "locked."
  • Show Changes (Toggle showing changes): Lets you show or hide the Annotations portion of the XML Editor.
  • Marker Width: Opens the Set Marker Width dialog.

Format Menu

To access:

  • Click the Window menu
  • Press [ALT] + F on the keyboard

Format Menu Items (Tooltips in parentheses)

  • Font… (Font Properties): Opens the Font Properties dialog, which lets you edit the font properties for the selected content.
  • Paragraph…/Picture… (Edit Paragraph Properties/Edit Picture): In most cases, this option invokes the Paragraph Properties dialog. If an image is selected, the option will change to "Picture…" and invoke the Edit Picture dialog.
  • Conditions… (Select Conditional Text Tags): Invokes the Condition Tags dialog which lets you apply condition tags to the selected content.
  • Language… (Select Language): Opens the Select Language dialog, which lets you choose a language for the selected content
  • Name… (Name the element): Opens the Manage Named Elements, which you can use to name the current selection.
  • Create Style Class… (Create a new CSS style class): Opens the Create Style dialog, which lets you create a new style for the content selected in the topic.
  • Pick Style (Pick style): Opens the floating Style Picker, which lets you select and apply a style to the selected content.
  • Create Snippet… (Create a new snippet): Invokes the Create Snippet dialog populated with the current element
  • List: Invokes the List sub-menu (See below)
  • Group… (Group selected items): Invokes the Create Group dialog
  • Bold (Bold): Applies bold typeface to the content selected in the topic.
  • Italic (Italic): Applies italic typeface to the content selected in the topic.
  • Underline (Underline): Underlines the content selected in the topic.
  • Color: Lets you choose and apply a color to the selected content.
  • Background Color: Lets you choose and apply a color to the background of the selected content.

  • Locks: Invokes the Locks sub-menu

    • Mark Element(s) to be Locked (Mark/unmark selected elements to be locked): Places a gray background behind the selected content in the topic, which means that the content can now be locked. In order to see this gray background, you must select the option below to show locked elements.
    • Locks (Disabled/Enabled) (Switch to locked/unlocked mode): Locks or unlocks the content designated with the gray background. After content has been locked, it cannot be edited, and annotations cannot be added to it.
    • Show Locked Elements (Show locked elements): Allows you to show or hide the gray background associated with elements that have been locked or selected to be locked.

 

Format | List Menu

Format | List Menu Items (Tooltips in parentheses)

  • List Actions: Invokes the List Actions sub-menu

    • Merge With Previous List (Merge list with Previous list): Combines a list with another list immediately before it.
    • Merge With Next (Marge list with next list): List Combines a list with another list immediately following it.
    • Sort List (Sort list): Reorders the items in the list alphabetically.
    • Reverse List (Reverse list): Reorders the items in the list so that they appear in reverse order (i.e., first item is last, last item is first).
    • Unbind List (Unbind list - replace it with the items): Removes the list designation from the content so that it displays as regular text.
    • Continue Sequence (Don't restart the sequence number, i.e. continue where previous list left off): Ensures that the next list you create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content).
    • List Start #: Lets you specify a number to start a numbered list. (After you enter a number in this selection, you need to click the blue "enter" arrow at the end to apply the effect.)
    • Item #: Lets you specify a number to start a selected item in a numbered list.
    • Move Up (Move list item up one step): Moves the selected item in the list above the item before it.
    • Move Down (Move list item down one step): Moves the selected item in the list below the item after it.
    • Make Simple Item(s) (Make Simple Items - remove any embedded paragraph tags): Lets you return a line in a list from a paragraph item to a simple item (an <li> tag without a <p> tag following it).
    • Make Paragraph Item(s) (Make Paragraph Items - inject paragraph elements inside the list items ): Lets you add a <p> tag after the <li> tag in a numbered or bulleted list.
  • Bullet List (Bullet List): Applies a bulleted list to the selected content.
  • Circle Bullet List (Circle Bullet List): Applies a bulleted list to the selected content. The bullets are circles.
  • Square Bullet List (Square Bullet List): Applies a bulleted list to the selected content. The bullets are squares.
  • Numbered List (Numbered List): Applies a numbered list to the selected content.
  • Lower-alpha Numbered List (Lower-alpha Numbered List): Applies a numbered list to the selected content where lowercase alpha characters are used (a., b., c., d., etc.).
  • Upper-alpha Numbered List (Upper-alpha Numbered List): Applies a numbered list to the selected content where uppercase alpha characters are used (A., B., C., D., etc.).
  • Lower-roman Numbered List (Lower-roman Numbered List): Applies a numbered list to the selected content where lowercase Roman numerals are used (i., ii., iii., iv., etc.).
  • Upper-roman Numbered List (Upper-roman Numbered List): Applies a numbered list to the selected content where uppercase Roman numerals are used (I., II., III., IV., etc.).
  • Outdent Items (Outdent Items): Lets you shift indented content to the left, thus "outdenting" it.
  • Indent Items (Indent Items): Lets you shift selected content to the right, thus "indenting" it.
  • Custom Format: 
    • Create Custom Format...: Opens the "Create New List Format" dialog
    • (reset class): Resets the class of selected list
    • [custom list styles in your project] All list styles will show here, click to apply the class to your selected list

Table Menu

To access:

  • Click the Window menu
  • Press [ALT] + A on the keyboard

Table Menu Items (Tooltips in parentheses)

  • Insert: Invokes the Insert submenu

    • Table (Run the Insert Table dialog): Invokes the Insert Table dialog.
    • Columns to the Left (Insert new column to the left of the selected column): Inserts a new column to the left of the selected column
    • Columns to the Right (Insert new column to the right of the selected column): Inserts a new column to the right of the selected column.
    • Rows Above (Insert new row above the selected row): Inserts a new row above the selected row.
    • Rows Below (Insert new row below the selected row): Inserts a new row below the selected row.

  • Delete: Invokes the Delete sub-menu

    • Table (Delete table): Deletes the entire table.
    • Columns (Delete selected column(s)): Deletes the selected columns in a table.
    • Rows (Delete selected row(s)): Deletes the selected rows in a table.

     

  • Select: Invokes the Select sub-menu

    • All Cells (Select all cells): Selects the entire table.
    • Table (Select table): Selects the entire table.
    • Column (Select column): Selects the column where the cursor is placed in a table.
    • Row (Select row): Selects the row where the cursor is placed in a table.
    • Cell (Select cell): Selects the cell where the cursor is placed in a table.

     

     

  • Clear Cells (Clear selected cells): Removes the text from the cells selected in a table.
  • Merge Cells (Merge selected cells): Turns the selected cells in a table into one big cell. You can merge any number of rows and columns in a table, as long as they are adjacent to one another.
  • Split Cells… (Split selected cells): Turns the selected cell in a table into many smaller cells by splitting the cell. When you split a cell, you are essentially inserting a smaller table into that cell.
  • Cell Content Style…(Select style for cell contents): Opens the Table Cell Content Style dialog, which lets you choose specific table or paragraph styles to use for the selected cells.
  • Reset Local Cell formatting: Resets the local cell formatting.

    • Note: This option resets formatting placed on the entire cell (e.g., a background color added to cells would be removed). However, local formatting on content (e.g., bold text) is not affected. If you would like to remove local formatting for cells as well as cell content, you can use the Remove Inline Formatting button instead.

  • Convert to Text: Converts the selected table to text.

  • Sort Rows: Sorts rows in either ascending or descending order.

    • Ascending (Sort the selected rows in ascending order): Sorts the selected rows in ascending order.
    • Descending (Sort the selected rows in descending order): Sorts the selected rows in descending order.

  • Caption: Invokes the Caption submenu

    • Default (Display table caption): Adds a placeholder for you to type a caption above the table.
    • Above Table (Display table caption above the table): Adds a placeholder for you to type a caption above the table.
    • Below Table (Display table caption below the table): Adds a placeholder for you to type a caption below the table.
    • None (Remove table caption): Removes a caption from the table.

  • Table Style: Invokes the Table Style sub-menu

    • Add Table Style... (Create a new Table Style file): Opens the Add File dialog with Table Style selected.
    • Edit Style (Edit the Style document): Opens the table style in the Table Style Editor so that you can edit styles in the style sheet.
    • Reset Table Style (Reset Table Style): Removes local formatting (e.g., color background shading) from cells in a table.
  • Table Properties (Run the Table Properties dialog): Invokes the Table Properties dialog, which lets you edit properties of the table (e.g., table size, caption, auto-fit behavior, style sheet).
  • Hide Gridlines (Toggle show gridlines): Shows and hides the gridlines in the table.
