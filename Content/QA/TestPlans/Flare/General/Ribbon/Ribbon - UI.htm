Ribbon - Ribbon UI

For the purposes of this test plan, verify all menu items function as they should and perform the Menu Test Cases for each menu. All menu items will be tested in more detail in the test plans of the modules which they support.

Accessing the Ribbon (If Ribbon is not enabled)

  1. Go to !Tools | Options
  2. In the "Interface" tab, select "Ribbon"
  3. Click OK

Home Menu

To access:

  • Click Home
  • Press ALT+H on the keyboard

 

Clipboard:

  • Paste: Pastes the selection from the clipboard to the place where the cursor is inserted.
  • Cut: Cuts the selection and sends it to the clipboard.
  • Copy: Copies the selection and sends it to the clipboard.
  • Delete: Deletes the selection.

 

Font:

  • Font: Lets you select a font type (e.g., Arial, Times New Roman), applying it to the content selected in the topic.
  • Font Size: Lets you select a font size, applying it to the content selected in the topic.
  • Bold: Applies bold typeface to the content selected in the topic.
  • Italic: Applies italic typeface to the content selected in the topic.
  • Underline: Underlines the content selected in the topic.
  • Superscript: Raises the selected content in the topic above the normal line of text.
  • Subscript: Lowers the selected content in the topic below the normal line of text.
  • Unformat: Removes local formatting in a selection or even an entire topic.
  • Font Color:Lets you choose and apply a color to the selected content.
  • Background Color: Lets you choose and apply a color to the background of the selected content.
  • Redact: Selecting this will cause selected content to be permanently eliminated from a printed or electronic document.
  • Font Properties: Opens the Font Properties dialog, which lets you edit the font properties for topic content directly.

 

Paragraph:

  • Bullet List: Lets you select one of several kinds of bullet or numbered lists, applying it to the selected content.
  • Outdent Items: Removes the group tag from a paragraph.
  • Indent Items: Lets you use a group tag such as <div> to indent content.
  • List Actions: Lets you apply miscellaneous actions to selected content in a list.
  • Group: Opens the Create Group dialog, which lets you create a group from selected content and apply one of the following tags to it: <blockquote>, <div>, <fieldset>, <form>.
  • Left: Aligns the selected content to the left.
  • Center: Aligns the selected content in the center.
  • Right: Aligns the selected content to the right.
  • Justify: Spaces the selected content so that both the left and right sides are aligned.
  • Default: Resets the alignment of the selected text to the default (inherited) setting.
  • Paragraph Properties: Opens the Paragraph Properties dialog, which lets you edit the paragraph properties for topic content directly.

 

Styles:

  • Style Window: Opens the Styles window, which displays styles contained in the stylesheets that are associated with the current topic.
  • Formatting Window: Opens the Formatting window, which displays style properties for the selected content in a topic.
  • Create Style Class: Opens the Create Style dialog, which lets you create a new style for the content selected in the topic.
  • Stylesheet Links: Opens the Stylesheet Links dialog, which lets you apply one or more stylesheets to the active topic.
  • Style: Lets you select a style from a style sheet, applying that style to the content selected in the topic.

 

Snippet:

  • Create Snippet: Opens the Create Snippet dialog, which lets you create a new snippet from the content selected in the topic.

 

Attributes:

  • Attributes Window: Opens the Attributes window, which is used to set attributes for a tag.
  • Conditions: Opens the Condition Tags dialog, which lets you apply condition tags to content in the topic.
  • Language: Opens the Select Language dialog, which lets you choose a language for the selected content. This affects things such as the spell check feature.
  • Name: Opens the Manage Named Elements dialog, which you can use to name content when creating togglers.

 

Editing:

  • Quick Find: Opens the Find widget, with the Replace field hidden
  • Quick Replace: Opens the Find and Replace widget, the Replace field selected
  • Find and Replace in Files: Opens a sub-menu, which lets you choose one of several "Find and Replace" options.
    • Find and Replace in Files (Ctrl + Shift + F): Opens the Find and Replace in Files window
    • Find Results 1: Opens the Find Results 1 window
    • Find Results 2: Opens the Find Results 2 window

 

Properties:

  • Properties: Opens the Properties dialog for the selected item in the Content Explorer or Project Organizer.

 

Insert Menu

To access:

  • Click Insert
  • Press ALT+I on the keyboard

 

Multimedia:

  • Image: Opens the Insert Image dialog, which lets you insert an image at the location of the cursor in the XML Editor.
  • Multimedia: Opens a sub-menu, which lets you select options for inserting multimedia (audio and video): Flash Movie, Windows Media Player, Quicktime Movie, Mimic Movie
  • Screen Capture: Opens the Screen Capture dialog, which lets you insert a screen capture image into the XML Editor.
  • QR Code: Opens the Insert QR Code dialog, which lets you create and insert a QR code into the XML Editor.
  • Slideshow: Inserts a Slideshow template into the XML Editor. Clicking this while the Slideshow is selected in the XML Editor will open the Edit Slideshow dialog.

 

Table:

  • Table: Clicking the button opens the Insert Table dialog which lets you insert a table into the XML Editor. Clicking the fly-out invokes a grid, which lets you insert a table with a selected number of rows and columns (up to 10 columns by 8 rows) into the XML Editor.

 

Links:

  • Hyperlink: Clicking the button opens the Insert Hyperlink dialog, which lets you insert a text hyperlink into the XML Editor. Clicking the fly-out gives you the options to insert a hyperlink which opens the Insert Hyperlink dialog or insert a Topic Popup which opens the Insert Topic Popup dialog.
  • Cross Reference: Opens the Insert Cross-Reference dialog, which lets you insert a cross-reference into the XML Editor.
  • Bookmark: Opens the Manage Bookmarks dialog, which lets you insert a bookmark into the XML Editor.
  • Named Destination: Opens the Named Destination dialog, where you can name and insert a named destination for use in Adobe PDF output. This allows you to point to a specific place in a PDF from the selected location.
  • Glossary Term Link: Opens the Create Glossary Term dialog, which lets you create a glossary term and definition, and insert a link to the definition into the XML Editor. In order for this option to be enabled, you first need to select text in the topic that you want to use as the glossary term link.
  • Keyword: Creates and inserts an index keyword at the location in your topic where text is selected. To enable this option, you must first select the text in the topic that you want to use as the index keyword.

 

Single Source:

  • Snippet: Opens the Insert Snippet Link dialog, which lets you insert a snippet into the XML Editor.
  • Variable: Opens the Variables dialog, which lets you insert a variable into the XML Editor.

 

Text:

  • Toggler: Opens the Insert Toggler dialog, which lets you insert a toggler into the XML Editor.
  • Drop-Down Text: Clicking the button opens the Insert Drop-Down dialog, which lets you insert a Drop-Down text into the XML Editor. Clicking the fly-out gives you the options to insert a Drop-Down Text which opens the Insert Drop-Down dialog, or a Drop-Down Hotspot (to insert a Hotspot, you must select text inside a Drop-Down Head).
  • Expanding Text: Opens the Insert Expanding Text dialog, which lets you insert expanding text into the XML Editor.
  • Text Popup: Opens the Insert Text Popup dialog, which lets you insert a text popup into the XML Editor.
  • Text Box: Opens the Insert Text Box dialog, which lets you insert a box into the XML Editor. A text box can be positioned in a variety of places on a page.
  • Footnote: Opens the Insert Footnote/Endnote dialog, which lets you insert a footnote or endnote. A footnote is typically added at the end of a page, but you can also insert an endnote at the end of a document, chapter, section, or book.
  • Rule: Opens the Insert Rule dialog, which lets you insert a rule (horizontal line) into the XML Editor.

 

Help Control:

  • Related Topics Control: Opens the Insert Related Topics Control dialog, which lets you insert a related topic link into the XML Editor.
  • Concept Link (A-link): Opens the Insert Concept Link Control dialog, which lets you insert a concept (See Also) link into the XML Editor.
  • Keyword Link Control (K-link): Opens the Insert Keyword Link Control dialog, which lets you insert a keyword link into the XML Editor.
  • Shortcut Control: Opens the Insert Shortcut Control dialog, which lets you insert a shortcut control into the XML Editor.

 

Symbols:

  • Equation: Opens the Equation Editor, which lets you insert a mathematical equation into the XML Editor.
  • Character: Clicking the button opens the Character dialog which lets you insert special characters or symbols into the XML Editor. Clicking the fly-out invokes a sub-menu that gives you more direct access to "Favorite" and "Recently Used" special characters and symbols to insert into the XML Editor.

 

Script:

  • Script: Opens the Insert Script dialog, which lets you insert a script into the XML Editor.

 

Proxy:

  • Proxy: Opens a sub-menu, which lets you select a proxy to insert into the XML Editor. A proxy is a placeholder for a certain type of content.
  • Page Header: Opens the Page Header dialog, which lets you insert a page header proxy into a Masterpage.
  • Page Footer: Opens the Page Footer dialog, which lets you insert a page footer proxy into a Masterpage.

 

View Menu

To access:

  • Click View
  • Press ALT+V on the keyboard

 

Explorer:

  • Content Explorer: Opens the Content Explorer, which is used to hold all of the content-related items in your project.
  • Project Organizer: Opens the Project Organizer, which is used to hold all of the project-related items in your project.
  • File List: Opens the File List, which lets you view all of the content files in your project in a grid view.

 

Common:

  • Start Page: Opens the Start Page, which provides links to high-level tasks and information.
  • Messages: Opens the Messages window, which displays important information as you work. For example, if a necessary file is missing, this window pane lets you know.
  • Backups: Opens the Backups window so that you can see the files for which you have created backups and restore (roll back) any of those files if necessary.

 

Reference:

  • Glossary Terms: Opens the Glossary Terms window, which you can use to see all of the glossary terms and definitions that you have added to the project.
  • Index Window: Opens the Index window, which is used to create index keywords and insert them into topics.
  • Concept Window: Opens the Concepts window, which is used to create concepts and insert them into topics.
  • Link Viewer: Opens the Link Viewer window, which is used to view other files linked to and from the active file.

 

Print Layout:

  • Footnotes: Opens the Footnotes window, which lets you see all of the footnotes that have been added in the active topic.
  • Page Frame Contents Pane: Opens the Frame Contents pane, which lets you add content to the selected frame in a page layout.

 

Resources:

  • External Resources: Opens the External Resources window, which lets you work with external resource files that you want to bring into your projects.
  • SharePoint Explorer: Opens the SharePoint Explorer, which lets you connect to a SharePoint server and work with files that you want to bring into your projects.
  • Contributions: Opens the Contributions window, which lets you see documents received from contributors that are pending acceptance. It also lets you see a list of contribution templates that you have created and send them to others.

 

Analysis:

  • Project Analysis: Opens a sub-menu, which lets you select various options to analyze your project. Each option opens the Project Analysis window, which then displays the appropriate data.

 

Info:

  • Info Bar: Hides or shows special information pertaining to the editor or window pane.
  • Status Bar: Opens or closes the status bar at the bottom of the program window.

 

Project Menu

To access:

  • Click Project
  • Press ALT+P on the keyboard

 

Content:

  • New: Clicking the button will open the Add File dialog with Topic pre-selected as the File Type. You can also click the fly-out and choose one of the other items. Each option will open the Add File dialog with that file type pre-select in the File Type field.
  • Import: Opens a sub-menu with options for importing documents, as well as files from another project (MS Word Documents, FrameMaker Documents, DITA Documents, Flare Project, HTML Files)

 

Build:

  • Build Primary: If you click on the button, output is generated for the target that is designated as the "primary" target. If you click the fly-out, you can select a specific target (not necessarily the primary target) to generate.
  • View Primary: Lets you view the output for the target that is designated as the "primary" target. If the target has not yet been generated, or if changes have been made to the project since it was last generated, you will be asked if you want to build the output. If you click the fly-out, you can select a specific target (not necessarily the primary target) to view.
  • Publish Primary: Opens the Publish Target dialog, which lets you publish the output for the "primary" target to its destinations. If the target has not yet been generated, or if changes have been made to the project since it was last generated, you will be asked if you want to build the output. If you click the fly-out, you can select a specific target (not necessarily the primary target) to publish.
  • Clean Project: Deletes the Output folder for the project and all output files inside. The next time you build a target, the Output folder will be created again to hold the new output files.
  • Open Output Folder: Opens the project Output folder in Windows so that you can see the output subfolders and files.

 

Primary Files:

  • Primary Target: Opens the primary target in the Target Editor.
  • Master TOC: Opens the master TOC in the TOC Editor.
  • Master Stylesheet: Opens the Stylesheet Editor and displays the master stylesheet specified in your primary target. If you have not specified a master stylesheet in that target, it opens the master stylesheet specified for the project. If you have not specified a master stylesheet for either of those, this option is disabled.
  • Master Page Layout: Opens the Page Layout Editor and displays the master page layout specified in your primary target. If you have not specified a master page layout in that target, it opens the master page layout specified for the project. If you have not specified a master page layout for either of those, it opens the first page layout listed in your project.

 

Save:

  • Export Project: Clicking the button opens the Export Project Wizard, which lets you export the entire project, or parts of the project based on targets/conditions/file tags/selected files to an output project file, project template, or zip file.
  • Zip Project: Clicking the button opens the Create Package dialog which lets you "pack" a copy of your project to a single zip file, which makes it easier to move it and send it to other individuals who have Flare installed. Clicking the fly-out invokes a sub-menu, which lets you simply zip your project or Zip and Email your project. The Zip and Email option will also open the Create Package dialog, but will then follow the zipping process by launching Outlook and attaching the zipped project to an outgoing email message.
  • Save Project As Template: Opens the Save Project as Template Wizard, which lets you save the project and its files in another location as a template. When you create a new project in the future, you can select the template project as a starting place for the new project.

 

External Resources:

  • Synchronize Files: Opens the Synchronize Files dialog, which lets you synchronize files in your project that are linked to source files elsewhere.
  • External Resources: Opens the External Resources window, which lets you work with external resource files that you want to bring into your projects.
  • SharePoint Explorer: Opens the SharePoint Explorer, which lets you connect to a SharePoint server and work with files that you want to bring into your projects.
  • Manage Mappings: Opens the Map Files and Folders dialog, which lets you create and edit mappings for files in your project that are linked to source files elsewhere.

 

Locate:

  • Locate in TOC: Opens the TOC Editor and highlights the location of the active topic in the TOC. If the topic is contained within multiple TOCs, a dialog will appear letting you select which TOC to open.
  • Locate in Explorer: Opens the Content Explorer and highlights the location of the active content file.

 

Properties:

  • Project Properties: Opens the Project Properties dialog

Review Menu

To access:

  • Click Review
  • Press ALT+R on the keyboard

 

Packages:

  • Send For Review: Opens the Send Topic for Review Wizard, which lets you specify settings and send topics to others so that they can review it.
  • Topic Reviews: Opens the Topic Reviews window pane, which lets you see topic files that have been reviewed from others and are pending acceptance. It also lets you see a list of topics that you have sent out for review.
  • Import Review Package: Opens an Open File dialog, which lets you import reviewed topics (*.fltrev) into your project. After that, you can view and/or accept the topic. Only packages created from the current project can be imported into the current project.
  • Review Packages: Opens the Review Packages window, which displays all of the topic files contained in a review package received from another Flare author. From this window pane you can open the topics to review and edit them.

 

Changes:

  • Track Changes: Enables or disables the track changes feature.
  • Accept Change: Accepts tracked changes or annotations in the active document, making them part of that document. You can click the down arrow to accept a single change or all of them.
  • Reject Change: Rejects tracked changes or annotations in the active document, removing them from that document. You can click the down arrow to reject a single change or all of them.
  • Show Changes: Opens a sidebar and displays any tracked changes or annotations (notes) that have been attached to any content in the topic.
  • Previous Change: Navigates to the previous change or annotation.
  • Next Change: Navigates to the next change or annotation.

 

Annotations:

  • Insert Annotation: Adds an annotation rectangle, extended to a sidebar to the right, with a line pointing to the selected content. This rectangle will hold annotation text after you type it. Also, the selected content is shaded, indicating that the annotation refers to that text. However, the shading will not be displayed in the output; it is for internal use only.
  • Delete Annotation: Removes annotations from the document. You can click the down arrow to delete a single annotation or all of them.

 

Locks:

  • Lock: Opens a sub-menu, which lets you select options for locking content in topics so that it cannot be edited.

 

Options:

  • Review Options: Opens the Review tab in the Options dialog, which lets you set various options for viewing tracked changes and annotations.

Tools Menu

To access:

  • Click Tools
  • Press ALT+T on the keyboard

 

Spell Check:

  • Spell Check Window: Opens the Spell Check window, which lets you perform spell checking manually.
  • Spell Check While Typing: Activates a feature that lets you check your spelling as you type in your topics. Questionable words are underlined with a jagged red line. You can take action with the word by right-clicking on it.
  • Ignored Words: Opens the Ignored Words dialog, which lets you create a list of words to be ignored when using the spell checking feature in the application.

 

Hyphenation:

  • Hyphenation Window: Opens the Hyphenation window, which lets you add custom hyphenation rules for words so that they are hyphenated at the end of lines after specific letters.

 

Templates:

  • Manage Language Skins: Opens the Language Skins dialog, which lets you localize (associate a language with) skins for some outputs (e.g. WebHelp outputs and in some cases, HTML Help)
  • Manage Templates: Opens the Template Manager, where you can perform a variety of tasks to manage your template files.

 

Context Sensitive Help:

  • Test CSH API Calls: Opens the Context-Sensitive Help API Tester dialog, which lets you test the context-sensitive Help links that you've created in the Alias Editor.
  • Export Header File(s): Opens the Export Header Files dialog, which lets you export existing header files in your project to another location. You can export header files to the following formats: C/C++ (.h), Visual Basic (.bas), Java (.properties), Delphi Pascal (.inc).

 

Update:

  • Update Cross-References: Opens the Update Cross-References dialog, which lets you update the cross-references that you have added to topics. By updating the cross-references, you can ensure that they reflect any changes in your project that might affect them.
  • Regenerate Captured Images: Regenerates (updates) all of the images in your project. The primary reasons for using this option are to remove watermarks and to modify PROPS files.

 

Launch:

  • MadCap: Opens a submenu, which lets you open various integrated MadCap Software products: MadCap Capture, MadCap Mimic, MadCap Feedback Explorer, MadCap Analyzer.

Source Control Menu

To access:

  • Click the Source Control menu
  • Press[ALT] + S on the keyboard

 

Changes:

  • Pending Check-Ins: Opens the Pending Check-Ins window pane, which displays files in the project that are pending check in. In other words, you can use this window pane to see which files you have checked out, which files have been created but not yet added to source control, which files other users have checked out, and which files are out of date.

 

File:

  • Add: Opens the Check In dialog, which lets you add the selected files to the integrated source control application.
  • Check In: Opens the Check In dialog, which lets you check files into source control.
  • Check Out: Opens the Check Out dialog, which lets you check out files from source control.
  • Get Latest Version: Opens the Get Latest Version dialog, which lets you "get" files from source control. After you bind a Flare project to a source control application, you can get the latest version of any of the source control files. When you do this, you are copying the most current files stored in the source control application to your local Flare project without necessarily checking out the files.
  • Undo Check Out: Opens the Undo Check Out dialog, which lets you reverse the check-out of the files. If you have files checked out from source control but do not want them checked out anymore, you can use the "Undo Check Out" option instead of checking in the files.
  • Show Differences: Opens a window pane that lets you view the content and code differences for files. If you select a single file and choose this option, the differences between the local version of the file and the source control version of the file are shown. If you select two files and choose this option, the differences between the two files are shown.
  • View History: Opens the History dialog, which lets you view the history of a particular source control file, including all of the occasions when the file has been checked in. You can then use the dialog to view the differences between older versions of the file or roll back to a particular version.
  • Undelete: Undeletes a file, allowing you to add it to source control.
  • Refresh: Refreshes the window pane so that the latest information is shown.

 

Project:

  • Check In All: Opens the Check Out dialog, which lets you check out all files in the project from source control.
  • Check Out All: Opens the Check Out dialog, which lets you check out all files in the project from source control.
  • Get Latest Version All: Opens the Get Latest Version dialog, which lets you "get" all files in the project from source control.
  • Undo Check Out All: Opens the Undo Check Out dialog, which lets you reverse the check-out of all files in the project.

 

Instant Messaging:

  • Instant Messages: Opens the Instant Messages window pane, which displays instant messages from other users and lets you send and configure instant messages.
  • Request Check In: Opens the Send Request Check-In Message Dialog, which lets you ask another user to check in the selected files so that you can edit them.
  • Request View File: Opens the View File Request dialog, which lets you ask another user to view the selected file.

Table Menu

To access:

  • Click the Table menu
  • Press[ALT] + B on the keyboard

 

Table:

  • Insert Table: If you click on the face of the button, the Insert Table dialog opens, which lets you insert a table and specify various settings for it. If you click the down arrow, you can use a grid to select the number of rows and columns for a simple table and insert it.
  • Select Table: Opens a submenu, which lets you choose options for selecting a table, column, row or cell.
  • Show Gridlines: Shows and hides the gridlines in the table.
  • Table Properties: Opens the Table Properties dialog, which lets you edit properties of the table (e.g. table size, caption, auto-fit behavior, stylesheet).

 

Rows & Columns:

  • Delete Table: Opens a submenu, which lets you select options for deleting a table, columns or rows.
  • Rows Above: Inserts a new row in a table. The new row is inserted above the area where the cursor is inserted.
  • Rows Below: Inserts a new row in a table. The new row is inserted below the area where the cursor is inserted.
  • Columns to the Left: Inserts a new column in a table. The new column is inserted to the left of the area where the cursor is inserted.
  • Columns to the Right: Inserts a new column in a table. The new column is inserted to the right of the area where the cursor is inserted.

 

Cells:

  • Merge Cells: Turns the selected cells in a table into one big cell. You can merge any number of rows and columns in a table, as long as they are adjacent to one another.
  • Split Cells: Turns the selected cell in a table into many smaller cells by splitting the cell. When you split a cell, you are essentially inserting a smaller table into that cell.
  • Clear Cells: Removes the text from the cells selected in a table.
  • Cell Content Style: Opens the Table Cell Content Style dialog, which lets you choose specific table or paragraph styles to user for the selected cells.
  • Reset Local Cell Formatting: Removes local cell formatting in a table. The local formatting is removed from the entire table or from selected cells.

 

Style:

  • Apply Table Style: Opens a drop-down menu where you can select an existing table style to apply to selected content.
  • New Table Style: Opens the Add File dialog, where the Table Style file type is already selected. After completing the dialog, the TableStyle Editor opens.
  • Edit Table Style: Opens the table style in the Table Style Editor so that you can edit styles in the stylesheet.
  • Reset Table Style: Applies the table style to the table again.

 

Data:

  • Sort Rows: Opens a submenu, which lets you select options for sorting table rows (Ascending, Descending).
  • Convert to Text: Converts a table to text. This option is enabled only if you have placed the cursor within an existing table.

 

Caption:

  • Caption: Opens a submenu, which lets you select options for inserting or removing a table caption (Default, Above Table, Below Table, None).

Window Menu

To access:

  • Click the Window menu
  • Press[ALT] + H on the keyboard

 

Open Windows:

  • Window: Opens the Manage Windows dialog, which lists the files that are currently open in your project and lets you activate (i.e. switch to), save, or close a particular file.
  • Close All Documents: Closes open files. You can click the down arrow and select one of the following (Close All Documents, Close All Documents Except This One).

 

Active Pane:

  • Dock: If you select the Dock option when a window pane is active, the Drag Pane area is shown or hidden in the window pane accordingly. The Drag Pane area lets you move the window pane and dock it elsewhere in the program window.
  • Auto Hide: "Pins" the active window pane or editor to the side of the interface so that you have more room for your workspace. You can show the window pane by hovering over the thin bar on the side of the interface. If you select the pin button at the top-right of the window pane, the window pane returns to its previous state.
  • Float: Converts the active window pane or editor to a floating window, which means it is no longer attached to the interface.

 

Window Layout:

  • Layout: Opens a submenu that lists the layouts in your project. Select a layout to display it in the interface.
  • Save Layout: Saves the window layout. You can click the down arrow and select one of the following: Save Window Layout, Save Window Layout As.
  • Reload Layout: If you are working in a particular layout and have opened different interface elements or moved interface elements around, you can select this option to return to the saved configuration of the layout.
  • Reset Layout: Returns the current configuration of the interface to the factory layout settings (the original layout when you first installed and launched Flare).
  • Auto-save Layout: Automatically saves the layout of the workspace when you exit Flare. The next time you launch Flare, that same layout will be displayed.

Help Menu

To access:

  • Click the Help menu
  • Press[ALT] + H on the keyboard

 

Topics:

  • Dynamic Help: Opens the Dynamic Help window pane, which automatically displays the Help topic related to the active element in the interface.
  • Help Topic: Opens Help topics. You can click the down arrow and select one of the following: Help Topic, New Help Topic Window.

 

Contents:

  • Contents: Opens the Help TOC window pane, which lets you locate and access topics in Flare's online help.
  • Locate in TOC: Lets you manually synchronize the current Help topic with the Help TOC. In other words, the TOC book containing that topic also opens to show you where the topic resides in the TOC hierarchy.
  • Auto-Sync TOC: Automatically synchronizes the current Help topic with the Help TOC. In other words, when a particular Help topic is displayed, the TOC book containing that topic also opens to show you where the topic resides in the TOC hierarchy.

 

Panes:

  • Index: Opens the index. You can click the down arrow and select one of the following: Index, Index Results.
  • Search: Opens the Help Search window pane, which lets you search for information in Flare's online help.
  • Favorites: Opens the Help Favorites window pane, which lets you quickly access your favorite Flare Help topics.
  • Glossary: Opens the Help Glossary window pane, which lets you view glossary terms and definitions in Flare's online Help.

 

Tutorials:

  • PDF Guides: Opens a submenu, which lets you access one of the printed guides related to Flare.
  • Video Tutorials: Opens a topic that provides access to video tutorials in the online Help.

 

Community:

  • Help Community: Opens the MadCap Forums website, where you can obtain additional information about Flare and communicate with other Flare users.

 

Product:

  • Activate Product: Opens the Activate Product Wizard, allowing you to activate the product.
  • Deactivate Product: Deactivates the currently used license for the machine running Flare.
  • Register Product: Lets you register your version of Flare if you have not done so already.
  • Check for Updates: Opens the New Updates Available dialog, which lets you download Flare updates from MadCap.
  • About: Displays a popup that shows information about the current version of Flare. Click the popup to make it disappear.
  • Report a bug: Opens your default internet browser to the MadCap Software bug submission web page, where you can report the bug.
