Every Icon in each of the Ribbon's (including items in the file drop down) shall be able to be placed on the Quick Access Toolbar and have the same functionally as the icon on the ribbon. For the Quick Access Toolbar Do the following :

1. A control can be added to the QAT and the Icon for the control shall appear on the QAT and must look reasonably similar to the original icon .

2. A control must perform the same action when added to the QAT as it would when it is present in the Ribbon.

3. When Flare is closed and re-opened, the previously added control shall remain in the QAT. 

4. The control shall work when invoked from the QAT after Flare is closed and re-opened
