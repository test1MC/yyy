Blaze

Bug Base Product Area:

  • Blaze

 

Description:

Blaze is a stripped down version of Flare that focuses on the Print targets exclusively. Most issues with Blaze will also be an issue with Flare and will be tested with each specific module during Flare testing. However there have been issues that were Blaze specific. That said, all the bugs identified in this Test Plan should ALSO BE VERIFIED IN FLARE, where appropriate. We need to make sure every bug fixed in Blaze is still fixed in Flare.
