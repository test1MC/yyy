Eclipse Help Target Editor: General Tab

UI Controls

 

  • Output Type: Sets the output type for this Target.
  • Comment: Lets you enter a comment for the Target.
  • Master TOC: Select a TOC file to be the Master TOC for this target output.
    • : Opens the currently selected TOC in the TOC Editor.
  • Master Stylesheet: Select a Stylesheet to be used for the target.
    • : Opens the currently selected Stylesheet in the Stylesheet Editor.
  • Output Folder: Lets you set the folder of the output file for this target.
    • : Opens a window which lets you browse for a folder to set as the Output Folder.
  • Automatically get latest version... : If user has bound project to a source control program, user can select this check box to automatically get the latest version of all files prior to generating the target. User might use this option if users are working with a team of authors and want to make sure that user include the latest changes from other writers in the output (without having to manually get those files).
  • Disable auto-sync... : Lets you disable the Auto-sync functionality if your project is setup to re-import every time you compile.

Eclipse Help Target Editor: Advanced Tab

UI Controls

  • Output Options:
    • Insert Mark of the Web: Select this check box if you want to add Mark of the Web (MOTW) when generating WebHelp output.
    • Do not use "Content" folder in output: Omits the “Content” folder from the generated output.
    • Generate resized copies of scaled images: If enabled, Flare will create new copies of images wherever you have specified resizing.
    • Generate "web-safe" images: Converts images to web-safe formats (e.g., GIF, JPEG, PNG) when you generate online output.
    • Replace reserved characters with underscores in filenames: Converts spaces or unusual characters in your file names or folders to underscores in the output.
    • Use custom file extension for topics: Lets you set a specific file extension for topics in the output.
    • Use lowercase filenames: Converts output file names and folders (and links pointing to the files) to all lowercase letters.
    • Exclude content not linked directly or indirectly from the target: Use this option to control which content is included in the output, based on direct and indirect links in the target.
    • Use empty ALT text for images that do not have ALT text: Use to add alternative text to images for 508 compliance.
    • Auto-size Capture Objects to Fit Text:Use to ensure Capture text is displayed appropriately.
    • Add meta tags to content: Insert meta tags into EclipseHelp output files
      • Current Default: <meta http-equiv="X-UA-Compatible" content="IE=edge" />
  • Master Page: Select a master page if you want to associate one with the target.
    • : Opens the currently selected Master Page in the XML Editor.
  • Stylesheet Medium: Select a style sheet medium type if you want to associate one with the target.
  • Alias File: If you are incorporating context-sensitive Help into your online Help, select an alias file to associate it with the target.
    • : Opens the currently selected Alias file in the Alias Editor.

Eclipse Help Target Editor: Eclipse Options Tab

UI Controls

  • Plug-in Options
    • Provider Name: Lets you set the Provider Name for the Eclipse Help system.
    • Plug-in Name: Lets you set the Plug-In name for the Eclipse Help system.
    • Plug-in ID: Lets you set the Plug-In ID for the Eclipse Help system.
    • Plug-in Version: Lets you set the Plug-In version for the Eclipse Help system.
    • Help System Name: Allow the user to name the help system
      • Use Label: Select this option to provide a label for the Eclipse TOC entry. If this option is selected and the field is left empty, the target filename will be used as the label.
      • Use Topic: Select this option to provide a topic link for the Master Book. When the Master Book is viewed in the Eclipse Help Viewer, the selected topic will display.

        • [Browse]: Invokes the Link to Topic dialog which lets you select a topic file in the current project to be the Master Book Topic.

    • Generate Index: This option should be enabled by default. Enable this option to include keywords from the Flare project in the Index which appears in the Eclipse Help Viewer. Disable the option to exclude project keywords from the Eclipse Help Viewer's Index.
    • Archive to JAR: Enable this option to generate Eclipse output to a single *.jar file as opposed to a set of files.
  • Viewing Options:
    • Eclipse Location: Lets the user provide the path to the local instance of Eclipse that should be used when viewing output. A local instance of Eclipse is required to view output, however it is not required to build output.
      • [Browse]: Invokes a Windows Open dialog which lets you browse for a local instance of Eclipse.exe
      • Download: Clicking this should take you to a redirect page on the MadCap Software website which will send the user to a place where they can download Eclipse.
