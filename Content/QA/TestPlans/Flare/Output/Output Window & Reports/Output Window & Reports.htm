Output Window & Reports

Custom Compiler Warnings context menu needs to be added to this Test Plan

 

Bug Base Product Area:

  • Output: General

 

Description:

When Flare generates output, the messages and errors are displayed in the Output window (aka Build progress dialog). If errors are generated the user has the ability to save the information as a report file (*.mclog) that can be opened in the Log Viewer.

Prior to the release of Analyzer, Log files were commonly referred to as "Reports" so it is important to recognize the difference between the Output Log files addressed in this test plan (*.mclog) and Report files created from analyzing your project (*.flrep). Any issues involving *.flrep files should not be added to this test plan.
