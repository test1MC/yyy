Log Files - UI

Log Files are saved to the Project/Reports folder.

 

Log file right-click menu:

 

  • Menu items are correct
  • Spelling and Grammar are correct
  • No duplicate accelerators
  • All accelerators function
  • Keyboard can navigate and select options
  • Menu is easy to use and understand

 

Right-click menu items (Tooltips in parentheses):

  • Open (Open): Opens the selected Project file in the MDI.
  • Open with: Gives options to open the Project file in the Internal Text Editor, Internal Browser or another application.
  • Delete (Delete): Removes the selected Project file from the project.
  • Rename (Rename): Allows you to rename the Project file.

Opening a Log file:

Double-click the file in the Project Organizer

Right-click the file in the Project Organizer and select Open

Select the file in the Project Organizer and press <Enter>

UI Controls

  • Description: Displays a description of the log file.
  • Date: Displays the date and time that the log file was created.
  • Messages tab: Displays messages from the generation of the target that was built.

  • Errors tab: Displays errors that were encountered during the build process.

    Columns

    • Description: A description of each error is given.
    • Subsystem: The Subsystem where the error occurred.
    • File: The file which caused the error.
    • Line: The line number in the file where the error occurred.
    • Char: The character number (within the line) where the error occurred.
