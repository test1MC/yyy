Output DITA - Output Files

The following is an outline of all files and directories that should be generated when building a DITA output.

Root-Level Files

  • Ditamap.ditamap

Content Folder

This Content folder should contain all of the topic files. The folder structure in this Content folder should mirror the folder structure of your topics in the Content Explorer.

Resources Folder

  • Glossary.dita
  • Images folder (if Images are used)
  • Snippets folder (if snippets are used)
