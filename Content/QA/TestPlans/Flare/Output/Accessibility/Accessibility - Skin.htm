Accessibility - Skin

Styles

  • There are several skin elements that can have their tooltip and alternate text set. There are defaults for many of these to be 508 compliant. Test to make sure that setting each of the following output correctly. Also, check a few of these in other languages to make sure they are being written when the project language or target language is set to something other than English.
