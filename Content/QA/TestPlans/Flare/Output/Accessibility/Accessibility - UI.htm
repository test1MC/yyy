Accessibility - UI

Images

Right-click an image and select Properties | General Tab

  • Screen Tip:Set the screen tip of an image
  • Alternate Text: Set alternate text for an image
  • Apply the alternate text and screen tip to all image references:Apply the alternate text and screen tip of this image to all other instances of this image in the project
Sample XML

<img src="../../Resources/Images/80s1337ness.jpg" style="width: 193px;height: 190px;" alt="Alternate Text" title="Screen Tip" />

Tables

Right-click a table and select Properties | General Tab

  • Text:Set the caption text of a table (Note: no matter what the position of the caption is relative to the table, the caption markup should be right after the table tag and before the column tags to be 508 Compliant.)
    • Side: Above Table:Places the caption at the top of the table
    • Side: Below Table:Places the caption at the bottom of the table
    • Continuation:Caption text appears at the top of each table for tables that span multiple pages
  • Table Summary:Set the summary of the table. The summary is not a visible attribute, however it is read by a screen reader.
  • Header Element:You can add a header row in the table properties dialog, or you can make any td into a th using the internal text editor.
Sample XML

<table style="width: 100%;caption-side: top;" summary="Summary">

<caption>Caption</caption>

<col />

<col />

<thead>

<tr>

<th>Header Row</th>

<th>&#160;</th>

</tr>

</thead>

<tbody>

<tr>

<td>&#160;</td>

<td>&#160;</td>

</tr>

<tr>

<th>Header Element</th>

<td>&#160;</td>

</tr>

</tbody>

</table>

Equations

Right-click an equation and select Properties | General Tab

  • Screen Tip:Set the screen tip of the equation
  • Alternate Text:Set the alternate text of the equation
Sample XML

<MadCap:equation xmlns:dsi="http://www.w3.org/1998/Math/MathML" alt="Alternate Text" title="Screen Tip">

QR Codes

Right-click a QR Code and select Properties | General Tab

  • Screen Tip:Set the screen tip of the QR Code
  • Alternate Text:Set the alternate text of the QR Code

Image Maps (Area Elements)

Right-click a Image Map and select Image Map | Select or draw an area element | Right-click the area element in the Image Map Editor and select Properties | Link Properties

  • Screen Tip:Sets the alternate text of the selected area element
Sample XML

<MadCap:qrCode alt="Alternate text" title="Screen Tip">QR Code</MadCap:qrCode>

Forms and Frames

Currently, you cannot set any of the required 508 compliant attributes for these objects in our UI. However, you can set them with the internal text editor.

Should look something like this:

Sample XML

Iframe and frame with title or name:

 

<iframe title="iframe"></iframe>

 

<frame name="frame"></frame>

 

A few examples of form elements with labels:

 

<form>

<input type="button" value="Button 1" />

<input type="image" src="../../Resources/Images/CustomIcon.jpg" alt="Next page" />

<button>Go <img src="../../Resources/Images/CustomIcon.jpg" alt="Next page" /></button>

<label>First Name:

<input type="text" /></label>

<label for="LastName">Last Name:</label>

<input id="LastName" type="text" />

<label>Option a:

<input type="radio" name="RadioGroup" value="a" /></label>

<label for="OptionB">Last Name:</label>

<input id="OptionB" type="radio" value="b" />

<label>Option c:

<input type="checkbox" value="c" /></label>

<label>Password:

<input type="password" /></label>

</form>
