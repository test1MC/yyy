Accessibility - Notes

General Notes (Remove Bugs When Fixed)

  • input type="button" does not need a label

    input type="image" does not need a label but does need alt text

  • Accessibility - Output is an ongoing list of things that may be overlooked while testing
  • There is an analyzer report for accessibility but that should be covered in the analyzer test plans.
  • There is a filter on the right side of the bug base thats called 'Improved 508 Support' (its next to the fogbugz number).

  • Alt text of images and links are not written to the output for word and frame bug 42043

  • There are some issues with the screen reader working in Chrome.
  • JAWS requires a lot of setup to get it to do what you want.
  • PDF's will only be tagged on output when the option 'Generate Tagged PDF' under PDF options in the target.
  • Use Inspect Element when trying verify things like checking Alt text on elements in the output. Inspect Element is now in the context menus for current versions of all three major browsers (IE 11 (new for IE), FF 27, and Chrome 33)
  • Make sure to use the Tab key and arrow keys to navigate output while JAWS is running
  • Make sure to test hitting Enter key and Spacebar key while using JAWS to see if they open and close things such as popups. (Currently there are issues in HTML5 that are bugged)
  • With v10, we added the ability to HTML5 to be more 508 compliant
  • With v10, we added a transparent.gif behind the image icons for glossaries, dropdowns, expanding text, togglers and help controls, so that Alt text can be added to these nav links and so there open/close states can be heard by screen readers. The alt text can be modified in the stylesheet and the Language skin (currently...there is a bug to move the HTML5 version of that into the UI Text section of the HTML5 skin to be more consistent)
  • In addition to that, they are now wrapped in <a> tags in order for user to tab to them. We ran into alot of issues where they were then inheriting from <a>. We cleaned that up quite a bit, but keep an eye out for anything that maybe different than v912.
