  • Output: WebHelp

 

Description:

WebHelp is a Web-based Help format that can run on almost any browser or platform. Use WebHelp to create online documentation for the Internet or an intranet, as well as for desktop applications.

 

The WebHelp output consists of a collection of files that you will distribute to users. The output will be displayed in the user's Internet browser window. The main entry file has an .htm extension.

 

WebHelp is a good choice if:

  • You are writing online documentation for distribution on the Internet or an intranet.
  • You want to be able to produce an output interface in various languages. This is possible through the use of language skins.
  • You want the flexibility to include the online documentation in a desktop application.
  • Your users have different Internet browsers on their systems.
  • Your users are working on different platforms.
