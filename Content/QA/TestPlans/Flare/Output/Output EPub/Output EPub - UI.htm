UI Controls

  • MetaData
    • Title: Enter the title for the EPUB document. This information is included in the final EPUB output (e.g., shown in the Contents tab and title bar).
    • Author: Enter the name of the author. This information is included in the final EPUB output (e.g., shown in the Contents tab).
    • Publisher(s): Enter the publisher information. Where this displays in the output is dependent on the EPUB viewer.
    • Subject: Enter the subject information. Where this displays in the output is dependent on the EPUB viewer.
    • Genre: Enter the genre information. Where this displays in the output is dependent on the EPUB viewer.
    • Description: Enter the description information. Where this displays in the output is dependent on the EPUB viewer.
    • Rights: Enter the rights information. Where this displays in the output is dependent on the EPUB viewer.
    • ISBN: Enter the ISBN information. Where this displays in the output is dependent on the EPUB viewer.
    • Publish Date: Enter the publish date information. Where this displays in the output is dependent on the EPUB viewer.
    • Identifier: Enter the identifier information. Where this displays in the output is dependent on the EPUB viewer. If you leave this field blank, a randomly generated ID will be used.
    • Coverage: Enter the coverage information. Where this displays in the output is dependent on the EPUB viewer.
    • Related To: Enter the related information. Where this displays in the output is dependent on the EPUB viewer.
    • Based On: Enter the based on information. Where this displays in the output is dependent on the EPUB viewer.
  • Cover Page
    • Cover Page Image Preview: Displays a preview of the currently selected cover page image. The cover page will display when viewing the final EPUB output in an EPUB viewer library.
    • Image Location: Displays the relative path of the image file within the project ("Content\...").
    • :  Invokes the Insert Image dialog which lets you select an image file that exists within the current project or browse for an external image. If an external image is selected, that image will be copied to the project.
  • Validate Epub3 output
    • Enable Epub 3 validation: Select this option to enable the EPUB check during compile. If this option is enabled and the path set below is empty or invalid, you will get the following compile warning "Error: Invalid epub checker...validation aborted!"
    • EpubCheck Path: Displays the absolute path of your EpubCheck file. The path is written to HKEY_CURRENT_USER|Software|MadCap Software|Shared|ExternalTools in the Windows Registry. Therefore, you should only need to set this for one target and it should appear in all EPUB targets in all projects. Setting a path here or changing the existing path should not dirty the target file.
    • : Invokes a Windows browser which lets you browse for an EpubCheck file (*.jar)
    • Download EpubCheck: Launches your web browser and opens www.madcapsoftware.com/downloads/epubcheck.aspx which will automatically forward the user to a site where they can download EpubCheck (currently https://github.com/IDPF/epubcheck/releases)
  • Generate MOBI output
    • Generate MOBI file: Select this option to generate a MOBI file (for Amazon Kindle) during compile. If this option is enabled and the path set below is empty or invalid, you will get the following compile warning "Error: Invalid KindleGen path. Mobi file generation could not be completed."
    • KindleGen Path: Displays the absolute path of your KindleGen executable file. The path is written to HKEY_CURRENT_USER|Software|MadCap Software|Shared|ExternalTools in the Windows Registry. Therefore, you should only need to set this for one target and it should appear in all EPUB targets in all projects. Setting a path here or changing the existing path should not dirty the target file.
    • : Invokes a Windows browser which lets you browse for an KindleGen application file (*.exe)
    • Download KindleGen: Launches your web browser and opens www.madcapsoftware.com/downloads/kindlegen.aspx which will automatically forward the user to a site where they can download EpubCheck (currently http://www.amazon.com/gp/feature.html?ie=UTF8&docId=1000765211)
  • Optimization Settings
    • Enable MathML to PNG conversion: Select this option to convert equations to PNG image files during compile. Leaving this option deselected with preserve MathML in the EPUB output. Not all EPUB readers will support MathML.
    • Embed Fonts: Select to include all of your used fonts in the final EPUB output file. Therefore, if a reader that does not have your fonts installed, your content will still be displayed as you designed it.  However, this is not fully supported in all EPUB viewers.
