Publishing/Destinations

Bug Base Product Area:

  • Publishing/Destinations

 

Description:

"Publishing" in the context of Flare is the process of copying output files and placing them in a location where others can access them, such as on a network or a website. Publishing settings are stored in a Destination file (*.fldes).
