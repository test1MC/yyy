Header File - Editor & Exporting

Opening a Header File

Header Files in the Text Editor

Header files should be edited via the Alias Editor. However, if you do open a header file using any of the above methods, it will open in the Internal Text Editor.

Test Cases

  • Verify that edits made to a header file in the Internal Text Editor can be saved and changes are reflected in the Alias Editor.

Header File Export

The Export Header file dialog can be opened by:

  • Project Organizer: Right-click a header file and select Export Header File(s)…
  • Tool Strip interface: !Build | Export Header File(s)…
  • Ribbon interface: !Tools | Export Header File(s)

Export Header File - UI Test Cases

UI Controls

  • C/C++ (.h): If this option is selected, a copy of the header file will be created with an .h file extension.
  • Visual Basic (.bas): If this option is selected, a copy of the header file will be created with a .bas file extension.
  • Java (.properties): If this option is selected, a copy of the header file will be created with a .properties file extension.
  • Delphi Pascal (.inc): If this option is selected, a copy of the header file will be created with an .inc file extension.
  • Header File Grid: Check the box next to the header files you want to include in the export.
  • Export Files To: 
    • Drop-down: The drop-down will provide paths to which you have previously exported header files
    • Browse… : Invokes the Browse For Folder dialog which lets you select a folder to which the header files will export
  • Export: Closes the dialog and exports the selected header files
  • Cancel: Closes the dialog and does not export the selected header files
