Testing CSH - CHM

This document will allow testing of CSH calls to a CHM project with minimal CSH knowledge needed. The only method for testing CSH calls is using the internal CSH tester API.

Test Files

Copy down these test files locally for testing.

  • CSH_Testing Flare Project :
    • \\Dc2\qa\Test Projects\Test Plan Projects\CSH\CHM CSH Testing\CSH_Testing\
  • CHM CSH tester application:
    • \\Dc2\qa\Test Projects\Test Plan Projects\CSH\CHM CSH Testing\WindowsApplication1.exe

Compile and Testing

  1. Open the project in Flare
  2. Generate HTML Help (CHM)
  3. !Build | Test CSH API Calls
  4. The API tester should load. Clicking the button will launch the CHM, focused on the Help topic specified in the Topic column of the CSH API tester.
  5. Click each button and verify the correct topic is shown
  6. Change the skin selection for any of the IDs to the HideNav skin. Pressing for that ID will open the correct topic, but with the navigation hidden.
