Testing CSH - WebHelp

This document will allow testing of CSH calls to a WebHelp project with minimal CSH knowledge needed.

Test Files

Copy down these test files locally for testing.

  • CSH_Testing Flare Project :
    • \\Dc2\qa\Test Projects\Test Plan Projects\CSH\CSH Project\CSH_Testing\
  • CSH_Calls.htm file :
    • \\Dc2\qa\Test Projects\Test Plan Projects\CSH\Web CSH Testing\

Compile and Testing

  1. Open the project in Flare
  2. Generate WebHelp
  3. Copy the CSH_Calls.htm file from above into the root webhelp output folder (where Default.htm is located)
  4. Open the CSH_Calls.htm file in IE, FireFox and Chrome*
  5. Click on each link, comparing each output to the Expected Result: section of each link
  6. All links are currently working as advertised. Any link that does not open, in the manner described, is a bug

 

*Chrome support is currently very limited. Verify with the QA lead what results should be expected.
