Output Frame

Bug Base Product Area:

  • Output: Frame

 

Description:

FrameMaker is an output type where the generated project is exported to Adobe FrameMaker in one of the following formats: BOOK, FM, PDF. However, you can also create PDF output directly, without going through FrameMaker.

 

Unless you specify otherwise, the FrameMaker target will create BOOK or FM files only. If you want to create a PDF file from your Flare project.

 

Note: Flare supports FrameMaker 7.0 and newer versions.

Note: If you have multiple version of FrameMaker on your machine, Flare will use whatever FrameMaker was last installed when creating the output.
