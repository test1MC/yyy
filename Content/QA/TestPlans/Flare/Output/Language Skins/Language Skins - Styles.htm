Formats Styles

Verify all customized Language Skin settings appear in HTML5, HTML Help, DotNet Help, WebHelp, WebHelp Mobile, WebHelp AIR, and WebHelp Plus outputs. Additionally, Cross Reference formats set here should appear in all print outputs. All Format language skin styles will only work if the corresponding style is not customized in the project Stylesheet. That is, the language skin is only used to substitute a translated string for the factory default English strings. If any of these are set at the CSS level, the CSS will always be used.
