Backups/Diffs

Bug Base Product Area:

  • Backups/Diff

 

Description:

You can manually create temporary backups of your files by sending them to the Backups Pane. Flare sometimes creates automatic backups of your files as you work on them. Additionally, you can view any differences between an older backup file and the newer file in the project in the Differences Viewer. Color coding makes it easy to see where differences occur. You can also switch between the tag view and WYSIWYG view, as well as ignore the case and whitespace.
