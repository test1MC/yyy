Color Picker-Launched by clicking “More Colors” from any color UI or ‘popup’ UI

Note: this dialog does not re-size

 

 

Controls

  • # (Color): Displays the #RRGGBB code of the color selected in the UI. – Allows user to type their own value in.
    • Boundary test this field – Large amounts of text, odd characters, letters, numbers, spaces, etc…
    • Color preview should update correctly when typing in the above field
  • Save: Adds the selected color to a list called "Saved"
  • Pick Color: Launches the “Screen Color Picker” and allows the user to pick a color currently viewable anywhere on the screen
  • Hue | Saturation | Brightness: Displays the values of the selected or typed color.
    • Boundary test these fields – Large amounts of text, odd characters, letters, numbers, spaces, etc…
    • Color preview should update correctly when typing in the fields
  • Red | Green | Blue: Displays the values of the selected or typed color
    • Boundary test these fields – Large amounts of text, odd characters, letters, numbers, spaces, etc…
    • Color preview should update correctly when typing in the fields
  • Enable CMYK: Activates the C, M, Y, K fields below so that you can use the CMYK color model.
    • Boundary test these fields – Large amounts of text, odd characters, letters, numbers, spaces, etc…
    • Color preview should update correctly when typing in the fields
  • Named: Displays the colors and values that have textual names.
    • "Name" column is sortable (A-Z and Z-A).
  • Web: Displays a list of web safe colors and values.
    • "Name" column is sortable (A-Z and Z-A).
  • Saved: Displays the Colors that user has added from the “save” button
    • "Name" column is sortable (A-Z and Z-A).
  • Recent: Displays the most recent colors that the user has used. To use one of these colors, click it.
    • "Name" column is sortable (A-Z and Z-A).
  • Current/New: The current applied color shows in the top half of this box, the color currently selected shows below.
  • OK: Applies the selected settings in the dialog to the selection or dialog it was called from
  • Cancel: Cancels the actions in the dialog and doesn’t apply any changes

 

Screen Color Picker

This UI allows you to select a color that is visible on the screen and use it as a color in the UI

 

  • The color at the tip of the cursor arrow is the color selected
  • The color should be displayed in the ‘preview’ box
  • The color should also be represented correctly in RRGGBB code
  • Clicking will select or use the color
  • Verify that it is added to the UI or code correctly
  • <ESC> closes the picker without applying the selection
  • No other keyboard input will work
