Team Foundation Server (TFS) 2012- Testing Matrix

Authentication Testing

Clear any login cache currently stored by deleting the information stored at the following location
Team Foundation Server: C:\Users\<username>\AppData\Local\Microsoft\Team Foundation
On newer systems, it may need to be cleared from Control Panel | Credential Manager

Functionality Testing

Before proceeding with the following tests, install Microsoft Visual Studio so that status of files in the repository can be checked with Source Control Explorer
