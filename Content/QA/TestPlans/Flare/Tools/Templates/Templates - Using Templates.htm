Templates - Using Templates

Using Templates to Add Files to a Project

This UI will allow users to browse all file templates in any folder set up to be included as a 'template folder'. Our "Factory templates" folder will be included by default.

  • Install Path\Flare.app\Resources\English\Templates
  • Test that if the default language for our application is set that the templates are pulled from the correct locations:
    • French: Install Path\Flare.app\Resources\French\Templates
    • German: Install Path\Flare.app\Resources\German\Templates
    • Japanese:Install Path\Flare.app\Resources\Japanese\Templates

Opening the Add File dialog

  • !Project | Add {File Type}
  • Right-clicking in the Content Explorer or Project Organizer and selecting 'Add {File Type}'
  • !File | New Project | Select a project template page of the wizard.
  • Getting Started Wizard | New Project | Select a project template page of the wizard.
  • Content Explorer Toolbar | Add Topic...
UI Test Cases:

Content File Dialogs: will have 'Folder' and 'File Name' fields (Topics will have additional options).

Project File Dialogs: will have only a 'File Name' field (Target will have an additional option).
