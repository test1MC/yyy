Template Management - UI

To fully test templates, you will need to create a few template folders. It is important that all template functionality is tested with template folders at a variety of locations. Use the Template Manager dialog (outlined below) to create a template folder at a:

  • Local path
  • UNC path
  • Mapped network drive
  • Secondary drive

Opening the dialog

  • Start New Project Wizard: In the "Select a project template" screen of the wizard, click the Manage Templates… button
  • Add File dialog: Open the dialog by any means, click the Manage Templates… button in the dialog
  • Tool Strip interface:
    • !Tools | Manage Templates…
    • !File | Save as Template, click the Manage Templates… button in the dialog
    • !Project | Save Project as Template, click the Manage Templates… button in the dialog
  • Ribbon interface:
    • !Tools | Manage Templates
    • !File | Save | Save as Template, click the Manage Templates… button in the dialog
    • !Project | Save Project as Template, click the Manage Templates… button in the dialog

Test Cases

  • You should not be able to manage the 'Factory' template folder. Verify it does not display in the Template Manager.
  • !Tools | Manage Templates... will be DISABLED if no project is open in Flare.
    • Ribbon interface
    • Toolstrip interface
  • Managing templates from the New Project Wizard should be disabled if no project is opened.

Template Manager - UI Test Cases

UI Controls

  • Templates: Allows you to browse/select all of a template or just files within a template.
    • (Show/Hide Folders): Toggles between showing or hiding the Tree View in the Template Manager.
    • (Show/Hide Files): Toggles between showing and hiding the Files View in the Template Manager.
      • If both options above are enabled, the Template Manager displays is what is commonly referred to as "Split View." The left half displays folders only. The right half shows folders and files in the currently selected folder with additional details for the files. As you navigate through folders on the right side, the containing folder should highlight in the left half of the Template Manager.
      • Verify that you can never disable both views. If only one view is enabled (you are not in "Split View"), attempting to disable the current view will disable that view and enable the other view.
    • (Up): Moves up one folder in the hierarchy in the Template Manager. This button becomes enabled if you use the Show Folders or Split View and browse into a folder.
    • (Delete):
      • Template Folder selected: Will remove the selected template folder from from being a template folder, so it no longer shows up in Flare's template UI. The folder WILL NOT be deleted from disk.
      • Template File selected: Will send the selected file to the recycle bin. Verify that this option prompts for confirmation.
    • (Add Template File): Invokes a Windows Open dialog which allows you to browse for a template file. Upon selecting a file, the Save as Template dialog opens to let you select to which template folder the file should be saved and optionally provide a description.
      • In the Open dialog, verify that all supported file types appear in the Template files filter.
      • Verify that any new file types that have been introduced to the latest version of Flare appear in the Template Files filter.

    • (New Template Folder): Invokes the Select Folder dialog which lets you browse for and select a folder to designate as a Template Folder.

      NOTE: Template folder locations are written to the registry and shared by all MadCap products: HKEY_CURRENT_USER|Software|MadCap Software|Shared|TemplateFolders. When a folder is "removed" from being a template, the path is not immediately removed from the registry. But the registry count ("Length") is changed to reflect the change. When another template folder is added, the new folder path will replace the removed path in the registry.

    • Filter: Lets you filter the visible files in the Template Manager to a particular file set.
      • All Template Files: All supported file types display
      • Project Data Templates: Only "Project" files (files that occur in the Project Organizer within a project) display.
      • Project Content Templates: Only "Content" files (files that occur in the Content Explorer within a project) display.
      • Project Templates: Only Flare project files (*.flprj) display
  • Preview/Description: Displays the preview or description for the selected template file.
    • This area should be disabled when a folder node is selected
    • This area should preview items that can be previewed (Topics, Snippets, Masterpages, Images and Multimedia) and show descriptions for items that cannot be previewed. (all other supported file types)
    • If there is no description the UI will display: "[No description found in template file]"
    • (Refresh Preview): Refreshes the preview or description.
    • (Edit Template Description): Invokes the Edit Template Description dialog which allows you to edit the description of the selected template file, when applicable. This option will be disabled when a folder node or a file that cannot contain a description is selected.

Template Manager - Right-click Menus

Right-clicking the root folder in the Template Manager should produce the following menu:

 

Right-click menu items (Tooltips in parentheses):

  • New Template Folder (New Template Folder): As detailed above, invokes the Select Folder dialog which lets you browse for and select a folder to designate as a Template Folder.

Right-clicking a template folder in the Template Manager should produce the following menu:

 

Right-click menu items (Tooltips in parentheses):

  • Open Folder in Windows (Open a Windows Explorer): Opens the selected folder in a Windows browser.
  • Paste (Paste): Pastes the files from the clipboard into the selected folder.
  • Delete (Delete): As detailed above, removes the selected folder from being a template folder, but does not delete anything from disk.
  • Add Template File (Add Template File): As detailed above, invokes a Windows Open dialog which allows you to browse for a template file.
  • New Template Folder (New Template Folder): As detailed above, invokes the Select Folder dialog which lets you browse for and select a folder to designate as a Template Folder.

Right-clicking a file in the Template Manager should produce the following menu:

 

Right-click menu items (Tooltips in parentheses):

  • Open (Open): Opens the selected template file in the Flare interface for editing.
  • Cut (Cut): Cuts the selected template files to the clipboard.
  • Copy (Copy): Copies the selected template files to the clipboard.
  • Paste (Paste): Pastes the files from the clipboard into the current template folder.
  • Delete (Delete): Sends the selected template file to the recycle bin. Verify a confirmation prompt is given.
  • Rename (Rename): Allows you to rename the template file.

Save as Template dialog - UI Test Cases

UI Controls

  • Template Folder: Select the template folder to which you want to save a copy of the selected template file. Verify all designated template folders appear in this list.
  • Template Description: If adding a file that supports template descriptions (any supported file besides Topics, Snippets, Masterpages, images and multimedia), this field will let you add a description for the template file. If adding a Topic, Snippet, Masterpage, image or multimedia files, this section will be disabled.
  • : In this context, when the Save as Template dialog is invoked from the Template Manager, this button should be disabled.
  • Template Name: Allows you to rename the file that is being copied to your template folder. By default, this field should display the original name of the file you selected, but you may change the name that will used for the template file here.
  • OK: Dismisses the dialog and copies the selected file to the selected template folder using the file name and description set in the Save as Template dialog.
  • Cancel: Dismisses the dialog and does not copy the template file.

Edit Template Description dialog - UI Test Cases

UI Controls

  • [description field]: Allows user to enter a text description for the selected template file.
  • OK: Dismisses the dialog and applies changes to the template file.
  • Cancel: Dismisses the dialog and does not apply changes to the template file.

Editing Template Files

Template files can be opened for editing inside of Flare but ONLY if there is a project currently open.

To open a template for editing:

  • Open the Template Manager and double-click a template file (or right-click and select Open):
    • Template Projects: Will open in another instance of Flare
    • Template files: Will close all dialogs and open the template in the MDI in the correct editor. There will be a shaded 'info' bar at the top of the file containing a blurb indicating the file is a template file and not a file in the current project.
