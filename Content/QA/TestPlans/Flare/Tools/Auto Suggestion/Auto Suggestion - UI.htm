Auto Suggestions - UI

Adding a Auto Suggestion file to the project

  • !Project |Advanced | Add Auto Suggestion List File…
  • In the Project Organizer, right click the Advanced folder and select Add Auto Suggestion List File…
  • In the Project Organizer, right click the Project folder and select Add | Advanced | Add Auto Suggestion List File…
  • Right click a word or phrase in a topic or snippet | Create auto suggestion | Prompt for new auto suggestion file if one doesn't exist

Enabling Auto Suggestions in Flare

!Tools | Options | Auto Suggestions

 

UI Controls

  • Settings:

    • Enable Auto Suggestion:Enables Auto Suggestions in your project
    • Enable Snippet Auto Suggestion: Enables Auto Suggestions from your snippets
    • Max Suggestions: Max number of suggestions to display in the popup (from 1 to 25)
    • Min Character Length: Select the minimum number of characters that must be typed before the popup opens with suggestions (from 3 to 12)
    • Enable/Disable files: Choose which files you will pull your auto suggestions from (including variable files, auto suggestion list files, and snippets)

Auto Suggestion Popup in the XML Editor

Auto Suggestions must be enabled and have phrases in an auto suggestion file list.

Just start typing one of the phrases on one of the enabled auto suggestion lists...

 

UI Controls

  • Hitting enter inserts the selection

  • Hitting the x closes the window
  • Hitting the icon at the bottom left opens the options menu
  • Click and holding the gray area at the bottom allows you to resize the popup window+
  • Hitting ESC should close the window

Test Cases:

  • Auto Suggestion popup should popup about 3/4 inch below and a little to the right of the start of the word
    • Make sure it behaves correctly on dual monitor display
  • Popup displays both capitalized and lowercase versions if both are present in the auto suggestion file
  • Popup only shows suggestions from selected files in the auto suggestion options
  • Test that it asks you what auto suggestion file to add your word to if there are more than one present
  • Add 2 similar suggestions that start with the same word, verify that you can navigate between them using arrow keys
