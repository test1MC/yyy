Auto Suggestion List Editor - UI Test Cases

Editor

UI Controls

  • Auto Suggestion List Editor: Displays and lets you create and edit Auto Suggestion phrases.

  • Local Toolbar:
    • Info Bar: Displays a blurb about limits and enabling the auto suggestion feature.
    • New Item: Inserts a new phrase
    • Cut: Cut the selected phrase(s)
    • Copy: Copy the selected phrase(s)
    • Paste: Paste the selected phrase(s)
    • Delete: Delete the selected phrase(s)

Test Cases:

  • Make sure that you cannot add duplicate words (should be an appropriate error message)

    • Make sure you can add the same word with different capitalizations
  • Verify that the 256 character limit is imposed by an error message when you try to save it.
  • Verify that you cannot create a autosuggestion for 2 character words (the minimum length for autosuggestion hints is 3 characters)

Auto Suggestion- General Editing Test Cases

Drag and Drop

There is currently no drag/drop editing functionality within the Auto Suggestion Editor

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

Cut, Copy and Paste functionality should work within an auto suggestion file and across multiple auto suggestion list files

Test Cases:

  • Verify cutting one phrase from an auto suggestion file and pasting into another works

    • Verify the original phrase is removed.
    • Do the same thing but copy and paste.
    • Verify that copy and pasting from the XML editor correctly pastes into an auto suggestion list file.

Workflow

Type the word that you want to add to the auto suggestion file

Select the word and click 'Create Auto Suggestion'

Click Ok to create auto suggestion file if you don't have one in your project already
