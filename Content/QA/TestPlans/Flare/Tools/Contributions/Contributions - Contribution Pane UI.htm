Contributions - UI

UI Controls:

  • Refresh: Refresh the Contributions Window
  • Delete: Deletes the selected document
  • Accept Topic...: Opens the Accept contribution wizard which allows the user to copy the file to the project.

Columns:

  • Document: Name of the document
  • Author: User name of the author of the document
  • Received: date/time the document was received

 

Accept contribution wizard:

 

  • File Name: Displays the file name of the contribution file.
  • Import Path: Displays the path where the file will be imported to. The file name should be the same as the File Name above.
  • Select folder: Allows you to select a folder within the Flare project
  • New folder: Inserts a new folder with the folder name selected and ready for immediate renaming.
  • Back: Disabled
  • Next: Moves to next Window (Comparing resources to project files...)
  • Finish: Disabled
  • Cancel: Closes the dialog.
