Contributions

Bug Base Product Area:

  • Contributions

 

Description:

Contributions, introduced in Flare 4.0, allow multiple authors to contribute content to a Flare project. The Flare author can create a contribution template (*.mccot) which contains the project stylesheet (and possibly other files) so that the contributing author can open it in Contributor and create new contribution files (*.mcco) using the styles from the Flare project. The contributing author can then submit the contribution to the Flare author for inclusion in the project. Also, a *.mcdoc file created from scratch in Contributor can serve as a contribution to a Flare project.
