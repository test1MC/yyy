Zipping a Project

Zip a project and save it locally:

!File | Zip | Zip Project… - Invokes the Create Package dialog for the tool strip

!Project | Save Group | Zip Project Invokesthe Create Package dialog for the Ribbon

 

Zip and e-mail a project:

!File | Zip | Zip and Email Project… - Immediately initiates the packaging process (Creating Package dialog), followed by opening an outgoing e-mail message with the *.flprjzip file attached and the name of the file in the e-mail's Subject field.

Create Package - UI Test Cases

UI Controls

  • Source Project: Displays the name and path of the current project to be zipped.
  • Package File: Displays and lets you edit the path and filename to which the *.flprjzip file will be save.
  • Browse…: Allows user to browse for a different location to which the *.flprjzip file will be saved.
  • Create: Closes the dialog and begins the packaging process. This will be indicated by the Creating Package dialog.
  • Cancel: Closes the dialog and does not package the project.

Creating Package… - UI Test Cases

UI Controls

  • Messages (tab): Displays messages throughout the packaging process.
  • Errors (tab): Displays errors during the packaging process (if any).
  • Save Log: If the zipping process produces errors, use the Save Log button to save a log file to the project.
  • Cancel: Lets you cancel the packaging process.

 

After the project is successfully packed, nothing will happen unless you selected Zip and Email. In that case, an outgoing e-mail message should appear at this point with the *.flprjzip file already attached and the name of the file in the Subject field.

Unzipping a Project

To unzip a project, simply open a *.flprjzip file either by:

  • !File | Open and then browse for a *.flprjzip file
  • Double-click any *.flprjzip file on disk

Unpack - UI Test Cases

UI Controls

  • Package to Unpack: Displays the name and path of the zipped project file.
  • Project Name: Displays and lets you edit the filename of the new Flare project.
  • Unpack to Folder: Displays and lets you edit the folder path to which the new Flare project will be saved.
  • Browse…: Allows user to browse for a different location to which the new Flare project will be saved.
  • Unpack: Closes the dialog and begins the unpackaging process. This will be indicated by the Unpacking dialog.
  • Cancel: Closes the dialog and does not unpack the project.

 

Unpacking… - UI Test Cases

UI Controls

  • Messages (tab): Displays messages throughout the packaging process.
  • Errors (tab): Displays errors during the packaging process (if any).
  • Save Log: If the zipping process produces errors, use the Save Log button to save a log file to the project.
  • Cancel: Lets you cancel the unpacking process.
