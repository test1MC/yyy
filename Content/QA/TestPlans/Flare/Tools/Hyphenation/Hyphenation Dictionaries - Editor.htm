Hyphenation Rule Editor - UI Test Cases

Local Toolbar:

Button (Tooltips in Parentheses):

  • (New item): Creates a new empty row
  • (Cut): Cuts the selected item(s) to the clipboard.
  • (Copy): Copies the selected item(s) to the clipboard.
  • (Paste): Pastes the item(s) from the clipboard.
  • (Delete): Deletes the selected item(s)

Hyphenation Rule Editor grid: Displays a list of words and custom hyphenation rules.

  • Word: Displays each word that has a custom hyphenation rule.
  • Rule: Displays the custom hyphenation rule for the corresponding word.

  • Keyboard can navigate and select options

Hyphenation Dictionary - General Editing Test Cases

Drag and Drop

  • There is currently no drag/drop editing functionality within the Hyphenation Rule Editor (as of Flare V5)

Deleting

Item:

  • Select the Hyphenation Rule row and press Delete on the keyboard
  • Select the Hyphenation Rule row and click in the local toolbar.
  • Select the Hyphenation Rule row and click in the Standard toolbar.
  • Select the Hyphenation Rule row and select Delete from the Edit menu.

Tooltips

  • If any Word or Rule is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.

Conditional Tags

  • Conditional Tags cannot be applied to individual Hyphenation Rules. Conditional Tags can be applied to the actual Hyphenation Dictionary file (*.mchyph), however, as of Flare V5, this tag can only be used for the purposes of Advanced Global Project importing.

Code

<?xml version="1.0" encoding="utf-8"?>
<HyphenationRules>
    <Hyphenation>
        <Word>Hyphenation</Word>
        <Rule>Hy-phenation;Hyphen-ation;Hyphena-tion</Rule>
    </Hyphenation>
    <Hyphenation>
        <Word>editor</Word>
        <Rule>ed-itor;edit-or</Rule>
    </Hyphenation>
</HyphenationRules>
