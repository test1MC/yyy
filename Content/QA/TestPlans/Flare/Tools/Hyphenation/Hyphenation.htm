Hyphenation/Hyphenation Dictionaries

Bug Base Product Area:

  • Hyphenation/Hyph. Dictionaries

 

Description:

Hyphenation lets you specify whether words at the end of a line in a paragraph should be hyphenated before continuing to the next line. Hyphenation Rules are created in a Hyphenation Dictionary by language. By default, Flare includes a Hyphenation Dictionary for English only. Hyphenation Rules can be created for other languages, however no default rules exist for languages other than English. Flare's Hyphenation Rules only work for our native print outputs: PDF, XPS and XHTML Book.

 

Another aspect of Hyphenation is determining minimum word and character settings for when hyphenation should actually be applied for your print output. This portion of Hyphenation functionality is covered along with Page Break control, Widow/Orphan controls and Short Line controls in the XML Editor: Print Layout test plan and will not be addressed in this test plan.
