Hyphenation - Styles

  • By default, Hyphenation is enabled for all elements. You can customize the hyphenation settings for any element locally in the Paragraph Properties dialog or globally in the Stylesheet Editor.

Paragraph Properties | Hyphenation Tab

  • If you want to set custom Hyphenation properties for a particular paragraph (or heading or list item, etc) you can select the tag in the XML Editor and select !Format | Paragraph Properties. Then, select the Hyphenation tab.

 

UI Controls (Default settings in parentheses)

  • Enable Hyphenation (Always): Select whether you always or never want words to be hyphenated at the end of lines.
  • Length of shortest word that should be hyphenated (5): Lets you define the minimum number of characters that have to be in a word in order for hyphenation to take affect.
  • The minimum number of characters remaining on the initial line (3): Lets you define the minimum number of characters that should remain on the initial line of a hyphenated word.
  • The minimum number of characters carried over to the next line (3): Lets you define the minimum number of characters that can be carried over to a new line in a hyphenated word.
  • The maximum number of adjacent lines that can end in a hyphenated word (2): Lets you define the maximum number of consecutive lines in which a hyphenated word can appear.

Stylesheet Editor | Hyphenation Styles

Stylesheet Editor in Simplified view

To access:

  • Open the Stylesheet Editor in Simplified View. Select the style you want to edit and run Properties (by any means; F4, double-click the style, etc)

Hyphenation Styles (Default settings in parentheses)

  • Enable Hyphenation (default): Select whether you always or never want words to be hyphenated at the end of lines.
  • Length of shortest word that should be hyphenated (not set): Lets you define the minimum number of characters that have to be in a word in order for hyphenation to take affect.
  • The minimum number of characters remaining on the initial line (not set): Lets you define the minimum number of characters that should remain on the initial line of a hyphenated word.
  • The minimum number of characters carried over to the next line (not set): Lets you define the minimum number of characters that can be carried over to a new line in a hyphenated word.
  • The maximum number of adjacent lines that can end in a hyphenated word (not set): Lets you define the maximum number of consecutive lines in which a hyphenated word can appear.

Stylesheet Editor in Advanced view

To access:

  • Open the Stylesheet Editor in Advanced View. Select the style you want to edit under Styles (on the left). On the right, use the pull-down to display properties by Property Groups or Alphabetical List. If you choose Property Groups, the hyphenation styles will appear in the "Unclassified" group.

Hyphenation Styles (Default settings in parentheses):

  • mc-hyphenate (default): Select whether you always or never want words to be hyphenated at the end of lines.
  • mc-hyphenate-maximum-adjacent-line-count (not set): Lets you define the maximum number of consecutive lines in which a hyphenated word can appear.
  • mc-hyphenate-shortest-prefix (not set): Lets you define the minimum number of characters that should remain on the initial line of a hyphenated word
  • mc-hyphenate-shortest-suffix (not set): Lets you define the minimum number of characters that can be carried over to a new line in a hyphenated word
  • mc-hyphenate-shortest-word (not set): Lets you define the minimum number of characters that have to be in a word in order for hyphenation to take affect.
