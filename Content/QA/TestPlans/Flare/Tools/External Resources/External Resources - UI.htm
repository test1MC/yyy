Adding an External Resource folder

In order to test anything in the External Resources pane, you'll first need to add an External Resource folder.

  1. Click the Add New External Resource button in the External Resources pane's local toolbar
  2. In the Select Folder dialog, browse for any folder to which you have access
  3. Click OK to close the Select Folder dialog and add the selected folder to the External Resources pane

Local Toolbar Buttons (Tooltips in parentheses):

  • (Add a new external resource folder): Add a folder to the External Resource pane.
  • (Copy file to your project): Copy the selected external file to your project
  • (Manage mappings of project files to external resources): Invokes the Map Files and Folders dialog which lets you manage the mappings of files in your project to those in your External Resources or any other external location.
  • (Select files and folders to import, export, or synchronize): Invokes the Synchronize Files dialog which lets you select mapped project files or folders to import, export or synchronize with those in your External Resources or any other external location.
  • (Show/Hide Folders): Toggles between showing or hiding the folders in the External Resources pane.
  • (Show/Hide Files): Splits the External Resources pane into two halves. The right half shows additional details for the files. When you click a file on the right side, a red circle shows the location in the External Resources pane. See a movie about this feature.
  • (Collapse All): Collapses all of the folders so that you can no longer see all of the files under each one.
  • (Up): Moves up one folder in the hierarchy in the External Resources pane. This button becomes enabled if you use the Show Folders or Split View and browse into a folder.
  • (Open): Opens the selected file. A file opened from the External Resources pane should display an Info Bar indicating that it exists outside the project.
  • (Run Properties dialog for this item): Opens the Properties dialog for the selected file.

Show Files grid

Right-Click Menus

Right-clicking the root folder you added in the External Resources Pane should produce the following menu:

 

 

Right-click menu items:

  • Open Folder in Windows:Open the selected folder in Windows
  • Copy to Project...: Copy the selected folder and all files therein to your project
  • Remove External Resources Folder: Removes the external resource folder from your External Resources Pane. Does not delete the folder from your hard drive!
  • Properties: Opens the Properties dialog window for the selected folder
  • Refresh: Refreshes the selected folder

Right-clicking a folder that is not the root folder you added in the External Resources Pane should produce the following menu:

  • Properties: Opens the Properties dialog window for the selected folder
  • Refresh: Refreshes the selected folder

Right-clicking a file in the External Resources Pane should produce the following menu:

Map Files and Folders dialog

File mapping is not limited to files in your External Resources pane. Mappings that you create to a designated external resource will remain in the project if you remove the external resource folder containing that file.

Opening the dialog

  • External Resources: Click Manage Mappings button in the local toolbar
  • Synchronize Files dialog: Click "Manage Mappings…" button
  • Tool Strip interface: !Tools | Manage Mappings
  • Ribbon interface: !Project | Manage Mappings

Map Files and Folders dialog - UI Test Cases

UI Controls

  • Click to enter a new project path: Invokes the Select Project Path dialog allowing you to choose which project file/folder you want to map
  • Click to enter a new external path: Invokes the Select External Path dialog allowing you to choose where to map the selected file/folder mapped from your project to some external place
  • Remove: Removes selected mapped path

Select Project Path/External Path dialog - UI Test Cases

UI Controls

  • (Show/Hide Folders): Toggles between showing or hiding the folders in the External Resources Pane.
  • (Show/Hide Files): Splits the External Resources Pane into two halves. The right half shows additional details for the files. When you click a file on the right side, a red circle shows the location in the External Resources Pane.
  • (Toggle show Hidden Files/Folders): Lets you hide or show files or folders that are marked as hidden in Windows Explorer.
  • (Up): Moves up one folder in the hierarchy in the External Resources Pane. This button becomes enabled if you use the Show Folders or Split View and browse into a folder.

Synchronize Files dialog

  • External Resources: Click the Synchronize Files button in the local toolbar
  • Tool Strip interface: !Tools | Synchronize Mapped Files
  • Ribbon interface: !Project | Synchronize Files

Synchronize Files dialog - UI Test Cases

UI Controls

  • Dropdown:
    • Synchronize Files: Files in your project that were more recently modified than external files will be copied to external location. External files that were more recently modified than files in your project will be copied to your project.
    • Import Files:Export files that were more recently modified than files in your project will be copied to your project.
    • Export Files: Files in your project that were more recently modified than external files will be copied to external location.
  • Refresh Button: Refreshes the list of mapped files.
  • Equals Button: Show/hide synchronized files.
  • Manage Mappings: Invokes the Manage Mappings dialog to manage the mappings in your project.
  • Synchronize: Synchronizes the selected files/folders in your project with the external file/folder (If nothing is mapped, this button should be greyed out).
