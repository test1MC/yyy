Spell Check/Dictionaries - Baseline

Verify that you can add words to multiple language dictionaries by creating projects of various languages, creating topics of various languages, and inserting language tagged elements inside of topics (including spans). Verify that in each case, the term is added to the correct language dictionary.

  • Be sure to include "(never spell check)" in your testing.
  • Test local language settings as well as setting the mc-language property in the stylesheet on various elements.
  • MAKE SURE TO TEST SOME BUGS WITH THE NEW RIGHT TO LEFT LANGUAGES AND/OR DICTIONARIES
