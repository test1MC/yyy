Spell Check - Styles

In addition to applying a language tag to an element in your content, you may prefer to create a style the is associate with a particular language or a style that should never be spell checked. This can be done in the Stylesheet Editor (Advanced view only).

 

Create one or more style classes and set the mc-language property to any valid xml:lang property ("none" for never spell check, "it" for Italian, "fr" for French, etc.).

Use these styles in your content and verify that Spell Check works correctly.
