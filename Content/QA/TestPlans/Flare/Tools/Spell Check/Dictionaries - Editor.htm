Dictionaries - Editor

Opening a Dictionary File

  • Tool Strip interface: !Tools| Edit Dictionary | Select Project or Global Dictionary from dropdown.
  • Ribbon interface: !Tools| Edit Dictionary| Select Project or Global Dictionary from dropdown.

 

Additional ways to open Project Dictionaries:

   

Open Dictionary Dialog

 

 

UI Controls

  • Select Language Dictionary (Global or Project): Select from the drop down to view a list of Global or Project dictionaries
  • OK: Opens selected dictionary in the editor.  If you open a Global dictionary the warning "You are currently editing the global dictionary" will show in the editor.
  • Cancel: Closes the dialog without opening a dictionary

Test Cases

  • Global dictionary should show all dictionaries saved in the Global dictionary location (by default it is C:\Users\[username]\AppData\Roaming\MadCap Software\Flare\Spelling Dictionaries, but it can be changed to any location in the Options | Spelling dialog).
  • Project dictionary should show all dictionaries in the Project Organizer | Advanced folder, including any nested in folders.

Dictionary Editor - UI Test Cases

Local Toolbar:

Button (Tooltips in Parentheses):

  • (New item): Creates a new empty row
  • (Cut): Cuts the selected item(s) to the clipboard.
  • (Copy): Copies the selected item(s) to the clipboard.
  • (Paste): Pastes the item(s) from the clipboard.
  • (Delete): Deletes the selected item(s)

Dictionary Editor grid: Displays a list of words added to the Dictionary.

  • Word: Displays each word that has been added to the Dictionary.
  • Comment: Lets the user add a comment for the corresponding word. This comment is strictly for use within the Dictionary Editor.

Tooltips

If any Word is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.

Conditional Tags

Conditional Tags cannot be applied to individual Dictionary terms. Conditional Tags can be applied to the actual Dictionary file (*.mcdic), however, as of Flare V5, this tag can only be used for the purposes of Advanced Global Project importing.

Code

Dictionary File

<?xml version="1.0" encoding="utf-8"?>
<SpellingDictionary>
<Word
Comment="user comment here...">MadCap</Word>
<Word>stylesheet</Word>
</SpellingDictionary>

 

Ignored Words are listed in a topic's html tag in the MadCap:ignoredWords attribute:

<html xmlns:MadCap="http://www.madcapsoftware.com/Schemas/MadCap.xsd" MadCap:ignoredWords="MadCap;stylesheet">
