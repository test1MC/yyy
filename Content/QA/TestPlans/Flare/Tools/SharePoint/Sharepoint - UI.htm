Adding a SharePoint Server

In order to test anything in the SharePoint Explorer, you'll first need to add a SharePoint Server.

  1. Click the Connect to SharePoint button in the SharePoint Explorer's local toolbar
  2. In the Connect to SharePoint Server dialog, enter the SharePoint site you want to connect to. At the time this test plan was last update, you could use http://appserver01:8080/ (or any sub-site therein)
  3. Click the button to verify the SharePoint Server
  4. Click OK to close the Connect to SharePoint Server dialog and add it to the SharePoint Explorer

Local Toolbar Buttons (Tooltips in parentheses):

  • (Connect to SharePoint Server): Invokes the Connect to SharePoint Server dialog allowing you to add a SharePoint Server.
  • (Show/Hide Folders): Toggles between showing or hiding the folders in the SharePoint Explorer Pane.
  • (Show/Hide Files): Splits the SharePoint Explorer Pane into two halves. The right half shows additional details for the files. When you click a file on the right side, a red circle shows the location in the SharePoint Explorer Pane. See a movie about this feature.
  • (Collapse All): Collapses all of the folders so that you can no longer see all of the files under each one.
  • (Up): Moves up one folder in the hierarchy in the SharePoint Explorer Pane. This button becomes enabled if you use the Show Folders or Split View and browse into a folder.
  • (Open): Opens the selected file.
  • (Run Properties dialog for this item): Opens the Properties dialog for the selected file.

Right-Click Menus

Right-clicking a server should produce the following menu:

 

 

Right-click menu items:

  • Open Folder in Windows: Open the SharePoint site in the Internal Browser
  • Remove: Removes the selected SharePoint site from the SharePoint Explorer
  • Refresh: Refreshes the list of subsite, folders and files that appear under the selected site

Right-clicking a subsite should produce the following menu:

 

 

Right-click menu items:

  • Open: Open the selected subsite in the Internal Browser
  • Properties: Opens the Properties dialog window for the selected folder
  • Refresh: Refreshes the list of folders and files that appear under the selected subsite

Right-clicking a folder should produce the following menu:

 

 

Right-click menu items:

  • Open: Open the SharePoint site for the selected folder in the Internal Browser
  • Copy to Project...: Copy the selected folder and all files contained therein to your project
  • Properties: Opens the Properties dialog window for the selected folder
  • Refresh: Refreshes the selected folder

Right-clicking a file should produce the following menu:

 

 

Right-click menu items:

  • Open: Open the selected file
  • Copy to Project...: Copy the selected file to your project
  • Check Out...: Check out the selected file from SharePoint
  • Check In...: Check in selected file to SharePoint
  • Undo Check Out...: Undo the check out of selected file from SharePoint
  • Properties: Opens the Properties dialog window for the selected file
