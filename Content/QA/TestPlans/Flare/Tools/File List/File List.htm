File List

Bug Base Product Area:

  • Tools/Options

 

Description:

File List allows user to view all of the content files in the project in alphabetical order. User can also limit the files that are displayed by using the file type filter and perform certain tasks like opening files, applying condition tags, and applying style sheets.
