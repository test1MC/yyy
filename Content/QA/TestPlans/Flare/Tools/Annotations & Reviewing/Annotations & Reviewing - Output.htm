Annotations & Reviewing - Output

Support in Web Outputs:

Support in HTML5 Outputs:

  • Annotations should not be present in any form in any output.
  • Add an annotation to a topic and generate that topic to all output types.
    • Verify that for all output types, the MadCap:annotation tag is simply unbound from its text. That is, make sure that all tagged content is included in the output, but that any information related to the annotation itself (initials, annotation text, etc) is not included in the output.
