Annotations & Reviewing - UI

You can add an annotation to content in a p tag.  Once you click Insert Annotation, Show Changes will be selected.  You can add multiple annotations to the same selection of content.

Adding an Annotation

  • Tool Strip interface: !Review |  Insert Annotation
  • Ribbon interface: !Review |Insert Annotation

Deleting Annotation(s)

  • Tool Strip interface: !Review |  Delete Annotation
  • Tool Strip interface: !Review |  Delete All Annotations
  • Ribbon interface: !Review |Delete Annotation
  • Ribbon interface: !Review |Delete Annotation drop down | Delete Annotation
  • Ribbon interface: !Review |Delete Annotation drop down | Delete All Annotations

 

  • XML Editor: R-click the annotation | Delete Annotation

 

 

 

Code

<MadCap:annotation MadCap:createDate="2014-04-22T15:44:22.5871722-07:00" MadCap:creator="ebrown" MadCap:initials="EB" MadCap:comment="first annotation" MadCap:editor="ebrown" MadCap:editDate="2014-04-22T15:44:26.2935428-07:00">This is my topic content</MadCap:annotation>
