Annotations - Editing

Opening the UI

  • !Review | Insert Annotation
  • Turn on Track Changes, highlight some text and right-click that text | insert annotation

Context Menu (removed in Flare v.7)

Evoking the Context Menu

  • There should be no circumstance under which you are able to evoke this context menu or see an annotation block bar element. Make you cannot access the menu or cause the block bar to appear in the following scenarios:
    • Right-clicking the annotation box (when you have show changes on) should not evoke the context menu
    • Multi-clicking the element should not allow you to select the annotation tag and evoke this menu
    • Cut/Copy & Pasting should not duplicate this element
    • Selecting the element should not cause a corresponding caret tag to appear

Verify that Togglers are removed or unbound from Outputs depending on the Condition tag settings.
