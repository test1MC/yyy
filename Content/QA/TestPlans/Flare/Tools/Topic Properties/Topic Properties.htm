Topic Properties

Bug Base Product Area:

  • Tools/Options

 

Description:

A dialog used to make property changes for a Flare topic. Project Properties lets you view basic information about the topic, such as filename, file folder, file size, etc. Topic Properties also lets you set the topic title, stylesheet association and style class, searchable status (for web outputs), and topic language (used for spell checking the topic). You can also set a topic level condition tag using Topic Properties as well as use Snippet Condition to condition out content contained inside a snippet contained in the topic.
