Standard Toolbar - UI Controls

Toolbar Items (Tooltips in parentheses)

  • (Toggle Tracking Changes) Toggles Tracked Changes on/off

  • (Toggle Show Changes): Toggles showing tracked changes
  • (Navigate to the Previous Change): Navigates to the previous tracked change
  • (Navigate to the Next Change): Navigates to the next tracked change
  • (Accept the Current Change): Accepts the currently selected change
    • Dropdown Arrow: Accept All Changes: Accepts All Changes in the Document

  • (Reject the Current Change): Rejects the currently selected change

    • Dropdown Arrow: Reject All Changes: Rejects All Changes in the Document
  • (Insert Annotation): Inserts an annotation for the selected text
  • (Delete the Annotation): Deletes the selected annotation
    • Dropdown Arrow: Deletes All Annotations: Deletes All Annotations in the Document

Note: The track changes setting is not global in a project (i.e., it does not affect other content files open at that time). For example, let's say you have two topics open. With focus on topic A, you enable the track changes feature. If you bring focus to topic B, the track changes option is not enabled for that topic yet. However, if you then open other topics after that point, the track changes feature will be enabled in those topics.

When you turn on track changes, show changes will also be turned on. If you turn off show changes, your changes are still being tracked, you just can't see what specifically was changed until you toggle on show changes.

Test Cases

  • Hide and show tracked changes
  • Change user name and initials: When you first install and launch Flare, your login user name is captured and stored in the Options dialog. The first two letters of the user name are also stored as your initials. This information is employed whenever you track changes. For example, if you add some text in a topic with track changes enabled, your initials are displayed at the beginning of the tracked change so that other users know where the edit came from. You can always change your Flare user name or initials to something else.
  • Change display: You can change the way tracked changes are displayed in the interface (e.g., in balloons, inline).
  • Change user color: The tracked changes for each user can be shown in a separate color. You also have the option of choosing a specific color for all users.
  • Find changes: You can use navigation buttons in the Review toolbar to quickly navigate to the previous or next tracked change or annotation in the file.
  • Accept or reject tracked changes You can right-click on tracked changes and use a context menu to accept or reject them individually. You can also use buttons in the Review toolbar or options in the Review menu to accept or reject each tracked change or all tracked changes in a file. The reject option also removes annotations.
