Track Changes - Editing

XML Editor

A sample of what the XML editor should look like if you have made an addition and a deletion. Right-clicking the tracked change balloon should invoke the above menu.

Note: The track changes feature is intended only for self-contained operations within the WYSIWYG (XML Editor). For example, if you make changes to index keywords or concepts in the Index and Concept window panes (e.g., renaming, assigning them to topics), those modifications are not indicated as tracked changes because they are considered project-wide "link updating" processes.

 

This is what a 2 column layout looks like with track changes on. Notice that the two changes on the left column will appear above the changes made on the right column.

Code

<MadCap:changeData></MadCap:changeData> tags will only appear once per topic around all of the tracked changes change data

What the code appears like inline for a tracked change:

<MadCap:change MadCap:changes="3">XXXXXXXXX</MadCap:change>

 

Addition:

<MadCap:changeData>

<MadCap:AddChange MadCap:userName="mbuckalew" MadCap:initials="MB" MadCap:id="3" MadCap:timestamp="2011-03-23T14:43:45.4738000-07:00" />

</MadCap:changeData>

 

Deletion:

<MadCap:changeData>

<MadCap:RemoveChange MadCap:userName="mbuckalew" MadCap:initials="MB" MadCap:id="1" MadCap:timestamp="2011-03-23T14:39:49.4778000-07:00" />

</MadCap:changeData>

 

Wrapping text/element in a span (ex. selecting some text and applying bold to it):

<MadCap:changeData>

<MadCap:BindChange MadCap:userName="mbuckalew" MadCap:initials="MB" MadCap:id="4" MadCap:timestamp="2011-03-23T15:18:39.1658000-07:00" />

</MadCap:changeData>

 

Modified (ex. changing an existing equation to a different one):

<MadCap:changeData>

<MadCap:ReplaceChange MadCap:userName="mbuckalew" MadCap:initials="MB" MadCap:id="7" MadCap:timestamp="2011-03-23T15:21:10.5008000-07:00">

</MadCap:changeData>

 

Attribute change (ex. changing multiple styles of text like applying bold and changing the font from Arial to Sans-Serif:

<MadCap:changeData>

<MadCap:AttributesChange MadCap:userName="mbuckalew" MadCap:initials="MB" MadCap:id="2" MadCap:timestamp="2011-03-23T15:24:45.1728000-07:00" MadCap:attribute="style" MadCap:value="font-family: sans-serif;" />

</MadCap:changeData>
