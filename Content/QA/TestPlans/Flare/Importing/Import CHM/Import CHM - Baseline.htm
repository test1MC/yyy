Import CHM - Baseline

Many issues that you'll find importing CHM files will also be issues when importing htm or html files. As much as possible, try to limit the bugs reported in this area to those which are specific to importing a CHM.

The Imported Project

Verifying a successful import

After importing a CHM you should compare the Flare project to the original CHM

  • Compare the original project with the new one
  • Compare the original output with the new projects output

 

Here is a list of things to check that import correctly (Make your own CHMs if necessary)

  • Topics (Specifically the content and formatting of the content)
  • Lists
    • Bulleted
    • Numbered
    • Images
  • Hyperlinks
  • Bookmarks
  • Index
  • TOC
  • Glossary
  • Images in topics
  • Tables
  • Colors
    • Font
    • Background
  • Styles (if you selected the correct option above)
  • All topics are converted correctly to XML
  • Shortcuts
  • Anything not listed here!
