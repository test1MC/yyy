Import CHM - Wizard

To Launch the Import Project Wizard

To create a new project by importing CHM files, launch the Import Project Wizard by:

  • Tool Strip interface: !File | Import Project | HTML Help File (CHM)…
  • Ribbon interface: !File | New Project | HTML Help File (CHM)
