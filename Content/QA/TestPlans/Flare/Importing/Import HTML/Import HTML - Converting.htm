Import HTML - Converting HTML/XHTML

Converting HTML Documents in an existing project - UI

To convert an HTML files in an existing project, launch the HTML-> XHTML Conversion Dialog by:

  • Right-click the HTML file in the Content Explorer or File List and select "Convert to XML"

 

Double-clicking the html file that is not converted will invoke this dialog:

UI Test Cases:

Single File:

UI Controls

  • Show Source Code: If this option is toggled on, the HTML-> XHTML Conversion dialog will display the original file and the (to be) converted file in code view. If this option is toggled off, the topics will display in WYSIWYG view.
  • Create Backup File: Creates a backup of the HTML file.
  • Convert: Proceeds with the conversion of all files in this list.
  • Cancel: Cancels the conversion operation and closes the dialog.

Multiple Files:

UI Controls

  • Open Selected: Opens the HTML file(s) in the default program for that file type (usually a web browser)
  • Preview Conversion: Creates a preview of what the conversion will look like.
  • Remove: Removes the selected file from the conversion dialog.
  • Create Backups of Original Files: Creates backups of the original HTML files.
  • Convert: Proceeds with the conversion of all files in this list.
  • Cancel: Cancels the conversion operation and closes the dialog.
