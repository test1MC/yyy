Import HTML

Bug Base Product Area:

  • Import: HTML/XML

 

Description:

Gives the users the ability to import and reimport HTML files (*.htm, *html, or *.xhtml files). If files are non-XML, there will also be converted to XML. You can create a new project or add topics to an existing project by importing HTML files. Import is designed to create XML files that can be used in all outputs. You can map directly to the files or you can map to a source folder that contains the files, this allows the user to update the source files while still maintaining the setup for where and how they import into the Flare project. Additionally the Import HTML Files Wizard gives the option to also import any linked resource files (images, stylesheets, etc).
