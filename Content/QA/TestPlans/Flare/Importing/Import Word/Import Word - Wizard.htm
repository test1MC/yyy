Import Word - Wizard

Launching the Importing Microsoft Word Wizard

Scenario 1: Import to a New Project

To create a new project by importing Word files, launch the Import Microsoft Word Wizard by:

  • Toolstrip interface: !File | Import Project | MS Word Documents…
  • Ribbon interface: !File | New Project | MS Word Documents
  • Ribbon interface: Open any project, and then !Project | Import | MS Word Documents, and select the radio button option: "Import into a new project"

Scenario 2: Import into an Existing Project

To import Word files into an existing project, first open the project you want to import files into, and then launch the Import Microsoft Word Wizard by:

  • Toolstrip interface: !File | Import Project | MS Word Documents…, and the radio button option "Import into this project" should be preselected
  • Ribbon interface: !Project | Import | MS Word Documents, and the radio button option "Import into this project" should be preselected

Import Microsoft Word Wizard - Dialog 1a (Scenario 1): Import Options

UI Controls

  • Project Name: Lets you enter a name for the project that will be created from the imported documents.
  • Project Folder: Lets you set the location of the project

  • Finish: Closes the Import Microsoft Word Wizard, invokes the Import Progress dialog and proceeds to import the Microsoft Word files. (Should be grayed out at this point)
  • Cancel: Closes the Import Microsoft Word Wizard and does not import anything.

Import Microsoft Word Wizard - Dialog 4: Options

UI Controls

  • Add "Topic Continued" links when appropriate: Select this option if you want a link called "Topic Continued" to be placed at the bottom of pages when a long topic has been split into multiple topics.
    • Cross-Reference Format: Use this field to specify the format for the "Topic Continued" links.
    • Edit: If you want to modify the cross-reference format provided, click this button, which opens the Cross-Reference Format dialog.
  • Add "Topic Continued From…" links when appropriate: Select this option if you want a link called "Topic Continued From" to be placed at the top of continued pages when a long topic has been split into multiple ones.
    • Cross-Reference Format: Use this field to specify the format for the "Topic Continued From" links.
    • Edit: If you want to modify the cross-reference format provided, click this button, which opens the Cross-Reference Format dialog.
  • Split Long Topics: Select this option if you have long source documents and want to make sure that they are converted to multiple topics
    • Threshold: Enter the maximum number of characters to be converted to a topic before a new topic is created.
  • Avoid Creating Empty Topics: Select this option if you want to ensure that new topics are not created when large sections are found in the Word documents without any content.
    • Threshold: Enter the maximum number of empty character spaces allowed in a topic.
  • Auto-reimport before 'Generate Output': Also called "Easy Sync," this option will tell Flare to automatically re-import Microsoft Word files when you attempt to build output.
  • Approximate Filename Length: Enter the maximum number of characters to use for naming new topic files that are automatically created after splitting a long topic.
  • Convert all tables to"Auto-Fit to Contents": Select this option if you want to automatically set tables to "Auto-Fit to Contents" when they are imported into Flare.
  • Create a Page Layout for each section header/footer: Select this option to create a unique Page Layout file during the import for every section break that occurs in the Word document.
  • Automatically set topic title: Enabled by default. Select this option to have Flare automatically use the first heading of a topic as its title at the point of compile. If you deselect this option, a topic title will be injected during import.
  • Back: Returns to the previous dialog in the Import Microsoft Word Wizard.
  • Next: Proceeds to the next dialog in the Import Microsoft Word Wizard.
  • Finish: Closes the Import Microsoft Word Wizard, invokes the Import Progress dialog and proceeds to import the Microsoft Word files.
  • Cancel: Closes the Import Microsoft Word Wizard and does not import anything.

Import Microsoft Word Wizard - Dialog 6: Paragraph Style Mapping

UI Controls

  • Left-hand Columns
    • MS Word Style: Lists all styles used in the Word documents you have added to the import.
    • Flare Style: Displays the style name that the Word style will adopt after the documents are imported.
  • Right-hand Column
    • Flare Styles: List of Flare styles available to be mapped to.
  • Map: Lets you map the selected MS Word Style(s) on the left to the selected Flare Style from the far right.
  • Unmap: Lets you remove the mapped Flare Style from the selected MS Word Style(s).
  • (Left) Preview: Displays a preview of the style selected in the MS Word Style list.
  • (Right) Preview: Displays a preview of the style selected in the Flare Styles list.
  • Back: Returns to the previous dialog in the Import Microsoft Word Wizard.
  • Next: Proceeds to the next dialog in the Import Microsoft Word Wizard.
  • Finish: Closes the Import Microsoft Word Wizard, invokes the Import Progress dialog and proceeds to import the Microsoft Word files.
  • Cancel: Closes the Import Microsoft Word Wizard and does not import anything.
