Importing a HHP into a New Project

To create a new project by importing HHP files, launch the Import Project Wizard by:

  • Tool Strip interface: !File | Import Project | HTML Help Project (HHP)…
  • Ribbon interface: !File | New Project | HTML Help Project (HHP)
