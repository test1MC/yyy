Import Flare - Add File

There are two ways to import a Flare project into an existing project:

  • Option 1: Create a Flare Project Import File to the project first and then import the project from within the Project Import Editor
  • Option 2: Import the project first through the Import Flare Project Wizard which will then create an import file at the end

 

This test module covers Option 1.
