Frame Import Styles - UI

To open this dialog, click the "Conversion Styles..." button from the Import FrameMaker Wizard or the Stylesheet Tab of the Frame Import Editor.

UI Test Cases:

Import Styles Editor

UI Controls

  • Standard Toolbar Buttons (Tooltips in parentheses) 
    • OK (OK): Closes the dialog and saves any changes.
    • Cancel (Cancel): Closes the dialog and does not save changes.
    • (Cut): Removes the selected item and copies it to the clipboard.
    • (Copy): Copies the selected item to the clipboard.
    • (Paste): Pastes the item from the clipboard.
    • (Delete): Deletes the selected item.
  • (Styles) Dropdown: Lets you select which set of styles should be displayed below (currently "Show All Styles" is the only option in this dialog)
  • (Styles) List of styles: Displays a list of all FrameMaker styles. Select one to
  • (Show) Property Groups: Select this option to display all of the properties for the selected style in groups (with other similar properties).
  • (Show) Alphabetical List: Select this option to display all of the properties for the selected style in alphabetical order.
Properties (by Group)

Font:

  • FontFamily: Sets the font family (opens the Font Family Picker dialog).
  • FontSize: The font size.
  • FontWeight: The font weight.
  • FontStyle: The font style.
  • Color: The font color.
  • BackgroundColor: The background color.
  • Underline: Set the underline property to "true" or "false."
  • Overline: Set the underline property to "true" or "false."
  • LineThrough: Set the underline property to "true" or "false."
  • LetterSpacing: Set the Letter spacing value.

Format: (for Cross-Reference styles only)

  • Format: The cross-reference format.

Borders:

  • BorderLeft: The left border.
  • BorderRight: The right border.
  • BorderTop: The top border.
  • BorderBottom: The bottom border.
  • BorderCollapse: Border-collapse property.

Padding:

  • PaddingLeft: The left padding.
  • PaddingRight: The right padding.
  • PaddingTop: The top padding.
  • PaddingBottom: The bottom padding.

Margin:

  • TextIndent: First line indent. (for Paragraph styles only)
  • MarginLeft: The left margin.
  • MarginRight: The right margin.
  • MarginTop: The top margin.
  • MarginBottom: The bottom margin.

AutoNumber: (for Paragraph styles only)

  • AutoNumberFormat: The auto-number format.
  • AutoNumberPosition: The position of the auto-number.
  • AutoNumberOffset: The offset from the (left) edge of the paragraph.
  • AutoNumberClass: The CSS class for the span tag around the auto-number.
