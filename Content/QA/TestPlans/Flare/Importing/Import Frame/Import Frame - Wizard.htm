Import Frame - Wizard

Importing Frame Documents into a New Project - UI

To create a new project by importing FrameMaker files, launch the Import FrameMaker Wizard by:

!File | Import Project | Import FrameMaker Documents…

Import FrameMaker Wizard dialog 1: Import Options

UI Controls

  • Import into a new project:
    • Project name This is where you name the Flare project
    • Project folder Select where you want to store the Flare project
    • Output Type Select the type of target you want to start the project with
  • Back: Returns to the previous dialog in the Import FrameMaker Wizard. (Should be grayed out at this point)
  • Next: Proceeds to the next dialog in the Import FrameMaker Wizard.
  • Finish: Closes the Import FrameMaker Wizard, invokes the Import Progress dialog and proceeds to import the FrameMaker files. (Should be grayed out at this point)
  • Cancel: Closes the Import FrameMaker Wizard and does not import anything.

Import FrameMaker Wizard dialog 2: Select FrameMaker documents to import

UI Controls

  • Selected FrameMaker documents

    Columns

    • FrameMaker Files: Displays each *.fm, *.mif or *.book file added to the import
    • Status: ???
  • : Opens the selected file in the default application. Button will gray out with a multi-selection.
  • Link Generate Files To Source Files: This option creates a connection between the original FrameMaker files and the files that are created as a result of the import.
  • : This moves the selected document higher in the list (if you have more than one document to import). The document at the top is used for the name of the content folder holding the imported topics in Flare. Also, the order determines how the imported documents are arranged in the Flare TOC that is created as a result.
  • : This moves the selected document lower in the list (if you have more than one document to import).
  • : This removes the selected document(s) from the list.
  • : Produces the Open dialog which lets you browse for *.fm, *.mif or *.book files to add to the import.
  • Back: Returns to the previous dialog in the Import FrameMaker Wizard.
  • Next: Produces the Scanning Source Documents dialog and then proceeds to the next dialog in the Import FrameMaker Wizard. (Should produce a warning if you try to proceed without adding any files)
  • Finish: Closes the Import FrameMaker Wizard, invokes the Import Progress dialog and proceeds to import the FrameMaker files. (Should be grayed out at this point)
  • Cancel: Closes the Import FrameMaker Wizard and does not import anything.

Import FrameMaker Wizard dialog 3: New Topic Styles

UI Controls

  • Used FrameMaker Styles: Lists all styles used in the FrameMaker documents you have added to the import.
  • 'New Topic' Styles: Lists all styles which should split the original FrameMaker document(s) into separate topics.
  • [ >> ] : Lets you add the Used FrameMaker Style(s) selected from the left side and add them to the 'New Topic' Styles list.
  • [ << ] : Lets you select a 'New Topic' Style from the right side and remove it from the 'New Topic' Styles list.
  • Preview: Displays a preview of the style selected in the Used FrameMaker Styles list.
  • Back: Returns to the previous dialog in the Import FrameMaker Wizard.
  • Next: Proceeds to the next dialog in the Import FrameMaker Wizard.
  • Finish: Closes the Import FrameMaker Wizard, invokes the Import Progress dialog and proceeds to import the FrameMaker files.
  • Cancel: Closes the Import FrameMaker Wizard and does not import anything.

Test Cases

  • Columns are sortable (A-Z and Z-A)
  • Horizontal and vertical scrollbars should be working correctly
  • Should be able to multi-select files using <Shift>-Click or <CTRL>-Click
  • Double-clicking a style in the left-hand list will move it to the right-hand list
  • Double-clicking a style in the right-hand list will remove it from the right-hand list

Import FrameMaker Wizard dialog 4: Options

UI Controls

  • Add "Topic Continued" links when appropriate: Select this option if you want a link called "Topic Continued" to be placed at the bottom of pages when a long topic has been split into multiple topics.
    • Cross-Reference Format: Use this field to specify the format for the "Topic Continued" links.
    • Edit: If you want to modify the cross-reference format provided, click this button, which opens the Cross-Reference Format dialog.
  • Add "Topic Continued From…" links when appropriate: Select this option if you want a link called "Topic Continued From" to be placed at the top of continued pages when a long topic has been split into multiple ones.
    • Cross-Reference Format: Use this field to specify the format for the "Topic Continued From" links.
    • Edit: If you want to modify the cross-reference format provided, click this button, which opens the Cross-Reference Format dialog.
  • Split Long Topics: Select this option if you have long source documents and want to make sure that they are converted to multiple topics
    • Threshold: Enter the maximum number of characters to be converted to a topic before a new topic is created.
  • Avoid Creating Empty Topics: Select this option if you want to ensure that new topics are not created when large sections are found in the FrameMaker documents without any content.
    • Threshold: Enter the maximum number of empty character spaces allowed in a topic.
  • Generate Images for Anchored Frames when needed: Use this option to generate new images during the import process when an anchored frame includes text (such as a callout).
  • Preserve Image Size: If this option is enabled, the <img> tags in the imported files are modified to closely reflect the height and width of the image in the FrameMaker document. If disabled, all images are referenced at 100%.
  • Auto-reimport before 'Generate Output': Also called "Easy Sync," this option will tell Flare to automatically re-import FrameMaker files when you attempt to build output.
  • Approximate Filename Length: Enter the maximum number of characters to use for naming new topic files that are automatically created after splitting a long topic.
  • Convert equations to MathML: Importing from Frame version 11 and earlier, equations created in Frame are written in MathFullForm, a markup unique to FrameMaker. Selected by default, this will convert equations to the web-standard MathML.  Deselected, equations will import as images.
  • Convert Table Styles: If you have tables in your FrameMaker documents that you have formatted in a certain way, select this check box if you want to create matching table styles as a result of the import
  • Enable 'Passthrough' Markers: If you have created passthrough markers in your FrameMaker source documents, enable this check box to be sure that information is imported.
    • Passthrough Marker Format: After you enable passthrough markers, click the down arrow in this field and select the type of format that you want to use for the import (text, fragment or xml)
  • Back: Returns to the previous dialog in the Import FrameMaker Wizard.
  • Next: Proceeds to the next dialog in the Import FrameMaker Wizard.
  • Finish: Closes the Import FrameMaker Wizard, invokes the Import Progress dialog and proceeds to import the FrameMaker files.
  • Cancel: Closes the Import FrameMaker Wizard and does not import anything.

Import FrameMaker Wizard dialog 5: Stylesheet

UI Controls

  • Stylesheet...: Produces the Open dialog which lets you browse for a *.css file.
  • Source Styles:
    • Preserve FrameMaker Styles: This retains any styles from your FrameMaker documents so that you can continue to use them in your new project.
    • Don't Preserve FrameMaker Styles: This does not keep the styles that were used in the FrameMaker documents.
    • Conversion Styles: Produces the Import Styles Editor. See "Frame Import Styles - UI"
  • Back: Returns to the previous dialog in the Import FrameMaker Wizard.
  • Next: Proceeds to the next dialog in the Import FrameMaker Wizard.
  • Finish: Closes the Import FrameMaker Wizard, invokes the Import Progress dialog and proceeds to import the FrameMaker files.
  • Cancel: Closes the Import FrameMaker Wizard and does not import anything.

Import FrameMaker Wizard dialog 6: Paragraph Style Mapping

UI Controls

  • Left-hand Columns
    • FrameMaker Style: Lists all styles used in the FrameMaker documents you have added to the import.
    • Flare Style: Displays the style name that the FrameMaker style will adopt after the documents are imported.
  • Right-hand Column
    • Flare Styles: List of Flare styles available to be mapped to.
  • Map: Lets you map the selected FrameMaker Style(s) on the left to the selected Flare Style from the far right.
  • Unmap: Lets you remove the mapped Flare Style from the selected FrameMaker Style(s).
  • (Left) Preview: Displays a preview of the style selected in the FrameMaker Style list.
  • (Right) Preview: Displays a preview of the style selected in the Flare Styles list.
  • Back: Returns to the previous dialog in the Import FrameMaker Wizard.
  • Next: Proceeds to the next dialog in the Import FrameMaker Wizard.
  • Finish: Closes the Import FrameMaker Wizard, invokes the Import Progress dialog and proceeds to import the FrameMaker files.
  • Cancel: Closes the Import FrameMaker Wizard and does not import anything.

Test Cases

  • Columns are sortable (A-Z and Z-A)
  • Horizontal and vertical scrollbars should be working correctly
  • Should be able to multi-select files in the left-hand list using <Shift>-Click or <CTRL>-Click
  • Should not be able to multi-select files in the right-hand list
  • Double-clicking a style in the left-hand list will do nothing
  • Double-clicking a style in the right-hand list will map it to the current selection in the left-hand list

Import Progress

Accept Imported Documents

UI Controls

  • Generated documents

    Columns

    • File: Displays each new Flare file that will be created as a result of the import.
    • Type: Displays the file extension of each file.
    • Folder: Displays the location where the file will exist in the Flare project.

    Test Cases

    • Columns are sortable (A-Z and Z-A)
    • Horizontal and vertical scrollbars should be working correctly
  • (Preview): Displays a preview (if available) of the file selected on the left.
  • Show Source Code: Shows the source code of the selected file in the preview box.
  • Accept: Closes the dialog and closes the Import FrameMaker Wizard. The resulting Flare project should now be open.
  • Cancel: Closes the dialog, does not complete the import and returns to the Import FrameMaker Wizard

 

Once the FrameMaker Import is complete a FrameMaker Import file is created in the Project Organizer. An alternative to importing FrameMaker files to create a new project is to create the FrameMaker Import file in an existing project. See Import Frame - Add File.
