DITA Import Editor: Local Toolbar

Controls:

  • Import/Reimport...: Imports or re-imports the set of *.dita and *.ditamap files added to the DITA Import file.

DITA Import Editor: Source Files tab

UI Controls

  • DITA Files grid:

    Columns

    • DITA Files: Displays each *.dita or *.ditamap file added to the import
    • Status: ???
  • Open File: Opens the selected file in the default application. Button will gray out with a multi-selection.
  • Link Generate Files To Source Files: This option creates a connection between the original DITA files and the files that are created as a result of the import.
  • : This moves the selected document higher in the list (if you have more than one document to import). The document at the top is used for the name of the content folder holding the imported topics in Flare. Also, the order determines how the imported documents are arranged in the Flare TOC that is created as a result.
  • : This moves the selected document lower in the list (if you have more than one document to import).
  • : This removes the selected document(s) from the list.
  • : Produces the Open dialog which lets you browse for *.dita or *.ditamap files to add to the import.

DITA Import Editor: Options tab

UI Controls

  • Import all Content files to one folder: Select this option if you want all of the imported DITA file content to be placed in just one folder in the Content Explorer.
  • Auto-reimport before 'Generate Output': Also called "Easy Sync," this option will tell Flare to automatically re-import DITA files when you attempt to build output.
  • Preserve ID attributes for elements: Every element in a DITA file can have an ID attribute. Since Flare does not use these, select this option to make sure the ID is preserved.

DITA Import Editor: Stylesheet tab

UI Controls

  • Conversion Styles...: Produces the DITA Import Styles Editor. See "DITA Import Styles - UI" on page 1
  • Stylesheet...: Produces the Open dialog which lets you browse for a *.css file.

DITA Import Editor: Imported Files tab

UI Controls

  • Imported Files grid:

    Columns

    • Imported File: Displays each project file created from the DITA import.
    • Folder: Displays the location of the imported file within the project.
  • Open: Opens the selected file(s)
