Import DITA - Location & Properties

The DITA Import Editor can be used to view and edit settings of a set of DITA documents that was imported to create a new project. Or, you can add a new DITA Import file to import DITA files into a pre-existing project. In both cases, the DITA Import Editor also lets you change import settings and re-import content.

DITA Import File - Location in Project

DITA Import files are saved to the Project/Imports folder.
