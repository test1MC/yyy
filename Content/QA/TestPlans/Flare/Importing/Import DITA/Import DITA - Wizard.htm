Import DITA - Wizard

There are two ways to import DITA content:

  • Option 1: Import DITA content to create a new project with the DITA Import Wizard
  • Option 2: Import DITA content into an existing project by adding a DITA Import File

 

This test module covers Option 1.

Importing DITA Wizard - UI Test Cases

To create a new project by importing DITA files, launch the Import DITA Wizard by:

  • (Standard Toolbar) !File | Import Project | DITA Document Set…
  • (Ribbon) !File | New Project | DITA Document Set

Import DITA Wizard dialog 1: Import Options

UI Controls

  • Project Name: Lets you enter a name for the project that will be created from the imported documents.
  • Project Folder: Lets you set the location of the project
  • Output Type: Lets you select the primary target for the project
  • Back: Returns to the previous dialog in the Import DITA Wizard. (Should be grayed out at this point)
  • Next: Proceeds to the next dialog in the Import DITA Wizard.
  • Finish: Closes the Import DITA Wizard, invokes the Import Progress dialog and proceeds to import the Microsoft Word files. (Should be grayed out at this point)
  • Cancel: Closes the Import DITA Wizard and does not import anything.

Import DITA Wizard dialog 2: Select DITA documents to import

UI Controls

  • Selected DITA documents

    Columns

    • DITA Files: Displays each *.dita or *.ditamap file added to the import

Import DITA Wizard dialog 3: Options

UI Controls

  • Import all Content files to one folder: Select this option if you want all of the imported DITA file content to be placed in just one folder in the Content Explorer.
  • Auto-reimport before 'Generate Output': Also called "Easy Sync," this option will tell Flare to automatically re-import DITA files when you attempt to build output.
  • Preserve ID attributes for elements: Every element in a DITA file can have an ID attribute. Since Flare does not use these, select this option to make sure the ID is preserved.
  • Back: Returns to the previous dialog in the Import DITA Wizard.
  • Next: Proceeds to the next dialog in the Import DITA Wizard.
  • Finish: Closes the Import DITA Wizard, invokes the Import Progress dialog and proceeds to import the DITA files.
  • Cancel: Closes the Import DITA Wizard and does not import anything.

Import DITA Wizard dialog 4: Stylesheet

UI Controls

  • Conversion Styles...: Produces the DITA Import Styles Editor. See "DITA Import Styles - UI" on page 1
  • Stylesheet...: Produces the Open dialog which lets you browse for a *.css file.
  • Back: Returns to the previous dialog in the Import DITA Wizard.
  • Next: Proceeds to the next dialog in the Import DITA Wizard.
  • Finish: Closes the Import DITA Wizard, invokes the Import Progress dialog and proceeds to import the DITA files.
  • Cancel: Closes the Import DITA Wizard and does not import anything.

Import DITA Wizard dialog 5: Ready to start the import

UI Controls

  • Back: Returns to the previous dialog in the Import DITA Wizard.
  • Next: Proceeds to the next dialog in the Import DITA Wizard. (Should be grayed out at this point)
  • Finish: Closes the Import DITA Wizard, invokes the Import Progress dialog and proceeds to import the DITA files.
  • Cancel: Closes the Import DITA Wizard and does not import anything.

Import Progress

 

Accept Imported Documents

UI Controls

  • Generated documents

    Columns

    • File: Displays each new Flare file that will be created as a result of the import.
    • Type: Displays the file extension of each file.
    • Folder: Displays the location where the file will exist in the Flare project.

    Test Cases

    • Columns are sortable (A-Z and Z-A)
    • Horizontal and vertical scrollbars should be working correctly
  • (Preview): Displays a preview (if available) of the file selected on the left.
  • Show Source Code: Displays the source code (if available) of the file selected on the left.
  • Accept: Closes the dialog and closes the Import DITA Wizard. The resulting Flare project should now be open.
  • Cancel: Closes the dialog, does not complete the import and returns to the Import DITA Wizard

Once the DITA Import is complete a DITA Import file is created in the Project Organizer. An alternative to importing DITA files to create a new project is to create the DITA Import file in an existing project.
