The Imported Project

Verifying a successful import

After importing a RoboHelp project you should compare the Flare project to the original RoboHelp Project

  • Compare the original project with the new one
  • Compare the original output with the new projects output

 

Here is a list of things to check that import correctly (Make your own RH projects if necessary)

  • Topics (Specifically the content and formatting of the content)
  • Lists
    • Bulleted
    • Numbered
    • Images
  • Hyperlinks
  • Bookmarks
  • Index’s
  • Defined targets (Flash help not supported)
  • TOC
  • Glossary
  • Variables
  • Dictionary
  • Images in topics
  • Tables
  • Colors
    • Font
    • Background
  • Headers
  • Footers
  • Styles (if you selected the correct option above)
  • Folder structure is maintained
  • All topics are converted correctly to XML
  • Merged Projects (slaves) are imported as well
  • Publishing destinations
  • Shortcuts
  • Anything not listed here! (We import EVERYTHING except skins and Flash Help)
