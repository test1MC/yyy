Import RoboHelp

Bug Base Product Area:

  • Import: RoboHelp

 

Description:

Flare can import any of these file types: *.mpj, *.hhp and *.xpj. Flare should be able to correctly import everything from a RoboHelp project with the exception of Skins (and Flash Help stuff, since we don’t support Flash Help).

 

Test Plans:

  • Import RoboHelp - Baseline
  • Import RoboHelp - Wizard

 

Dependencies:

RoboHelp projects will be needed to import. It would be helpful to have RoboHelp installed; however, very few machines are able to have it installed.
