Bookmarks

Bug Base Product Area:

  • Links: Hyperlinks/Bookmarks

 

Description:

A bookmark is a marker, or flag, that lets you create hyperlinks to specific locations within topics (rather than to another topic file in general). You can insert a bookmark at a specific location in your topic (e.g., at a subheading) and then insert a hyperlink that "connects" to that bookmark. This is a useful feature, for example, if you have a somewhat lengthy topic and want the user to be able to locate a specific place in the topic quickly.
