Bookmarks - UI

Opening the UI

  • Tool Strip interface: !Insert | Bookmark
  • Ribbon interface: !Insert | Bookmark

Manage Bookmarks Dialog - UI Test Cases

UI Controls:

  • New bookmark: Type the name of the bookmark.
  • Rename: Rename a bookmark(grayed out when inserting a new bookmark)
  • Add: Adds the bookmark to the topic and displays a bookmark icon at the location(as long as your markers are turned on.)
  • Delete: Delete the selected bookmark(grayed out when inserting a new bookmark)
  • GoTo: GoTo the selected bookmark(grayed out when inserting a new bookmark)
  • Close: Closes the dialog without adding a bookmark to the topic.

 

Test Cases:

  • Insert in paragraphs selected in various ways (Partial selection, just the insertion point, entire selection)
  • Insert in various formatted paragraphs (Paragraph styles, character styles, Local formatting, etc)
  • Insert in paragraphs with various elements in them (multimedia, Form elements, Special Characters, Help Controls, etc)
  • Insert with various elements selected (Table Cells, Lists, Divs, Drop-down/Expanding areas, etc.)
  • Insert in many types of Elements (Tables, Forms, Textboxes, Marquees, Lists, Drop-down/Expanding areas, etc.)
  • Make sure Addappears and not move when inserting a new bookmark into a topic for the fist time.

Manage Bookmarks Dialog when a bookmark already exist in a topic.

Place your cursor in a different location and select !Insert|Bookmark to display the Manage Bookmark Dialog.

 

UI Controls:

  • New bookmark: (do not use spaces): Type the name of the bookmark.
  • Rename: Rename a bookmark(grayed out when inserting a new bookmark)
  • Move: Move the selected bookmark to the new location.
  • Delete: Delete the selected bookmark(grayed out when inserting a new bookmark)
  • GoTo: GoTo the selected bookmark(grayed out when inserting a new bookmark)
  • Close: Closes the dialog without adding a bookmark to the topic.

 

Test Cases:

  • Insert in paragraphs selected in various ways (Partial selection, just the insertion point, entire selection)
  • Insert in various formatted paragraphs (Paragraph styles, character styles, Local formatting, etc)
  • Insert in paragraphs with various elements in them (multimedia, Form elements, Special Characters, Help Controls, etc)
  • Insert with various elements selected (Table Cells, Lists, Divs, Drop-down/Expanding areas, etc.)
  • Insert in many types of Elements (Tables, Forms, Textboxes, Marquees, Lists, Drop-down/Expanding areas, etc.)
  • Make sure Move appears and not Add when inserting a bookmark after a bookmark is already in a topic.
