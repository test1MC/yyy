UI

Insert Footnote/Endnote Dialog

Controls

In the first part of the dialog, you can select the location where the accompanying footnote or endnote will be placed in the output. However, it is best practice to set the location in the style sheet and apply it to your entire project.

  • End of Page This places the footnote comment at the end of the page where the in-text footnote number is inserted.
  • End of Topic This places the footnote comment at the end of the topic where the in-text footnote number is inserted. This is similar to the previous option. However, a single topic may result in multiple pages in the output. This option ensures that the footnote is placed at the end of the topic, not necessarily on the same page where the corresponding number or symbol is placed.
  • End of Section This places the footnote comment at the end of the section where the footnote number is inserted. You can create section breaks in the TOC.
  • End of Chapter This places the footnote comment at the end of the chapter where the footnote is inserted. You can create chapter breaks in the TOC.
  • Endnote Proxy This is designed to place the footnote comment at a specific location in the output, such as at the end of the book. It works in conjunction with an endnotes proxy, which you create separately. The comment displays wherever you insert the endnotes proxy.

Footnote Properties Footnote Dialog -Right click a footnote in WSY and select Footnote Properties

Controls

 

  • Comment position: (Footnote tab, Position field) You can change the location where a footnote comment is placed in the output (e.g., end of page, document, chapter, section, book).
  • Number format or symbol: (Footnote tab, Format fields) By default, a decimal number (e.g., 1, 2, 3) is used when you insert a footnote. However, you can select a different format. For example, you can change the format to uppercase alpha (e.g., A, B, C) or lowercase Roman (e.g., i, ii, iii). You can also replace the number with a symbol, such as an asterisk.
  • Number: (Footnote tab, Style CSS Span Class fields) You can change the look of the footnote numbers (both in-text and in-comment), by creating and modifying <span> style classes. Alternatively, you can use the <MadCap|footnote> style or a class of it to modify the look of both the in-text and in-comment footnote numbers. The difference is that the <span> style lets you change the look of one or the other, rather than both at the same time.
  • Number restart: (Footnote tab, Number field) When you insert footnotes, the numbers are incremented throughout your manual. However, you can restart the numbering at a certain location (e.g., at the beginning of the next chapter or section).
  • Style (CSS'span' class for number): Footnote number in the topic itself. 
  • Style(CSS 'span' class for in-comment number, defaults to in-text style): in the footnote area of the topic where the footnotes are listed.

Footnotes Font Dialog

 

  • Comment font settings: (Font tab) You can modify the font in a footnote comment. This includes changing the font family, size, color, case, and more. See Editing Footnote Comment Font Settings.

Footnotes Advanced Dialog

Controlse

  • Comment spacing: (Advanced tab) You can modify the spacing for the footnote comment text. This includes the ability to change the spacing above or below the comment text, the white space between words or characters, or the indentation before the number and text.
  • Complete Letter Spacing and/or Word Spacing sections: For either section, you can select Expanded or Condensed and enter a number in the field below. Watch the preview area at the bottom to see how the settings affect the spacing.
