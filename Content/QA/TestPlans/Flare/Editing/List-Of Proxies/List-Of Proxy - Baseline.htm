List-of/List-of Proxies - Baseline

  • Add several class styles to each of the available styles <p>, <a>. Verify you can select them and that they work correctly.
  • Use tagged text that’s contained in tables, conditions, headings, etc. Verify the results are what is expected.
  • Create several style sheet classes for the generated lists. Make sure those styles are applied correctly in all outputs.
  • Create several Classes for MadCap|listOfProxy. Alter the Margins, Borders to verify the entire list is affected by these changes.
  • Verify the page number is correct in the generated list in print outputs.
  • Make changes to the leaders to alter how they look in print outputs.
  • You're only limited to the styles you make in the CSS.
  • Be sure to test on <DIV> objects and its text.
