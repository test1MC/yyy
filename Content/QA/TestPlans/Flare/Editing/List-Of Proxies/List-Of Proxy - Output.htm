Support in DITA Output:

  • DITA does offer an equivalent element to a List-Of Proxy. When generating DITA output, confirm that we simply strip the proxy from the topic. Verify that the output topic which contained the stripped List-Of Proxy is output as valid DITA.

Compiler errors:

  • Including a List-Of Proxy in DITA output:
    MadCap:ListOfProxy found in file: "Topic.htm". It will be removed from the output.
