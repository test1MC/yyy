Additional condition tag tests for List-Of Proxies:

Apply condition tags to one or more of the elements that are to be included in the generated List-Of Proxy. That is, if your List-Of Proxy is set to list all p.figureCaption tags, condition out at least one of those elements and confirm it is not included in the generated List-Of Proxy in the output.

Also, condition out some inline text in such elements.

Code

No Class:

<MadCap:listOfProxy style="mc-list-of-tag: caption;" />

 

With Class and Conditions and all properties set:

<MadCap:listOfProxy style="mc-list-of-tag: caption;mc-list-of-class: figures;mc-list-of-paragraph-class: class1;" class="class1" />
