Text Popup - Editing

To edit the Text Popup, place the insertion point in either the Popup Head and type.

 

 

 

 

 

 

 

 

Code

No Class:

<MadCap:popup>
    <MadCap:popupHead>Click Me!</MadCap:popupHead>
    <MadCap:popupBody>popupBody content</MadCap:popupBody>
</MadCap:popup>

 

With Class:

<MadCap:popup class="StyleClass">
    <MadCap:popupHead>Click Me!</MadCap:popupHead>
    <MadCap:popupBody>popupBody content</MadCap:popupBody>
</MadCap:popup>
