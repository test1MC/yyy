Text Popup - Output

Support in Web Outputs:

Support in Web Help Outputs:

  • Verify Text Popup works for all web output types as well as topic previews (no java script errors).
    • Test with "Condense JavaScript Files" disabled in the Performance tab of the target
    • Test with "Condense JavaScript Files" enabled in the Performance tab of the target
  • Verify that the entire <MadCap:popup> tag and all of its properties are removed when conditionalized out.
  • Verify that applied styles are in outputs

Support in Web Help Mobile Outputs:

  • Popups are not supported by Mobile devices so Text Popups must be converted to topics for Mobile output. Verify that Text Popups output to Mobile create topics in the "Content\MCGeneratedPopups" folder in the output.
  • Identical Text Popups should not create multiple topics. Verify all references to identical Text Popups generate working links.
  • Test with all configurations of Advanced target options (lowercase, use underscores, etc)
  • Verify Text Popup works for all web output types as well as topic previews (no java script errors).
    • Test with "Condense JavaScript Files" disabled in the Performance tab of the target
    • Test with "Condense JavaScript Files" enabled in the Performance tab of the target
  • Verify that the entire <MadCap:popup> tag and all of its properties are removed when conditionalized out.
  • Verify that applied styles are in outputs

Support in Print Outputs:

Support in PDF Outputs:

  • The behavior of Popup text in print outputs is determined by the Popup Text Effects option in the Target Editor in the Printed Output tab.

 

  • Verify that Text Popup is converted to footnotes on selecting "Convert to Footnotes*"
  • Verify that Text Popup is converted to expdanded text inline on selecting "Expand Text Inline"
  • Verify that Text Popup has the body removed on selecting "Remove the Body"

* If "Convert to footnotes" is selected, the generated footnote's format, style, position, etc can be customized in the Stylesheet. See Text Popup - Styles
