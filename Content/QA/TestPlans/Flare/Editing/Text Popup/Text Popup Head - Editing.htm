Text Popup Head - Editing

To edit the Text Popup, place the insertion point in either the Popup Head and type.

 

 

 

Selecting:

You should not be able to select the MadCap:popupHead tag (aside from selecting the content of the tag).

 

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

You should not be able to cut or copy the MadCap:popupHead tag (aside from copying the selected content of the tag).

 

Drag and Drop

You should not be able to drag the MadCap:popupHead tag outside of its parent MadCap:popup tag.

 

Deleting

You should not be able to delete the MadCap:popupHead tag aside from deleting the entire MadCap:popup tag.

 

 

Conditional Tags

You should not be able to apply a conditional tag to the MadCap:popupHead tag. You should only be able to apply a condition to the parent MadCap:popup tag.

 

Code

No Class:

<MadCap:popup>
    <MadCap:popupHead>Click Me!</MadCap:popupHead>
    <MadCap:popupBody>popupBody content</MadCap:popupBody>
</MadCap:popup>

 

With Class:

<MadCap:popup class="StyleClass">
    <MadCap:popupHead>Click Me!</MadCap:popupHead>
    <MadCap:popupBody>popupBody content</MadCap:popupBody>
</MadCap:popup>
