Page Layouts - Editor

Opening a Page Layout File

Any Page Layout file:

 

The Page Layout file associated with a topic:

Open a topic in Print Layout mode and:

  • Select Edit Page Layout Document from the XML Editor's "Layout" pull down menu in the local toolbar
  • Right-click anywhere in the margin area of the page and select Edit Page Layout Document
  • !View | Layout Mode | Edit Page Layout Document
  • Locate the topic in the TOC. Select that TOC node and view its Properties. In the Printed Output tab, click the "Open File" button next to the specified Page Layout file for the chapter or page layout break.

The Master Page Layout:

  • CTRL-Shift-F8
  • Target Editor: In the Basic tab of the Target Editor, click the "Edit" button next to Master Page Layout
  • Project Properties dialog: !Project | Project Properties to open the dialog, then in the Defaults tab, click the "Edit" button next to Master Page Layout
  • Tool Strip interface:
    • Click "Open Master Page Layout" in the Project Toolbar
    • !View | Open Master Page Layout
  • Ribbon interface:
    • !Project | Master Page Layout

Editing a Page Layout File

Local Toolbar:

 

Button (Tooltips in Parentheses):

  • (File Properties): Opens the Properties dialog for the current Page Layout file.

  • (Page Properties…): Invokes the Page Layout "Page" drop-down menu:

    • Copy Page: Places a copy of the open page on the clipboard so that you can paste it, thus creating a copy of the page.
    • Paste Page: Creates a copy of a page, based on the most recent page that you have added to the clipboard (using the Copy option).
    • Duplicate Page: Creates a duplicate of the open page. Copying a page accomplishes the same results as duplicating a page. The only difference is that duplicating a page requires one less step.
    • Remove Page: Removes the open page from the page layout.
    • Add Page: Invokes a sub-menu which lets you select a specific page type to add to the page layout file.
    • Mirror Page: Creates a new page, mirroring the settings of the open page
    • Page Properties: Opens the Page Properties dialog for the open page

     

     

  • (Run Properties Dialog for this item): Opens the Frame Properties dialog for the selected frame.

  • (New Frame Mode): Invokes the Page Layout "New Frame" drop-down menu:

    • New Frame Mode: The first time you select this option, it will allow you to draw a body frame. The second time you select this option, it will allow you to draw a header frame. The third time, it will allow you to draw a footer frame. And the fourth time, it will allow you to draw a decoration frame.
    • New Header Frame Mode: This option allows you to draw a header frame
    • New Footer Frame Mode: This option allows you to draw a footer frame.
    • New Body Frame Mode: This option allows you to draw a body frame.
    • New Decoration Frame Mode: This option allows you to draw a decoration frame.
    • New Image Frame Mode: This option opens the Insert Picture dialog and lets you insert a decoration frame with the selected image set as the background.

     

     

  • (Select Mode): Enables "Select" mode so that you can click on frames that appear in the editor.
  • (Snap frames to other display positions): If this button is enabled, moving and resizing frames will snap to other items on the page (center and edges of other frames as well as the center and edges of the page itself and its margins).
  • (Edit Text): Lets you add or modify text in the frame. This button is enabled only if you first select a header, footer, or decoration frame.
  • (Pick Content Template): Opens the Select Frame Text dialog, which lets you choose an existing frame text snippet for a frame.

  • Layout: Invokes the Page Layout "Layout" drop-down menu:

    • Rotate: Opens another submenu with options for rotating frames.
      • Rotate Clockwise: Turns the frame 1 degree clockwise.
      • Rotate 45 Clockwise: Turns the frame 45 degrees clockwise.
      • Rotate 90 Clockwise: Turns the frame 90 degrees clockwise.
      • Rotate Counter Clockwise: Turns the frame 1 degree counter clockwise.
      • Rotate 45 Counter Clockwise: Turns the frame 45 degrees counter clockwise.
      • Rotate 90 Counter Clockwise: Turns the frame 90 degrees counter clockwise.
      • Reset Rotation: Returns the frame to its original setting before it was rotated.
    • Depth: Opens another submenu with options for floating and sinking frames
      • Float: Floats the selected frame, bringing it forward one level if at least one other frame overlaps it (above it).
      • Sink: Sinks the selected frame, sending it backward one level if at least one other frame overlaps it (below it).
      • Sink to Bottom: Sinks the selected frame, sending it backward to the bottom level if at least one other frame overlaps it (below it).
      • Float to Top: Floats the selected frame, bringing it forward to the top level if at least one other frame overlaps it (above it).
    • Align on Canvas: Opens another submenu with options for aligning the frames on the page canvas.
      • Center on Canvas: Centers the selected frame(s) both vertically and horizontally on the canvas.
      • Vertically Center on Canvas: Centers the selected frame(s) vertically on the canvas.
      • Horizontally Center on Canvas: Centers the selected frame(s) horizontally on the canvas.
    • Make Same: Opens another submenu with options for aligning frames on the page in relation to each other. All positioning and resizing options below are based on the first frame that you select.
      • Same Left: Horizontally aligns the frames so the left border is at the same location.
      • Same Right: Horizontally aligns the frames so the right border is at the same location.
      • Same Top: Vertically aligns the frames so the top border is at the same location.
      • Same Bottom: Vertically aligns the frames so the bottom border is at the same location.
      • Same Width: Resizes the width of the selected frames so that they are all the same.
      • Same Height: Resizes the height of the selected frames so that they are all the same.
      • Same Size: Resizes the width and height of the selected frames so that they are all the same.
      • Same Center: Vertically aligns the selected frames so that the center of each is at the same location.
      • Same Middle: Horizontally aligns the selected frames so that the middle of each is at the same location.
    • Vertically Align: Opens another submenu with options to align the text in a frame vertically.
      • Top: Aligns the text horizontally along the top edge of the frame.
      • Middle: Aligns the text horizontally at the middle point of the frame.
      • Bottom: Aligns the text horizontally along the bottom edge of the frame.
    • Align: Opens another submenu with options to align the text in a frame horizontally.
      • Left: Aligns the text vertically along the left edge of the frame.
      • Center: Aligns the text vertically at the center point of the frame.
      • Right: Aligns the text vertically along the right edge of the frame.
      • Justify: Spaces the text in the frame so that both the left and right sides are aligned.
      • Default: Resets the alignment of the text in the frame to the default (inherited) setting.

  • View: Invokes the Page Layout "View" drop-down menu:

    • Page Overview Bar: Lets you show or hide the Page Overview bar on the right side of the Page Layout Editor.
    • Anchor Lines: Lets you show or hide anchor lines when an anchored frame is selected. Anchor lines will only display for the currently selected frame.
    • Bleed: Lets you show or hide the bleed area around the outside of the page.
    • Labels: Lets you show or hide page layout frame labels ("header," "body," "footer," or "decoration" that appear in the lower-right corner of the frames).
    • Margin: Lets you show or hide the page margins. You must set margins on the page to see these lines.
    • Padding: Lets you show or hide page layout frame padding. You must set padding values on a frame in order to see these lines.
    • Word Overlay: Displays the page layout in a way that represents how Word output will interpret the layout. Word output will not, for example, allow unique margins for the page frames. So this view will display all frames with the same margins.
    • Hide Vertical Ruler: Lets you show or hide the vertical ruler along the left side of the Page Layout Editor.
    • Hide Horizontal Ruler: Lets you show or hide the horizontal ruler along the top side of the Page Layout Editor.
    • Scale to 100%: Changes the Page Layout Editor view to show the page at 100%.
    • Fit Width: Changes the Page Layout Editor view to fit the width of the page.
    • Fit Height: Changes the Page Layout Editor view to fit the height of the page.

 

 

 

 

 

Overview Bar:

  • This bar on the right side of the editor displays a small rectangle that represents each page in the layout.
  • The circled letter which appears in the center of each rectangle corresponds to the page Type ('L' for left, 'R' for right, 'F' for first, etc)
  • If you hover over a rectangle, a popup displays the Name of the page and the page Type.
  • If you left-click a rectangle, that page opens in the editor so that you can modify it.
  • Right-clicking a rectangle invokes the Overview Bar context menu.

 

 

 

 

 

 

 

 

 

 

Overview Bar Context Menu:

  • Copy Page: Copies the selected page to the clipboard
  • Paste Page: Pastes the page from the clipboard
  • Duplicate Page:Creates a new page which is an exact duplicate of the selected page.
  • Remove Page: Removes the selected page from the page layout file.
  • Add Page: Creates a new page with no frames. The new page will be the same page type as the currently selected page.
  • Mirror Page:Creates a new page which vertically mirrors the selected page. The new page should contain a set of frames equivalent to the selected page but with mirrored margins, padding, etc. The new page type will be the same as the selected unless the page type is "Left" or "Right" in which case, a "Left" page will produce a "Right" page and vice versa.
  • Page Properties: Opens the Page Properties dialog

 

 

Editing Page Layout Pages

Page Properties

Opening Page Properties for the page currently open in the Page Layout Editor:

  • Select Page Properties… from the Page drop-down menu in the Page Layout Editor's local toolbar
  • Right-click anywhere in the Page Layout Editor and select Page Properties…
  • Double-click anywhere in the Page Layout Editor that is not inside a frame

Opening Page Properties for another page:

  • Right-click on the specific page in the Overview Bar and select Page Properties…
Page Properties dialog: All Tabs
  • Page: This drop-down displays the Page Name for the page which is currently being modified by the dialog. If you click the drop-down, you can switch the dialog to edit another page's properties, or you can select <all> to modify properties for every page within the page layout file.
    • Verify that when changing from one page to another that you are prompted to save changes.
    • Verify that if you elect to save changes by clicking "Yes" that changes are saved.
    • Verify that if you elect not to save changes by clicking "No" that changes are not saved.
Page Properties dialog: General Tab

UI Controls

  • Page Size:
    • Page Size: Lets you set standard size for the page (e.g., Letter, Legal) or select (custom) to specify a non-standard width and height.
    • Custom Width: If you selected "(custom)" from the Page Size field, enter a specific width for the page.
    • Custom Height: If you selected "(custom)" from the Page Size field, enter a specific height for the page.
    • Orientation: Lets you set the orientation (portrait or landscape) of the page
  • Margins:
    • Left: Enter the length of the left margin for the page.
    • Right: Enter the length of the right margin for the page.
    • Top: Enter the length of the top margin for the page.
    • Bottom: Enter the length of the bottom margin for the page.
  • Page Name: Lets you provide a name for each page that you create in a layout.
  • Page Type: Lets you assign one of the following types to determine how that page behaves in the output: Normal, First, First Left, First Right, Left, Right, Title, Empty, Empty Left, Empty Right
Page Properties dialog: Background Tab

UI Controls

  • Color:
    • (Gradient): Lets you set the style of the background color to Solid or a gradient (Top to Bottom, Left to Right, etc)
    • Fill Start: Sets the background color (if Solid is used) or the start color if a gradient is used
    • Fill End: Sets the end color if a gradient is used
  • Image:
    • Browse: Opens the Insert Image dialog which lets you browse for an image to use as the background
    • Repeat: Lets you set a repeat style on the background image.
    • (Position) X: Lets you set the horizontal image position.
    • (Position) Y: Lets you set the vertical image position.
Page Properties dialog: Print Marks Tab

UI Controls

  • Bleeds:
    • Left: Enter the length of the left bleed for the page.
    • Right: Enter the length of the right bleed for the page.
    • Top: Enter the length of the top bleed for the page.
    • Bottom: Enter the length of the bottom bleed for the page.

Editing Page Layout Frames

Frame Properties dialog

Opening the Frame Properties dialog:

  • Double-click any Frame
  • Right-click any Frame and select Properties
  • Select a Frame and click the Properties button in the Page Layout Editor's local toolbar
  • Select a Frame and press F4
  • Tool Strip interface: 
    • Select a Frame and click the Properties button in the Standard toolbar
    • Select a Frame and !Edit | Properties
  • Ribbon interface:
    • Select a Frame and !Home | Properties
Frame Properties dialog: Frame Tab

UI Controls

  • Size and Position:
    • Left: Lets you set the horizontal position of the frame (from the left edge).
    • Top: Lets you set the vertical position of the frame (from the top).
    • Width: Lets you set the (horizontal) width of the frame.
    • Height: Lets you set the (vertical) height of the frame.
  • Padding:
    • Left: Lets you set the left padding for the frame.
    • Right: Lets you set the right padding for the frame.
    • Top: Lets you set the top padding for the frame.
    • Bottom: Lets you set the bottom padding for the frame.
  • Anchors:
    • Right: Lets you set the horizontal position of the frame from the right edge (if the right anchor is enabled).
    • Bottom: Lets you set the vertical position of the frame from the bottom (if the bottom anchor is enabled).
    • : Lets you enable/disable anchors for the frame
Frame Properties dialog: Columns Tab

UI Controls

  • Columns:
    • Count: Lets you set the number of columns the frame should be divided into
    • Gap: Sets the width of space that should exist between columns
  • Column Divider:
    • Style: Sets the style of the column divider (a vertical line that will appear centered inside the column gap)
    • Width: Sets the width of the column divider
    • Margin Top: Sets the top margin of the column divider
    • Margin Bottom: Sets the bottom margin of the column divider
    • Color: Sets the color of the column divider
Frame Properties dialog: Background Tab

UI Controls

  • Color:
    • (Gradient): Lets you set the style of the background color to Solid or a gradient (Top to Bottom, Left to Right, etc)
    • Fill Start: Sets the background color (if Solid is used) or the start color if a gradient is used
    • Fill End: Sets the end color if a gradient is used
  • Image:
    • Browse: Opens the Insert Picture dialog which lets you browse for an image to use as the background
    • Repeat: Lets you set a repeat style on the background image.
    • (Position) X: Lets you set the horizontal image position.
    • (Position) Y: Lets you set the vertical image position.
  • Preview: Displays a preview of the background.
Frame Properties dialog: Borders Tab

UI Controls

Borders

  • Left: Lets you set the style of the frame's left border.
  • Right: Lets you set the style of the frame's right border.
  • Top: Lets you set the style of the frame's top border.
  • Bottom: Lets you set the style of the frame's bottom border.
  • : Lets you set the style of the all of the frame's borders simultaneously

Border Radius

  • Top-Left: Lets you set the shape of the of the frame's top-left border. Default value is 0.
  • Top-Right: Lets you set the shape of the of the frame's top-right border. Default value is 0.
  • Bottom-Right: Lets you set the shape of the of the frame's bottom-right border. Default value is 0.
  • Bottom-Left: Lets you set the shape of the of the frame's bottom-left border. Default value is 0.
  • : Lets you set the shape of the all of the frame's borders simultaneously
Frame Properties dialog: Flow Tab

UI Controls

  • Flow: Lets you set the flow or the type of the frame.
Frame Properties dialog: Alignment Tab (Only present for Header, Footer and Decoration Frames)

UI Controls

  • Horizontal Alignment: Lets you set the horizontal alignment of text contained in the frame.
  • Vertical Alignment: Lets you set the vertical alignment of text contained in the frame.

Page Layouts - General Editing Test Cases

Drag and Drop

  • Should be able to click on and drag page layout frames within the Page Layout Editor
  • Should not be able to drag content or project files from the Content Explorer, Project Organizer or File List into the Page Layout Editor

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

  • Right-click on the frame, select Cut or Copy and then Paste
  • Select the frame and CTRL-X or CTRL-C and then CTRL-V
  • Tool Strip interface: 
    • Select the frame and then select !Edit | Cut or Copy and then Paste
    • Select the element and then Cut or Copy and then Paste using the Standard Toolbar buttons
  • Ribbon interface:
    • Select the frame and then select !Home | Cut or Copy and then Paste

Deleting a Frame

  • Right-click the frame and select Delete
  • Right-click the frame and select Cut
  • Select the frame and press Delete
  • Select the frame and press CTRL-X
  • Tool Strip interface: 
    • Select the frame and !Edit | Delete
    • Select the frame and click the Delete button on the Standard Toolbar
  • Ribbon interface:
    • Select the frame and !Home | Delete

Tooltips

  • Hovering over a frame displays the frame flow ("Body Frame," "Header Frame," etc).

Flow Arrows

  • Clicking the arrow between multiple body frames will reverse the flow order between the two frames.  Flow order is FlowIndex="#" in the source code.

Conditional Tags

Conditional Tags can be applied to the actual Page Layout file (*.flpgl), however, as of Flare V9, this tag can only be used for the purposes of Advanced Global Project importing. You cannot condition out a Page Layout file.

You can apply condition tags to content inside a frame and that must be tested.

  • Add condition tags to content in a header, footer and decoration frame and verify that this content can be conditioned out from the output.
