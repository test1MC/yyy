Insert Script Dialog: UI Test Cases

UI Controls

  • Language: select a scripting language (text/javascript, text/jscript, text/vbscript)
  • Script Code: Type in the script code here.
  • Script Link: Select a script file already in the project.
  • Browse...: Browse to a scrip file elsewhere on your network.
  • OK: Accepts the selected style class and inserts the ruler.
  • Cancel: Closes the Dialog without accepting any changes and will not insert a ruler..

 

Test Cases: 

  • Insert in paragraphs selected in various ways (Partial selection, just the insertion point, entire selection)
  • Insert in various formatted paragraphs (Paragraph styles, character styles, Local formatting, etc)
  • Insert in paragraphs with various elements in them (multimedia, Form elements, Special Characters, Help Controls, etc)
  • Insert with various elements selected (Table Cells, Lists, Divs, Drop-down/Expanding areas, etc.)
  • Insert in many types of Elements (Tables, Forms, Textboxes, Marquees, Lists, Drop-down/Expanding areas, etc.)

 

Open Script Dialog

UI Controls

  • Hide Files: Hides the files from view
  • Hide Folders: Hides the folders from view
  • File name: path of file
  • Files of type: type of file you can open from this dialog. These include only Script Files (*.js,*.vb), java script (*.js), and visual basic (*.vs).
