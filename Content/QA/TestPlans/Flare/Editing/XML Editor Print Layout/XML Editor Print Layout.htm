XML Editor: Print Layout

Bug Base Product Area:

  • XML Editor: Print Layout

 

Description:

This test plans covers the XML Editor UI and a few features that affect content in the Print Layout view of the XML Editor. In addition to the basic Print Layout UI, this test plan covers page and column breaks, widow/orphan control, and short line controls.

 

A separate test plan which focuses on the XML Editor in general addresses editing features which apply to both layout views of the XML Editor.
