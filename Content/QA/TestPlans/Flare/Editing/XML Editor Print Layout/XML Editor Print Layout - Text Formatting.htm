XML Editor: Print Layout - Text Formatting

This portion of the test plan covers formatting text to determine how it will appear in the XML Editor in Print Layout mode. Functionality covered in this test plan includes Page Breaks, Column Breaks, Widow/Orphan control, Short/Long Line control and some Hyphenation control.

Formatting an element for Print - Paragraph Properties:

Select the element (p, div, ul, etc) that you want to format in the XML Editor,right click, then select “Paragraph” to open the Paragraph Properties dialog.In Tool Strip mode, you can also select !Format | Paragraph to open the Paragraph Properties dialog.

Breaks Tab

Formatting options in this tab let you determine if and how your content should be split across page breaks (and column breaks if you have a multi-column page layout file). You can also determine if a particular content element should precede or be followed by a page (or column) break.

Hyphenation Tab

Flare has a built-in (English) hyphenation dictionary that will automatically hyphenate words in your content where appropriate. Flare will hyphenate any word it can at the end of each line in Print Layout mode. The Hyphenation Tab of the Paragraph Properties dialog essentially lets you create rules which will tell Flare when not to hyphenate. Creating and editing the Hyphenation dictionary is covered in the Hyphenation test plan.

UI Controls

  • Enable Hyphenation: Select whether you always or never want words to be hyphenated at the end of lines for the selected content.
  • Length of shortest word that should be hyphenated: Enter the minimum number of characters that a word must have in order to be hyphenated for the selected content.
  • The maximum number of characters remaining on the initial line: Enter the minimum number of characters that must remain on the initial line when a word is hyphenated for the selected content.
  • The minimum number of characters carried over to the next line: Enter the minimum number of characters that must be carried over to the second line when a word is hyphenated for the selected content.
  • The maximum number of adjacent lines that can end in a hyphenated word: Enter the maximum number of lines next to each other that are allowed to end with a hyphenated word for the selected content.

Short Line Tab

Short Line options allow you

UI Controls

  • Enable Short Line Elimination: Select whether to enable short line elimination, as well as how it will work.
    • Never: Lines will not be adjusted.
    • Tighten: Flare will try to decrease the kerning (i.e., the white space will become more narrow).
    • Tighten, Loosen: Flare will first try to decrease the kerning (i.e., the white space will become more narrow) to the maximum setting. Then it will try to increase the kerning (i.e., the white space will become wider).
    • Loosen, Tighten: Flare will first try to increase the kerning (i.e., the white space will become wider) to the maximum setting. Then it will try to decrease the kerning (i.e., the white space will become more narrow).
    • Loosen: Flare will try to increase the kerning (i.e., white space becomes wider).
  • Length of "short line": Specify the length at which a line is considered "short," and therefore the short line settings come into play.
  • Length of "long line": Specify the minimum length at which a line is considered "long enough."
  • Step-Kerning By: Specify the number of pixels of kerning should be used incrementally on the entire paragraph until the desired effect is achieved.
  • Maximum to Tighten: Specify the maximum number of pixels to tighten during short line elimination.
  • Maximum to Loosen: Specify the maximum number of pixels to loosen during short line elimination.

Formatting an element for Print - Stylesheet:

All properties described above can also be set in the Stylesheet.

Simplified View

In the Simplified View of the Stylesheet Editor, double-click the element to which you want to apply these properties to open the Properties dialog. The tabs in the Properties dialog that opens from the stylesheet are the same as above: Breaks, Hyphenation and Short Line.
