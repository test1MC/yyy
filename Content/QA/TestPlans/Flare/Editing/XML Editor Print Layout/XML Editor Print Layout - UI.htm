XML Editor: Print Layout - UI

Top open the XML Editor in Print Layout mode, open a topic and select Print Layout from the Layout drop-down menu.

XML Editor Test Cases

Local Toolbar

The local toolbar is the same if the XML Editor is in Web Layout or Print Layout Mode. However, all toolbar functions should be tested in both layouts.

Buttons (Tooltips in parentheses):

  • (Preview compiled topic): Lets you preview the topic and lets you see how the topic will look with condition tags associated with it.
  • (Select the conditional expression for viewing this topic in the XML Editor): Opens the Conditional Text dialog where you can choose to include/exclude conditions from view in this topic. Does not affect the output and resets when you reopen the topic.

  • Layout drop-down: Clicking this button lets you toggle between the Web Layout mode and the Print Layout mode. If you select the down arrow, a sub-menu lets you choose between the following options:

    Layout drop-down menu:

    • Web Layout: Displays how the topic will look in online outputs.
    • Print Layout: This lets you see how the pages will look in print outputs with a page layout applied to it.
    • Edit Page Layout Document: This opens the page layout associated with the topic
    • Edit Master Stylesheet: This opens the style sheet associated with the topic

     

  • Medium drop-down: Clicking the down arrow on this button lets you select a medium so that you can view the topic with the styles from a specific medium.

  • Page Layout drop-down: In Print Layout Mode, clicking the down arrow on this button invokes a sub-menu which lets you select from any of the Page Layout files in the project (for XML Editor display purposes only; selecting different page layouts here will not affect the output). Page Layout files should appear in alphabetical order by directory with Page Layouts in the root Content folder displaying at the top:

    Layout drop-down menu:

    • (All Page Layouts in the project): Displays how the topic will look with the selected Page Layout file.

     

     

     

     

     

     

     

  • (Insert a hyperlink): Opens the Insert Hyperlink dialog, which lets you insert a text hyperlink on the selected text in the topic.
  • (Insert a Cross-Reference): Opens the Insert Cross-Reference dialog, which lets you insert a cross-reference into the topic at the location of the cursor.
  • (Insert a picture): Opens the Insert Picture dialog, which lets you insert an image at the location of the cursor.
  • (Insert a snippet): Opens the Insert Snippet Link dialog, which lets you insert a snippet at the location of the cursor.
  • (Insert a variable): Opens the Variables dialog, which lets you insert a variable at the location of the cursor.

  • (Quick character): If you click the -a- portion of the button, opens the Character dialog which lets you select a special character to insert at the location of the cursor. If you click the down arrow, the Character sub-menu appears:

    Quick-character drop-down menu:

    • Quick Character
      • Insert Quick Character [the character and associated font] (Insert the 'Quick Character' [the character name and associated font]: Inserts the quick character into the topic at the location of the cursor.
      • Non-breaking Space (Insert a non-breaking space character): Inserts a non-breaking space into the topic at the location of the cursor.
      • Non-breaking Hyphen (Insert a non-breaking hyphen character): Inserts a non-breaking hyphen into the topic at the location of the cursor.
    • Favorites: This section displays up to 16 special characters that the user has defined as a "Favorite" using the Character dialog. Hovering over a Favorite character should display a tooltip indicating the character name and the associated font. By default, Flare provides the 16 characters pictured on the left as the out-of-the-box Favorite characters.
    • Recently Used: This section displays up to 16 special characters that the user has recently inserted using the Character dialog. Characters inserted from the Favorites group should not appear as Recently Used (unless that were added to the Favorites from the Recently Used). Hovering over a Recently Used character should display a tooltip indicating the character name and the associated font.
    • More Characters (Insert character): Opens the Character dialog which lets you select a special character to insert at the location of the cursor.
  • (Toggle Index Entry Mode): Turns Index Entry Mode on and off.
  • (Toggle Concept Entry Mode): Turns Concept Entry Mode on and off.
  • (Toggle show space): Turns paragraph and space markers on and off.

  • (Show tags): If you click the <t> portion of this button, the XML Editor switches to "Show Tags" mode, showing the XML tags surrounding the content. If you click the down arrow, you can select the following options:

    Show drop-down menu:

    • Show Caret Tag Neighborhood: Turns this feature on and off.
    • Show Markers: Turns markers on or off.
    • Show Inline Markers: You can show or hide markers for all inline elements in the active topic. This option is only active if the preceding setting, Show Markers, is turned on.
    • Show invisible tags: Turns invisible tags (e.g., title) on or off in the active topic. This option is enabled only if the XML Editor is in "Show tags" mode.
    • Show Comments: Turns comment tags on or off in the active topic.
    • Show All Images As Thumbnails: Lets you show or hide images in your project as thumbnails.
    • Show tags in mono-color: Lets user choose whether to display tags in a color or in black. This option is enabled only if the XML Editor is in "Show tags" mode.
    • Show tags in mono-font: Lets user choose whether to display all tags in the same font or in the font actually applied to each area. This option is enabled only if the XML Editor is in "Show tags" mode.
    • Enable Object Positioning: Turns object positioning on and off.
    • Enable Orphan/Widow Control: Turns widow and orphan control on or off.
    • Show Hidden Elements: Shows content that has been designated as "hidden."
    • Show Conditional Indicators: Toggles between showing and hiding conditional indicators.
    • Show Variable Names: Toggles between showing and hiding variable names.
    • Highlight Variables: Toggles between displaying variables highlighted or not.
    • Show Bookmark Names: Toggles between showing and hiding bookmark names.
    • Show Named Elements: Lets user choose whether to see the yellow flag icon next to named elements in user’s markers.
    • Locks (Disabled/Enabled): Locks or unlocks the content designated with the gray background.
    • Show Locked Elements: Allows you to show or hide the gray background associated with elements that have been locked or selected to be locked.
    • Show Annotations: Toggles between showing and hiding annotations.
    • Marker Width: Lets user adjust the width of markers in the active topic to see more or less of the markers when they are visible.

Lower Toolbar - Print Layout

  • (Word Count:) Accurately tracks the number of words in the topic.

Buttons (Tooltips in parentheses):

  • (Go to the first page): Displays the first page of the topic in Print Layout mode..
  • (Go to the previous page): Displays the previous page of the topic in Print Layout mode..
  • (Go to the next page): Displays the next page of the topic in Print Layout mode..
  • (Go to the last page): Displays the last page of the topic in Print Layout mode..
  • (Page count): Displays the current page of the topic, as well as the total number of pages in the topic. You can enter another number in the field or use the arrow buttons to navigate to a different page.

  • : The Zoom/Scale button (which looks different depending on the current selection) opens a drop-down menu so that you can select from the following options:

    Menu options:

    • 100%: Displays the page at 100% of its actual size
    • 100% (2 pages): Shows two pages of a topic at a time, displaying each one at 100%.
    • Fit Width: Resizes the page so that it displays in proportion to the width of the editor window pane.
    • One Page: Displays only one page at a time.
    • Two Pages: Displays two pages at a time.
    • Four Pages: Displays four pages at a time.
    • Eight Pages: Displays eight pages at a time.

     

     

  • Zoom out (Ctrl + -): Zooms out in the increments of 10% (Minimum is 10%)
  • Zoom in (Ctrl + +): Zooms in in increments of 10% (Maximum is 300%)
  • Set scaling: Lets you select a specific percentage to scale the page.
  • (Hide/show conditional Indicators): Lets you show or hide the colored shading of content to which condition tags have been applied.
  • (Toggle show blocks): Lets you show or hide the tag block bars.
  • (Toggle show spans): Lets you show or hide the span bars.
  • (Toggle show the horizontal ruler): Lets you show or hide a measurement ruler horizontally at the top of the editor. Clicking on the ruler in the XML Editor lets you change the ruler units between pixels, points, centimeters or inches.
  • (Toggle show the vertical ruler): Lets you show or hide a measurement ruler vertically at the left of the editor. Clicking on the ruler in the XML Editor lets you change the ruler units between pixels, points, centimeters or inches.
