Multimedia - Add File

Adding a Multimedia File to the Project

Add File dialog: Multimedia - UI Test Cases

 

Copy to Project Dialog

 

Columns:

  • Action: Displays the action being performed such as copy.
  • File: Displays the name of the file. If the file is renamed projects containing those files may show a missing file so the user will have to re-add the file.
  • Type: Displays the file type.
  • Path: Displays the path where the file is stored. If the file is moved this path should update automatically.

 

Button/Section Description:

  • The following files will be copied to the project:Displays the files to be copied. If the file being added is dependent on other files to operate correctly (e.g., if it is linked to another topic file), those necessary files are also listed in the dialog (and therefore copied as well).
  • Add to Source Control:Select this check box if you want the copied file(s) to be added to the integrated source control application. If you select this box, the Add File(s) dialog will open next, giving you the opportunity to automatically check the files out after they are added to source control.
  • Cancel: Closes the dialog without adding the new multimedia file.

  • Add: Adds the new multimedia file to the project.

    • MadCap Capture Icon:(Edit this file with in MadCap Capture) Opens up MadCap Capture if you have it installed else it will prompt the user that they need MadCap Capture installed.
    • Hovering the mouse over the multimedia name will display the file path.

Test Cases:

  • Insert in paragraphs selected in various ways (Partial selection, just the insertion point, entire selection)
  • Insert in various formatted paragraphs (Paragraph styles, character styles, Local formatting, etc)
  • Insert in paragraphs with various elements in them (multimedia, Form elements, Special Characters, Help Controls, etc)
  • Insert with various elements selected (Table Cells, Lists, Divs, Drop-down/Expanding areas, etc.)
  • Insert in many types of Elements (Tables, Forms, Textboxes, Marquees, Lists, Drop-down/Expanding areas, etc.)

Edit Multimedia Dialog

General Tab

 

UI Controls

  • Select File
    • Multimedia in Project:
      • Show All Files in Project: Shows all of the files in the project in a list in the area below. If you click the button again, it switches to a folder tree view. In the list, you can click the "File," "Type," or "Path" column headers to sort the list alphabetically by that column data.
      • Show/Hide Folders: Shows or hides the folders that the files are stored in.
      • Show/Hide Files: Shows or hides the files. If you click this button when the "Show Folders" button is selected, the area splits into two halves. The folder is shown on the left side, and the files and subfolders within it are shown on the right.
      • Up: If the "Show Files" button is the only one selected, you can click this button to move up one folder level.
      • (Filter): Lets you filter the area below to show only certain kinds of files.
      • (Path Dropdown): Click to view recently used multimedia files from this project.
      • (Browse):Browse for a multimedia file outside of the project.
    • Multimedia from Web: Type or Paste a link to a video from YouTube or Vimeo
  • Options
    • Style Class: Select a style class for the multimedia.
    • Screen Tip: Apply a screen tip to the multimedia.

Test Cases

YouTube Video

UI Controls

  • Auto Hide: The video controls will automatically hide after a video begins playing.
  • Auto Play: The video will play automatically once the player/website loads.
  • Full Screen: The full screen button displays in the lower right corner of the controls.

    • Note: Bug 95312:Full screen button on YouTube videos does not display in Chrome browser

  • Show Related Videos: The player will show related videos when playback of the initial video ends.
  • Annotations: Annotations are clickable text overlays on YouTube videos. https://www.youtube.com/yt/playbook/annotations.html
  • Modest Branding: The YouTube player will not show a YouTube logo on the control bar. Note that a small YouTube text label will still display in the upper-right corner of a paused video when the user's mouse pointer hovers over the player.

    • Note: Bug 94429:Turning "Modest Branding" to "On" is not removing the "YouTube" logo from the video when being viewed from Chrome

  • Show Info: The player will display information like the video title and uploader before the video starts playing.
  • Progress Bar Color: The color that will be used in the player's video progress bar to highlight the amount of the video that the viewer has already seen. Note: Setting the color parameter to white will disable the modestbranding option.
  • Controls: Indicates if the video player controls will display
    • On (Always): Player controls display in the player. For IFrame embeds, the controls display immediately and the Flash player also loads immediately.
    • On (Only during playback): Player controls display in the player. For IFrame embeds, the controls display and the Flash player loads after the user initiates the video playback.
    • Off: Player controls do not display in the player
  • Theme: This parameter indicates whether the embedded player will display player controls (like a play button or volume control) within a dark or light control bar.
    • Dark: dark theme
    • Light: light theme

Vimeo Video

Note: If your embedded video is less than 375px wide, some of the embed elements will be hidden. Mini Mode is activated if the player is smaller than 375px wide. In Mini Mode, the portrait, volume slider, Vimeo logo, and custom outros are hidden. Tiny Mode is activated if the smaller than 300px wide or 170px tall. In Tiny Mode, the portrait, title, control bar, Vimeo logo, Like / Watch Later / Share buttons, and custom outros are hidden. PRO Members: If you’re using a custom logo in the player, the criteria for Mini Mode and Tiny Mode are a bit different. Mini mode is activated when the player is smaller than 450px wide. Tiny mode is activated if the player is smaller than 338px or 190px tall.

Note: If your advanced settings aren't working, it may be because the owner of the video is a Plus member and has chosen to override settings with their preferences. https://developer.vimeo.com/player/embedding#universal-parameters

UI Controls

  • Auto Pause: Enables or disables pausing this video when another video is played. 

    • Note: Bug 94466:"Auto pause" in the "Advanced" features of a Vimeo file does not stop a currently playing movie when another starts

Note: When resizing objects, you can ensure that the aspect ratio is maintained. For example, if you want certain objects to be resized so that each is exactly 3 inches high, you can make sure that the width of each object is adjusted accordingly so that it stays in proportion. To do this, you would first set the height at 3 inches. Then for the width property, you would select Automatic (instead of "Length") from the top drop-down list. In the same way, if you were to specify an exact width, you could maintain the aspect ratio by setting the height to "Automatic."

Minimum:Set a minimum width and/or height:

  • If the original multimedia is smaller than the minimum width or height that is set, it will be enlarged so that it reaches the minimum value. If the original multimedia is larger than the minimum width or height, it will not be resized.

UI

Insert Multimedia Dialog - Position

UI Controls

UI How to position Multimedia:

  1. Open a topic where you have inserted a multimedia.
  2. Right-click on the multimedia and from the menu, select Edit multimedia. The Edit multimedia dialog opens.
  3. Select the Position tab.
  4. In the Position section, you can select a Float and a Clear setting. You can also set the Vertical Alignment of the multimedia.

 

Controls:

 

Float: Use this field to specify where to place the multimedia on the page.

  • None: Does not place the multimedia in a specific location.
  • Left: Positions the multimedia on the left side of the page frame, allowing you to type text to the right of the multimedia.
  • Right: Positions the multimedia on the right side of the page frame, allowing you to type text to the left of the multimedia.
  • Center: of Column Positions the multimedia in the center of the column on the page.
  • Outside Left Margin: Positions the multimedia beyond the left margin of the topic text.
  • Outside Right Margin: Positions the multimedia beyond the right margin of the topic text.
  • Outside Frame Positions: the multimedia outside of the page frame.
  • Outside Frame, Top Align: Positions the multimedia outside of the page frame, as well as aligning it with the top of the frame.
  • Left of Frame: Positions the multimedia to the left of the page frame.
  • Right of Frame: Positions the multimedia to the right of the page frame.
  • Center of Frame: Positions the multimedia both vertically and horizontally in the middle of the page frame.
  • Clear: Use this field to position a multimedia so that it is "clear" of an adjacent multimedia. For example, let's say you have already inserted a multimedia and applied the float left property to it. If you then insert another multimedia immediately after the first multimedia, you want to make sure that the second multimedia doesn't rest next to the first multimedia. Instead, you want the second multimedia to be placed completely below the first multimedia. Therefore, you can apply a clear property to the second multimedia.
  • None: Does not apply the clear property to the multimedia.
  • Left Side: The multimedia will be placed below the bottom outer edge of a previous multimedia that is floating left.
  • Right Side: The multimedia will be placed below the bottom outer edge of a previous multimedia that is floating right.
  • Both Sides: The multimedia will be placed below the a previous multimedia, whether it is floating left or right.

Vertical Alignment: Use this field to adjust where the item is positioned vertically.

  • Baseline: The baseline of the box will be aligned with the baseline of the parent box.
  • Text Top: The top of the box will be aligned with the top of the parent element's font.
  • Text Bottom: The bottom of the box will be aligned with the bottom of the line box.
  • Top: The top of the box will be aligned with the top of the line box.
  • Middle: The vertical midpoint of the box will be aligned with the baseline of the parent box, plus half the x-height of the parent.

  • Bottom: The bottom of the box will be aligned with the bottom of the line box.

      5. Click Ok.

      6. Press CTRL+S or click to save your work.

Note: If you want to adjust the space between the object and the text that flows next to it, you can modify the margins on the object (using the Borders & Margins tab in the dialog).

Insert Multimedia Dialog - Borders & Margins

UI Controls

UI Borders & Margins Controls:

  • Margin: Margin is the distance from the edge of the document to the edge of the content.
    • Click in any of the individual fields (Left, Right, Top, Bottom) to specify the settings for the margins around the multimedia. If you click the down arrow to the right of all the fields, the settings will be applied to all of margin fields. When you click that down arrow, a small popup displays. Use the lower-left area of the popup to enter a number for the amount of margin. Use the lower-right area to select a unit of measurement (e.g., point, pixel, centimeter) for the number you entered. When you are finished, click OK in the small popup.
  • Border: Border is a line drawn around the content.
    • Click in any of the individual fields (Left, Right, Top, Bottom) to specify the settings for the multimedia border. If you click the down arrow to the right of all the fields, the settings will be applied to all of border fields. When you click that down arrow or in one of the individual fields, a small popup displays. Use the lower-left area of the popup to enter a number for the thickness of the border. Use the lower-middle area to select a unit of measurement (e.g., point, pixel, centimeter) for the number you entered. Use the upper-right area to select a color for the border. Use the lower-right area to select a line type (e.g., solid, double, dashed) for the border. When you are finished, click OK in the small popup.
  • Padding:Adds extra space between an multimedia's border and the multimedia itself.
    • Click in any of the individual fields (Left, Right, Top, Bottom) to specify the settings for the padding in the multimedia. In the left side of the field, enter a number for the amount of padding. In the right side of the field, select a unit of measurement (e.g., point, pixel, centimeter) for the number you entered. If you click the down arrow to the right of all the fields, the settings will be applied to all of padding fields. When you click that down arrow, a small popup displays. Use the lower-left area of the popup to enter a number for the amount of padding. Use the lower-right area to select a unit of measurement. When you are finished, click OK in the small popup.
  • Preview:Displays a preview of how a multimedia will look as you specify settings.
  • : sets the top, bottom, left, right all at the same time with the same number.

 

Test: test using the drop down setting selection as well as setting each area manually one at a time and all together. Some combinations may cause a problem. Don't just use the drop down and think that the rest work.

Note: Different browsers may treat margin and padding settings differently. For example, Internet Explorer 8 and Firefox honor padding settings more than they honor margin settings. If you were to set a left margin at, say, 1 inch, Internet Explorer 7 would show it that way. However, in order to get the same results in Internet Explorer 8 or Firefox, you would also need to set the left padding at 1 inch.

Insert Multimedia Dialog - Backgrounds

UI Controls

UI Backgrounds Control

  • Color: Sets the color of the background. In the Color field, click the down arrow and select a color from the popup. For advanced color options, select More Colors and use the fields in the Color Picker dialog.
  • Multimedia: Choose a multimedia for the background. Next to the multimedia field click the drop down menu to choose a recently added multimedia that is already in the project or click the Browse button to insert a new multimedia.
  • Repeat: repeats a background multimedia.
  • X: Select or enter the position (in pixels) of the multimedia in the background. This positions the background multimedia horizontally (i.e., on the X axis).
  • Y: Select or enter the position (in pixels) of the multimedia in the background. This positions the background multimedia vertically (i.e., on the Y axis).
  • Preview: Displays the appearance of the background as you make selections in the tab.

Insert Multimedia Dialog - Position

UI How to position Multimedia:

  1. Open a topic where you have inserted a multimedia.
  2. Right-click on the multimedia and from the menu, select Edit multimedia. The Edit multimedia dialog opens.
  3. Select the Position tab.
  4. In the Position section, you can select a Float and a Clear setting. You can also set the Vertical Alignment of the multimedia.

 

Controls:

 

Float: Use this field to specify where to place the multimedia on the page.

  • None: Does not place the multimedia in a specific location.
  • Left: Positions the multimedia on the left side of the page frame, allowing you to type text to the right of the multimedia.
  • Right: Positions the multimedia on the right side of the page frame, allowing you to type text to the left of the multimedia.
  • Center: of Column Positions the multimedia in the center of the column on the page.
  • Outside Left Margin: Positions the multimedia beyond the left margin of the topic text.
  • Outside Right Margin: Positions the multimedia beyond the right margin of the topic text.
  • Outside Frame Positions: the multimedia outside of the page frame.
  • Outside Frame, Top Align: Positions the multimedia outside of the page frame, as well as aligning it with the top of the frame.
  • Left of Frame: Positions the multimedia to the left of the page frame.
  • Right of Frame: Positions the multimedia to the right of the page frame.
  • Center of Frame: Positions the multimedia both vertically and horizontally in the middle of the page frame.
  • Clear: Use this field to position a multimedia so that it is "clear" of an adjacent multimedia. For example, let's say you have already inserted a multimedia and applied the float left property to it. If you then insert another multimedia immediately after the first multimedia, you want to make sure that the second multimedia doesn't rest next to the first multimedia. Instead, you want the second multimedia to be placed completely below the first multimedia. Therefore, you can apply a clear property to the second multimedia.
  • None: Does not apply the clear property to the multimedia.
  • Left Side: The multimedia will be placed below the bottom outer edge of a previous multimedia that is floating left.
  • Right Side: The multimedia will be placed below the bottom outer edge of a previous multimedia that is floating right.
  • Both Sides: The multimedia will be placed below the a previous multimedia, whether it is floating left or right.

Vertical Alignment: Use this field to adjust where the item is positioned vertically.

  • Baseline: The baseline of the box will be aligned with the baseline of the parent box.
  • Text Top: The top of the box will be aligned with the top of the parent element's font.
  • Text Bottom: The bottom of the box will be aligned with the bottom of the line box.
  • Top: The top of the box will be aligned with the top of the line box.
  • Middle: The vertical midpoint of the box will be aligned with the baseline of the parent box, plus half the x-height of the parent.

  • Bottom: The bottom of the box will be aligned with the bottom of the line box.

Insert Multimedia Dialog - Borders & Margins

UI Borders & Margins Controls:

  • Margin: Margin is the distance from the edge of the document to the edge of the content.
    • Click in any of the individual fields (Left, Right, Top, Bottom) to specify the settings for the margins around the multimedia. If you click the down arrow to the right of all the fields, the settings will be applied to all of margin fields. When you click that down arrow, a small popup displays. Use the lower-left area of the popup to enter a number for the amount of margin. Use the lower-right area to select a unit of measurement (e.g., point, pixel, centimeter) for the number you entered. When you are finished, click OK in the small popup.
  • Border: Border is a line drawn around the content.
    • Click in any of the individual fields (Left, Right, Top, Bottom) to specify the settings for the multimedia border. If you click the down arrow to the right of all the fields, the settings will be applied to all of border fields. When you click that down arrow or in one of the individual fields, a small popup displays. Use the lower-left area of the popup to enter a number for the thickness of the border. Use the lower-middle area to select a unit of measurement (e.g., point, pixel, centimeter) for the number you entered. Use the upper-right area to select a color for the border. Use the lower-right area to select a line type (e.g., solid, double, dashed) for the border. When you are finished, click OK in the small popup.
  • Padding:Adds extra space between an multimedia's border and the multimedia itself.
    • Click in any of the individual fields (Left, Right, Top, Bottom) to specify the settings for the padding in the multimedia. In the left side of the field, enter a number for the amount of padding. In the right side of the field, select a unit of measurement (e.g., point, pixel, centimeter) for the number you entered. If you click the down arrow to the right of all the fields, the settings will be applied to all of padding fields. When you click that down arrow, a small popup displays. Use the lower-left area of the popup to enter a number for the amount of padding. Use the lower-right area to select a unit of measurement. When you are finished, click OK in the small popup.
  • Preview:Displays a preview of how a multimedia will look as you specify settings.
  • : sets the top, bottom, left, right all at the same time with the same number.

 

Test: test using the drop down setting selection as well as setting each area manually one at a time and all together. Some combinations may cause a problem. Don't just use the drop down and think that the rest work.

Note: Different browsers may treat margin and padding settings differently. For example, Internet Explorer 8 and Firefox honor padding settings more than they honor margin settings. If you were to set a left margin at, say, 1 inch, Internet Explorer 7 would show it that way. However, in order to get the same results in Internet Explorer 8 or Firefox, you would also need to set the left padding at 1 inch.
