Multimedia

Bug Base Product Area:

  • Images/Multimedia

 

Description:

You can add many types of multimedia files (both video and audio) to your project, which can then be inserted into topics or snippets. Multimedia files are stored in the Resources\Multimedia subfolder in the Content Explorer, unless you specify otherwise.

 

YouTube/Vimeo links can be inserted into a topic or snippet and w

 

Following are the file types supported.

  • Flash file:
    • .swf, .swf2, .swf3

 

  • Windows Media files:
    • .asf, .midi, .mpeg, .wma, .asx, .mp3, .mpg, .wmv, .au, .mp4, .wav, .wmx, .avi, .mpa, .wax, .mid, .mpe, .wm

 

  • QuickTime files:
    • .mov, .qt

 

  • 3D Models:
    • .u3d
