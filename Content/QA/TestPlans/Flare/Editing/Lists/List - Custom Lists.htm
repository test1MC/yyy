Create New List Format

UI Controls

  • General
    • Name: Name of the class (cannot be left empty)
    • Comment: Lets you add a comment to the style class.
    • Stylesheet: Allows for the selection of the stylesheet to which the new class/style will be added, if the topic is associated with multiple stylesheets.  Otherwise this will be grayed out.
  • Formatting
    • Level: Lets you choose which list level to modify, up to Level 10
    • Content: Content field is blank by default for the bullet list.
    • (List Styles List):  shows a drop down menu
      • (bullet list): Nothing is added to the content field.
      • (numbered list): {n}. is added to the content field.
      • (lower-alpha numbered list): {a}. is added to the content field.
      • (upper-alpha numbered list): {A}. is added to the content field.
      • (lower-Roman numbered list): {r}. is added to the content field.
      • (upper-Roman numbered list): {R}. is added to the content field.
      • (select a custom bullet image): The path to the image file is added to the content field.
    • (Character): Lets you insert a special character or symbol. Clicking the Character icon opens the Character dialog.  Clicking the drop down shows the Character fly-out menu
    • (Font Properties): Opens the Font Properties dialog. This lets you change the look of the list marker, such as the font size or type, color, and effects.
  • OK: Closes the dialog and creates or modifies the list style
  • Cancel: Closes the dialog and does not create or modify the list style

Character

  • Quick Character
    • Insert Quick Character [the character and associated font] (Insert the 'Quick Character' [the character name and associated font]): Inserts the quick character into the topic at the location of the cursor.
    • Non-breaking Space (Insert a non-breaking space character): Inserts a non-breaking space into the topic at the location of the cursor.
    • Non-breaking Hyphen (Insert a non-breaking hyphen character): Inserts a non-breaking hyphen into the topic at the location of the cursor.
  • Favorites (Insert [the character name and associated font]): This section displays up to 16 special characters that the user has defined as a "Favorite" using the Character dialog. Hovering over a Favorite character should display a tooltip indicating the character name and the associated font. By default, Flare provides the 16 characters pictured on the left as the out-of-the-box Favorite characters.
  • Recently Used (Insert [the character name and associated font]): This section displays up to 16 special characters that the user has recently inserted using the Character dialog. Characters inserted from the Favorites group should not appear as Recently Used (unless that were added to the Favorites from the Recently Used). Hovering over a Recently Used character should display a tooltip indicating the character name and the associated font.
  • More Characters (Insert character): Opens the Character dialog which lets you select a special character to insert at the location of the cursor.

 

  • Inserting a character will apply that font to the list's Content field.  Adding a second character from a different font will prompt you to choose which font
    • Yes: Character will be inserted and font will change in the Content field
    • No: Character will not be inserted

Editing in the Stylesheet

Custom lists created through this UI are located in the (Custom Lists) group in the stylesheet, in Advanced View only. Expanding this list, the first level has the list-style-type set, you will see ul.[listName].  This level is identical to ul.[listName] under the ul style. Expanding this, you will see (level 1) - (level 10) which will again show the list-style-type.  Expanding this, you will see (marker) which is where all other properties are set (ex: color, content, etc.).  This level is identical to (Complex Selectors) | ul.[ListName] > li::marker.

  • Rename: will launch Rename Class dialog (note: Bug 51962:No way to rename a style and have it update topics using the style.)
  • Edit Custom Format...: will launch the Create New List Format dialog
  • Cut: Cuts the selected style to the clipboard.
  • Copy: Copies the selected style to the clipboard.
  • Delete: Removes the selected style.
