Lists

Bug Base Product Area:

  • Lists

 

Description:

Flare lets you work with numbered and bulleted lists in a variety of ways, such as the following.

  • Simple lists: You can create simple lists that are numbered or bulleted.
  • Multi-level lists: You can create numbered and bulleted lists with multiple levels (such as an outline). There are multiple ways to create a multi-level list.
  • Continue sequence: Use this to ensure that the next list you create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content).
  • Merge: Use this to combine a list with another list immediately before it.
  • Notes or comments (paragraph item): Use this to include comments between items without interrupting the flow of the list.
  • Restart numbering: Use this to specify a number to start a numbered list, or to start a selected item within a numbered list.
  • Reverse: Use this to reorder the items in a list so that they appear in reverse order (i.e., first item is last, last item is first).
  • Sort: Use this option to reorder the items in a list alphabetically.
  • Unbind (remove): Use this to remove the list designation from the content so that it displays as regular text.
  • Edit list styles: You can modify the look of lists by making adjustments to the tags and style classes used for them. By editing list styles, you can affect things such as the alignment, background, bullet images, and indentation of lists.
