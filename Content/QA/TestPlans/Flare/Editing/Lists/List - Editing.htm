UL/OL/List Context Menu

 

  • Select: Selects the element or the textual content of the element as appropriate
  • Cut: Cuts the selected text/element.
  • Copy: Copies the contents of the element to the clipboard
  • Paste Before: Pastes the content from the clipboard before the selected tag.
  • Paste After: Pastes the content from the clipboard after the selected tag.
  • Unbind [Element]: Unbinds the element, converting it to normal text
  • Delete: Deletes the element
  • Move: Invokes the move submenu
  • Type: Invokes the type submenu
  • Bullet List: Applies a bulleted list to the selected content.
  • Circle Bullet List: Applies a bulleted list to the selected content. The bullets are circles.
  • Square Bullet List: Applies a bulleted list to the selected content. The bullets are squares.
  • Numbered List: Applies a numbered list to the selected content.
  • Lower-alpha Numbered List: Applies a numbered list to the selected content where lowercase alpha characters are used (a., b., c., d., etc.)
  • Upper-alpha Numbered List: Applies a numbered list to the selected content where uppercase alpha characters are used (A., B., C., D., etc.)
  • Lower-roman Numbered List: Applies a numbered list to the selected content where lowercase Roman numerals are used (i., ii., iii., iv., etc.)
  • Upper-roman Numbered List: Applies a numbered list to the selected content where uppercase Roman numerals are used (I., II., III., IV., etc.)
  • Merge With Previous List: Merge the selected list with the previous list
  • Merge With Next List: Merge the selected list with the next list
  • Continue Sequence: Ensures that the next list you create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content).
    • List Start #: Lets you specify a number to start a numbered list. (After you enter a number in this selection, you need to click the blue "enter" arrow at the end to apply the effect.)
  • Sort List: Reorders the items in the list alphabetically.
  • Reverse List: Reorders the items in the list so that they appear in reverse order (i.e., first item is last, last item is first).
  • Unbind List: Removes the list designation from the content so that it displays as regular text.
  • Make Simple Item(s):Removes p tags from the li.
  • Make Paragraph Item(s):Inserts a p into the li.
  • Conditions: Launches the Conditions Tags dialog
  • Name: Name the element
  • Paragraph… (Edit Paragraph Properties): Invokes the Paragraph Properties dialog which lets you set inline paragraph styles for the current block level element.
  • Font… (Font Properties): Invokes the Font Properties dialog which lets you set inline font styles for the current element.
  • Create Snippet: Launches the Create Snippet dialog
  • Style Class: Invokes the Style Class submenu
    • Create Style Class: Launches the Create Style dialog 
    • Edit Style Class: Opens the associated CSS and select the element or element.Class
    • Reset Style Class: Active only if a class is selected. Removes the class from the tag.
    • Class List: Lists all of the element Classes in the linked CSS. The Class with have a check next to it if it is applied to the element
