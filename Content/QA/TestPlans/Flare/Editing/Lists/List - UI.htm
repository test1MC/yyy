List Menu items-shortcuts in parenthesis

List drop down (Alt, o, s, l):

  • List Actions:
    • Merge With Previous List: Merge the selected list with the previous list
    • Merge With Next List: Merge the selected list with the next list
    • Sort List: Reorders the items in the list alphabetically.
    • Reverse List: Reorders the items in the list so that they appear in reverse order (i.e., first item is last, last item is first).
    • Unbind List: Removes the list designation from the content so that it displays as regular text.
    • Continue Sequence: Ensures that the next list you create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content).
      • List Start Number: Lets you specify a number to start a numbered list. (After you enter a number in this selection, you need to click the blue "enter" arrow at the end to apply the effect.)
      • Item Number:Change the numbering for a specific item in a numbered list (After you enter a number in this selection, you need to click the blue "enter" arrow at the end to apply the effect.)
    • Move Up:Move the li up above the li that is currently above it
    • Move Down:Move the li down below the li that is currently below it
    • Make Simple Item(s):Removes p tags from the li.
    • Make Paragraph Item(s):Inserts a p into the li.
  • Bullet List (Alt, o, s, b): Applies a bulleted list to the selected content.
  • Circle Bullet List (Alt, o, s, c): Applies a bulleted list to the selected content. The bullets are circles.
  • Square Bullet List (Alt, o, s, s): Applies a bulleted list to the selected content. The bullets are squares.
  • Numbered List (Alt, o, s, n): Applies a numbered list to the selected content.
  • Lower-alpha Numbered List (Alt, o, s, o): Applies a numbered list to the selected content where lowercase alpha characters are used (a., b., c., d., etc.)
  • Upper-alpha Numbered List (Alt, o, s, u): Applies a numbered list to the selected content where uppercase alpha characters are used (A., B., C., D., etc.)
  • Lower-roman Numbered List (Alt, o, s, w): Applies a numbered list to the selected content where lowercase Roman numerals are used (i., ii., iii., iv., etc.)
  • Upper-roman Numbered List (Alt, o, s, p): Applies a numbered list to the selected content where uppercase Roman numerals are used (I., II., III., IV., etc.)
  • Outdent Items (Alt, o, s, t): This lets you shift indented content to the left, thus "outdenting" it.
  • Indent Items (Alt, o, s, i): This lets you shift selected content to the right, thus "indenting" it.
  • Custom Format: 
    • Create Custom Format...: Opens the "Create New List Format" dialog
    • (reset class): Resets the class of selected list
    • [custom list styles in your project] All list styles will show here, click to apply the class to your selected list
