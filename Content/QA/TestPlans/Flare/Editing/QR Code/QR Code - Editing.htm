Test Cases

  • Pasting another QR Code over the existing one should replace the selected QR Code – Verify that it renders and display in the block bar correctly.
  • Pasting another element or text over a selected QR Code will replace it with that data. – Verify that there is no malformed XML from this
  • Inserting another element or typing over a selected QR Code will replace it with that data. – Verify that there is no malformed XML from this
  • Test lots of different characters including foreign characters and make sure they all render correctly
