Hyperlinks & Popups - Notes

General Notes (Remove Bugs When Fixed)

  • If you set the target frame to be popup on a hyperlink to an email address, it doesn't work
  • If you hyperlink to a CHM that is included in the project at the root content folder and attempt to edit it, the link for the chm gets corrupted. Works correctly if the CHM is in a subfolder (bug 45213)
  • If you copy a CHM from the network you have to right click it and click properties and click unblock to view it.
  • Topic Popup has the same dialog as hyperlinks, except you cannot set it to Place in this Document or Email.
