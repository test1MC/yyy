Hyperlinks - UI

Inserting a Hyperlink without the dialog

Drag and drop method:

  • Content Explorer: Drag and drop a topic from the Content Explorer into another topic that is open in the XML Editor. If dragging a topic into an insertion point, the XML Editor must be in Web Layout Mode.
    • Drop topic at a an insertion point. The file name will be used as the default link text.
    • Drop topic onto selected text. The selected text should convert to a hyperlink.
    • Test using topics of varying depth within the Content folder and verify paths are written correctly.
  • File List: Drag and drop a topic from the File List into another topic that is open in the XML Editor. The XML Editor must be in Web Layout Mode.
    • Drop topic at a an insertion point. The file name will be used as the default link text.
    • Drop topic onto selected text. The selected text should convert to a hyperlink.
    • Test using topics of varying depth within the Content folder and verify paths are written correctly

Test making a wide variety of selections:

  • Text
  • Span
  • Text with span
  • Whole paragraphs
  • List items
  • Etc.

You should only be able to insert a hyperlink using the "Drag and drop" method with a valid selection. If selection is invalid (multiple paragraphs, a table, a whole list, etc), dropping a topic on the selection should not create a hyperlink.

Quick Links method:

  • XML Editor: Right-click in the XML Editor at an insertion point or with a text selection and select Quick Link | (test all sub-menu options):

    • Bookmarks: Invokes a sub-menu that displays all bookmarks in the current topic. Select a bookmark to insert a hyperlink to that bookmark.
    • Same Folder: Invokes a sub-menu that displays all topic files that exist in the same folder as the current topic. Select a topic to insert a hyperlink to that topic.
    • Open Documents: Invokes a sub-menu that displays all topic files that are currently open. Select a topic to insert a hyperlink to that topic.
    • New Topic…: Invokes the Add File dialog which lets you add a new topic to the project and automatically inserts a hyperlink to that topic.
      • Verify that when the Add File dialog is opened in this manner that you cannot change the File Type from Topic.
    • Topic...: Invokes the Link to Topic dialog which lets you select any topic in the project. Select a topic to insert a hyperlink to that topic.

Test making a wide variety of selections:

You should only be able to insert a hyperlink using the "Quick Link" method with a valid selection. If selection is invalid (multiple paragraphs, a table, a whole list, etc), the Quick Link menu options should not appear.

Inserting a Hyperlink with the dialog

In all cases listed below, you can optionally select the link text in the XML Editor prior to inserting the hyperlink.

Insert a hyperlink with the Insert Hyperlink dialog:

  • Keyboard: Press CTRL+K
  • XML Editor: Click the Insert Hyperlink button in the XML Editor's local toolbar
  • XML Editor: Right-click in the XML Editor and select Hyperlink from the context menu
  • Tool Strip interface: !Insert | Hyperlink
  • Ribbon interface: !Insert | Hyperlink
