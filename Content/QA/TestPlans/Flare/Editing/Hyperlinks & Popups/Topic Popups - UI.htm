Topic Popups - UI

Inserting a Topic Popup

Topic Popups are basically hyperlinks that have the Target Frame set to "Popup Window".

In all cases listed below, you can optionally select the link text in the XML Editor prior to inserting the hyperlink.

Insert a hyperlink with the Insert Hyperlink dialog:

  • Tool Strip interface: !Insert | Topic Popup
  • Ribbon interface: !Insert | Hyperlink | Topic Popup

Test Cases

  • Test all Insert Hyperlink methods at an insertion point
  • Test all Insert Hyperlink methods using a text selection

Test making a wide variety of selections:

  • Text
  • Span
  • Text with span
  • Whole paragraphs
  • List items
  • Etc.

You should only be able to insert a Topic Popup with a valid selection. If selection is invalid (multiple paragraphs, a table, a whole list, etc), the menu option and button should be grayed out.

  • Verify that the selected text populates the "Link Text" field of the Insert Topic Popup dialog

Insert Topic Popup dialog - UI Test Cases

The UI of the Insert Topic Popup dialog is essentially a subset of the Insert Hyperlink dialog and all functionality is essentially the same. However, Topic Popups should be tested independently.

UI Controls

  • Class: Displays the selected class from the "Defined Classes" list, but also lets you manually enter a class name.
  • Defined Classes: Displays a list of all defined classes of the a style that exist in the project stylesheets.
    • Add a custom class in the stylesheet and verify that it shows up in this dialog.
