Grouping

Bug Base Product Area:

  • General Editing

 

Description:

blockquote: Enclosed text is a quote block consisting of one or more paragraph.

div: Denotes a generic “division” within the document. This element is used to add a customizable block element to the document. The content within the div element is typically given a name via a class or id attribute and then formatted with stylesheets.

fieldset: Groups related form controls and labels. Fieldset elements are placed within the form element. It is similar to div but is specifically for grouping fields. It was introduced to improve form accessibility for users with alternative browsing devices.

form: Indicates an interactive form that contains controls for collecting user input and other page content. There can be more than one form per html document, but forms cannot be nested inside one another.

 

Flare minimally supports these tags and currently provides no UI for editing form elements. The tags must be modified in the source to produce desired effects.
