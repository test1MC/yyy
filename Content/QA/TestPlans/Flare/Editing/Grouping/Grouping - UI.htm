Grouping - UI

Opening the UI

 

  1. Select one or more blocks of content (such as paragraphs).
  2. Select Format>Group or press the Tab key on your keyboard.
    1. The Create Group dialog opens.
  3. Select a tag from the list:
    1. blockquote: This tag is typically used to format text used as a quotation. Usually the <blockquote> tag has margin indentations to set it apart from the rest of the content in the topic.
    2. div A <div> tag is used to define logical divisions in your topics or to hold objects that can be "floated" (such as text boxes). It is similar to a paragraph tag, but it divides the page into larger sections. You can use <div> tags to name those larger sections of a topic and then affect them with style sheets or Dynamic HTML.
    3. fieldset: This tag is used to combine multiple tags into a group, drawing a box around all of the content.
    4. form: This tag is used to create a form for user input. After you create the group with the <form> tag, you can open the topic in the Internal Text Editor to supply the necessary code for the form fields.
  4. Click OK.

 

Test Cases:

  • Insert in paragraphs selected in various ways (Partial selection, just the insertion point, entire selection)
  • Insert in various formatted paragraphs (Paragraph styles, character styles, Local formatting, etc)
  • Insert in paragraphs with various elements in them (Images, Form elements, Special Characters, Help Controls, etc)
  • Insert with various elements selected (Table Cells, Lists, Divs, Drop-down/Expanding areas, etc.)
  • Insert in many types of Elements (Tables, Forms, Textboxes, Marquees, Lists, Drop-down/Expanding areas, etc.)
