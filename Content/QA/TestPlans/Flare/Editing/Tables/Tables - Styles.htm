Table - Styles

Applying or changing a style class

  • Click on the element, select Table Class and select Create Style Class – Launches Create Style Dialog – Then open the CSS Editor
  • Click on the element, select Table Class and select Edit Style Class – Opens the CSS Editor with the style/class selected
  • Click on the element, select Table Class and select Reset Style Class – Removes the class applied to the element
  • Click on the element, select Table Class and select a style from the Style Class list
  • Selecting the element and applying (reset class) from the Style Drop-down on the formatting toolbar
  • Selecting the element and applying a style from the Style Drop-down on the formatting toolbar
  • Verify that Classes applied to the element render correctly in the Editor and all outputs

Create Style Dialog – Launched from a Table object.
