AutoNumbering

Bug Base Product Area:

  • AutoNumbering

 

Description:

Auto-Numbering gives the users the ability to create style classes that contain Auto-numbering commands. This includes counters: Counters are commands (such as {n}, {n=1}, {n+}, {r}, {A}, and {Gn}) that provide information about what types of numbers should be used and how they should be incremented. Other commands allow the user to organize their project and apply number sequences to them. The sequences can be formatted and used to display where you are in a project (i.e. Chapter 1, Chapter 2, and Chapter 3) or to create and track a series (i.e. Figure 1-1, Table 1-1, and Image 1-1). The user can also modify the auto number commands buy adding format, text, page or file commands
