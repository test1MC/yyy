Autonumbering - UI

(TOC node) Properties - Printed Output tab

UI Controls:

  • Section Break:
    • Start a new section: Lets you designate the selected TOC node as the beginning of a new chapter. In doing so, lets you begin using a different Master Page file (for Word and Frame only) and lets you set auto-numbering properties for sections (in the Auto-numbers tab).
    • Configure section using this Master Page: Select the Master Page file you want to associate with this section
    • Open File: Opens the selected Master Page file in the XML Editor
  • Chapter Break:
    • Start a new chapter document: Lets you designate the selected TOC node as the beginning of a new chapter. In doing so, lets you begin using a different Page Layout file and lets you set auto-numbering properties for chapters (in the Auto-numbers tab) and page number properties in this tab.
    • Configure chapter using this Page Layout: Select the Page Layout file you want to associate with this chapter.
    • Open File: Opens the selected Page Layout file in the Page Layout Editor
    • Page Type: Lets you designate the page type (from the selected Page Layout file) that the chapter should begin with.
    • Auto-end on 'left' page: Lets you disable the use of the "Empty" page. See the Help for more information.
    • Chapter File Name: Lets you specify a particular file name for the chapter being created.
  • Page Number:
    • Reset to: If you want the first page number in the chapter to begin at a specific number, you can select this option and enter the number in the field. If you select the "text" option from the Format field, you can type any kind of text into this field. That text will be used instead of numbering.
    • Continue from previous: If you want the first page number in the chapter to continue from the previous chapter, you can select this option.
    • Format: If you want a particular format of numbering to be used (e.g., decimal, upper Roman, lower alpha, text), select it from this drop-down. If you select the "text" option, you can type any kind of text into the "Reset to" field. That text will be used instead of numbering.

(TOC node) Properties - Auto-numbers tab

UI Controls:

  • Section Number:
    • Reset to: Select this option if you want the section numbers in topics starting at this point in the TOC to begin at a specific number. Enter the number in the blank field. If you select the "text" option from the Format field, you can type any kind of text into this field. That text will be used instead of numbering.
    • Same as previous: Select this option if you want the section numbers in topics starting at this point in the TOC to begin at the same number as the previous section.
    • Continue from previous: Select this option if you want the section numbers in topics starting at this point in the TOC to begin at the next number after the previous section.
    • Format: Select the type of numbering to be used for the section auto-numbers (e.g., decimal, Roman, uppercase, lowercase, text). If you select the "text" option, you can type any kind of text into the "Reset to" field. That text will be used instead of numbering.
  • Chapter Number:
    • Reset to: Select this option if you want the chapter numbers in topics starting at this point in the TOC to begin at a specific number. Enter the number in the blank field. If you select the "text" option from the Format field, you can type any kind of text into this field. That text will be used instead of numbering.
    • Same as previous: Select this option if you want the chapter numbers in topics starting at this point in the TOC to begin at the same number as the previous chapter.
    • Continue from previous: Select this option if you want the chapter numbers in topics starting at this point in the TOC to begin at the next number after the previous chapter.
    • Format: Select the type of numbering to be used for the chapter auto-numbers (e.g., decimal, Roman, uppercase, lowercase, text). If you select the "text" option, you can type any kind of text into the "Reset to" field. That text will be used instead of numbering.
  • Volume Number:
    • Reset to: Select this option if you want the volume numbers in topics starting at this point in the TOC to begin at a specific number. Enter the number in the blank field. If you select the "text" option from the Format field, you can type any kind of text into this field. That text will be used instead of numbering.
    • Same as previous: Select this option if you want the volume numbers in topics starting at this point in the TOC to begin at the same number as the previous volume.
    • Continue from previous: Select this option if you want the volume numbers in topics starting at this point in the TOC to begin at the next number after the previous volume.
    • Format: Select the type of numbering to be used for the volume auto-numbers (e.g., decimal, Roman, uppercase, lowercase, text). If you select the "text" option, you can type any kind of text into the "Reset to" field. That text will be used instead of numbering.
