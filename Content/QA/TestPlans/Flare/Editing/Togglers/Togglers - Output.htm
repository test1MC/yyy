  • Verify Togglers work for all web output types as well as topic previews (no java script errors).
    • Test with "Condense JavaScript Files" disabled in the Performance tab of the target
    • Test with "Condense JavaScript Files" enabled in the Performance tab of the target
  • Verify that the entire <MadCap:toggler> tag and all of its properties are removed when conditionalized out.
  • Verify that applied styles are in outputs
