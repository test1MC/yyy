Togglers - Editing

Opening the UI

Click on the Toggler and select Edit Toggler…

Right-click the Toggler and select Edit Toggler…

Select the Toggler !Insert | Toggler

Right-click on the Toggler Block Bar node and select Edit Toggler…

Context Menu

 

Verify that Togglers are removed or unbound from Outputs depending on the Condition tag settings.

Code

No Class:

<MadCap:toggler targets="NamedElement">

 

Multiple Targets:

<MadCap:toggler targets="NamedElement;NamedElement2">

 

With Class and Condition:

<MadCap:toggler targets="NamedElement" class="className1" MadCap:conditions="Default.ScreenOnly">
