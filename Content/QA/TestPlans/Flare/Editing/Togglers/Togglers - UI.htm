Insert Toggler dialog - UI Test Cases

  • Test all possible elements as Toggler targets: paragraphs, lists, tables, groupings, pictures, etc.
  • If a toggler is inserted without selecting any text, the default hotspot text is “Click Me!”
  • If a toggler is inserted with text selected, the selected text should appear in the hotspot text field.

UI Controls

  • The hotspot text: Displays the text that serves as the link to the toggled content.
  • Toggler class: Displays the toggler style class.
  • Toggler targets: Displays all Named Elements in the topic. Allows user to select an element (or elements) to be the target of the toggler hotspot. All columns should be sortable and resizable (#10134).
  • Checkbox column: A check mark in the check box associates that named element with the hotspot.
  • Name Elements: Displays the name of the element
  • Tag: Displays the tag of the named element
  • Text: Displays the actual text in the named element
  • Go To: Highlights the text in the topic for the toggler element that is selected.
