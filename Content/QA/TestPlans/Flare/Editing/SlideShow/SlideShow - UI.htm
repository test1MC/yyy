SlideShow - UI

 Slideshow Editor: General Tab

UI Controls

  • Navigation:

    • Bullets: Shows the default bullet which uses rounded border property
    • Thumbnails: Default thumbnail is a black box - customize the image on each slide below.
    • None: No navigation will display.
  • Arrow Overlays: Toggles on/off Previous and next navigation arrows embedded in the slide
  • Show Captions: Toggles on/off caption display
  • Start on random slide: Toggles on/off the ability to start the SlideShow on a random slide
  • Slides displayed: Click the up/down arrows or type how many slides you want to show at once. The maximum number is the total number of slides.
  • Width: Allows control over size of the Slideshow presentation. Default is full page width. Pixels is the only unit option (by design).
  • Adaptive Height: Toggles on/off Slideshow's ability to adjust to slides of different heights
  • Responsive size: Toggles on/off the Slideshow's ability to adjust to any browser's size
  • Slides: Displays all slides contained in the slide show.
    • New File (New File (Ctrl + N)): Inserts a new slide
    • (Delete (Del)): Deletes selected slide(s)
    • (Move item up): Moves selected slide(s) up
    • (Move item down): Moves selected slide(s) down
    • Caption: Allows for selected slide to be captioned - Used in conjunction with "Show captions"
    • Thumbnail: Allows for an image thumbnail to be set as a navigation point- Used in conjunction with Navigation: Thumbnails
      • (Insert Thumbnail Image): Opens Insert Image dialog
    • Preview Window: previews the selected slide's content.

Test Cases

  • New File - Test all insertion points
  • Delete - Test deletion of both single and multiple selections.
  • Arrows - Test moving both single and multiple selections.
  • Try displaying more slides than exist in the slideshow - should correct to total number of slides.

Slideshow Editor: Playback Tab

UI Controls

  • Loop: Toggles on/off continuous loop. Default state is checked (on)
  • Start automatically: Toggles on/off auto start of slideshow slides
    • Delay (ms): Sets a timed delay for the start of Slideshow slides. 1000 Milliseconds = 1 second
  • Pause on hover: Will cause the Slideshow slides to pause when mouse pointer is placed over a slide. Will start back up again when pointer is removed.
  • Play/Pause button: Enables the Play/Pause buttons. Controls start and stop of Slideshow.
  • Slide Duration (ms): Controls the speed at which the slide moves through the Slideshow. Default is 4,000 milliseconds.
  • Direction: Controls the direction in which the slide(s) move through the Slideshow
    • Next: Slides will move in the next direction (Slide 1, Slide 2, Slide 3...)
    • Previous: Slides will move in the previous direction (Slide 1, Slide 3, Slide 2...)
  • Transition: Controls how the next slide will be revealed
    • horizontal: Scrolls left to right
    • vertical: Scrolls up and down
    • fade: Fades out current slide, fades in next/previous slide
  • Transition speed (ms): Controls how fast the next slide will be revealed.
