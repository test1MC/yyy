Slideshow - Styles

Applying or changing a style class

  • Right click on the element, select Style Class | Create Style Class – Launches Create Style Dialog – Then open the CSS Editor
  • Right click on the element, select Style Class | Edit Style Class – Opens the CSS Editor with the style/class selected
  • Right click on the element, select Style Class | Reset Style Class – Removes the class applied to the element
  • Right click on the element, select Style Class | (select a style from the Style Class list)
  • Selecting the element and applying (reset class) from the Style Drop-down on the formatting toolbar
  • Selecting the element and applying a style from the Style Drop-down on the formatting toolbar
  • Verify that Classes applied to the element render correctly in the XML Editor and all outputs:
    • HTML5 / WebHelp 2
    • WebHelp
    • WebHelp Plus
    • WebHelp Air
    • WebHelp Mobile
    • DotNet
    • CHM

  • MadCap|slideshowBullet- Style properties for navigation Bullets
    • Background
    • Border
    • Box
    • Active- Background properties for active bullet
      • (Pseudo classes)- No settings for this
        • hover- Background properties for "Active" Bullet's hover state
      • (Pseudo Classes)- No setting for this
        • hover- Background properties for Bullet's hover state

 

  • MadCap|slideshowCaption- Style properties for the slide captions
    • Font
    • Background
    • Border
    • Box

 

  • MadCap|slideshowThumbnail- Style properties for navigation Thumbnails
    • Background
    • Border
    • Box
    • Active- Background properties for active thumbnail
      • (Pseudo classes)- No settings for this
        • hover- Background properties for "Active" thumbnail's hover state
      • (Pseudo Classes)- No setting for this
        • hover- Background properties for thumbnail's hover state

       

Test Cases:

  • Verify style set in MadCap|slide overrides styles set in MadCap|slideshow (change font color in both and verify that the correct color displayed is from the color set in MadCap|slide)
  • Create two classes in Madcap|slideshow. Insert two slideshows into the same topic. Apply different style class to each and verify that they are correct in output.
  • Create two classes in MadCap|slide, apply them to two different slides in the same slideshow and verify that the individual slides display the properties of their assigned class.
  • Apply border styles to both MadCap|slideshow and MadCap|slide and verify that both are displayed correctly in the output.
  • Insert different size thumbnail images into a Slideshow, then set the height property in MadCap|slideThumbnail->Box. View the output and verify that all thumbnails display at the same size
