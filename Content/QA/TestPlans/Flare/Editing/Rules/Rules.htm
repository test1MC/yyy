Rules (HR)

Bug Base Product Area:

  • General Editing

 

Description:

A rule is a horizontal line (or bar) that you can insert into a topic. You might use a rule, for example, as a design element to separate your topic title from the content. At the bottom of this topic is a rule followed by a footer with contact information for MadCap Software.

 

Flare provides the <hr> style tag, which you select when inserting a rule. If you want, you can change the way a rule looks (size, color, etc.) by editing the <hr> style properties in the Stylesheet Editor.
