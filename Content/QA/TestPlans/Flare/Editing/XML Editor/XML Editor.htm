XML Editor

Bug Base Product Area:

  • XML Editor

 

Description:

This test plans covers the XML Editor UI and General Editing & Formatting in the XML Editor. General Editing and Formatting includes allowing users to edit and format content in WYSIWYG in general, drag and drop in editor, show/hide indicators in WYSIWYG, simple editing such as cutting, copying, deleting and pasting content, editing shortcuts.

 

A separate test plan which focuses on the Print Layout XML Editor addresses additional features which apply exclusive to the Print Layout (page and column breaks, widows/orphans, etc).

Notes and things to add to the testing resources:

  • Add v11 Feature: Added support for the CSS “position: absolute” setting, which lets you remove content from the main flow of a document and place it anywhere you want. You can drag absolutely positioned elements to reposition them (by holding down CTRL) or resize them using similar UI to frames in the page layout editor
  • Add v11 Feature: When your cursor is at the start of a paragraph with no text selected, pressing tab will increase the text-indent of that paragraph by 0.5 inches. All other selection types (e.g. place the cursor anywhere else in the paragraph, select some of the paragraph or multiple paragraphs) will continue to invoke the “create group” dialog.
  • Add v11 Feature: Block Bar Improvements - 1) You can now shift-click on blocks to extend the current selection to that block. 2) Block bars will now highlight correctly when your selection includes multiple blocks. Previously only the first block in the selection would be highlighted. Now blocks that are fully within your selection have a darker blue background, while partially selected blocks that your selection starts or ends on have a lighter blue background
  • Add v11 Feature: Insert flyout menu
