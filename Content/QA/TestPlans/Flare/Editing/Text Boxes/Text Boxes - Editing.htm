Text Boxes- Editing

text box menu

 

 

 

If you remove either the height or the width from the code of a text box it will become a div and won't have the Text Box... option in the context menu.

 

Test Cases

  • Pasting another text box object over the existing one should replace the selected text box object – Verify that it renders and display in the block bar correctly.
  • Pasting another element or text over a selected text box object will replace it with that data. – Verify that there is no malformed XML from this
  • Inserting another element or typing over a selected text box object will replace it with that data. – Verify that there is no malformed XML from this

Misc

Code

If you remove either the width or the height the 'text box' becomes just a div.

No Class:

 

<div style="width: 200px;height: 200px;clear: right;mc-float: none;">

<p>&#160;</p>

</div>

 

With Class:

 

<div class="text1" style="width: 260px;height: 251px;">

<p>&#160;</p>

</div>
