Locked Elements - Editing

Lock/Unlock an Element via the Stylesheet

Set the "mc-locked" property of any elements to "locked".

Lock/Unlock an Element Inline

How to lock elements:

  1. Open a topic.

  2. Place your cursor on a paragraph that you want to lock, or highlight multiple paragraphs.

    Note: Only entire blocks of content or paragraphs can be locked, not portions of paragraphs.

  3. Select !Format | Locks | Mark Element(s) to be Locked for the tool strip mode
  4. Select high light text and the select "!Review | Locks Group |Locks " to select the text to be locked and the select "!Review | Locks Group |Enable Locks " to lock text
  • The paragraphs display a gray background (which will not show up in the output). If you want to hide this shading from the XML Editor, but still keep the locks on the paragraphs, select Format>Locks>Show Locked Elements.
  • Notice that you can still click in the paragraphs and edit the text. That is because the elements have only been marked for locking, but they have not yet been locked.
  1. Select Format | Locks | Locks (Disabled)
    • The paragraphs marked for locking are now locked, and you cannot edit the text within them.
  2. Press CTRL+S or click to save your work.

Note: Annotations, which are used in the topic review process, cannot be added to areas of a topic that have been locked. See About Topic Reviews and Contributions.

Note: This feature is to be used for locking areas of a file, not the file itself.

 

How to unlock elements:

  1. Open a topic where some elements have been locked.
  2. Select Format>Locks>Locks (Enabled) for the toolstrip mode "Review|Locks|Enable Locks" to uncheck the lock enable in ribbon mode
  3. The elements are now unlocked and you can edit them. However, the paragraphs continue to be "marked for locking," meaning they can be easily be locked again by selecting this option again.
  4. (Optional) If you want to remove the "marks" on paragraphs that prepare them for locking, place your cursor on the paragraph or highlight multiple paragraphs. Then select Format>Locks>Unlock Element(s).
  5. Press CTRL+S or click to save your work.

Code

Locked paragraph (inline):

<p class="note" style="mc-locked: locked;">This is the paragraph content.</p>

 

Unlocked paragraph (inline):

<p class="note" style="mc-locked: unlocked;">This is the paragraph content.</p>
