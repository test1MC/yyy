Redacted Text - Editing

Text Redaction is like bold, italic or color in that is applied to an element as a style. Flare uses the style "mc-redacted" for redacted text.

Applying Text Redaction to text inline

Select some text in the Editor and:

  • Click the Text Redaction button in the Format Toolbar or in the Home Ribbon.

  • !Format | Font to open the Font Properties dialog and in the Advanced tab select "Redacted"

    Font Properties dialog:

 

Applying Text Redaction to an element inline

Select an element in the Editor and:

  • Click the Text Redaction button in the Format Toolbar.

  • !Format | Font to open the Font Properties dialog and in the Advanced tab select "Redacted"

    (Screen shot same as above)

 

Applying Text Redaction to an element using the CSS

Advanced View:

In the Stylesheet Editor, select the style or class you want redacted. Display Properties by Property Groups. Find "mc-redacted" in the Redaction group. Available options should display as "(default)," "inherit," "none" and "redacted." Set to "redacted" to redact the style class.

Simplified View:

In the Stylesheet Editor, double-click the style or class you want redacted. This will open the (Style) Properties dialog. In the Advanced Tab, find the Redacted Text section, and select "Redacted."

Redacted Text in the XML Editor

Redacted Text will display in the XML Editor with a gray highlight. Redaction applied to an element will simply display the element with a gray background. Redaction applied to a span will also display inline markers (assuming Inline Markers are enabled in the Show menu).

<p style="mc-redacted: redacted;">Redacted paragraph.</p>
