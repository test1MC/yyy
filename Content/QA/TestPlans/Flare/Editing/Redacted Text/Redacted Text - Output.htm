Support in EPUB Output:

  • Redacted Text is not supported in this output. Verify that a compile error is given for all Web output types (see below).

Support in DITA Output:

  • DITA does offer an equivalent feature to Redacted Text. When generating DITA output, confirm that we simply remove any redacted styles from any element it is applied to (for example, redact an entire p tag or a div). Currently, all spans are completely removed from DITA output.

Compiler errors:

  • Generating EPUB, HTML5, Eclipse, WebHelp, WebHelp Plus, WebHelp AIR, WebHelp Mobil, DotNet Help, HTML Help, Word and Frame with Redacted Text should produce:
    "Redacted text found in file. Current target type does not support this feature. The redacted text will be included in the output."
