With the Index Window

Steps to Insert

  1. Open the Index window
    • Press F9
    • Toolstrip interface: Select !Tools | Index | Index Window
    • Ribbon interface: Select !View | Index Window
  2. Open the topic to which you wish to add the keyword
  3. Place the insertion point where you would like to insert the keyword
  4. Select an empty field in the Terms section of the Index window
  5. Type the keyword
  6. Press Enter

With Index Entry Mode

Index Entry Mode lets add keywords quickly throughout your topics by letting you type them directly in the XML Editor.

 

Steps to Insert

  1. Open the topic to which you wish to add the keyword
  2. Enable Index Entry Mode
    • Toggle "on" the Index Entry Mode button in the XML Editor's local toolbar
    • Toolstrip interface: !Tools | Index | Index Entry Mode
  3. Notice the cursor changes, displaying a “i” next to it
  4. Click at the place in your topic where you want to add the keyword
  5. Type the word or phrase that you want to add as the keyword
  6. When you start typing, the Index window opens and the phrase is added to the first empty field under the Terms column
  7. Notice the cursor does not lose focus in the XML Editor
  8. Press Enter
  9. Place the insertion point where you want to enter the next keyword (in the same topic or another) and repeat.
  10. When you are finished, disable Index Entry Mode
    • Toggle "on" the Index Entry Mode button in the XML Editor's local toolbar
    • Right-click in the XML Editor to toggle off Index Entry Mode
    • Toolstrip interface: !Tools | Index | Index Entry Mode

With Drag and Drop

The Drag and Drop method requires that you have an existing keyword, and you now want to add it to a different topic.

  1. Open the Topic to which you wish to add the keyword
  2. Open the Index window (see above)
  3. In the lower portion of the Index Window, click the keyword you want and drag it to the location in the topic where you want to insert it.

Auto-complete option

The "Auto-complete Index Entries" option applies to both Index Keywords and Concepts. To enable or disable "Auto-complete Index Entries":

  • !Tools | Index | (toggle on) Auto-complete Index Entries
  • In the XML Editor's local toolbar, expand the menu under the Index Entry button , and toggle on Auto-complete Index Entries

 

If "Auto-complete Index Entries" is enabled, index keywords and concepts will auto-complete when you start typing them in the Index Window and Concept Window.

Editing a Keyword

To edit a keyword (the actual word(s)), you need to do so in the Index Window. It is possible to edit a single instance of a keyword within a topic (using the upper portion of the Keyword Window), or you can globally edit all instances of a keyword (using the lower portion of the Index Window).

Editing a single instance of a keyword:

  • With the Index Window open, select an existing keyword. Any keyword(s) in the selected MadCap:keyword tag should appear in the Index Window pane for editing
  • Right-click an existing keyword in the XML Editor and select Edit Index Keyword…. This should open the Index Window pane and display any keywords in the selected MadCap:keyword tag for editing.
  • Right-click on the MadCap:keyword Span Bar node and select Edit Index Keyword… . This should open the Index Window pane and display any keywords in the selected MadCap:keyword tag for editing.
  • Right-click on the MadCap:keyword Caret Tag Neighborhood node and select Edit the Keyword Association… . This should open the Index Window pane and display any keywords in the selected MadCap:keyword tag for editing.

Editing all instances of a keyword:

In the lower portion of the Index Window…

  • Right-click any existing keyword and select Rename Keyword.
  • Select any existing keyword and press [F2].
  • Slowly click on any existing keyword twice.

This will affect all instances of that keyword in all topics throughout the project. This is covered in greater detail in the Index - UI test plan.
