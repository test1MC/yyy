Inserting a Keyword Link

!Insert | Help Control | Keyword Link Control (K-link)...

UI Test Cases:

UI Controls

  • Selected Keywords: Displays a list of Keywords associated with the selected Keyword Link.
  • All Keywords: Displays a list of all Keywords present in the current Flare project. Any keywords which are present in the current topic should appear at the top of this list when the dialog opens.
  • [ >> ] : Associates the selected Keyword (in the All Keywords column) with the Keyword Link. (The selected keyword should then appear in the Selected Keywords column.)
  • [ << ] : Disassociates the selected keyword (in the Selected Keywords column) with the Keyword Link.
  • Remove Control: Removes the Keyword Link from the topic.
  • Options: Will open the options dialog for the help control

 

 

 

 

Keyword Link Options

UI Controls

  • Style Class:: Displays a list of style classes for the MadCap:keywordLink style
    • Selecting one will apply that style class to the help control, once OK is pressed.
  • Label: Will set an inline label for the help control.
    • This field should be populated with the default label, as set on the MadCap:keywordLink, mc-label style property
    • Typing your own label will only add it as an inline label. This will NOT change the mc-label style property on the associated Style.
  • Display Topics in: Drop down selection for how topic list should be displayed
    • (default): Will use what is set on the MadCap:keywordLink mc-help-control-display property
    • Popup Menu: This option will display the topics in a popup window, when the help control is clicked.
    • List: This option will display the topics in a list format, when the help control is clicked.
  • Target Frame : Drop down selection for where the topic list should show up.
    • (default): Just means Parent Frame
    • Parent Frame: Will open the selected topic in the same window
    • New Window: Will open the selected topic in a new window
    • Same Frame: Will open the selected topic in the same frame
    • Top Window: Will open the selected topic in the top most window
    • Popup Window: Will open the selected topic in a popup window
