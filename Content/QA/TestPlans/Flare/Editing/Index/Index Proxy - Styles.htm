Index Proxy - Styles

The styles detailed below all contribute to the look of the generated index in the output (in place of the Index Proxy). Of the styles outlined here, only MadCap|indexProxy will allow you to create unique classes. All other styles are generated styles used by all Index Proxies regardless of class.

The styles applied here should be tested in all outputs, however, keep in mind that these styles only affect the Index Proxy placed inside a topic. These styles do not affect the look of the Index pane that is generated for Web, DotNet and HTML Help outputs. Styles for the Index pane in Web, DotNet, and HTML Help outputs are set in the Skin and will be covered in the Skins test plan.
