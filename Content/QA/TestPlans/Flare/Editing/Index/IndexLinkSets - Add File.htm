Index Link Set - Add File

An Index Link Set lets you create "See" and "See also" references from one index entry to another. For example, if you have used the keyword "canine" throughout your project, but you did not use "dog," you may want to include an entry in your generated index that indicates "Dog, See Canine." You may also have two keywords that are closely related. In that case you may want to use a "See also" reference," for example "MadCap Flare, See also MadCap Blaze."
