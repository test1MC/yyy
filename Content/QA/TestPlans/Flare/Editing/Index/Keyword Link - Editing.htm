Editing a Keyword Link

To edit a keyword link, you need to re-open the Insert Keyword Link Control dialog for an existing keyword link. This is done by invoking the MadCap:keywordLink context menu and selecting Edit Keyword Link…

 

Context Menu

 

 

 

 

 

 

 

Code

One term, no class:

<p>
    <MadCap:keywordLink term="keyword" externalHelpSystems="" />
</p>

 

Multiple terms, class, local label and conditions:

<p>
    <MadCap:keywordLink term="keyword;bagel;donut" externalHelpSystems="" style="mc-label: 'My Concept Links';" MadCap:conditions="Default.ScreenOnly" class="class1" />
</p>
