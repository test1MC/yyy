UI Test Cases:

UI Controls

  • Generate Headings: Select whether you want the printed index to include headings (e.g., A, B, C).
  • MS Word and FrameMaker Only:
    • Column Count: Select the number of columns that you want to be used in Microsoft Word or Adobe FrameMaker output to display the index.
    • Header Height: Enter the amount of space to be occupied by the index header (i.e., the index title). This lets you control the spacing between the header and the actual index. This feature is for Adobe FrameMaker output only.
  • Stylesheet class for proxy: Lets you select a class for the proxy.
