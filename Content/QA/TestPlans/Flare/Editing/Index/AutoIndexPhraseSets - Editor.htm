Auto-Index Editor - UI Test Cases

Local Toolbar

Button (Tooltips in Parentheses):

  • (New item): Opens the (Auto-index Phrase) Properties dialog for a new entry. Clicking Cancel in this dialog should not create a new entry.
  • (Run Properties dialog for this item): Opens the Properties dialog for the selected auto-index phrase.
  • Cut: Cuts the selected auto-index phrase(s) to the clipboard.
  • Copy: Copies the selected auto-index phrase(s) to the clipboard.
  • (Paste): Pastes the auto-index phrase(s) from the clipboard.
  • (Delete): Deletes the selected auto-index phrase(s)
  • Warning

UI Controls

  • Auto-index Editor grid: Displays and lets you create and edit Auto-index Phrases.

    Columns

    • Phrase: Enter a phrase that exists in your content where you would like to have the Index Term added as a keyword throughout the project in the output.
    • Index Term: Enter the Index keyword that you would like to have injected at every occurrence of the Phrase in the output.

Right-click menu

Right-clicking anywhere in the Auto-index Phrase Editor should produce the following menu:

Right-click menu:

 

Right-click menu items:

  • New: Opens the (Auto-index Phrase) Properties dialog for a new entry. Clicking Cancel in this dialog should not create a new entry.
  • Cut: Cuts the selected auto-index phrase(s) to the clipboard.
  • Copy: Copies the selected auto-index phrase(s) to the clipboard.
  • Paste: Pastes the auto-index phrase(s) from the clipboard.
  • Delete: Deletes the selected auto-index phrase(s)
  • Properties: Opens the Properties dialog for the selected auto-index phrase.

(Auto-index Phrase) Properties dialog

To open:

  • Double-click any row in the editor
  • Select any row in the editor and click in the local toolbar
  • Select any row in the editor and go !Edit | Properties
  • Select any row in the editor and press F4 on the keyboard

UI Controls:

  • Phrase: Enter the phrase to which the auto-index term should be injected
  • Index Term: Enter the index term that should be injected into the content containing the phrase

Test Cases

  • For Phrase, enter single words, multiple words, numbers, words with international characters, etc.; confirm these work in all outputs
  • For Index Term, enter single words, multiple words, numbers, words with international characters, etc.; confirm these work in all outputs
  • For Index Term, enter root Index keywords as well as sub-entries; confirm both work in all outputs.
  • For Index Term, enter existing and non-existing keywords (root and sub); confirm both work in all outputs.

Tooltips

If any Phrase or Index Term is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.

Conditional Tags

Conditional Tags cannot be applied to individual Auto-index Phrases. Conditional Tags can be applied to the actual Auto-index file (*.flaix), however, as of Flare V5, this tag can only be used for the purposes of Advanced Global Project importing.

Code

<?xml version="1.0" encoding="utf-8"?>
<CatapultAutoIndexSet>
<AutoIndex
Phrase="This is my auto-indexed phrase"
IndexTerm="Linked To This" />
<AutoIndex
Phrase="This is the term"
IndexTerm="Other Term:Subterm"></AutoIndex>
<AutoIndex
Phrase="MadCap Flare"
IndexTerm="MadCap Software:MadCap Flare"></AutoIndex>
<AutoIndex
Phrase="MadCap Mimic"
IndexTerm="MadCap Software:MadCap Mimic"></AutoIndex>
<AutoIndex
Phrase="MadCap Lingo"
IndexTerm="MadCap Software:MadCap Lingo"></AutoIndex>
</CatapultAutoIndexSet>
