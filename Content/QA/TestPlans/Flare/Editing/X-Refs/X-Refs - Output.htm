X-Refs - Output

Support in Web Outputs:

Support in Web Help Outputs:

  • Verify all target frame settings for Xrefs work for all web output types as well as topic previews.
    • Test the "Popup" option with "Condense JavaScript Files" disabled in the Performance tab of the target
    • Test the "Popup" option with "Condense JavaScript Files" enabled in the Performance tab of the target
  • Verify that the entire Xref and all of its properties are removed when conditionalized out.
  • Verify that the entire Xref and all of its properties are unbound when conditionalized out with the "Unbind" behavior.
  • Verify that applied styles are in outputs

Support in Web Help Mobile Outputs:

  • Although most target frame settings are not supported in Mobile output, verify that regardless of target frame set for a Xref that all Xrefs work in Mobile outputs.
