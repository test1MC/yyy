X-Refs - Editing

Editing the text of a X-Ref

A X-Ref gets its textual content from the text to which it links, and uses a style to set its format beyond that. Therefore you cannot edit the text of a X-Ref. Although it may be allowable in the XML Editor, when you build output, the X-Ref will update itself to the format and content reference to which it is set.

Code

Simple X-Ref with Screen Tip:

<MadCap:xref href="Topic.htm" target="" title="The screen tip" alt="The screen tip">See "Topic Title"</MadCap:xref>

 

Simple X-Ref to "Place in this Document":

<MadCap:xref href="#Heading2" target="" title="" alt="">See "Heading 2"</MadCap:xref>

 

X-Ref With Bookmark, Target Frame, Class and Screen Tip:

<MadCap:xref href="Topic.htm#bookmark" target="_popup" title="The screen tip" alt="The screen tip" class="class">Link Text</MadCap:xref>
