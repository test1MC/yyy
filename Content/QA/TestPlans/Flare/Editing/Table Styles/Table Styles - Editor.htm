Table Styles - Editor

Opening a Table Style File

The Table Style file associated with a particular table:

Open a topic, masterpage or snippet which contains a table associated with a table style:

  • XML Editor:
    • Right-click the "table" block bar and select Table Style | Edit Style
    • In the Toolstrip interface, place the insertion point inside the table and select !Table | Table Style | Edit Style
    • In the Ribbon interface, place the insertion point inside the table and select !Table | Edit Table Style

Table Style Editor - UI Test Cases

Local Toolbar

 

Button (Tooltips in Parentheses):

  • Medium (Medium:([current medium])): Lets you select the medium of the table stylesheet that you want to edit. The list of available mediums should include any medium added to any stylesheet in the project.
    • Apply Style… (Apply Style…): Invokes the Apply Table Style dialog. See Table Styles - Applying.

General Tab

UI Controls

  • Table Margins: Lets you specify the settings for the table margins (the amount of space around the table).
    • Left: Sets the left margin.
    • Right: Sets the right margin.
    • Top: Sets the top margin.
    • Bottom: Sets the bottom margin.
    • : Lets you apply settings to all of the table margin fields.
  • Outer Borders: Lets you specify the settings for the table border in the style sheet.
    • Left: Sets the left border style.
    • Right: Sets the right border style.
    • Top: Sets the top border style.
    • Bottom: Sets the bottom border style.
    • : Lets you apply settings to all of the border fields.
  • Background: Use this section to set general background properties for the entire table style.
    • Color: Select a color to be used in the background of the table.
    • Image: Select an image to be used in the background of the table.
    • : Invokes the Insert Picture dialog which lets you select any image already included in the project or lets you browse for an external image (which will then be added to the project).
    • Repeat: If a background image is selected above, used this option to set the repeat behavior of the background image.
    • X: If a background image is selected above, used this option to set the horizontal starting point of the upper-left corner of the image.
    • Y: If a background image is selected above, used this option to set the vertical starting point of the upper-left corner of the image.
  • Border Radius: Used to curve the corners of a table
    • Top-Left: Sets the top-left border radius style.
    • Top-Right: Sets the top-right border radius style.
    • Bottom-right: Sets the bottom-right border radius style.
    • Bottom-left: Sets the bottom-left border radius style.
  • Cell Border Collapse: Select whether you want to collapse the cell borders in the style sheet.
    • Collapse cell borders: If you collapse the cell borders, the row and cell borders of a table are joined in a single border.
    • Do not collapse cell borders: If you do not collapse the cell borders, the row and cell borders of a table are detached.
  • Cell Border Spacing: Use this area to increase or decrease the amount of spacing for a cell border.
    • Vertical: Set vertical spacing for a cell border.
    • Horizontal: Set horizontal spacing for a cell border.
  • Advanced:
    • Hide bottom ruling when table crosses a page break:
    • True: Will hide the bottom border when the table continues on another page
    • False: Default setting
    • Overflow: Determines what happens if the content overflows the table.
    • Visible: The overflow is not clipped. It renders outside the table. This is the default state.
    • Hidden: The overflow is clipped and the rest of the content will be invisible. (**This must be set if using border-radius**)
    • Scroll: The overflow is clipped, but a scroll-bar is added to see the rest of the content.
    • Auto: If overflow is clipped, a scroll-bar shoud be added to see the rest of the content.
    • Inherit: The value of the overflow property is inherited from the parent element.
  • Cell Padding: Lets you specify the settings for the cell padding (the amount of space between the edge of the table cell and the content in the cell).
    • Left: Sets the left padding.
    • Right: Sets the right padding.
    • Top: Sets the top padding.
    • Bottom: Sets the bottom padding.
    • : Lets you apply settings to all of the cell padding fields.
  • Print Options: Opens the Breaks dialog.

Breaks Dialog (from General tab)

Rows Tab

UI Controls

  • Patterns grid: Lets you create patterns for the table style

    Vertical Toolbar (tooltips in parentheses)

    • (New item):Creates a new pattern.
    • (Delete): Deletes the selected patterns.
    • (Move up): Moves the selected patterns up in the sequence.
    • (Move down): Moves the selected patterns down in the sequence.

    Columns

    • Name: Set the pattern name.
    • Type: Set the pattern type.
    • Repeat: Set the number of times the pattern will repeat in the table style before proceeding to the next pattern.
  • Row Style: Lets you set properties for the pattern selected above.
    • Height: Select a pattern from the section above. Then click this field to open a small popup, which lets you set properties for the row height.
    • Font: Select a pattern from the section above. Then click this field to open a small popup, which lets you set properties for the text.
    • Ruler: Select a pattern from the section above. Then click this field to open a small popup, which lets you set properties for a ruler between the rows or columns within the pattern. The "Repeat" value of the pattern must be 2 or greater for this option to be enabled.
    • Separator: Select a pattern from the section above. Then click this field to open a small popup, which lets you set properties for a separator between the final row or column in the pattern and the first row or column in the next pattern.
    • Alignment: Select an option for aligning text in the row or column horizontally.
    • Vertical Alignment: Select an option for aligning text in the row or column vertically.
    • Background: Select a pattern from the section above. Then use this area to set background properties for the table rows for that pattern.
      • Color: Select a pattern from the section above. Then select a color to be used in the background of the table rows for that pattern.
      • Image: Select a pattern from the section above. Then select an image to be used in the background of the table rows for that pattern.
      • : Invokes the Insert Picture dialog which lets you select any image already included in the project or lets you browse for an external image (which will then be added to the project).
      • Repeat: If a background image is selected above, used this option to set the repeat behavior of the background image.
      • X: If a background image is selected above, used this option to set the horizontal starting point of the upper-left corner of the image.
      • Y: If a background image is selected above, used this option to set the vertical starting point of the upper-left corner of the image.
  • Cell Padding: Lets you specify the settings for the cell padding (the amount of space between the edge of the table cell and the content in the cell).
    • Left: Sets the left padding.
    • Right: Sets the right padding.
    • Top: Sets the top padding.
    • Bottom: Sets the bottom padding.
    • : Lets you apply settings to all of the cell padding fields.
  • Print Options Button: Invokes the Breaks dialog. Break styles set in the General tab apply to the whole table. Break styles set from the Rows tab apply to the selected Row Styles. The Breaks dialog invoked from the General tab will include the Widows & Orphans section while the Breaks dialog invoked from the Rows tab will not. (same as above but setting Break styles here only apply to the selected row pattern)
