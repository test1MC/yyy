UI Controls

  • Template Folders: Select a template folder
  • Templates: Choose a template to create a table style from
  • Source File: Select the source file for the template (if using one that is not in the template folders already selected)
  • New Style Name: Apply a name to your new table style
  • Preview: Displays the preview of the selected table style template.
