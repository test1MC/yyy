  • Phone

Support in Print Outputs:

Support in PDF

  • Verify all applicable “Link to:” image map area properties work correctly.

Note: PDFs are limited to rectangular shaped links. Circles and polygons will be drawn as rectangles in the output (examples below). Users will be alerted of the limitation through compile warnings. 80002: PDF only supports rectangular shaped image map areas. All non-rectangular shapes will be converted.

 

 

Corner case: first drawn link in output will be completely overridden by second drawn link.

Support in DITA Output:

Image maps are not supported in Dita output.
