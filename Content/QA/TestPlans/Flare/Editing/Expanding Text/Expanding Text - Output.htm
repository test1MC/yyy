Expanding Text - Output

Support in Web Outputs:

Support in Web Help Outputs:

  • Verify Expanding Text works for all web output types as well as topic previews (no java script errors).
    • Test with "Condense JavaScript Files" disabled in the Performance tab of the target
    • Test with "Condense JavaScript Files" enabled in the Performance tab of the target
  • Verify that the entire <MadCap:expanding> tag and all of its properties are removed when conditionalized out.
  • Verify that applied styles are in outputs

Support in Print Outputs:

Support in PDF Outputs:

The behavior of Expanding Text in print outputs is determined by the Expanding Text Effects option in the Target Editor in the Printed Output tab.

 

 

  • Verify Expanding Text is converted to footnotes*
  • Verify Expanding Text is expanded text inline
  • Verify Expanding Text has its body removed

* If "Convert to footnotes" is selected, the generated footnote's format, style, position, etc can be customized in the Stylesheet. See Expanding Text - Styles
