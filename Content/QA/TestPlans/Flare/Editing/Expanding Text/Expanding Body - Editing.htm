Expanding Body - Editing

To edit the Expanding Text Body, place the insertion point inside the MadCap:expandingBody tag and type.

Context Menu

 

 

Selecting

You should not be able to select the MadCap:expandingBody tag (aside from selecting the content of the tag).

 

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

You should not be able to cut or copy the MadCap:expandingBody tag (aside from copying the selected content of the tag).

 

Drag and Drop

You should not be able to drag the MadCap:expandingBody tag outside of its parent MadCap:expanding tag.

 

Deleting

You should not be able to delete the MadCap:expandingBody tag aside from deleting the entire MadCap:expanding tag.

 

 

Conditional Tags

You should not be able to apply a conditional tag to the MadCap:expandingBody tag. You should only be able to apply a condition to the parent MadCap:expanding tag.

 

Code

No Class:

<MadCap:expanding>
    <MadCap:expandingHead>Delete this text</MadCap:expandingHead>
    <MadCap:expandingBody> and replace it with your own content.</MadCap:expandingBody>
</MadCap:expanding>

 

With Class and Condition:

<MadCap:expanding class="className1" MadCap:conditions="Default.ScreenOnly">
    <MadCap:expandingHead class="className1">Delete this text</MadCap:expandingHead>
    <MadCap:expandingBody class="className1"> and replace it with your own content.</MadCap:expandingBody>
</MadCap:expanding>
