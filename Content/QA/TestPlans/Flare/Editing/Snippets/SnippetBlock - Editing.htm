Block Snippet - Editing

This section deals with editing a Snippet, specifically a Snippet Block, in the context of other content, for example in a Topic. You should not be able to edit or even place the insertion point within the text of a Snippet in the context of a Topic (or Page Layout, etc). You should only be able to edit (cut, copy , move, etc) the entire snippet block.
