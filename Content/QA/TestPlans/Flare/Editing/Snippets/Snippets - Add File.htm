Creating a Snippet from existing content

Create a snippet from a selection:

Select some content in the XML Editor and go !Format | Create Snippet… in the tool strip or !Home |Snippet | Create Snippet in the Ribbon menu.

Create a snippet from a tag:

Invoke the content menu for any element (p, span, ul, etc) and select Create Snippet…

Create Snippet - UI Test Cases

UI Controls

  • Snippet File: Enter a file name for the snippet
  • Project Folder: The file path to the folder location in which the new snippet will be created. By default, we should use Resources/Snippets.
  • Snippet: Preview window allows user to see content selected to include in snippet.
  • Replace Source Content with the New Snippet: If this option is selected, upon creation of the snippet file, the selected source content will be replaced with a link to the new snippet.
  • Create: Closes the dialog and creates the snippet (*.flsnp) file in the assigned folder.
  • Cancel: Closes the dialog and does not create a new snippet.

Inserting a Snippet

Using the Insert Menu:

!Insert |Snippet…-

Using the Ribbon Menu:

!Insert |Single Source | Snippet

Drag & Drop:

Click on a Snippet in the Content Explorer and drag it into a topic.

Both methods will invoke the Insert Snippet Link dialog.

Insert Snippet Link - UI Test Cases

UI Controls

  • Select Snippet: Displays a list of all snippet files in the project.

    • Shows all of the snippet files in the project in a list in the area below.
    • Shows or hides the folders that the snippet files are stored in.
    • Shows or hides the snippet files. If you click this button when the "Show Folders" button is selected, the area splits into two halves. The folder is shown on the left side, and the files and subfolders within it are shown on the right.
    • If the "Show Files" button is the only one selected, you can click this button to move up one folder level.
    • *.flsnp;*.mcsnp: displays the filetypes that will show in the list. You can't change this.

    Columns

    • File: Displays the filename of the snippet file.
    • Path: Displays the path of the snippet file.
  • [File]: Should populate with the file path to the selected snippet file, or the user can type in the path.
  • : Launches the Open dialog with File of type set to Snippet Files (*.flsnp)
  • Preview: Displays a preview of the snippet file selected to the left.
