Text Snippet - Editing

This section deals with editing a Snippet, specifically a Text Snippet, in the context of other content, for example within a paragraph in a Topic. You should not be able to edit or even place the insertion point within the text of a Snippet in the context of a Topic (or Page Layout, etc). You should only be able to edit (cut, copy , move, etc) the entire text snippet tag.
