Snippets - Output

Support in Web Outputs:

Support in HTML5 Outputs:

  • In all Web outputs, the Snippet content should replace the MadCap:snippetBlock (or MadCap:snippet Text) tag wherever it is placed in any Topic or Master Page.
  • Verify Snippets work for all web output types as well as topic previews.
  • Verify that the entire <MadCap:snippetBlock> or <MadCap:snippetText> tag and all of its properties are removed when conditionalized out.
  • Default use case: Snippets in the Resources/Snippets folder
  • Snippets in sub-folders of the Resources/Snippets folder
  • Snippets in folders outside the Resources folder
  • Test using Snippets in Topics in the root directory as well as one, two, three, etc folders deep.
  • Test Snippets in Page Layouts and Master Pages
  • Test Snippets in Snippets (in Snippets in Snippets...).
