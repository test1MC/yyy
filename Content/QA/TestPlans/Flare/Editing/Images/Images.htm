Bug Base Product Area:

Description:

You can insert a picture into a topic to help explain something. Flare supports the following types of raster and vector image files: BMP, EMF, EPS, EXPS, GIF, HDP, JPG, JPEG, PNG, PS, SVG, SWF, TIF, TIFF, WDP, WMF, XAML, XPS, PDF
