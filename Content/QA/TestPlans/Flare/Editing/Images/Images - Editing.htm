Images - Editing

Image menu

 

You can also select the image and then go to !Format| Image to open the Image Properties Dialog.

General Editing

As of V7. Users now are able to copy/paste images from the internet (png format). Test copying and pasting images from the internet.

 

Additional Tests:

  • Hovering the mouse over an image will display a tooltip with the name and file path.

 

Test Cases

  • Test copying and cutting an image from one topic and pasting it into another topic in a different folder. Verify the link updates.
  • Pasting another image over the existing one should replace the selected image – Verify that it renders and display in the block bar correctly.
  • Pasting another element or text over a selected image will replace it with that data. – Verify that there is no malformed XML from this
  • Inserting another element or typing over a selected image will replace it with that data. – Verify that there is no malformed XML from this
