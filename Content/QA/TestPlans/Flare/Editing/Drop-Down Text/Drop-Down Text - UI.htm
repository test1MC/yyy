UI Controls

  • Drop-Down Head/Select the Hotspot (Optional): Displays the textual content of the Drop-Down Head. The first paragraph contained in the selection will appear as the Drop-Down Head. If no selection is made when the dialog is invoked, Flare will create the default head content, "(This is the Drop-down Hotspot)"

    Optionally, you may also make a selection within the head to define the specific Drop-Down Hotspot text.

  • Drop-Down Body: Displays the textual content of the Drop-Down Body. The area should display the second paragraph (and any additional selected paragraphs if more than two paragraphs were selected) when the dialog is invoked. If only one paragraph or no paragraphs were selected when the dialog is invoked, Flare will create a default body, "(This is the Drop-down text)"
