Drop-Down Body - Editing

To edit the Drop-Down Body Text, place the insertion point in he Drop-Down Body and type.

Context Menu

 

 

Selecting:

You should not be able to select the MadCap:dropDownBody tag.

 

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

You should not be able to cut or copy the MadCap:dropDownBody tag.

 

Drag and Drop

You should not be able to drag the MadCap:dropDownBody tag.

 

Deleting

You should not be able to delete the MadCap:dropDownBody tag.

 

 

Conditional Tags

You should not be able to apply a conditional tag to the MadCap:dropDownBody tag. You should only be able to apply a condition to the parent MadCap:dropDown tag or any paragraph hlevel tags contained by the MadCap:dropDownBody tag.

 

Code

No Class:

<MadCap:dropDown>
    <MadCap:dropDownHead>
        <MadCap:dropDownHotspot>Delete</MadCap:dropDownHotspot> this text and replace it with your own content.</MadCap:dropDownHead>
    <MadCap:dropDownBody>
        <p>(This is the Drop-down text)</p>
        <p>(This is more Drop-down text)</p>
    </MadCap:dropDownBody>
</MadCap:dropDown>

 

With Class and Condition:

<MadCap:dropDown MadCap:conditions="Default.PrintOnly" class="class1">
    <MadCap:dropDownHead class="class1">
        <MadCap:dropDownHotspot MadCap:conditions="Default.ScreenOnly" MadCap:excludeAction="unbind" class="class2">Delete</MadCap:dropDownHotspot> this text and replace it with your own content.</MadCap:dropDownHead>
    <MadCap:dropDownBody class="class1">
        <p>(This is the Drop-down text)</p>
        <p>(This is more Drop-down text)</p>
    </MadCap:dropDownBody>
</MadCap:dropDown>
