Concept - Editing

Inserting a Concept…

With the Concept Pane

  • Open the topic you wish to add the concept to
  • Place the insertion point where you would like to insert the concept
  • Press Shift+F9 or select !Tools | Concept | Concept Window or select !View | Concept Window
  • Select an empty field in the Terms section (upper portion) of the Concept Window
  • Type the Concept keyword as you want it to appear in the concept link
  • Press Enter

 

With Concept Entry Mode

Concept Entry Mode lets add concepts quickly throughout your topics.

  • Open the topic to which you wish to add the concept
  • Select !Tools | Concept | Concept Entry Mode
  • Notice the cursor changes, displaying a “C” next to it
  • Click at the place in your topic where you want to add the Concept
  • Type the phrase that you want to add as the Concept
  • When you start typing, the Concepts Window opens and the phrase is added to the first empty field under the Terms column.
  • Notice the cursor does not lose focus in the XML editor.
  • Press Enter
  • Place the insertion point where you want to enter the next concept (in the same topic or another) and repeat.
  • When you are finished, right-click in the XML Editor to turn off Concept Entry Mode

 

For more testing for Index Entry Mode: See "Index and Concept Entry Mode - UI"

With Drag and Drop

The Drag and Drop method requires that you have an existing concept, and you now want to add it to a different topic.

 

  • Open the Topic to witch you wish to add the concept
  • Open the Concepts Window (Press Shift+F9 or select !Tools | Concept | Concept Window or select !View | Concept Window)
  • In the lower portion of the Concepts Window, click the concept you want and drag it to the location in the topic where you want to insert it. (If no concepts already exist from other topics, you can create a new one by right-clicking in the lower portion of the window and selecting Add Concept)

 

Auto-complete option

The "Auto-complete Index Entries" option applies to both Index Keywords and Concepts. To enable or disable "Auto-complete Index Entries":

  • !Tools | Index | (toggle on) Auto-complete Index Entries
  • In the XML Editor's local toolbar, expand the menu under the Index Entry button , and toggle on Auto-complete Index Entries

 

If "Auto-complete Index Entries" is enabled, index keywords and concepts will auto-complete when you start typing them in the Index Window and Concepts Window.

Editing a Concept

To edit a concept (the actual word(s)), you need to do so in the Concepts Window. It is possible to edit a single instance of a concept within a topic (using the upper portion of the Concepts Window), or you can globally edit all instances of a concept (using the lower portion of the Concepts Window).

Editing a single instance of a concept:

  • With the Concept Window open, open a topic and select an existing Concept. Any concept(s) in the selected MadCap:concept tag should appear in the upper portion of the Concepts Window for editing.
  • Right-click an existing Concept in the XML Editor and select Edit the Concept Association…. This should open the Concepts Window and display any concepts in the selected MadCap:concept tag for editing.
  • Right-click on the MadCap:concept Span Bar node and select Edit the Concept Association… . This should open the Concepts Window and display any concepts in the selected MadCap:concept tag for editing.
  • Right-click on the MadCap:concept Caret Tag Neighborhood node and select Edit the Concept Association… . This should open the Concepts Window pane and display any concepts in the selected MadCap:concept tag for editing.

Editing all instances of a concept:

In the lower portion of the Concepts Window…

  • Right-click any existing concept and select Rename Concept.
  • Select any existing concept and press [F2]
  • Slowly click on any existing concept twice

This will affect all instances of that concept in all topics throughout the project. This is covered in greater detail in the "Concepts - UI" test plan.
