Editing a Concept Link

To edit a concept link, you need to re-open the Insert Concept Link Control dialog for an existing concept link. This is done by invoking the MadCap:conceptLink context menu and selecting Edit Concept Link…

Context Menu

 

 

 

 

 

 

 

Code

One term, no class:

<p>
    <MadCap:conceptLink term="concept" externalHelpSystems="" />
</p>

 

Multiple terms, class, local label and conditions:

<p>
    <MadCap:conceptLink term="concept;breakfast;lunch" externalHelpSystems="" style="mc-label: 'My Concept Links';" MadCap:conditions="Default.ScreenOnly" class="class1" />>
</p>
