Concept Link (A-Link) - UI

Inserting a Concept Link

  • Toolstrip interface: !Insert | Help Control | Concept Link (A-link)...
  • Ribbon interface: !Insert | Concept Link (A-link)

UI Test Cases:

UI Controls

  • Selected Concepts: Displays a list of concepts associated with the selected Concept Link.
  • All Concepts: Displays a list of all Concepts present in the current Flare project. Any concepts which are present in the current topic should appear at the top of this list when the dialog opens.
  • [ << ] : Associates the selected Concept (in the All Concepts column) with the Concept Link. (The selected concept should then appear in the Selected Concepts column.)
  • [ >> ] : Disassociates the selected concept (in the Selected Concepts column) with the Concept Link.
  • Remove Control: Removes the Concept Link from the topic.
  • Options: Will open the options dialog.

 

Concept Link Options

UI Controls

  • Style Class:: Displays a list of style classes for the MadCap:conceptLink style
    • Selecting one will apply that style class to the help control, once OK is pressed.
  • Label: Will set an inline label for the help control.
    • This field should be populated with the default label, as set on the MadCap:conceptLink, mc-label style property
    • Typing your own label will only add it as an inline label. This will NOT change the mc-label style property on the associated Style.
  • Display Topics in: Drop down selection for how topic list should be displayed
    • (default): Will use what is set on the MadCap:conceptLink mc-help-control-display property
    • Popup Menu: This option will display the topics in a popup window, when the help control is clicked.
    • List: This option will display the topics in a list format, when the help control is clicked.
  • Target Frame : Drop down selection for where the topic list should show up.
    • (default): Just means Parent Frame
    • Parent Frame: Will open the selected topic in the same window
    • New Window: Will open the selected topic in a new window
    • Same Frame: Will open the selected topic in the same frame
    • Top Window: Will open the selected topic in the top most window
    • Popup Window: Will open the selected topic in a popup window
