Concepts Proxy - Output

Support in Web Outputs:

Support in Web Help Outputs:

  • The generated Concepts proxy will list each and every concept used throughout the project
    • Test using concepts in different contexts: Headings, sub-headings, paragraphs, table cells, drop-downs, etc.
    • Test using multiple concepts in the same context using compound concepts (multiple concepts in the same madcap:concept tag)
    • Test using multiple concepts in the same context using unique madcap:concept tags
  • For Web output types, the Topic Title of each topic that uses the concept will appear in alphabetical order under each concept
    • If a topic has a set Topic Title (in the Topic Properties dialog), that title will be used
    • If no Topic Title is set, the first heading will become the Topic Title in the output
    • If no Topic Title is set and the topic does not contain a single heading, the File Name will become the Topic Title in the output

Support in DITA Output:

  • Concepts Proxies are not included in DITA output. Verify that a compiler error is given indicating each Concept Proxy will not be present in the output for every instance of a Concepts Proxy in the project.

Compiler errors:

  • Attempting to include a Concept Proxy in DITA output:
    MadCap:conceptsProxy found in file: "Content/Topic.htm". It will be removed from the output.
