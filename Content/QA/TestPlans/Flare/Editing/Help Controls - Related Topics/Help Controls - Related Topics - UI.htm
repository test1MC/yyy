Help Controls: Related Topics - UI

Inserting a Related Topics Control/Opening the Dialog

  • Content Explorer: In Show Files mode, select and drag two or more topics from the Content Explorer into a topic in the XML Editor, the Insert Related Topics Control dialog should open with the selected files already displayed in the Related Topics list.
  • Tool Strip interface: !Insert | Help Control | Related Topics Control...
  • Ribbon interface: !Insert | Related Topics Control

UI Controls

  • All Topics: Displays a list of all topic files (*.htm, *.html and *.xml) that exist in the project's Content folder.

    Local toolbar:

    • (Show/Hide All Files in Project): Changes the view in the dialog to "Show All Files" view. This view display all topics in the project in one grid.
    • (Show/Hide Folders): Changes the view in the dialog to "Show Folders" view. This view displays all files in the project in a tree view, similar to the Content Explorer. This view can be combined with the "Show Files" view to offer a "Split" view.
    • (Show/Hide Files): Changes the view in the dialog to "Show Files" view. This view displays all files in the project one folder at a time, similar to the "Show Files" view in the Content Explorer. This view can be combined with the "Show Folders" view to offer a "Split" view.
    • (Up): Moves up one folder in the hierarchy within the project's Content folder. This button becomes enabled if you use the "Show Folders" or "Split" View and browse into a folder.
    • [Filter]: Use this filter to limit the view of content files to topics (*.htm or *.html) or enter text in the filter (and press Enter) to filter by filename.

  • Selected Topics grid: Displays a list of all topics set to be linked to as related topics.

    Columns

    • File: Displays the filename of the topic set to be a related topic.
    • Bookmark: Displays the bookmark (if included) in the topic that the related topic link points to.
    • Path: Displays the path of the topic set to be a related topic.
  • : Adds the topic selected in the All Topics section to the list of related topics on the right.
  • : Removes the topics selected in the related topics list from that list.
  • Options:
    • : Invokes the Select Bookmark dialog which lets you select a bookmark or heading in the selected topic to which the Related Topics Control will link
    • : Invokes the Related Topics Link Options dialog which lets the user set additional options for the Related Topics Control
    • : Moves the selected items in the Selected Topics list up in the sequence. Assuming custom ordering is used, the order of the topics in the Selected Topics list will determine the order in which they appear in the output.
    • : Moves the selected items in the Selected Topics list down in the sequence.
    • Use Custom Ordering: Lets the user determine if related topics should display in alphabetical order ("No") or the order in which the topics appear in this dialog ("Yes"). Setting this option to default will use whatever is set in the stylesheet (mc-use-custom-sort-order).
  • Remove Control: Deletes the Related Topics Control from the topic. (Only active when opening the dialog from an existing Related Topics Control. This button will not be active on initial insertion.)
  • OK: Closes the dialog and applies the changes.
  • Cancel: Closes the dialog and does not apply any changes.

You can add multiple files in the show files mode.

Use the up and down arrows to customize the order of topics in the related topic control.

Test Cases

  • Double-clicking a file in the All Topics list on the left will add it to the Selected Topics list on the right.
  • Double-clicking a file in the Selected Topics list on the right will open the Select Bookmark dialog for that topic's bookmarks.
