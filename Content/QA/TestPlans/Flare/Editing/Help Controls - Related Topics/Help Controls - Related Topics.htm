Help Controls: Related Topics

Bug Base Product Area:

  • Help Controls: Related Topics

 

Description:

A navigation link that lets users open topics that you've determined are related to the current topic. This is similar to a Concept link. You should use a Related Topic link if you are applying it to a topic that you want to associate with specific topics but you do not plan to reuse the same link in other topics. For example, let's say you have an introduction topic for your Help system. You want to create a Help control navigation link at the bottom of that topic that links users to three other topics that you feel are particularly useful to new users. However, you do not intend to use this same navigation link in other topics. So you decide to create a Related Topic link because it is quicker than creating a Concept link.
