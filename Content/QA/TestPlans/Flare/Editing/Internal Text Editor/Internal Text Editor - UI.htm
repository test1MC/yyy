Internal Text Editor - UI

To open the Internal Text Editor, do one of the following:

  • Right-click any file in the Content Explorer and select Open With | Internal Text Editor
  • Right-click any file in the Project Organizer and select Open With | Internal Text Editor
  • Right-click any file in the File List and select Open With | Internal Text Editor
  • Right-click any file in the Pending Check-Ins pane and select Open With | Internal Text Editor
