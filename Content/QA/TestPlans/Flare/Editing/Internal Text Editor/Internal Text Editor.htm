Internal Text Editor

Bug Base Product Area:

  • Internal Text Editor

 

Description:

This is a simple editor that you can use in place of Flare's other editors to view and modify the true code, including text, tags, and metadata. Most project files can be opened and modified in the internal text editor (including topics, snippets, header files, style sheets, etc.). For topic and snippet files, this editor also lets you show or hide line numbers, collapse or expand tagged areas (outlines), show or hide syntax coloring, and wrap text.
