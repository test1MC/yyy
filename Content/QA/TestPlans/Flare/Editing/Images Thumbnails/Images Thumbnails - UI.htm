Images: Thumbnails - UI

Thumbnail Properties can be set locally on an <img> tag using the Edit Image dialog.

 

Starting in Flare 11, the original image for a thumbnail can be previewed in the XML Editor by clicking on it. The preview is centered over the image if the whole preview will be visible. If a part of the preview would be cut off, then the position is adjusted so that the whole preview is visible.

Opening the dialog

  • Right-click an image and select Edit Image… from the context menu
  • Right-click an img block or span bar and select Edit Image… from the context menu
  • Select an image and go !Format | Image
  • Select an image and press <CTRL><Alt>-B

 

Then, select the Thumbnails tab.

UI Test Cases

UI Controls

  • (Behavior): Sets the behavior of the thumbnail in the output
    • (default): Uses the default behavior as it is set in the Stylesheet.
    • (as is): Used mainly when viewing properties of multiple items. Leaves the previous setting unchanged.
    • None: The image will not be shown as a thumbnail.
    • Inherit: The image will use whatever thumbnail settings are applied to the parent <img> tag
    • Mouse Over: The end user can see the picture in its full size by hovering the mouse over the thumbnail. As a result, the full size of the image will be shown in a popup window.
    • Hyperlink: The end user can see the picture in its full size by clicking the thumbnail. As a result, the full size of the image will be shown in a new window.
    • Popup: The end user can see the picture in its full size by clicking the thumbnail. As a result, the full size of the image will be shown in a popup window.
  • Max Width*: Lets you set the size of the thumbnail by setting the maximum width of the image.
  • Max Height*: Lets you set the size of the thumbnail by setting the maximum height of the image.

* If both of these properties are set, the property that makes the thumbnail image smaller will be the only relevant property.
