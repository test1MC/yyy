Images: Thumbnails

Bug Base Product Area:

  • Images/Multimedia

 

Description:

A feature for Web outputs which lets you display an image as a thumbnail. Upon hovering over or clicking on the thumbnail image, the full-size image appears. This feature should not be confused with the "Show Images as Thumbnails" option in the XML Editor. When output is generated, any image set to use a thumbnail will actually generate two images, the original image and the thumbnail image. The thumbnail image is essentially a hyperlink to the original image. Keep this in mind when troubleshooting any output issues.
