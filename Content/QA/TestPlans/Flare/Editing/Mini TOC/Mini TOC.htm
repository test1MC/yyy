Mini-TOC

Bug Base Product Area:

  • Mini-TOC

 

Description:

A mini-TOC lets you display a portion of the complete TOC inside a topic. A mini-TOC proxy can be inserted in a master page (generally for online output) or directly inside a topic (generally for print output). When output is compiled, the proxy is replaced with a "mini TOC" that shows headings below the current topic, based on h1-h6 styles or the structure of your main TOC. You can set the "TOC Depth" of the mini-TOC proxy to set the number of heading levels to show in the mini TOC.

 

Dependencies:

In order for the mini-TOC to appear in online output, you must have a topic associated with a TOC Book. Additionally, you must either insert a mini-TOC proxy into that topic or use a Masterpage that includes a mini-TOC proxy.

 

In order for the mini-TOC to appear in print output, you must organize your topics using heading levels. Additionally, you must insert a mini-TOC proxy into the topics when it is appropriate. For Word and Frame, it is allowable to use a Masterpage.
