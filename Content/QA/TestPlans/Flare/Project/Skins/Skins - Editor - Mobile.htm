Skins - Editor (Mobile)

Opening a Skin File

  • Target Editor: In the General tab of the Target Editor, in the Skin section, select a Skin File and click the "Edit" button.

Skin Editor (Mobile) - UI Test Cases

Skin Editor (Mobile): Toolbar

  • Preview: Click the button to preview an example of what the skin will look like in WebHelp Mobile.
  • Add Class:Disabled unless a style is selected in the Style Tab: Click to add a new class to the selected style in the Style Tab.

Skin Editor (Mobile): General Tab

UI Controls

  • Caption: Insert a caption for your skin or select WebHelp Mobile in the dropdown for WebHelp Mobile caption.
  • Features: Select which features you want to appear in the output. Options include: TOC, Index, Search, Glossary, Browse Sequences.
  • Unreferenced Images: If an image is added to the skin (Styles tab) and then later replaced with another image, Flare does not automatically remove the initial image from the Skin file.
    • Keep: (The default behavior) Image that cease to be referenced by any skin style are kept in the skin so they can be references at a later time.
    • Remove On Save: If this option is selected, unreferenced images are removed from the skin the next time the skin file is modified and saved.
    • [Unused Images Count]: Displays the number of unreferenced images that are currently included in the skin file.

Test Cases

  • Verify that images used for any new skin styles do not get deleted when using the "Remove on Save" option. The mechanism used to delete images only checks specific styles, so if new style are added that reference images, we must account for those.
  • Verify that images associated with custom buttons (Toolbar Items) do not get deleted when "Remove on Save" is enabled.

Skin Editor (Mobile): About Tab

UI Controls

  • Include About in output: Checked by default. Select this option to include an About image in the output.
  • Browse: browse for an image file to replace the default MadCap image
  • Reset: Resets back to MadCap image.

Skin Editor (Mobile): Options Tab

UI Controls

  • Display number of sub-items next to TOC entries: Select this option if you want the number of sub-item TOC entries to be displayed in parentheses
  • Display number of sub-items next to Browse Sequence entries: Select this option if you want the number of sub-item Browse Sequence entries to be displayed in parentheses
  • Display number of linked topics next to Index entries: Select this option if you want the number of linked topics to Index entries to be displayed in parentheses

Skin Editor (Mobile): Topic Toolbar Tab

Skin Editor (Mobile): Styles Tab

 

UI Controls

  • Styles: All available styles for items that appear in the skin
  • Show Property Groups/Alphabetical List: Choose how you want the Properties to be displayed.
  • Preview: The Preview now shows below and the Style item that is selected will be highlighted with a label in the preview.
  • Properties: Displays the list of properties available for each style
