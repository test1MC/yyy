ToolbarItem:

 

The Topic Toolbar shares its Skin styles with the standard WebHelp Toolbar. These Styles are set in the "Styles" tab of the Skin Editor.

Only the items which are actually allowed to appear in the Topic Toolbar need to be tested.

ToolbarItem Styles that apply to the Topic Toolbar:

  • AddTopicToFavorites
  • Back
  • CollapseAll
  • CurrentTopicIndex
  • ExpandAll
  • Forward
  • Home
  • NextTopic
  • PreviousTopic
  • Print
  • QuickSearch
  • Refresh
  • RemoveHighlight
  • SelectBrowseSequence
  • SelectFavorites
  • SelectGlossary
  • SelectIndex
  • SelectSearch
  • SelectTOC
  • Separator
  • Stop
  • ToggleNavigationPane

 

Properties

Default values should exist for all Styles for Icon, PressedIcon, HoverIcon and ControlType. Test all styles in all outputs and all supported browsers.
