Glossary Terms pane - UI Test Cases

UI Controls

  • Glossary Terms grid: Displays all glossary terms from all glossary files in the project. Double-clicking a glossary term in this pane will insert a Glossary Term Link at the insertion point in the XML Editor.

    Columns

    • Term: Displays each glossary term that exists in the project.
    • Definition: Displays the text definition of the corresponding glossary term or the path and filename of the definition topic.
    • File: Displays the name of the project Glossary file that contains the corresponding glossary term.
