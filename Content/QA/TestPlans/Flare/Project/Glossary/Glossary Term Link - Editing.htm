Editing a Glossary Term Link

To edit a Glossary Term Link, you need to open the Glossary Term Link dialog for an existing Glossary Term Link. This is done by invoking the MadCap:glossaryTerm context menu and selecting Edit Glossary Term…

Glossary Term Link dialog - UI Test Cases

UI Controls

  • Term: Displays the term of the selected Glossary Term Link. This field should not be able to be edited.
  • Definition: Displays the definition of the selected Glossary Term Link. This field should not be able to be edited.
  • Glossary: Displays the name and path of the Glossary file of the selected Glossary Term Link. This field should not be able to be edited.
    • : Opens the Glossary file in the Glossary Editor
  • Select a style class… : Lets you select a style class for the Glossary Term Link
  • Remove Link: Unbinds the Glossary Term Link leaving the text but removing the MadCap:glossaryTerm tag

Code

Glossary term, no class:

<p>Delete this <MadCap:glossaryTerm>text</MadCap:glossaryTerm> and replace it.</p>

 

Glossary term with class, condition and exclude action:

<p>Delete this <MadCap:glossaryTerm class="Expanding" MadCap:conditions="Default.ScreenOnly" MadCap:excludeAction="unbind">text</MadCap:glossaryTerm> and replace it.</p>
