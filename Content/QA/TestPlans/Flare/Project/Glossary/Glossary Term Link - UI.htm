Glossary Term Link - UI

Inserting a Glossary Term Link

In order to insert a Glossary Term Link, you need to first open a topic and select some text.

  • CTRL-Shift-T on the keyboard
  • Toolstrip interface: !Insert | Glossary Term Link
  • Ribbon interface:  !Insert | Glossary Term Link

 

If the selected text has already been added to a Glossary file in the project, Flare will simply create a Glossary Term Link out of the selection. If the selected text does not already exist as a glossary term in any of the project Glossary files, the Create Glossary Term dialog will open.

UI Test Cases

UI Controls

  • Term: Displays the selected text that will become the new Glossary term. This text should not be able to be edited in this dialog.
  • Definition: Lets you enter a text definition for the Glossary term. You may opt to leave this blank at the point of creating the Glossary Term Link. You can edit the definition or give this term a topic link definition in the Glossary Editor.
  • Select Glossary File: Select the Glossary file to which the Glossary term should be added. All project Glossary files should appear in this list. The list should be sortable.
