Browse Sequence Node Properties - UI

Browse Sequence Node Properties - UI Test Cases

Opening the dialog

  • Double-click any Browse Sequence node in the Browse Sequence Editor (as long as the double-click behavior has not been toggled to open the linked file)
  • Select any Browse Sequence node in the Browse Sequence Editor and click in the editor's local toolbar
  • Select any Browse Sequence node in the Browse Sequence Editor and click in the Standard toolbar
  • Right-click any Browse Sequence node in the Browse Sequence Editor and select Properties from the Content Menu
  • Select any Browse Sequence node in the Browse Sequence Editor and go !Edit | Properties
  • Select any Browse Sequence node in the Browse Sequence Editor and press F4 on the keyboard

Browse Sequence Node Properties - General tab

UI Controls:

  • Appearance: This section sets the appearance of the Browse Sequence entry in the output
    • Label: Set the textual content of the Browse Sequence entry
    • Style Class: Lets you set the Browse Sequence node to a style class defined in the Skin
    • Mark as New: Designates the Browse Sequence entry to be "new." As a result, the icon for this node will display a extra glyph on it in the Browse Sequence Editor and in the output.
  • Link: This section sets the link for the Browse Sequence node
    • "Select" radio buttons: Lets you select the type of file to which the Browse Sequence node should link
      • Select Topic… : Selecting this button will invoke the Open File dialog which will let you browse for a topic within the project.
      • Select TOC… : Selecting this button will invoke the Link to TOC dialog which will let you browse for a TOC within the project.
      • Select Browse Sequence… : Selecting this button will invoke the Open File dialog which will display all Browse Sequence files that exist in the project.
      • Select Flare Project and Target… : Selecting this button will invoke the Link to Flare Project and Target dialog which will let you browse for a Flare project. This is covered in greater depth in the Merging test plan.
      • Select HTML Help...:Selecting this button will invoke the Link to HTML Help dialog which will let you browse for a chm.
      • Enter External Help System… : Selecting this button will invoke the Link to Help System dialog which will let you browse for a compiled WebHelp or DotNet. (.mcwebhelp and .mchelp) This is covered in greater depth in the Merging test plan.
      • Mimic Movie or Movie Collection as:… :Allows you to link to a mimic project or movie
    • Select Link… : Depending on which radio button is selected, this button invokes the Open File, Link to Flare Project and Target, or Link to Help System dialog.
    • Select Bookmark… : Opens the Select Place dialog which lets you select a bookmark or heading from the topic file associated with the selected Browse Sequence node. This button is only active for Browse Sequence nodes that link to a topic.
    • Open Linked File: Opens the linked file. This button is not active for a Browse Sequence node that has a broken link, is an unlinked node, or links to a Flare project.

Browse Sequence Node Properties - Auto-generate tab

UI Controls:

  • Compute sub-entries from linked file: For web outputs, you can enable this auto-generate feature to have Flare generate a Browse Sequence for you in the output based on the heading styles used in your topics.
    • Action: Lets you select the auto-generate behavior.
      • Insert Computed Entries: On output, adds Browse Sequence sub-entries directly below the selected node based on h1-h6 styles in the topic.
      • Replace with Computed Entries: On output, removes the current Browse Sequence node and replaces it with sub-entries based on the h1-h6 styles in the linked topic.
    • Depth: This number determines the number of levels of headings that should be included in the auto-generation.

Browse Sequence Node Properties - Advanced tab

UI Controls:

  • Open in Browser Frame: Lets you specify the browser frame in which a linked file should open when a user click the Browse Sequence entry in the output.
  • Open in Skin: Lets you associate the Browse Sequence entry with a particular skin.
  • HTML Help Icon: For HTML Help output only, this option lets you set a specific icon for the Browse Sequence node.
  • Auto-set title of linked file: Sets the Topic Title of the linked topic to be the Browse Sequence Label in the output, overriding the title that you may have provided in the (Topic) Properties dialog.
  • When merging, replace node with merged Browse Sequence: If the Browse Sequence node links to another Flare project, another Browse Sequence or a TOC, you can select this option to replace the Browse Sequence node with the merged TOC or Browse Sequence. By default, this option is disabled, and such a node will output as a book with the merged TOC or Browse Sequence inside the book.
  • Server-based Auto-Merge: For Webhelp Plus output only, use this option to determine where other Flare project outputs are to be auto-merged into the Browse Sequence. This is covered in greater depth in the Output WebHelp Plus test plan.
