Search Filters - Styles

There are two areas in the Skin Editor where you can change the look and text of the Search Filter Set.

Search Filter styles in the Skin

Control.SearchFiltersLabel

Default properties: (none)

General

Output

Label: The label for the item.

 

Font

Output

FontFamily: Name of the font used for an element’s text.

 

FontSize: Size of the font used for an element’s text.

 

FontWeight: Boldness of the font used for an element’s text.

 

FontStyle: Style of the font used for an element’s text.

 

Color: Color of an element’s text.

 

BackgroundColor: The background color of an element’s text.
