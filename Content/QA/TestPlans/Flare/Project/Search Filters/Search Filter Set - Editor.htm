Search Filter Set - Editor

Opening a Search Filter Set

  • Target Editor (HTML5, WebHelp, WebHelp AIR, WebHelp Plus, DotNet): In the Advanced tab of the Target Editor, in the Search Filters section, select a Filter File and click the "Edit" button.

Search Filter Set Editor - UI Test Cases

Local Toolbar:

Button (Tooltips in Parentheses):

  • (New item): Adds a new line to the Search Filter Set Editor using the default filter name "NewFilter."
  • Cut: Cuts the selected filter(s) to the clipboard.
  • Copy: Copies the selected filter(s) to the clipboard.
  • (Paste): Pastes the filter(s) from the clipboard.
  • (Delete): Deletes the selected filter(s)

UI Controls

  • Search Filter Set Editor grid: Displays and lets you create and edit Auto-index Phrases.

    Columns

    • Filter Name: Lists the search filters as they will appear in the filter list in the output.
    • Concepts: Displays the concept(s) associated with this filter which will be used to limit the search results.
    • Comment: Allows you to add a comment about the filter. Comments should not affect the output.

Test Cases

For Filter Name, enter single words, multiple words, numbers, words with international characters, etc.

Right-click menu

Right-clicking a row in the Search Filter Set Editor should produce the following menu:

Select Concepts For Search Filter dialog:

UI Controls

  • Selected Concepts: Displays all concepts associated with the selected search filter.
  • All Concepts: Displays all concepts contained in all topics throughout the project.
  • : Adds the concept(s) selected in the All Concepts list to the list of Selected Concepts on the left.
  • : Removes the concept(s) selected in the Selected list from that list.
  • OK: Closes the dialog and applies the selected concepts to the selected search filter.
  • Cancel: Closes the dialog and does not apply any changes.

Test Cases

  • Select Concepts and All Concepts lists are sortable.
  • Double-clicking an item in the All Concepts list adds it to the Selected Concepts list.
  • Shift-Click and CTRL-Click allow for multi-selection in both lists and multi-selections can be added to and removed from the Selected Concepts list.
  • Create concepts with various characters, symbols and international letters and make sure all concepts used in your project appear in the All Concepts list.
  • Create concepts in all files that allow them (topics, snippets, etc) and make sure all concepts used in your project appear in the All Concepts list.

Code

Default Search Filter Set:

<?xml version="1.0" encoding="utf-8"?>
<CatapultSearchFilterSet>
<SearchFilter Name="My Subset"></SearchFilter>
</CatapultSearchFilterSet>
 

Search Filter Set with two filters, one with two concept associations and a comment:

<?xml version="1.0" encoding="utf-8"?>
<CatapultSearchFilterSet>
<SearchFilter
Name="My Subset"
Concepts="concept1"></SearchFilter>
<SearchFilter
Name="NewFilter"
Comment="Two concept associations"
Concepts="concept2;concept3" />
</CatapultSearchFilterSet>
