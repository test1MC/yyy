File Tags - UI

Applying a File Tag to a file

To apply a File Tag to any file:

  1. Right-click any file in the Content Explorer or Project Organizer and select Properties or simply select any file and press F4
  2. Select the File Tags tab
  3. Select the tag you want to apply and click OK

File Tags in the File List

For each File Tag Set that exists in the project, a column will be added to the File List with the Tag Type and the File Tag Set Name as the column heading.
