File Tag Sets

Bug Base Product Area:

  • Tagging

 

Description:

File Tag Sets or "Tagging" allows the user to create various sets of tags which can then be applied to any file in a Flare or Blaze project. The most common uses of File Tagging are Author and Status tagging, but a File Tag Set can be created for anything. To use a File Tag, you must first create a File Tag Set in the Project Organizer. Then, you can apply those tags to any file via the file's Properties dialog.

 

Any File Tag Sets that exist in your project should also create a unique column in the project's File List. This allows a user to view all files in the project and sort them by Status, Author, or any other tag you can come up with.
