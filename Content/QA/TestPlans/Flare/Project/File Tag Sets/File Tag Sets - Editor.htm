File Tag Sets - Editor

Opening a File Tag Set File

  • Properties (any file): In the File Tags tab of the Properties dialog for any file, select a File Tag Set in the pull-down and click the "Edit Set" button.

File Tag Set Editor - UI Test Cases

Local Toolbar:

Button (Tooltips in Parentheses):

  • (New item): Adds a new tag to the File Tag Set using the default filter name "NewTag."
  • Cut: Cuts the selected tag(s) to the clipboard.
  • Copy: Copies the selected tag(s) to the clipboard.
  • (Paste): Pastes the tag(s) from the clipboard.
  • (Delete): Deletes the selected tag(s).
  • Tag Type: xxxxx (The type of tag for this set (Click to change)): Displays the Tag Type for the current File Tag Set. Clicking this button opens the Change Tag Type dialog.

UI Controls

  • File Tag Set Editor grid: Displays and lets you create and edit File Tags.

    Columns

    • Tag Name: Lists the File Tags created for that File Tag Set.
    • Comment: Allows you to add a comment about the tag. Comments should not affect the output.

 

Test Cases

  • For Tag Name, enter single words, multiple words, numbers, words with international characters, symbols, etc.
  • Test that tags with such characters can successfully be applied to a file.

File Tag Set - General Editing Test Cases

Conditional Tags

Conditional Tags cannot be applied to individual File Tags. Conditional Tags can be applied to the actual File Tag Set file (*.flsfs) for the purposes of Advanced Global Project importing.

Misc

Code

Default File Tag Set (using MyAuthorFileTagSet template):

<?xml version="1.0" encoding="utf-8"?>
			<CatapultFileTagSet Type="Author" Comment="My set of topic author tags">
			<FileTag Name="Author1" Comment="" />
			<FileTag Name="Author2" Comment="" />
		</CatapultFileTagSet>
 

File Tag Set with multiple tags and comments:

<?xml version="1.0" encoding="utf-8"?>
			<CatapultFileTagSet
			Type="Status"
			Comment="My set of topic status tags">
			<FileTag
			Name="Done"
			Comment="Content completed, but not yet verified." />
			<FileTag
			Name="Well Done"
			Comment="Completed and verified." />
			<FileTag
			Name="Rare"
			Comment="Created and outline started." />
			<FileTag
			Name="Medium Rare"
			Comment="Created and outline completed." />
			<FileTag
			Name="Medium Well"
			Comment="Outline complete; content started." />
			<FileTag
			Name="Bloody Rare"
			Comment="Created... that's it." />
		</CatapultFileTagSet>
