Batch Targets

Bug Base Product Area:

  • Targets

 

Description:

Batch Targets allow you to build multiple targets at once. In addition, Batch Targets allow you to schedule build and publish times ("triggers") for any associated targets. Triggers can be set to occur once or at a regular frequency (daily, weekly, monthly, every other Friday, etc).

 

Test Plans:

  • Batch Targets - Add File
  • Batch Targets - File Location & Properties
  • Batch Targets - Editor
  • Batch Targets - Baseline

 

Dependencies:

  • Batch Targets require ordinary targets to already be established in the project. If you are using Batch Targets to publish, you would also need to set your project up with Destinations for each target you want to setup with a publishing triggers.
  • Scheduling functionality is dependent on Windows Task Scheduler.
