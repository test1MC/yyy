Synonyms

Bug Base Product Area:

  • Synonyms

 

Description:

A phrase similar to another word that is found in a particular topic. By creating synonyms, users will be able to search for and find the same topic information using either word, even though the synonym phrase does not occur in the actual topic. Synonyms can be directional ("term A" finds "term B") or grouped ("term A" finds "term B" and "term B" finds "term A").

 

Synonyms can also be created "on-the-fly" in a Feedback-enabled Help system. Synonyms created in the Feedback Explorer are covered in the Feedback Explorer test plans. The test plans in this section will only address Synonyms in the context of Flare.
