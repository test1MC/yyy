Synonyms Editor - UI Test Cases

Directional Tab

UI Controls

  • Directional Synonyms grid: Displays and lets you create and edit directional Synonyms.

    Columns

    • Word: Enter a term that users might use instead of the term that actually occurs throughout your content.
    • Synonym: Enter the term that exists in your content that should be found when the corresponding "Word" is used as a search term
    • Stem: Enable this option to if you want Flare to accept other variations of the search term that have the same stem.

Groups Tab

UI Controls

  • Groups Synonyms grid: Displays and lets you create and edit Synonym Groups.

    Columns

    • Group: Enter a collection of terms (synonyms) that will produce the same search results for all of the words in the group.
    • Stem: Enable this option to if you want Flare to accept other variations of the search term that have the same stem.
