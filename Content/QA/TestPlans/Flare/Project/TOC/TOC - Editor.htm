TOC - Editor

Opening a TOC File

  • Target Editor: In the General tab of the Target Editor, in the Master TOC section, select a TOC File and click the "Edit" button.

TOC Editor - UI Test Cases

Local Toolbar

Button (Tooltips in Parentheses):

  • (Hide/show the Info Bar): Lets you toggle to display or hide the Info Bar.
  • (Export the data to a CSV file): Exports all of the data to a csv file.
  • (Export the selected data to a CSV file): Exports only the selected data to a csv file.
  • / (Toggle between tree view and grid view): Lets you switch between the Tree View of the TOC which lets you see the multiple levels that exist in your TOC from TOC Books, and the Grid View which is a flat, line-item grid view which is useful for quickly viewing the properties of every TOC node.
  • (Expand all): Expands all TOC Books in the TOC Editor.
  • (Collapse all): Collapses all TOC Books in the TOC Editor.
  • (Create a new topic and link to it): Opens the Add New Topic dialog, which lets you add a new topic and automatically add it to the TOC. If the selected TOC node is a book, the new TOC entry is created at the end of that book. If the selected TOC entry is not a book, the new TOC entry is created immediately after the selected node.
  • (New item): Creates a new, unlinked TOC node as a child of the selected node.
  • (New book): Creates a new, unlinked TOC Book node with one unlinked child node immediately preceding the currently selected TOC node.
  • (New top-level book): Creates a new, unlinked TOC Book node with one unlinked child node immediately preceding the currently selected TOC node's top-level book.
  • (Run Properties dialog for this item): Opens the Properties dialog for the selected TOC node.
  • (Move item to the left): Moves the selected TOC node to the left one level. This option is only active for TOC nodes inside a TOC Book.
  • (Move item to the left): Moves the selected TOC node to the right one level. If the selected node is preceded by a Book, this will move the node into that book. If the selected node is not preceded by a book, this will turn the preceding node into a book and move the selected node into it.
  • (Move item up): Moves the selected TOC node up in the TOC sequence above the first preceding TOC node of the same level.
  • (Move item down): Moves the selected TOC node down in the TOC sequence below the first acceding TOC node of the same level.
  • (Locate file in Content Explorer): Opens the Content Explorer and highlights the location of the selected content file within it.
  • (Send linked document to reviewers): Invokes the Send Topic for Review Wizard with all linked topics from the selected TOC nodes already included in the Wizard's Topic list. Using this method to send topics for review also includes the current TOC file.
  • (Open linked file): Opens the file to which the TOC node is linked. This should work for topics as well as linked TOC, Browse Sequence, Flare project, etc. Test for all.
  • (Show unlinked books): Toggle between showing unlinked TOC Books as unlinked items or not.
  • (Shows the current warning, if any): The toolbar item itself will display the number of broken links or (if there are no broken links) the number of unlinked items. If there are no such issues, this item will appear grayed out. Clicking on this button will invoke a Current Issues dialog which details the number of broken links, broken bookmarks, unlinked books and unlinked items.
  • (Navigate the previous warning): Lets you quickly navigate through the TOC to the previous broken link or unlinked item. If there are no such issues, this item will appear grayed out.
  • (Navigate to the next warning): Lets you quickly navigate through the TOC to the next broken link or unlinked item. If there are no such issues, this item will appear grayed out.
  • (Toggle the double-click behavior; open topic or display properties): Toggles the double-click behavior in the TOC Editor between opening the TOC node's linked file or opening the Properties dialog for the TOC node.

Grid View Test Cases

  • TOC grid: Displays all TOC Entries and associated properties in a single grid.

    Columns

    • A column can exist for each and every property that can be set on a TOC node. The set of visible columns can be changed in the Columns dialog. To open the Columns dialog:
      • Right-click anywhere in the Grid View TOC Editor and select Columns from the menu
      • Select Columns from the drop-down menu under Tree View/Grid View in the local toolbar

Right-click menu (Tree View)

Right-clicking anywhere in the Tree View of the TOC Editor should produce the following menu:

Right-click menu:

 

Right-click menu items (Tooltips in parentheses):

  • Expand/Collapse (Expand/collapse node): If a TOC Book is selected, this option expands the book, but only the first level.
  • Expand All/Collapse All (Expand/collapse node and child nodes): If a TOC Book is selected, this option expands the and any books contained therein.
  • Add New Topic… (Create a new topic and link to it): Opens the Add New Topic dialog, which lets you add a new topic and automatically add it to the TOC. If the selected TOC node is a book, the new TOC entry is created at the end of that book. If the selected TOC entry is not a book, the new TOC entry is created immediately after the selected node.
  • Link to New Topic… (Link to a new topic): Opens the Add New Topic dialog, which lets you add a new topic and automatically link to it from the selected TOC node.
  • Link to Topic (Link to a topic): Opens the Open File dialog which lets you browse for a topic within the project.
  • Locate in Explorer (Locate file in Content Explorer): Opens the Content Explorer and highlights the location of the selected content file within it.
  • Send For Review... (Send linked document to reviewers): Invokes the Send Topic for Review Wizard with all linked topics from the selected TOC nodes already included in the Wizard's Topic list. Using this method to send topics for review also includes the current TOC file.
  • Open Link (Open linked file): Opens the file to which the TOC node is linked. This should work for topics as well as linked TOC, Browse Sequence, Flare project, etc. Test for all.
  • Cut (Cut): Cuts the selected TOC node(s) to the clipboard.
  • Copy (Copy): Copies the selected TOC node(s) to the clipboard.
  • Paste (Paste): Pastes the TOC node(s) from the clipboard.
  • Delete (Delete): Deletes the selected TOC node(s)
  • Properties (Display properties for selected item): Opens the Properties dialog for the selected TOC node.

Right-click menu (Grid View)

Right-clicking anywhere in the Grid View of the TOC Editor should produce the following menu:

Right-click menu:

 

Right-click menu items (Tooltips in parentheses):

  • Columns… (Show/hide columns): Opens the Columns dialog to customize the columns
  • Export to CSV (Export the data to a CSV file): Exports all the available data to a CSV file
  • Export selection to CSV (Export the selected data to a CSV file): Exports the selected data to a CSV file
  • Locate in Explorer (Locate file in Content Explorer): Opens the Content Explorer and highlights the location of the selected content file within it.
  • Send For Review... (Send linked document to reviewers): Invokes the Send Topic for Review Wizard with all linked topics from the selected TOC nodes already included in the Wizard's Topic list. Using this method to send topics for review also includes the current TOC file.

  • Open Link (Open linked file): Opens the file to which the TOC node is linked. This should work for topics as well as linked TOC, Browse Sequence, Flare project, etc. Test for all.

  • Cut (Cut): Cuts the selected TOC node(s) to the clipboard.
  • Copy (Copy): Copies the selected TOC node(s) to the clipboard.
  • Delete (Delete): Deletes the selected TOC node(s)
  • Properties (Display properties for the selected item): Opens the Properties dialog for the selected TOC node.
