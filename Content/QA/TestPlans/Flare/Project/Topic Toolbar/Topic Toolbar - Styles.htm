Topic Toolbar - Styles

Topic Toolbar Styles are set in two places. If you want to apply properties that affect the entire toolbar, such as displaying a border around the toolbar or setting up the positioning of the toolbar, this can be done in the Stylesheet Editor by applying properties to the MadCap | topicToolbarProxy style. If you want to style the individual buttons contained within the Topic Toolbar, such as the button icon, the button label, or the button's border and positioning inside the toolbar, this can be done in the Skin Editor.

Skin Editor Styles

The Topic Toolbar shares its Skin styles with the standard WebHelp Toolbar. These Styles are set in the "Styles" tab of the Skin Editor.

Only the items which are actually allowed to appear in the Topic Toolbar need to be tested.

ToolbarItem Styles that apply to the Topic Toolbar:

  • AddTopicToFavorites
  • Back
  • CollapseAll
  • CurrentTopicIndex
  • ExpandAll
  • Forward

  • PreviousTopic
