Inserting a Topic Toolbar Proxy

  • Tool Strip interface: !Insert | Proxy | Insert Topic Toolbar Proxy...
  • Ribbon interface: !Insert | Proxy | Topic Toolbar Proxy

Topic Toolbar Proxy - UI Test Cases:

UI Controls

  • Stylesheet class for proxy: Lets you select a class for the proxy.
  • Buttons: Displays a list of buttons set to appear in this Topic Toolbar.
  • Select buttons...: Opens the Configure Toolbar dialog which lets you select the set of buttons for this Topic Toolbar

Configure Toolbar - UI Test Cases:

UI Controls

  • Available: Displays a list of all available buttons which can be added to the Topic Toolbar.
  • Selected: Displays a list of buttons set to appear in this Topic Toolbar.

  • : Moves the selected button in the "Selected" list down in the sequence
