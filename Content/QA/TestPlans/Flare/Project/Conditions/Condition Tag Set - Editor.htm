Condition Tag Set - Editor

Opening a Condition Tag Set File

  • Condition Tags dialog: Apply a condition to some content in a topic. In the Condition Tags dialog, select a Condition Tag Set in the first column and click the "Edit Set" button.
  • Target Editor: In any output type's Target Editor, select the Conditional Text tab. Then select a Condition Tag Set in the first column and click the "Edit Set" button.
  • Properties (any file): In the Conditional Text tab of the Properties dialog for any file, select a Condition Tag Set in the first column and click the "Edit Set" button.
  • Properties (Topic, Snippet or Master Page): In the Snippet Conditions tab of the Properties dialog for a topic, snippet, or Master Page, select a Condition Tag Set in the first column and click the "Edit Set" button.

Condition Tag Set - Editor

Local Toolbar:

Button (Tooltips in Parentheses):

  • (New item): Adds a new tag to the Condition Tag Set using the default filter name "NewTag."
  • (Cut): Cuts the selected tag(s) to the clipboard.
  • (Copy): Copies the selected tag(s) to the clipboard.
  • (Paste): Pastes the tag(s) from the clipboard.
  • (Delete): Deletes the selected tag(s).

UI Controls

  • Condition Tag Set Editor grid: Displays and lets you create and edit Condition Tags.

    Columns

    • Condition Tag: Lists the Condition Tags created for that Condition Tag Set.
    • Background: Lets you use the Color Picker to associate a color with the conditional tag.
    • Comment: Allows you to add a comment about the tag. Comments should not affect the output.

 

Test Cases

  • For Tag Name, enter single words, multiple words, numbers, words with international characters, symbols, etc.
  • Test that tags with such characters can successfully be applied to a file, element, etc.
