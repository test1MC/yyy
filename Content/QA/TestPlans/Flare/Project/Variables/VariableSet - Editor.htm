Variable Set - Editor

Opening a Variable Set File

  • Target Editor: In the Variables tab of the Target Editor any file, select a Variable Tag Set in the pull-down and click the "Edit Set" button.

Variable Set Editor - UI Test Cases

Local Toolbar:

Button (Tooltips in Parentheses):

  • (Add variable): Adds a new row for a Variable.
  • (Add DateTime variable): Adds a new row for a DateTime variable.
  • (Add variable definition): Adds a new variable definition.
  • Cut: Cuts the selected row(s) to the clipboard.
  • Copy: Copies the selected row(s) to the clipboard.
  • (Paste): Pastes the row(s) from the clipboard.
  • (Delete): Deletes the selected row(s).

UI Controls

  • Variable Set Editor grid: Displays and lets you create and edit Variable Tags.

    Columns

    • Name: Displays the variable name.
    • Definition: displays the definition of the variable.
    • Comment: Allows you to add a comment about the variable. Comments should not affect the output.

 

Test Cases

  • For Name, enter single words, multiple words, numbers, words with international characters, symbols, etc.

Right-click menu items:

Variable Set - General Editing Test Cases

Drag and Drop

There is currently no drag/drop editing functionality within the variable Tag Set Editor (as of Flare V6)

Cut/Copy & Paste – Paste options will only be active if there is data on the clipboard

Select one or more tags (CTRL-Click) from any combination of rows and then:

Right-click on the filter row | Cut/Copy Paste menu options

Cut/Copy Paste Standard Toolbar options

Cut/Copy Paste Keyboard keys (CTRL-X, CTRL-C, CTRL-V)

Cut/Copy Paste !Edit Menu options

 

Undo/Redo works and commands display correctly in the Standard Toolbar.

Deleting

Item:

  • Select the entry and press Delete on the keyboard
  • Select the entry and click the Delete button on the Standard Toolbar
  • Select the entry and click the Delete button on the Local Toolbar
  • Right-click the entry and select Delete from the context menu
  • Select the entry, !Edit | Delete
  • Select the entry and “Cut”

Tooltips

If any Tag Name or Comment is so long that it does not fit within the column, hovering over the cell will display the entire contents of that cell.

Conditional Tags

Conditional Tags cannot be applied to individual variable entries. Conditional Tags can be applied to the actual Variable Tag Set file (*.flsfs) for the purposes of Advanced Global Project importing.
