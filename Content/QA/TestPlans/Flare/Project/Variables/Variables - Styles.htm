Variables - Styles

Variable cannot be styled. If they are styled within a Flare project, that styling will all be lost when generating output since the variable tag is replaced with the textual content of that variable's definition.

 

Since the UI to create apply style classes is and has been present in Flare since V1, it is still important to test that style classes can at least be created and applied correctly. It is entirely possible that users may be using variable classes to tag variables in their content. But please note that none of these styles will be present in any output.
