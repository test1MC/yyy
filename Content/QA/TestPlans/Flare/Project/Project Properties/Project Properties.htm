Project Properties

Bug Base Product Area:

  • Tools/Options

 

Description:

A dialog used to make property changes for the Flare project. Project Properties lets you set the project's Master TOC, Master Page Layout and Master Stylesheet. Project Properties also lets you set the project language, used for spell checking and determining whether or not a language skin should be used when generating web outputs. Finally, Project Properties lets you view source control provider information for your project as well as letting you bind or unbind the project to source control, or letting you perform basic source control tasks for the main project file (*.flprj).
