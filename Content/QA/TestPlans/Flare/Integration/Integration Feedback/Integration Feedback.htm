Integration Feedback

Bug Base Product Area:

  • Integration: Feedback

 

Description:

Integration between Feedback and Flare. Enable Feedback for an output in the Target Editor. Then, the internal Feedback Explorer pane in Flare displays data from the Feedback Server associated with a projects Primary Target.

 

Test Plans:

  • Enabling Feedback - UI
  • Integration Feedback - Styles
  • Feedback Explorer - UI
  • Integration Feedback - Baseline

 

Dependencies:

  • Flare output connected to a Feedback Server and the source Flare project. In order to test all UI, you may need to perform various actions in the output in order to populate all of the Feedback Server data grids (ie, view topics, performs searches, make CSH calls, add comments and rate topics).

 

Testing Resources (Copy all files to local Machine before importing):

  • Test projects and generated outputs are available on FEEDBACKV1 and FEEDBACKV2 (and eventually FEEDBACKV3), or you can create a new project using those Feedback Servers.
