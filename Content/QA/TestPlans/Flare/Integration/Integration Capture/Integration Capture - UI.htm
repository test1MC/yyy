Relevant UI Controls:

Image Viewer

To open the Image Viewer double- click an image in the Content Explorer

 

The local toolbar will contain one button ("Edit this file with MadCap Capture") if it has not been created or saved in Capture or two buttons, with the second being "Regenerate Captured image".

  • Edit this file with MadCap Capture: Will open the file in MadCap capture.
  • Regenerate Captured image: Used to update image to its Capture state. (test: Edit Captured image in another program such as MS Paint, save, the click this button. The image should revert back to its Capture state and remove the changes made outside Capture)

 

Conditions and Variables

Like Mimic, conditions and variables from a Flare project linked to the image will be available for use in Capture. However, in Capture, the project is already linked to the image, so all variables and conditions will be available in Capture when editing an image that is in a Flare project in Capture.
