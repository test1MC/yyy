Details

Auto-size Rectangle to fit text

For Capture/Mimic, there is now an auto-size check box which will size the rectangle-based shape (rectangle, callouts, etc) to fit the text inside. With this option checked the user will not be able to manually size the shape, but may adjust the shape placement. This is to allow users who do automation coupled with translation, so that upon compilation of their projects in various languages, the dimensions of the shape will adjust to the caption.
