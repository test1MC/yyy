Feedback Keys, Trial Behavior and Releases

Keys:

LH - A Target Key – Used specifically for the server that we host for clients (the “Feedback Service” is what Sales calls it).

XX - A Target Key – Used for Trails and Training. These expire after so many days.

W1 - A Target Key – Used with a server that the client buys (that is not our service). When a customer buys a server they get 15 of these Target Keys to go with it.

Z1 - A Server Key – Used to install and run an installation of Feedback Server Admin.

* no key is necessary for any shipped version Feedback Explorer although pre-V1 builds did use an "FE" keyReleases:

Trial Behavior

Feedback Explorer - Free application. No trial limitations.

Feedback Server 1-2 - 30 day trial. No other limitations.

Feedback Service - No trial limitations.
